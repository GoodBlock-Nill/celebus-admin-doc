## 1. 화면 목적

기존 게임의 정보를 수정하는 폼 화면입니다. 생성 폼과 동일한 구조이나 기존 값이 채워져 있으며, 게임유형은 읽기 전용으로 변경이 불가합니다. 일부 필드는 게임 상태 및 참여자 유무에 따라 수정이 제한됩니다.

##### 주요 기능
- 게임 기본정보 수정 (타이틀, 설명, 힌트 링크 (PM 전용))
- 보상/참여/일정 설정 수정
- 결과 확인 기준 수정 (Prediction Market 전용)
- 퀴즈 수정 (Survival Trivia 전용, 진행중 상태에서는 수정 불가)
- 변경사항 저장 또는 취소

##### 접근 경로
- 게임 상세(CEB-BO-GZ-202) > [수정하기] 버튼 클릭 시 이동

##### 수정 가능 조건
- 게임 상태가 Draft, Ready, Active, Pending(PM 한정)일 때 수정 가능
- 상태에 따라 수정 가능한 필드가 제한됨 (상세는 4. 규칙/로직 참조)
- Closed, Ended 상태에서는 수정 불가 (수정 페이지 접근 시 상세 화면으로 리다이렉트)
- Survival Trivia는 게임 시작 후 자동으로 진행/종료되므로 Pending 상태가 없음

## 3. 화면 내 기능설명

### 3.1 다국어 입력 (게임유형 공통)

##### 타이틀 입력
- KO/EN/JP 탭으로 언어 전환
- 각 언어별 50자 제한
- 실시간 글자 수 표시: "25/50"
- 필수 입력: 모든 언어 필수 (KO/EN/JP)

##### 상세설명 입력
- KO/EN/JP 탭으로 언어 전환
- WYSIWYG 에디터 제공 (Toolbar + 편집 영역)
- Toolbar 기능: Bold, Italic, Underline, 링크 삽입, 순서 없는 리스트, 순서 있는 리스트, 이미지 삽입, 좌측/중앙/우측 정렬
- 각 언어별 500자 제한
- 필수 입력: 모든 언어 필수 (KO/EN/JP)

### 3.2 힌트 링크 입력 (Prediction Market 전용)

##### 힌트 링크
- Segmented Button: 사용/미사용
- "사용" 선택 시 URL 입력 필드 표시
- "미사용" 선택 시 URL 입력 필드 숨김
- Placeholder: "https://example.com/hint"
- 도움말 텍스트: "게임 관련 힌트 페이지 URL을 입력하세요."
- URL 형식 검증 (입력 시)

### 3.3 퀴즈 수정 (ST 전용)

##### 문제당 제한시간
- Number Input 사용
- 기본값: 기존 저장값
- 최소값: 5초, 최대값: 60초

##### 문제 텍스트
- KO/EN/JP 탭으로 언어 전환
- 각 언어별 200자 제한
- 실시간 글자 수 표시
- 필수 입력: 모든 언어 필수 (KO/EN/JP)

##### 선택지 입력
- 최소 2개, 최대 4개 (기본 4개)
- [+ 선택지 추가] 버튼으로 선택지 추가 가능 (4개 도달 시 버튼 Disabled)
- 각 선택지 우측 [삭제] 버튼으로 삭제 가능 (2개 남으면 버튼 Disabled)
- 각 선택지: KO/EN/JP 탭으로 언어 전환
- 각 언어별 100자 제한
- 필수 입력: 모든 언어 필수 (KO/EN/JP)

##### 정답 지정
- 각 문제별 Radio 버튼으로 등록된 선택지 중 정답 1개 지정
- 정답으로 지정된 선택지 삭제 시 첫 번째 선택지(선택지 1)로 정답 자동 초기화
- 정답 미지정 시 저장 불가

##### 정답결과 노출시간
- 각 문제별 정답 결과화면 카운트다운 시간 설정
- 기본값: 5초, 범위: 1~10초
- Number Input으로 입력

##### 문제 순서 재정렬
- 드래그앤드롭으로 문제 순서 변경 가능

##### 문제 추가 및 삭제

- 기본 10문제로 시작
- [+ 문제 추가] 버튼으로 문제 추가 가능 (최대 10문제, 초과 시 버튼 Disabled)
- 각 문제 헤더의 [삭제] 버튼으로 문제 삭제 가능 (최소 1문제, 1개 남으면 버튼 Disabled)
- 문제 삭제 후 문제 번호 자동 재정렬 (Q1~QN 순서 유지)

##### Active 상태에서의 퀴즈 수정 제한
- 게임이 진행중(Active) 상태인 경우 퀴즈 설정 전체(문제 텍스트, 선택지, 정답, 제한시간, 정답결과 노출시간, 순서)가 읽기 전용으로 전환됨
- [+ 문제 추가] 버튼 및 문제 [삭제] 버튼도 비활성화
- [+ 선택지 추가] 버튼 및 선택지 [삭제] 버튼도 비활성화
- 퀴즈설정 섹션 상단에 안내 문구 표시: "진행중(Active) 상태에서는 퀴즈를 수정할 수 없습니다."
- 해당 필드 회색 배경 처리, 입력 불가

### 3.4 보상 유형 선택 (ST 전용)

##### 보상 유형

- Segmented Button: 단계별 보상 / 비율별 보상 (기본값: 단계별 보상)
- **단계별 보상** 선택 시: 모집인원별 상금 단계 테이블 표시, 모집률에 따른 상금 비율을 단계별로 설정
- **비율별 보상** 선택 시: 모집인원별 상금 단계 테이블 숨김, 적용 상금풀 = 실제 모집인원 / 최대 모집인원 × 최대 상금풀 공식으로 자동 계산
- 보상 유형 변경 시 모집인원별 상금 단계 입력값 초기화

### 3.5 버튼 동작

##### 취소 버튼
- 위치: 하단 좌측
- 스타일: Secondary
- 동작: CEB-BO-GZ-202-MD-EDIT-CANCEL 모달 표시
- 단축키: ESC

##### 저장 버튼
- 위치: 하단 우측
- 스타일: Primary
- 동작:
  1. 수정 가능한 필드에 대한 유효성 검사
  2. 통과 시 API 호출
  3. 성공 시:
     - 현재 상태가 Draft이고 모든 필수 항목 입력 완료:
       - 게임 상태 Draft → Ready로 변경
       - Success 토스트: "게임이 저장되었습니다. 게시대기 상태로 변경되었습니다."
     - 현재 상태가 Ready, Active, Pending:
       - 상태 유지
       - Success 토스트: "게임 정보가 수정되었습니다."
     - 게임 상세(CEB-BO-GZ-202) 화면으로 복귀
  4. 실패 시:
     - 오류 메시지 토스트 표시
     - 폼 유지
- 단축키: Ctrl+S (Cmd+S on Mac)

## 5. 에러 및 예외상황

##### 필수 입력 검사
- 모든 필수(*) 필드 미입력 시:
  - 해당 필드 하단에 빨간색 오류 메시지 표시
  - 예: "타이틀을 입력해주세요."
  - 첫 번째 에러 필드로 자동 스크롤

##### 날짜/시간 검증 (Prediction Market)
- 투표 종료일시가 현재 시간 이전일 경우: "투표 종료일시는 현재 시간 이후여야 합니다."
- 결과 발표 예정일이 종료일시 이전: "결과 발표 예정일은 투표 종료일시 이후여야 합니다."

##### 날짜/시간 검증 (Survival Trivia)
- 게임 시작일시가 현재 시간 + 30분 이내일 경우: "게임 시작일시는 현재 시간으로부터 30분 이후여야 합니다."

##### 퀴즈 미입력 검증 (Survival Trivia)
- 문제 텍스트 미입력 시: "{n번} 문제 텍스트를 모든 언어로 입력해주세요."
- 선택지 미입력 시: "{n번} 문제의 선택지를 모두 입력해주세요."
- 정답 미지정 시: "{n번} 문제의 정답을 지정해주세요."

##### 퀴즈 수정 불가 안내 (Survival Trivia, Active 상태)
- 퀴즈 필드 편집 시도 시: 퀴즈설정 섹션 상단 안내 문구 표시 ("진행중(Active) 상태에서는 퀴즈를 수정할 수 없습니다.")
- 모든 퀴즈 관련 필드 비활성화 (회색 배경, 클릭 불가)

##### 숫자 범위 검증
- 총 상금 GP가 0 이하 (PM): "총 상금 GP는 1 이상이어야 합니다."
- 최대 상금풀이 0 이하 (ST): "최대 상금풀은 1 GP 이상이어야 합니다."
- 배수가 1 이하 (ST): "배수는 1배를 초과해야 합니다."
- 최대 모집인원이 0 이하 (ST): "최대 모집인원은 1명 이상이어야 합니다."
- 탈락자 응모권 수량이 0 이하 (ST): "탈락자 응모권 수량은 1장 이상이어야 합니다."
- 참여 정원이 0 이하 (PM): "참여 정원은 1명 이상이어야 합니다."
- 부스팅 배수가 범위 초과 (PM): "부스팅 배수는 1배 이상 10배 이하여야 합니다."
- 문제당 제한시간이 범위 초과 (ST): "문제당 제한시간은 5초 이상 60초 이하여야 합니다."

##### 네트워크 오류
- API 호출 실패 시: "게임 수정에 실패했습니다. 다시 시도해주세요." 토스트 표시
- 타임아웃: "요청 시간이 초과되었습니다. 다시 시도해주세요."

##### 권한 오류
- 수정 권한 없음: "게임을 수정할 권한이 없습니다." 토스트 표시 후 상세 화면으로 복귀
- 게임 상태가 Closed 또는 Ended: "종료된 게임은 수정할 수 없습니다." 토스트 표시 후 상세 화면으로 리다이렉트
