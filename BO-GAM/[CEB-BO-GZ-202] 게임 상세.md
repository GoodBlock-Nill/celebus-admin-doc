## 1. 화면 목적

게임의 상세 정보를 조회하고 게임 상태에 따른 액션(수정, 게시, 결과 입력, 보상 지급 등)을 수행하는 화면입니다.

##### 주요 기능
- 게임 기본정보 조회 (타이틀, 설명, 보상/참여/일정 설정)
- 참여자 내역 조회 (선택, 참여 GP, 부스팅 GP)
- 결과 및 보상 정보 조회
- 게임 상태별 액션 실행 (게시, 수정, 강제 종료, 결과 입력, 보상 지급)

##### 접근 경로
- 게임 목록(CEB-BO-GZ-201) > 게임 행 클릭 시 이동

## 3. 화면 내 기능설명

### 3.1 게임 상태 및 액션

| 상태 | 코드 | Badge | 가능 액션 |
| ---- | ---- | ---- | ---- |
| 임시저장 | Draft | 회색 | 수정, 삭제 |
| 게시대기 | Ready | 파란색 | 수정, 게시, 삭제 |
| 진행중 | Active | 초록색 | 수정, 강제 종료 |
| 결과 대기 | Pending | 주황색 | 수정, 결과 입력, 강제 종료 |
| 결과 확정 | Closed | 보라색 | 결과 수정, 보상 지급 |
| 종료 | Ended | 검정색 | 조회만 가능 |

### 3.2 탭 구성

###### 참여내역 탭 동작
- 기본 정렬: 참여일시 내림차순
- 페이지네이션: 20건/페이지
- ROW 클릭: 회원 상세(CEB-BO-USR-201) 화면으로 이동
- Empty 상태: "참여 내역이 없습니다." 메시지 표시

###### 보상 상세 테이블 동작
- 기본 정렬: 보상 GP 내림차순
- 결과가 입력되지 않았거나 정답자가 없을 경우 테이블 비표시

###### 보상 계산 로직
- 정답자 보상 = 총 상금 GP × (개인 배분 값 / 전체 합)
- 기본: 개인 배분 값 = 참여 GP
  - 부스팅 구매시: 개인 배분 값 = 참여 GP × 부스팅 배수
- 전체 합 = ∑(각 참여자의 배분 값)
- 참여 GP는 정답/오답 관계없이 전액 환급
- 부스팅 GP는 환급 없음, 배분 가중치로만 사용
- 소수점 8자리 이하 절삭, 단수는 시스템 귀속

### 3.3 액션 버튼별 동작

##### 수정하기
- 조건: 상태가 Draft, Ready, Active, 또는 Pending
- 동작: CEB-BO-GZ-202-EDIT 화면으로 이동

##### 결과 수정
- 조건: 상태가 Closed
- 동작: CEB-BO-GZ-202-MD-RESULT-EDIT 모달 표시

##### 게시하기
- 조건: 상태가 Ready
- 동작: CEB-BO-GZ-202-MD-PUBLISH 모달 표시

##### 삭제하기
- 조건: 상태가 Draft 또는 Ready
- 동작: CEB-BO-GZ-202-MD-DELETE 모달 표시

##### 강제 종료
- 조건: 상태가 Active 또는 Pending
- 동작: CEB-BO-GZ-202-MD-CLOSE 모달 표시

##### 결과 입력
- 조건: 상태가 Pending
- 동작: CEB-BO-GZ-202-MD-RESULT 모달 표시

##### 보상 지급
- 조건: 상태가 Closed
- 동작: CEB-BO-GZ-202-MD-REWARD 모달 표시

## 4. 규칙/로직

### 권한 구분

| 권한 | 접근 가능 영역 | 실행 가능 액션 |
| ---- | ---- | ---- |
| 읽기 | 모든 탭 조회 | 조회만 가능 |
| 쓰기 | 모든 탭 조회 | 수정, 게시, 결과 입력, 보상 지급 |
| 삭제 | 모든 탭 조회 | 삭제 추가 |

### 제약사항

- Closed 또는 Ended 상태에서는 수정 불가
- Ended 상태에서는 모든 액션 불가 (조회만 가능)
- 결과는 Closed 상태에서 수정 가능
- 보상 지급은 1회만 가능 (취소 불가)

## 5. 에러 및 예외상황

##### 접근 권한
- 읽기 권한 없음: 게임 목록으로 리디렉션, "접근 권한이 없습니다." 토스트 표시
- 쓰기 권한 없음: 액션 버튼 비활성화 또는 비표시

##### 데이터 로딩 실패
- 게임 정보 조회 실패 시 "게임 정보를 불러올 수 없습니다." 오류 메시지 표시
- [재시도] 버튼 제공

##### 네트워크 오류
- "네트워크 오류가 발생했습니다. 다시 시도해주세요." 토스트 표시
