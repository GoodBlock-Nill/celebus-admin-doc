## 1. 화면 목적

게임존 참여자들의 누적 GP 기준 랭킹을 조회하는 화면입니다. TOP 10 사용자를 하이라이트하여 표시하고, 전체 랭킹 목록을 제공합니다.

### 주요 기능

- GP 기준 전체 랭킹 조회
- TOP 10 하이라이트 섹션 (5열 그리드 카드형)
- 닉네임 검색
- 회원 상세 화면 연동

### 랭킹 집계 기준

**누적 GP**: 보상 GP + 환급 GP - 참여 GP - 부스팅 GP (순 수익)

**승률**: (정답 게임 수 / 참여 완료 게임 수) × 100%

## 4. 규칙/로직

### 4.1 누적 GP 집계

**공식**:

```text
누적 GP = (보상 GP + 환급 GP) - (참여 GP + 부스팅 GP)
```

**구성 요소**:

| 항목 | 설명 | 부호 |
| ---- | ---- | ---- |
| 보상 GP | 정답 맞힌 게임의 보상 | + |
| 환급 GP | 게임 취소/자동 환급 시 환급액 | + |
| 참여 GP | 게임 참여 시 지불한 GP | - |
| 부스팅 GP | 부스팅으로 지불한 GP | - |

**예시**:

- 보상 GP: 50,000
- 환급 GP: 10,000
- 참여 GP: 8,000
- 부스팅 GP: 2,000
- **누적 GP = (50,000 + 10,000) - (8,000 + 2,000) = 50,000 GP**

### 4.2 승률 계산

**공식**:

```text
승률 = (정답 게임 수 / 참여 완료 게임 수) × 100
```

**정답 게임**: 사용자가 선택한 선택지가 최종 결과와 일치한 게임

**참여 완료 게임**: 결과가 확정된 게임 (완료 상태)

**소수점**: 반올림하여 정수 표시 (예: 85%)

### 4.3 참여 게임 수

**집계 기준**: 결과가 확정된 게임 (완료 상태)

**제외**: 대기, 모집중, 진행중, 취소 상태 게임

### 4.4 최근 참여일

**집계 기준**: 사용자가 마지막으로 게임에 참여한 날짜

**형식**: YYYY.MM.DD

### 4.5 랭킹 갱신 주기

**기본**: 1시간마다 집계 (CEB-BO-GZ-402 설정에서 변경 가능)

**옵션**: 실시간 / 1시간 / 1일

### 4.6 최소 참여 횟수

**기준**: CEB-BO-GZ-402 설정에서 정의 (기본값: 1회)

**적용**: 최소 참여 횟수 미달 사용자는 랭킹에서 제외

### 4.7 닉네임 형식

**허용 문자**: 영문 소문자(a-z), 숫자(0-9), 밑줄(_)

**정규식**: `^[a-z0-9_]+$`

**적용**: 앱에서 회원가입 시 적용되는 규칙이며, 백오피스에서는 표시만 함

## 5. 에러 및 예외상황

### 데이터 로딩 실패

##### 상황
- API 호출 실패 또는 네트워크 오류

##### 처리
1. 에러 토스트 표시
2. [다시 시도] 버튼 제공

### 권한 오류

##### 상황
- 읽기 권한이 없는 관리자가 화면 접근 시도

##### 처리
1. 접근 차단
2. 권한 오류 페이지로 리다이렉트

### 검색 결과 없음

##### 상황
- 검색어에 해당하는 사용자가 없음

##### 처리
1. Empty State 표시: "검색 결과가 없습니다."
2. 필터 초기화 버튼 제공

### 랭킹 데이터 없음

##### 상황
- 최소 참여 횟수를 충족하는 사용자가 없음

##### 처리
1. Empty State 표시: "랭킹 데이터가 없습니다."
2. 안내 메시지: "최소 참여 횟수를 충족하는 사용자가 없습니다."
