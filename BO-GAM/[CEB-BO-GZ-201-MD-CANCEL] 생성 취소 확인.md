## 1. 화면 목적

게임 생성 화면에서 사용자가 [취소] 버튼을 클릭했을 때, 작성 중인 데이터가 저장되지 않음을 경고하고 최종 확인을 받는 모달입니다. 실수로 인한 데이터 손실을 방지합니다.

## 3. 화면 내 기능설명

### 모달 오픈

##### 트리거
- CEB-BO-GZ-201-CREATE (게임 생성) 화면에서 [취소] 버튼 클릭

##### 동작
1. 배경 화면 Dim 처리
2. 모달 중앙 정렬하여 표시
3. 모달 외부 스크롤 비활성화

### 계속 작성

##### 트리거
- [계속 작성] 버튼 클릭

##### 동작
1. 모달 닫기
2. CEB-BO-GZ-201-CREATE (게임 생성) 화면 유지
3. 입력된 데이터 모두 유지

### 취소하기

##### 트리거
- [취소하기] 버튼 클릭

##### 동작
1. 모달 닫기
2. 입력된 모든 데이터 폐기
3. CEB-BO-GZ-201 (게임 리스트) 화면으로 이동

### 모달 닫기 (취소)

##### 트리거
- ESC 키 입력
- 모달 배경 클릭

##### 동작
- 모달 닫기 (= [계속 작성]과 동일)

## 4. 규칙/로직

### 모달 표시 조건

| 조건 | 모달 표시 여부 |
| ---- | ------------- |
| 입력된 데이터가 있음 | 표시 (O) |
| 입력된 데이터가 없음 | 표시 안 함 (X), 바로 리스트로 이동 |

##### 입력된 데이터 판단 기준
- 모든 입력 필드가 기본값/빈 값인지 확인
- 하나라도 변경된 경우 "입력된 데이터 있음"으로 간주

### 버튼 스타일

| 버튼 | 스타일 | 의미 |
| ---- | ------ | ---- |
| [계속 작성] | Secondary | 안전한 선택, 데이터 유지 |
| [취소하기] | Danger | 위험한 선택, 데이터 삭제 |

### 포커스 관리

- 모달 오픈 시 [계속 작성] 버튼에 기본 포커스
- Tab 키로 버튼 간 포커스 이동
- Enter 키로 포커스된 버튼 실행

### 키보드 단축키

| 키 | 동작 |
| --- | ---- |
| ESC | 모달 닫기 (계속 작성) |
| Enter | 포커스된 버튼 실행 |
| Tab | 다음 버튼으로 포커스 이동 |
