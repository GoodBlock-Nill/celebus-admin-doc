### 2.2 영역별 구성요소

## 4. 규칙/로직

### 4.1 데이터 조회 기준

| 항목 | 기준 |
| ---- | ---- |
| "오늘" 기준 시간 | 서버 기준 00:00:00 ~ 23:59:59 (KST, UTC+9) |
| 데이터 갱신 주기 | 화면 진입 시 1회 조회, 자동 갱신 없음 |
| 숫자 포맷 | 천 단위 콤마(,) 표시 (예: 1,234) |

### 4.2 현황 항목별 집계 기준

| 항목 | 집계 기준 |
| ---- | --------- |
| 진행중 게임 | 상태가 Active인 전체 게임 수 |
| 결과 입력 필요 | 상태가 Pending인 전체 게임 수 |
| 오늘 GP 가져오기 | 오늘 CELB → GP 교환 완료된 GP 합계 |
| 오늘 CELB으로 보내기 | 오늘 GP → CELB 교환 완료된 GP 합계 |
| 진행중 게임 (PM) | 상태가 Active인 Prediction Market 게임 수 |
| 결과 입력 필요 (PM) | 상태가 Pending인 Prediction Market 게임 수 |
| 오늘 참여자 (PM) | 오늘 Prediction Market에 참여한 유저 수 |
| TOP 10 공개 | 랭킹 설정의 TOP 10 공개 여부 |
| TOP 1 | 누적 GP 기준 1위 유저의 닉네임 |
| TOP 1 누적 GP | 누적 GP 기준 1위 유저의 누적 GP |
| 오늘 GP 변동 건수 | 오늘 발생한 모든 GP 증감 내역 건수 |

### 4.3 게임 유형별 카드 노출 조건

| 게임 유형 상태 | 카드 노출 | 바로가기 | 상세 항목 |
| -------------- | --------- | -------- | --------- |
| Active | 노출 | 활성화 | 표시 |
| Inactive | 미노출 | - | - |
| Coming Soon | 노출 | 미표시 | 중앙에 아이콘 + "준비중" 텍스트 + 부가설명 |

### 4.4 권한별 분기

| 권한 | 동작 |
| ---- | ---- |
| 게임존 읽기 권한 있음 | 화면 접근 가능, 모든 현황 조회 가능 |
| 게임존 읽기 권한 없음 | 사이드 메뉴에서 "게임존" 메뉴 미노출 |

### 4.5 상태별 표시

| 상태 | 조건 | 표시 |
| ---- | ---- | ---- |
| Loading | 데이터 로딩 중 | 각 카드/항목별 Skeleton UI |
| Default | 데이터 정상 조회 | 조회된 값 표시 |
| Empty | 데이터 없음 (값이 0) | "0" 표시 |
| Error | 조회 실패 | 해당 카드에 "조회 실패" 메시지 + 재시도 링크 |

## 5. 에러 및 예외상황

| 상황 | 처리 |
| ---- | ---- |
| 전체 데이터 조회 실패 | 토스트: "데이터를 불러오는데 실패했습니다." |
| 특정 카드 데이터 조회 실패 | 해당 카드만 에러 표시, 나머지 카드 정상 표시 |

| 권한 없이 URL 직접 접근 | 권한 없음 페이지로 리다이렉트 |
