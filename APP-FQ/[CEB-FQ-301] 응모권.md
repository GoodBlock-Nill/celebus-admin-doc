## 1. 화면 목적

유저가 보유한 응모권(티켓) 잔액을 확인하고, 적립/사용 거래 이력을 조회하는 화면입니다.

##### 주요 기능
- 보유 응모권 잔액 실시간 확인
- 적립 이력 조회 (Fan Quest 보상, Survival Trivia 탈락 보상, Prediction Market 보상)
- 사용 이력 조회 (Raffle 응모)
- 반환 이력 조회 (Raffle 취소 시 반환)

##### 접근 경로
- 홈 메인(CEB-HOM-501-FQ) > 헤더 🎫 응모권 아이콘 탭 시 이동

## 2. UI 설명

### 2.1 공통 레이아웃

```text
┌──────────────────────────────────┐
│ Status Bar                       │
├──────────────────────────────────┤
│ [<] 응모권                       │
├──────────────────────────────────┤
│ 🎫 보유 응모권  {N}              │
│ (강조 카드, 보라색 배경)           │
├──────────────────────────────────┤
│ [전체] [적립] [사용]             │
├──────────────────────────────────┤
│ 이력 리스트 (무한 스크롤)         │
│                                  │
│ [{아티스트}] QUEST               │
│ {타이틀}                    +{N} │
│ {일시}                 잔여 {N}장│
│──────────────────────────────────│
│ [{아티스트}] RAFFLE              │
│ {타이틀}                    -{N} │
│ {일시}                 잔여 {N}장│
│──────────────────────────────────│
│ Survival Trivia                  │
│ {타이틀}                    +{N} │
│ {일시}                 잔여 {N}장│
└──────────────────────────────────┘
```

### 2.2 영역별 구성요소

##### Header 영역

| 항목 | 내용 |
| ---- | ---- |
| 좌측 | [<] 뒤로가기 버튼 (이전 화면으로 복귀) |
| 타이틀 | "응모권" |

##### 보유 응모권 카드

| 항목 | 내용 |
| ---- | ---- |
| 배경 | 보라색 강조 카드 |
| 아이콘 | 🎫 응모권 아이콘 |
| 수량 | 서버에서 호출한 유저의 보유 응모권 잔액 (실시간 표시) |
| 표시 형식 | "보유 응모권" 레이블 + 잔액 수량 (Big Font) |

##### 탭 메뉴

| 항목 | 내용 |
| ---- | ---- |
| 구조 | 상단 고정 탭 (3개) |
| Tab 1 | 전체 (적립 + 사용 전체) |
| Tab 2 | 적립 (+ 변동만) |
| Tab 3 | 사용 (- 변동만) |
| 초기 상태 | 전체 탭이 기본 선택된 상태로 진입 |

##### 이력 리스트

| 요소 | 설명 |
| ---- | ---- |
| 1행: 출처 라벨 | 아티스트/게임유형 표시 (예: [{아티스트}] QUEST, Survival Trivia, Prediction Market, [{아티스트}] RAFFLE) |
| 2행: 이벤트 타이틀 + 변동 수량 | 좌측 타이틀, 우측 +{N} (초록) 또는 -{N} (빨강) |
| 3행: 일시 + 잔여 수량 | 좌측 YYYY.MM.DD HH:MM:SS, 우측 "잔여 {N}장" |

> 상세 스타일은 피그마 참조

## 3. 화면 내 기능설명

### 3.1 [<] 뒤로가기

- 이전 화면(홈 메인)으로 복귀

### 3.2 탭 전환

- 전체 / 적립 / 사용 탭 전환 시 해당 조건에 맞는 이력만 필터링하여 표시
- 탭 전환 시 리스트 최상단으로 스크롤 초기화

### 3.3 리스트 스크롤

- 무한 스크롤 (20건씩 추가 로드)
- 스크롤 바닥 도달 시 추가 데이터 로딩

---

## 4. 규칙/로직

### 4.1 탭별 필터

| 탭 | 표시 내용 |
| ---- | ---- |
| 전체 | 적립 + 사용 전체 (최신 일시순) |
| 적립 | + 변동만 (Quest 보상, ST 탈락 보상, PM 보상, Raffle 반환) |
| 사용 | - 변동만 (Raffle 응모) |

### 4.2 이력 항목 유형

| 출처 | 라벨 형식 | 수량 표시 | 탭 분류 |
| ---- | ---- | ---- | ---- |
| Fan Quest 참여 보상 | [{아티스트}] QUEST + {Quest 타이틀} | +{N} (초록) | 적립 |
| Survival Trivia 탈락 보상 | Survival Trivia + {게임 타이틀} | +{N} (초록) | 적립 |
| Prediction Market 보상 | Prediction Market + {게임 타이틀} | +{N} (초록) | 적립 |
| Raffle 응모 | [{아티스트}] RAFFLE + {Raffle 타이틀} | -{N} (빨강) | 사용 |
| Raffle 취소/반환 | [{아티스트}] RAFFLE + {Raffle 타이틀} | +{N} (초록) | 적립 |

### 4.3 각 항목 표시 필드

| 행 | 좌측 | 우측 |
| ---- | ---- | ---- |
| 1행 | 출처 라벨 (아티스트/게임유형) | - |
| 2행 | 이벤트 타이틀 | 변동 수량 (+{N} 또는 -{N}) |
| 3행 | 일시 (YYYY.MM.DD HH:MM:SS) | 잔여 {N}장 |

### 4.4 정렬 기준

- 최신 일시순 (desc)

### 4.5 검색

- 없음

### 4.6 페이지네이션

- 무한 스크롤 (20건씩 추가 로드)
