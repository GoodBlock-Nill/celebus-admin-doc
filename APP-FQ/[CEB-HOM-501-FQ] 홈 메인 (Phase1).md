## 1. 화면 목적

CELEBUS APP의 메인 홈 화면으로, 사용자에게 현재 진행 중인 Raffle(래플)과 Quest(퀘스트) 정보를 한눈에 제공합니다. 사용자는 이 화면에서 참여 가능한 이벤트를 확인하고, 각 콘텐츠 상세 화면으로 빠르게 이동할 수 있습니다.

##### 주요 기능
- 보유 응모권 수량 확인
- 현재 진행 중인 Raffle 카드 리스트 조회 및 상세 이동
- 현재 진행 중인 Quest 카드 리스트 조회 및 상세 이동
- 누적 참여 이력 요약 표시
- 바텀 앱바를 통한 주요 화면 내비게이션

##### 접근 경로
- 앱 실행 시 기본 진입 화면
- 바텀 앱바 Home 탭 선택 시 이동
- CELEBUS 로고 탭 시 화면 최상단 스크롤

## 2. UI 설명

### 2.1 공통 레이아웃

```text
+------------------------------------------------------------------+
| Status Bar                                                        |
+------------------------------------------------------------------+
| [CELEBUS 로고]                        [🎫 {N}]  [🔔 알림 아이콘]  |
+------------------------------------------------------------------+
| 환영 배너                                                         |
| "어서오세요, {닉네임}님! ✨"                                       |
| "내 누적 참여 Quests {N} | RAFFLE {N}"                           |
+------------------------------------------------------------------+
| NOW RAFFLE                                                        |
| [카드1] [카드2] [카드3] ...  (가로 스와이프)                        |
+------------------------------------------------------------------+
| QUESTS  🎫 응모권 모으기                                           |
| [카드1] [카드2] [카드3] ...  (가로 스와이프)                        |
+------------------------------------------------------------------+
| 바텀 앱바                                                         |
| [Home]  [Collection]  [CELEBUS]  [Game]  [MY]                    |
+------------------------------------------------------------------+
```

### 2.2 영역별 구성요소

##### Header 영역

| 항목 | 내용 |
| ---- | ---- |
| CELEBUS 로고 | 앱 브랜드 로고 (좌측 상단). 탭 시 화면 최상단으로 스크롤 |
| 티켓 아이콘 + 수량 | 사용자 보유 응모권 수량 표시 (예: 🎫 12). 탭 시 CEB-FQ-301 화면으로 이동 |
| 알림 아이콘 | 푸시 알림 확인 버튼. 탭 시 CEB-NTF-510 화면으로 이동 |

##### 환영 메시지 영역 (배너)

| 항목 | 내용 |
| ---- | ---- |
| 환영 문구 | "어서오세요, {닉네임}님! ✨" (로그인 사용자의 닉네임 노출) |
| 누적 참여 정보 | "내 누적 참여 Quests {N} \| RAFFLE {N}" (사용자의 전체 참여 이력 요약) |

##### NOW RAFFLE 섹션

| 항목 | 내용 |
| ---- | ---- |
| 섹션 타이틀 | "NOW RAFFLE" (고정 텍스트) |
| Raffle 카드 리스트 | 가로 스와이프 가능한 카드 형태. 카드 탭 시 CEB-FQ-202 화면으로 이동 |

**Raffle 카드 구성요소**

| 요소 | 설명 |
| ---- | ---- |
| 대표 이미지 | Raffle 썸네일 이미지 (백오피스에서 등록) |
| 아티스트명 | 해당 Raffle의 아티스트명 |
| Raffle 타이틀 | 래플 이벤트명 (다국어 지원) |
| D-Day 표시 | "D-{N}" 형식으로 마감까지 남은 일수 표시 |
| 기간 표시 | "YYYY.MM.DD ~ YYYY.MM.DD" 형식 |
| 참여자 수 | "참여자 {N}명" |
| 누적 응모 수 | "응모 {N}" |
| 참여 상태 뱃지 | "참여중" (사용자가 이미 응모한 경우) |

##### QUESTS 섹션

| 항목 | 내용 |
| ---- | ---- |
| 섹션 타이틀 | "QUESTS" (고정 텍스트) |
| 섹션 부제 | "🎫 응모권 모으기" (퀘스트 목적 안내) |
| Quest 카드 리스트 | 가로 스와이프 가능한 카드 형태. 카드 탭 시 CEB-FQ-205 화면으로 이동 |

**Quest 카드 구성요소**

| 요소 | 설명 |
| ---- | ---- |
| 아티스트명 | 해당 Quest의 아티스트명 |
| Quest 타이틀 | 퀘스트 이벤트명 (다국어 지원) |
| 보상 정보 | "보상 수령 {N}" (응모권 또는 NFT 수량) |
| D-Day 표시 | "D-{N}" 형식으로 마감까지 남은 일수 표시 |
| 기간 표시 | "YYYY.MM.DD ~ YYYY.MM.DD" 형식 |
| 참여 상태 뱃지 | "참여중" / "심사중" / "참여완료" |
| 심사중 건수 | 심사 대기 중인 제출물 수량 표시 (예: "1 심사중") |

> Quest 카드에서는 연관 링크 정보를 표시하지 않음. 유저가 Quest 진행(제출) 화면(CEB-FQ-205)에 진입하면 백오피스에서 등록한 연관 링크가 라벨 버튼으로 노출됨.

## 3. 화면 내 기능설명

### 3.1 데이터 노출 조건

| 항목 | 조건 | 설명 |
| ---- | ---- | ---- |
| Raffle 카드 노출 | 상태가 Active인 Raffle만 노출 | Draft, Ended, Closed 상태 제외 |
| Quest 카드 노출 | 상태가 Active인 Quest만 노출 | Draft, Ended 상태 제외 |
| D-Day 계산 | 마감일시 - 현재일시 기준 | D-Day 지나면 메인 리스트에서 자동 제거 |

### 3.2 정렬 기준

| 섹션 | 정렬 기준 | 상세 설명 |
| ---- | ---- | ---- |
| NOW RAFFLE | 마감일 임박순 > 최신 생성순 | 마감이 가까운 Raffle이 상단에 노출 |
| QUESTS | 참여가능 > 반려 > 검토중 > 지급완료 > 최신순 | (1) 참여가능 (2) 반려 (재참여 가능) (3) 검토중 (4) 지급완료 순서로 정렬. 동일 상태 내에서는 최신 생성순 |

### 3.3 Quest 카드 상태 정의

| 상태 | 조건 | UI 표시 | 클릭 가능 여부 |
| ---- | ---- | ---- | ---- |
| 참여가능 (참여한적 없음) | 사용자가 해당 퀘스트에 한 번도 제출하지 않음 | stroke(inside, 1px, Border/Opacity), 그림자 적용 | O |
| 참여중 | 사용자가 제출 이력 있으나 추가 참여 가능 | 기본 카드 스타일 | O |
| 심사중 | 제출물이 관리자 검토 대기 상태 | "심사중" 뱃지 + 심사중 건수 표시 | 탭 시 토스트 메시지 표시 |
| 참여완료 | 퀘스트 참여 한도 도달 또는 마감 | "참여완료" 뱃지 | X (클릭 불가) |

### 3.4 Raffle 카드 상태 정의

| 상태 | 조건 | UI 표시 | 클릭 가능 여부 |
| ---- | ---- | ---- | ---- |
| 참여가능 | 사용자가 응모하지 않은 상태 | 기본 카드 스타일 | O |
| 참여중 | 사용자가 1회 이상 응모한 상태 | "참여중" 뱃지 | O |

### 3.5 카드 그림자 스타일 (Light/Dark 모드)

| 모드 | 그림자 컬러 | 비고 |
| ---- | ---- | ---- |
| Light Mode | #000000, 12% opacity | 베리어블 미등록 |
| Dark Mode | #FFFFFF, 8% opacity | 베리어블 미등록 |

> 상세 스타일은 피그마 참조
