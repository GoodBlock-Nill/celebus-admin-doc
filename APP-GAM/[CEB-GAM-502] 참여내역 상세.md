## 1. 화면 목적

게임 참여 관련 GP 변동 내역의 상세 정보를 보여주는 화면입니다. 참여, 부스팅, 환급, 보상 4가지 유형에 따라 다른 상세 정보를 표시하며, 게임 정보와 결과를 확인할 수 있습니다.

### 주요 기능

- 게임 참여 상세 정보 조회 (참여/부스팅/환급/보상)
- GP 변동량 및 게임 정보 확인
- 게임 상세 화면으로 이동
- 결과 확인 링크 제공 (보상 유형)

### 진입 경로

```text
CEB-GAM-501 GP 히스토리 > 참여내역 탭 > 내역 항목 탭
```

### 영역별 구성요소

#### Header 영역

- 좌측: < 뒤로가기 버튼 (GP 히스토리로 복귀)
- 중앙: "{유형} 상세" 타이틀
  - 참여 상세 / 부스팅 상세 / 환급 상세 / 보상 상세

#### 상단 카드 영역

- 게임 타이틀 (최대 2줄, 말줄임)
- GP 변동량 (큰 폰트, 색상)

#### 구분선

- 그라데이션 또는 단순 구분선

#### 상세 정보 영역

2컬럼 레이아웃 (Label : Value)

- 라벨: 왼쪽 정렬, 회색 텍스트
- 값: 오른쪽 정렬, 검정 텍스트

> 상세 스타일은 피그마 참조

### 3.2 GP 변동량 표시

| 변동 방향 | 표시 형식 | 색상 | 유형 |
| ---- | ---- | ---- | ---- |
| 증가 (+) | + {숫자} GP | 파란색 (#1E90FF) | 환급, 보상 |
| 감소 (-) | - {숫자} GP | 빨간색 (#FF4D4F) | 참여, 부스팅 |

- 천 단위 콤마 적용
- 폰트 크기: 본문보다 큰 사이즈

### 3.3 내 선택 표시

#### 일반 선택 (참여, 환급)

```
내 선택          Yes
```

- "Yes" 또는 "No" 텍스트만 표시

#### 부스팅 선택 (부스팅, 보상)

```
내 선택          Yes 🔥 부스팅 (2배)
```

- "Yes" 또는 "No" + 불 이모지(🔥) + "부스팅" + "(배수배)"
- 부스팅 적용 시에만 추가 표시

### 3.4 게임 상태 표시

| 유형 | 게임 상태 표시 | 추가 요소 |

| 환급 | 결과 발표 / 종료 | - |
| 보상 | 결과 발표 / 종료 | [결과보기 >] 링크 (보상만) |

### 3.8 사용자 시나리오

#### 시나리오 1: 게임 참여 상세 확인

1. GP 히스토리 > 참여내역 탭에서 "[참여]" 내역 탭
2. 참여 상세 화면 진입
3. Header: "참여 상세"
4. 상단 카드: 게임 타이틀 + "- 10 GP" (빨간색)
5. 내 선택: Yes
6. 게임 상태: 진행중
7. 참여일시: 2026.02.01. 09:53:32 (UTC+9)
8. 게임 정보: Prediction Market

#### 시나리오 2: 부스팅 상세 확인

1. GP 히스토리 > 참여내역 탭에서 "[부스팅]" 내역 탭
2. 부스팅 상세 화면 진입
3. Header: "부스팅 상세"
4. 상단 카드: 게임 타이틀 + "- 1 GP" (빨간색)
5. 내 선택: Yes 🔥 부스팅 (2배)
6. 게임 상태: 진행중
7. 참여일시: 2026.02.01. 09:53:32 (UTC+9)
8. 게임 정보: Prediction Market

#### 시나리오 3: 환급 상세 확인

1. GP 히스토리 > 참여내역 탭에서 "[환급]" 내역 탭
2. 환급 상세 화면 진입
3. Header: "환급 상세"
4. 상단 카드: 게임 타이틀 + "+ 10 GP" (파란색)
5. 내 선택: Yes
6. 결과: Yes (정답)
7. 게임 상태: 결과 발표
8. 환급일시: 2026.02.02. 15:00:12 (UTC+9)
9. 게임 정보: Prediction Market

#### 시나리오 4: 보상 상세 확인 및 결과 보기

1. GP 히스토리 > 참여내역 탭에서 "[보상]" 내역 탭
2. 보상 상세 화면 진입
3. Header: "보상 상세"
4. 상단 카드: 게임 타이틀 + "+ 1,000 GP" (파란색)
5. 내 선택: Yes 🔥 부스팅 (2배)
6. 결과: Yes (정답)
7. 게임 상태: 결과 발표 [결과보기 >]
8. [결과보기 >] 링크 탭 → CEB-GAM-203 결과 상세 화면 이동
9. 결과 확인 링크 및 결과 상세 정보 확인

#### 시나리오 5: 게임 상세 정보 확인

1. 참여내역 상세 화면에서 상단 카드의 게임 타이틀 탭
2. CEB-GAM-202 PM 상세 화면으로 이동
3. 게임 정보, 참여 현황, 결과 등 확인

### 표시 규칙

2. GP 변동량: 천 단위 콤마, 색상 적용
3. 내 선택: Yes/No + 부스팅 정보 (부스팅 유형만)
4. 결과 필드: 환급/보상 유형만 표시
5. 결과보기 링크: 보상 유형만 표시

| 조건 | 표시 |

| 부스팅 사용 (3배) | Yes 🔥 부스팅 (3배) |
| 부스팅 사용 (최대 10배) | Yes 🔥 부스팅 (10배) |

### 비즈니스 규칙

1. 참여내역 상세는 수정할 수 없습니다 (조회 전용)
2. 게임 삭제 시에도 타이틀은 표시됩니다
3. 환급은 참여 GP 전액 반환입니다
4. 보상은 정답자에게만 지급됩니다
5. 부스팅은 보상 가중치로 작용하며 환급되지 않습니다
6. 결과보기 링크는 결과 확인 링크(URL)가 있는 경우에만 표시됩니다
