## 1. 화면 목적

게임 참여 관련 GP 변동 내역의 상세 정보를 보여주는 화면입니다. 참여, 부스팅, 환급, 보상, 환불 5가지 유형에 따라 다른 상세 정보를 표시하며, 게임 정보와 결과를 확인할 수 있습니다.

### 주요 기능

- 게임 참여 상세 정보 조회 (참여/부스팅/환급/보상/환불)
- GP 변동량 및 게임 정보 확인
- 게임 상세 화면으로 이동
- 결과 확인 링크 제공 (보상 유형)

### 진입 경로

```text
CEB-GAM-501 GP 히스토리 > 참여내역 탭 > 내역 항목 탭
```

### 영역별 구성요소

#### Header 영역

- 좌측: < 뒤로가기 버튼 (GP 히스토리로 복귀)
- 중앙: "{유형} 상세" 타이틀
  - 참여 상세 / 부스팅 상세 / 환급 상세 / 보상 상세 / 환불 상세

#### 상단 카드 영역

- 게임 타이틀 (최대 2줄, 말줄임)
- GP 변동량 (큰 폰트, 색상)

#### 구분선

- 그라데이션 또는 단순 구분선

#### 상세 정보 영역

2컬럼 레이아웃 (Label : Value)

- 라벨: 왼쪽 정렬, 회색 텍스트
- 값: 오른쪽 정렬, 검정 텍스트

> 상세 스타일은 피그마 참조

### 3.2 GP 변동량 표시

| 변동 방향 | 표시 형식 | 색상 | 유형 |
| ---- | ---- | ---- | ---- |
| 증가 (+) | + {숫자} GP | 파란색 (#1E90FF) | 환급, 보상, 환불 |
| 감소 (-) | - {숫자} GP | 빨간색 (#FF4D4F) | 참여, 부스팅 |

- 천 단위 콤마 적용
- 폰트 크기: 본문보다 큰 사이즈

### 3.3 내 선택 표시

#### 일반 선택 (참여, 환급)

```
내 선택          Yes
```

- "Yes" 또는 "No" 텍스트만 표시

#### 부스팅 선택 (부스팅, 보상)

```
내 선택          Yes 🔥 부스팅 (2배)
```

- "Yes" 또는 "No" + 불 이모지(🔥) + "부스팅" + "(배수배)"
- 부스팅 적용 시에만 추가 표시

#### Survival Trivia

- Survival Trivia는 게임 참여 시 사전 선택(Yes/No)이 없으므로 "내 선택" 필드 자체를 표시하지 않습니다.

### 3.4 게임 상태 표시

| 유형 | 게임 상태 표시 | 추가 요소 |
| ---- | ---- | ---- |
| 참여/부스팅 | 진행중 / 결과 대기 / 결과 발표 / 종료 | - |
| 환급 | 결과 발표 / 종료 | - |
| 보상 | 결과 발표 / 종료 | [결과보기 >] 링크 (PM 보상만, ST 미표시) |
| 환불 | 취소 | - |

#### 결과보기 링크 (PM 보상만)

- 위치: "게임 상태" 필드 우측
- 표시: [결과보기 >] (파란색 링크)
- 탭 동작: CEB-GAM-203 결과 상세 화면 이동
- 전달 파라미터: 게임 ID
- **Survival Trivia는 결과보기 링크를 표시하지 않습니다.** (ST는 별도 결과 상세 화면이 없으므로)

#### 하트 변동 표시 (ST 전용)

- 위치: "게임 상태" 행 아래
- 표시: `❤️ -{소진 수} / ❤️ +{획득 수}`
- 소진 수: 게임 중 오답으로 잃은 하트 개수
- 획득 수: 게임 완주 성공(+1) 또는 관전 완료(+1)로 얻은 하트 개수
- 소진 0이면 "❤️ -0", 획득 0이면 "❤️ +0"
- **게임 진행중(Active)일 때는 하트 변동 행 미표시** (종료 후에만 확정값 표시)
- Prediction Market은 하트 변동 미표시

## 4. 규칙/로직

### 표시 규칙

1. 게임 타이틀: 최대 2줄, 초과 시 말줄임(...) 처리
2. GP 변동량: 천 단위 콤마, 색상 적용
3. 내 선택: Yes/No + 부스팅 정보 (부스팅 유형만)
4. 결과 필드: 환급/보상 유형만 표시
5. 결과보기 링크: 보상 유형만 표시

| 부스팅 사용 (3배) | Yes 🔥 부스팅 (3배) |
| 부스팅 사용 (최대 10배) | Yes 🔥 부스팅 (10배) |

| 조건 | Prediction Market | Survival Trivia |

| 환급 유형 (REFUND) | 정상 표시 | 발생하지 않음 |

| 하트 변동 | 미표시 | ❤️ -{N} / ❤️ +{N} (종료 후만) |

6. 결과보기 링크는 PM 보상 유형에서만 표시됩니다 (ST는 미표시)
7. ST 하트 변동은 게임 종료 후에만 표시됩니다 (진행중일 때 미표시)
