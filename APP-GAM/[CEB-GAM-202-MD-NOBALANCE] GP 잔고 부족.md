## 1. 화면 목적

게임 참여 시 사용자의 GP 잔액이 부족할 때 표시되는 모달입니다. 필요한 GP 금액과 현재 보유 GP를 명확히 표시하고, GP 교환소로 이동하여 충전할 수 있도록 유도합니다.

## 3. 화면 내 기능설명

### 모달 트리거

| 트리거 조건 | 동작 |
| ---- | ---- |
| 게임 상세(CEB-GAM-202)에서 YES/NO 선택 시 | GP 잔액 < 참여 비용일 때 본 모달 오픈 |
| 참여확정(CEB-GAM-202-MD-CONFIRMED)에서 참여 시도 | GP 부족 응답 시 본 모달 오픈 |
| 참여완료(CEB-GAM-202-MD-COMPLETED)에서 부스팅 시도 | GP 부족 응답 시 본 모달 오픈 |

### 버튼 동작

#### 취소 버튼

1. 본 모달 닫기
2. 게임 상세 화면으로 복귀
3. 상태 변경 없음

#### GP 충전하기 버튼

1. 본 모달 닫기
2. CEB-GAM-301 (GP 교환소) 화면으로 이동
3. 이동 시 현재 게임 ID를 쿼리 파라미터로 전달 (충전 후 복귀 경로)

### 모달 닫기 동작

- 취소 버튼 클릭 시
- 모달 외부 영역(딤) 탭 시
- 뒤로가기 제스처

## 4. 규칙/로직

### 필요한 GP 계산

#### 게임 참여 시

- 필요한 GP = 게임 참여 비용
- 예: 참여 비용이 1 GP인 경우, "필요한 GP: 1 GP"

#### 부스팅 시

- 필요한 GP = 부스팅 비용
- 예: 부스팅 비용이 1 GP인 경우, "필요한 GP: 1 GP"

### 보유 GP 표시

- 사용자 계정의 현재 GP 잔액을 실시간으로 조회하여 표시
- 형식: "{숫자} GP" (천 단위 콤마)
- 0 GP인 경우: "0 GP"

### 다국어 안내 문구

| 언어 | 텍스트 |
| ---- | ---- |
| KO | 예측에 참여하려면 {N}GP가 필요해요. GP를 충전한 뒤 다시 시도해 주세요. |
| EN | You need {N}GP to participate. Please recharge GP and try again. |
| JP | 参加するには{N}GPが必要です。GPをチャージしてからもう一度お試しください。 |

### GP 부족 판단 기준

| 시나리오 | 조건 | 처리 |
| ---- | ---- | ---- |
| 게임 참여 | 보유 GP < 참여 비용 | 본 모달 표시 |
| 부스팅 | 보유 GP < 부스팅 비용 | 본 모달 표시 |
| 참여 + 부스팅 동시 | 보유 GP < (참여 비용 + 부스팅 비용) | 본 모달 표시 (필요한 GP = 합계) |

### GP 교환소 이동 시 쿼리 파라미터

```
/exchange?returnTo=game&gameId={gameId}
```

- `returnTo`: 충전 후 복귀할 화면 (game 고정)
- `gameId`: 현재 게임의 ID

## 5. 에러 및 예외상황

### 클라이언트 에러

| 에러 상황 | 에러 메시지 | 처리 방법 |
| ---- | ---- | ---- |
| GP 정보 조회 실패 | "GP 정보를 불러올 수 없습니다." | 에러 토스트, 모달 닫기 |
| 게임 정보 조회 실패 | "게임 정보를 불러올 수 없습니다." | 에러 토스트, 모달 닫기 |

### 서버 에러

| HTTP 상태 | 에러 메시지 | 처리 방법 |
| ---- | ---- | ---- |
| 401 | "로그인이 필요합니다." | 로그인 화면으로 이동 |
| 404 | "게임을 찾을 수 없습니다." | 에러 토스트, 게임 리스트로 이동 |
| 500 | "서버 오류가 발생했습니다." | 에러 토스트, 모달 유지 |

### 네트워크 에러

| 에러 상황 | 처리 방법 |
| ---- | ---- |
| 타임아웃 | "요청 시간이 초과되었습니다. 다시 시도해주세요." 에러 토스트 |
| 연결 실패 | "네트워크 연결을 확인해주세요." 에러 토스트 |

### 데이터 정합성 에러

| 에러 상황 | 처리 방법 |
| ---- | ---- |
| 필요 GP가 음수 | "잘못된 요청입니다." 에러 표시, 모달 닫기 |
| 보유 GP가 음수 | GP 데이터 재조회, 실패 시 "0 GP" 표시 |
