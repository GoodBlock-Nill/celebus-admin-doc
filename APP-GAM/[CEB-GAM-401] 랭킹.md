## 1. 화면 목적

모든 게임(Prediction Market + Survival Trivia)을 통해 얻은 '누적 Game Point'와 그에 따른 현재 '내 순위'를 확인하는 화면이다. 보상 성과를 우선적으로 보여주며(순위 경쟁보다 내가 얻은 성과가 먼저 인지되도록), 상세 성과는 카드 확장 영역에서 확인 가능하다. 서비스 초기에는 데이터 부족으로 의미 있는 순위 제공이 어려워 개인 기록 중심의 화면을 우선 제공한다.

## 3. 화면 내 기능설명

### 3.1 뒤로가기

**트리거**: Header 좌측 < 버튼 탭

**동작**: 게임존 메인 (CEB-GAM-101) 화면으로 이동

### 3.2 누적 GP 안내

**트리거**: 내 랭킹 영역 상단 ⓘ 아이콘 탭

**동작**: 바텀시트 표시

**바텀시트 내용**:
- 타이틀: "누적 GP란?"
- 본문:
  - "누적 GP는 게임을 통해 지금까지 획득한 Game Point의 합계에요."
  - "게임 참여로 얻은 GP가 모두 합산되며, 랭킹은 누적 GP를 기준으로 정해져요."
  - "게임별 상세 성과는 카드 펼침 영역에서 확인할 수 있어요."

### 3.3 내 랭킹 펼치기/접기

**트리거**: 내 랭킹 영역의 펼침 영역 또는 화살표 아이콘 탭

**동작**:
- 접힌 상태 → 펼친 상태: 게임별 상세 성과 표시 (애니메이션)
- 펼친 상태 → 접힌 상태: 간략 정보만 표시 (애니메이션)

**애니메이션**: 0.3초 Ease-in-out

### 3.4 Pull-to-Refresh

**트리거**: 화면 상단에서 아래로 드래그

**동작**: 랭킹 데이터 재조회 (내 랭킹 + TOP 10)

**완료**: 로딩 인디케이터 사라짐, 데이터 갱신

## 4. 규칙/로직

### 4.1 누적 GP 집계

**공식**:

```text
누적 GP = 보상 GP + 환급 GP
```

**구성 요소**:

| 항목 | 설명 | 부호 |
| ---- | ---- | ---- |
| 보상 GP | 정답 맞힌 게임의 보상 | + |
| 환급 GP | 결과 발표 후 참여 GP 반환 | + |

**예시**:
- 보상 GP: 50,000
- 환급 GP: 10,000
- **누적 GP = 50,000 + 10,000 = 60,000 GP**

### 4.2 승률 계산

**공식**:

```text
승률 = (정답 게임 수 / 참여 완료 게임 수) × 100
```

**정답 게임**: 사용자가 선택한 선택지가 최종 결과와 일치한 게임

**참여 완료 게임**: 결과가 확정된 게임 (종료 상태)

**소수점**: 반올림하여 정수 표시 (예: 68%)

**참여 없을 시**: "-" 표시

### 4.3 상위 비율 계산

**공식**:

```text
상위 비율 = (내 순위 / 전체 랭킹 참여자 수) × 100
```

**소수점**: 첫째 자리까지 표시 (예: 13.2%)

**예시**:
- 내 순위: 123위
- 전체 참여자: 1,000명
- **상위 비율 = (123 / 1,000) × 100 = 12.3%**

### 4.4 TOP 10 정렬 기준

**기본 정렬**: 누적 GP 내림차순 (높은 순)

**동점 시 정렬**:
1. 누적 GP 동일 시: 누적 GP를 먼저 달성한 사용자 우선
2. 달성 시간도 동일 시: UID 오름차순

### 4.5 랭킹 갱신 주기

**기본**: 매일 00시 (KST, UTC+9) 집계

**안내**: TOP 10 섹션 헤더에 "순위는 매일 00시 업데이트됩니다." 표시

### 4.6 게임별 누적 GP (펼침 영역)

**Prediction Market**:
- 누적 GP: 해당 게임 유형의 누적 GP 합계
- 참여 횟수: 결과 확정된 게임 수
- 승률: (정답 게임 수 / 참여 완료 게임 수) × 100

**Survival Trivia** (Live 2.5 버전):
- 누적 GP: 해당 게임 유형의 누적 GP 합계
- 참여 횟수: 결과 확정된 게임 수
- 승률: (정답 게임 수 / 참여 완료 게임 수) × 100
- 참여 없을 시: "참여 0회  승률 -"
