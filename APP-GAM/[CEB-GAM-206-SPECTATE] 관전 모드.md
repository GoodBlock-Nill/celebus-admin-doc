## 1. 화면 목적

탈락 후 "관전하고 하트 받기"를 선택한 사용자가 다른 참여자의 게임 진행을 실시간으로 관전하는 화면입니다. 문제와 선택지를 볼 수 있지만 직접 응답할 수는 없습니다. 채팅은 계속 참여 가능합니다. 게임 종료 시까지 관전을 완료하면 하트 1개를 보상으로 받습니다.

> 이 문서는 탈락 후 관전 모드 진행 화면을 다룹니다. 게임 진행(CEB-GAM-206), 탈락 모달(CEB-GAM-206-MD-ELIMINATE), 나가기 모달(CEB-GAM-206-MD-LEAVE), 게임 결과(CEB-GAM-206-RESULT)는 별도 문서입니다.

## 3. 화면 내 기능설명

### 3.1 문제/선택지 관전

##### 동작
- 문제 텍스트, 선택지 4개가 실시간으로 표시됨
- 선택지는 비활성화 상태 (회색 처리, 탭 불가)
- 타이머 바도 동일하게 작동 (시각적으로 감소)

##### 규칙
- 관전자는 선택지를 탭해도 응답이 전송되지 않음
- 타이머 바는 표시만 하며 탈락 이벤트와 무관

### 3.2 정답 공개 관전

##### 트리거
- 서버에서 정답 + 선택지별 선택 비율(%) 수신

##### 동작
1. 정답 선택지: 파란 테두리 + ⭕ 마크
2. 모든 선택지에 선택 비율(%) 표시
3. "내 선택" 배지 없음 (관전자이므로 응답 내역 없음)
4. 공개 후 5초 대기 → 자동으로 다음 스테이지 전환

### 3.3 채팅 기능

##### 메시지 표시
- 화면 하단 채팅 영역에 실시간 메시지 스트림 표시
- 닉네임 + 메시지 형태로 좌측 정렬
- 새 메시지 수신 시 자동 하단 스크롤

##### 메시지 입력/전송
- 💬 버튼 탭 → 키보드 오픈 + 입력 필드 활성화
- ▶ 전송 버튼 탭 → 메시지 서버 전송

##### 제한 규칙

| 규칙 | 내용 |
| ---- | ---- |
| 욕설 필터링 | 금칙어 목록 기반 자동 마스킹 처리 ("***") |
| 발송 제한 | 초당 1회 (rate limit), 초과 시 발송 무시 |
| 문자 수 제한 | 최대 100자, 초과 입력 불가 |

##### 키보드 활성화 시
- 상태 바([관전중] 배지/스테이지/참여자) 숨김
- 타이머 바와 문제/선택지는 축소 표시 (비활성화 상태 유지)
- 키보드 닫기: 빈 영역 탭 또는 전송 완료 후

### 3.4 뒤로가기

##### 트리거
- Header 좌측 [<] 버튼 탭

##### 동작
1. CEB-GAM-206-MD-LEAVE (나가기 확인) 모달 표시
2. 모달 안내 문구에 "관전 보상 하트도 받을 수 없습니다" 문구 추가
3. 나가기 확인 시: CEB-GAM-204(트리비아 홈)로 이동, 하트 미지급

### 3.5 게임 종료 시 자동 전환

##### 트리거
- 마지막 스테이지 정답 공개 완료

##### 동작
1. CEB-GAM-206-RESULT (게임 결과) 화면으로 자동 전환
2. 관전자 결과 화면: 탈락자 결과 표시 + 하트 1개 지급 안내

## 4. 규칙/로직

### 관전 모드 진입 조건

- CEB-GAM-206-MD-ELIMINATE 모달에서 "관전하고 하트 받기" 버튼 선택

### 하트 보상 규칙

| 조건 | 하트 지급 | 비고 |
| ---- | -------- | ---- |
| 끝까지 관전 완료 | +1개 | 마지막 스테이지 정답 공개 이벤트를 서버로부터 수신한 시점에 자동 지급 |
| 중도 이탈 (나가기) | 미지급 | 나가기 모달에서 "나가기" 선택 시 |
| 네트워크 끊김 후 미복귀 | 미지급 | 재연결 실패 시 이탈로 처리 |
| 이미 최대 보유(3개) | 미지급 | 하트 보유 상한(3개) 초과 불가 |

> **관전 완료 판정 기준**: 마지막 스테이지의 정답 공개 이벤트를 서버로부터 수신한 시점을 기준으로 합니다. 클라이언트 상태가 아닌 서버 이벤트 수신 여부로 판단합니다.

### 관전 재진입 불가

- CEB-GAM-206-MD-ELIMINATE 모달에서 [나가기]를 선택한 사용자는 해당 게임의 관전 모드에 재진입 불가
- 한 번 나가기를 선택하면 관전 기회는 영구 소멸
- 재진입 시도 경로 자체가 존재하지 않음 (CEB-GAM-204에서 해당 게임은 이미 "참여 완료" 상태)

### [관전중] 배지

- 하트 아이콘 영역 대신 [관전중] 배지 표시
- 색상: 보라색 또는 회색 (피그마 참조)

### 정답/오답 시각 표시 (관전 모드)

| 상태 | 테두리 색상 | 아이콘 | 배지 | 비율 표시 |
| ---- | ---------- | ------ | ---- | --------- |
| 정답 선택지 | 파란색 | ⭕ | 없음 | O |
| 일반 선택지 | 기본 (회색) | 없음 | 없음 | O |

> "내 선택" 배지는 관전자에게 표시하지 않음

### 다국어 적용

| 요소 | 다국어 적용 | 비고 |
| ---- | --------- | ---- |
| [관전중] 배지 | O (KO/EN/JP) | KO: 관전중, EN: Spectating, JP: 観戦中 |
| 문제 텍스트 | O | 서버에서 해당 언어로 제공 |
| 선택지 텍스트 | O | 서버에서 해당 언어로 제공 |
| 채팅 메시지 | X | 사용자 입력 그대로 |

### 테마 지원

- 라이트/다크 테마 모두 지원
- 시스템 설정 또는 앱 내 설정에 따라 자동 전환
