## 1. 화면 목적

하트 소진(0개에서 오답) 또는 미응답(시간 초과, 하트 없음)으로 탈락 시 표시되는 모달입니다. 사용자에게 관전 모드로 전환하거나 게임을 나갈 수 있는 선택지를 제공합니다. 끝까지 관전하면 하트 1개를 보상으로 받을 수 있음을 안내합니다.

## 3. 화면 내 기능설명

### 모달 트리거

| 트리거 조건 | 동작 |
| ---- | ---- |
| CEB-GAM-206에서 하트 0개 + 오답 | 정답 공개 5초 표시 후 본 모달 오픈 |
| CEB-GAM-206에서 미응답(시간 초과) + 하트 없음 | 정답 공개 5초 표시 후 본 모달 오픈 (오답 탈락과 동일 플로우, 동일 타이밍). 정답 공개 화면에서 선택지에 "미선택" 배지(회색) 표시 |

### 버튼 동작

#### 나가기 버튼

1. 본 모달 닫기
2. CEB-GAM-204(트리비아 홈)로 즉시 이동
3. 하트 보상 미지급

#### 관전하고 하트 받기 버튼

1. 본 모달 닫기
2. CEB-GAM-206-SPECTATE(관전 모드)로 전환
3. 끝까지 관전 완료 시 하트 1개 지급 (서버 처리)

### 모달 닫기 동작

- 나가기 / 관전하고 하트 받기 버튼으로만 닫기 가능
- 외부 영역(딤) 탭: 닫히지 않음 (강제 선택 필요)
- 뒤로가기 제스처: 닫히지 않음

### 자동 닫기 규칙

| 항목 | 규칙 |
| ---- | ---- |
| 모달 표시 시점 | 정답 공개 5초 종료 후 표시 (정답 공개와 연속, 겹치지 않음) |
| 자동 닫힘 | 모달 표시 후 5초 이내 선택하지 않으면 자동으로 CEB-GAM-204로 이동 (결과 화면 자동 전환 없이 204에 머무름) |
| 카운트다운 표시 | 버튼 내 숫자 카운트다운: [관전하고 하트 받기 (5)] → (4) → (3) → (2) → (1) → 자동 닫힘. 매초 숫자만 갱신, 버튼 텍스트 유지 |
| [나가기] 버튼 | 카운트다운 없음. 고정 텍스트 |
| 총 소요 시간 | 정답 공개 5초 + 모달 유예 5초 = 총 10초 |
| 다음 스테이지 진행 | 다음 스테이지는 탈락 유저의 모달과 무관하게 다른 유저에게 정상 진행 |
| 사유 | 게임이 진행 중이므로 무한 대기 불가 |

### 정답 공개 → 모달 전환 애니메이션

| 항목 | 설명 |
| ---- | ---- |
| 정답 공개 5초 | CEB-GAM-206의 정답/오답 공개 화면 그대로 표시 (배경) |
| 모달 등장 | 정답 공개 5초 종료 즉시 딤 오버레이 0.3초 페이드인 + 모달 0.3초 스케일인 (center → full size) |
| 배경 | 정답 공개 화면이 딤 처리되어 배경으로 유지 |

## 4. 규칙/로직

### 탈락 조건

| 조건 | 하트 변화 | 결과 |
| ---- | -------- | ---- |
| 하트 1개 이상 + 오답 | -1개 | 계속 진행 (본 모달 미표시) |
| 하트 0개 + 오답 | 0 유지 | 탈락 → 본 모달 표시 |
| 미응답 (시간 초과) + 하트 있음 | -1개 | 다음 스테이지 진행 (본 모달 미표시) |
| 미응답 (시간 초과) + 하트 없음 | 0 유지 | 탈락 → 본 모달 표시 |

### 마지막 스테이지 예외

| 조건 | 동작 |
| ---- | ---- |
| 마지막 스테이지(10/10) 오답 + 하트 0 | 정답 공개 5초 표시 후 본 모달 **생략** → CEB-GAM-206-RESULT(D타입) 직행 |
| 마지막 스테이지(10/10) 미응답 | 하트 소진 후 정답 공개 5초 표시 후 본 모달 **생략** → CEB-GAM-206-RESULT(D타입) 직행 (다음 스테이지 없으므로 탈락) |
| 사유 | 마지막 스테이지이므로 관전할 다음 문제가 없음. 관전 선택지 제공이 무의미 |

> CEB-GAM-206 문서에 이미 명시된 규칙이며, 본 모달 문서에도 교차 참조로 기록
