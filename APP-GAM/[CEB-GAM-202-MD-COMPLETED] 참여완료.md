## 1. 화면 목적

게임 참여가 성공적으로 완료된 후, **토스트 메시지**로 완료를 알리고, **바텀시트**로 부스팅 참여를 유도하는 화면입니다.

## 3. 화면 내 기능설명

### 토스트/바텀시트 트리거

| 트리거 조건 | 동작 |
| ---- | ---- |
| CEB-GAM-202-MD-CONFIRMED에서 참여 성공 | 참여완료 토스트 표시 + 바텀시트 오픈 |
| 참여 API 성공 응답 | 토스트 표시 + 바텀시트 오픈 |

### 나중에 버튼

1. 바텀시트 닫기
2. 게임 상세 화면으로 복귀
3. 게임 상세 상태: 참여완료 (부스팅 미사용)
4. 게임 상세에서 부스팅 버튼 계속 노출 (나중에 부스팅 가능)

### 부스팅 버튼

1. 부스팅 API 호출
   - 게임 ID
   - 사용자 ID
   - 부스팅 비용 (GP 차감)

2. 성공 시:
   - 사용자 GP 차감 (부스팅 비용)
   - 부스팅 내역 생성
   - 토스트 메시지: "부스팅이 완료되었습니다." (Success)
   - 바텀시트 닫기
   - 게임 상세 화면 갱신 (참여완료 + 부스팅완료 상태)

3. 실패 시:
   - GP 부족: CEB-GAM-202-MD-NOBALANCE (GP 잔고 부족) 모달 오픈
   - 기타 에러: 에러 토스트 표시, 바텀시트 유지

### 바텀시트 닫기 동작

- 나중에 버튼 클릭 시
- 바텀시트 외부 영역(딤) 탭 시
- 뒤로가기 제스처

## 4. 규칙/로직

### 부스팅 비용 표시

- 게임 설정에서 정의된 부스팅 비용 값을 표시
- 기본값: 1 GP
- 형식: "{N}GP로 부스팅" (버튼 텍스트)

### 부스팅 가능 조건

| 조건 | 검증 시점 | 실패 시 처리 |
| ---- | ---- | ---- |
| 게임 상태 = Active | API 호출 시 | "부스팅 가능한 게임이 아닙니다." 에러 토스트 |
| 사용자 GP 잔액 >= 부스팅 비용 | API 호출 시 | CEB-GAM-202-MD-NOBALANCE (GP 잔고 부족) 모달 오픈 |
| 이미 부스팅하지 않음 | API 호출 시 | "이미 부스팅한 게임입니다." 에러 토스트 |
| 게임 참여 완료 상태 | API 호출 시 | "참여하지 않은 게임입니다." 에러 토스트 |
| 투표 종료일시 미경과 | API 호출 시 | "부스팅 가능 기간이 종료되었습니다." 에러 토스트 |

### 부스팅 계산 로직

- 부스팅 사용 시: 배분 값 = 참여 GP × 부스팅 배수
- 부스팅 미사용 시: 배분 값 = 참여 GP
- 배수는 게임별 설정값 (기본 2배, 최대 10배)
- 정답자 보상 = 총 상금 GP × (개인 배분 값 / 전체 합)
- 소수점 8자리 이하 절삭, 단수는 시스템 귀속

## 5. 에러 및 예외상황

### 클라이언트 에러

| 에러 상황 | 에러 메시지 | 처리 방법 |
| ---- | ---- | ---- |
| 게임 상태가 Active가 아님 | "부스팅 가능한 게임이 아닙니다." | 에러 토스트, 바텀시트 닫기, 게임 상세 갱신 |
| GP 잔액 부족 | - | CEB-GAM-202-MD-NOBALANCE (GP 잔고 부족) 모달 오픈 |
| 이미 부스팅한 게임 | "이미 부스팅한 게임입니다." | 에러 토스트, 바텀시트 닫기, 게임 상세 갱신 |
| 참여하지 않은 게임 | "참여하지 않은 게임입니다." | 에러 토스트, 바텀시트 닫기, 게임 상세 갱신 |

### 서버 에러

| HTTP 상태 | 에러 메시지 | 처리 방법 |
| ---- | ---- | ---- |
| 400 | "유효하지 않은 요청입니다." | 에러 토스트, 바텀시트 유지 |
| 403 | "부스팅 권한이 없습니다." | 에러 토스트, 바텀시트 닫기 |
| 404 | "게임을 찾을 수 없습니다." | 에러 토스트, 게임 리스트로 이동 |
| 409 | "이미 부스팅한 게임입니다." | 에러 토스트, 바텀시트 닫기, 게임 상세 갱신 |
| 422 | "부스팅 가능 기간이 종료되었습니다." | 에러 토스트, 바텀시트 닫기, 게임 상세 갱신 |
| 500 | "서버 오류가 발생했습니다." | 에러 토스트, 바텀시트 유지 |

### 네트워크 에러

| 에러 상황 | 처리 방법 |
| ---- | ---- |

| 연결 실패 | "네트워크 연결을 확인해주세요." 에러 토스트 |
