## 1. 화면 목적

Celeb 토큰을 GP로 교환하기 위한 입금 지갑 주소를 안내하는 바텀시트입니다. 사용자는 QR 코드 스캔, 주소 복사 또는 공유를 통해 Celeb를 입금할 수 있습니다.

### 주요 기능
- Celeb 입금 주소 QR 코드 표시 (Celeb 로고 포함)
- 네트워크 정보 배지 표시 (BSC)
- 지갑 주소 텍스트 표시 및 복사
- 주소 공유 (OS 공유 시트)
- 입금 시 주의사항 안내

### 접근 경로
- GP 교환소(CEB-GAM-301) > [Celeb 입금주소] 버튼 탭

## 3. 화면 내 기능설명

### 바텀시트 열기/닫기

##### 열기 트리거
- GP 교환소(CEB-GAM-301)에서 [Celeb 입금주소] 버튼 탭

##### 닫기 트리거
- 바텀시트 외부 영역(딤) 탭
- 바텀시트 드래그 핸들을 아래로 스와이프

##### 동작
1. 열기: 하단에서 슬라이드 업 애니메이션으로 표시, 배경 딤 처리
2. 닫기: 바텀시트 사라짐, GP 교환소(CEB-GAM-301) 화면 복귀

### QR 코드

##### 생성 규칙
- 입력 데이터: Celeb 입금 지갑 주소 (0x로 시작하는 42자)
- 중앙 로고: Celeb 로고 (보라색 C)
- 오류 정정 레벨: H (높음, 30% 복구 가능) — 로고 삽입을 위해 높은 레벨 적용
- 배경색: 흰색
- 전경색: 검정색

##### 동작
- QR 코드 롱프레스 시 이미지 저장 옵션 표시
- 저장 성공 시 토스트: "QR 코드가 저장되었습니다."

### 네트워크 배지

##### 표시 규칙
- 텍스트: "BSC (BEP-20)" (고정)
- 스타일: 배지(Badge) 형태
- 위치: QR 코드 하단 중앙
- 설명: Celeb 토큰이 발행된 블록체인 네트워크 정보

##### 네트워크 종류
- BSC (BEP-20): BNB Smart Chain의 BEP-20 토큰 표준 (현재 고정)
- 향후 다른 네트워크 추가 가능 (Ethereum, Polygon 등)

### 입금 주소 표시

##### 표시 규칙
- 형식: 0x로 시작하는 42자 Ethereum 주소
- 폰트: Monospace (고정폭)
- 표시: 1줄 (전체 주소 노출)
- 위치: 네트워크 배지 하단

### 안내사항

##### 표시 내용
1. **네트워크 제한**: "BSC(Binance Smart Chain) 네트워크로만 입금해주세요."
2. **복구 불가 경고**: "잘못된 네트워크 입금 시 복구가 불가능합니다."

##### 표시 규칙
- 불릿 포인트(•) 형식
- 다국어 지원 (KO/EN/JP)
- 색상: 회색 (Secondary Text)

### [복사] 버튼

##### 표시 규칙
- 라벨: "복사" (다국어)
- 위치: CTA 영역 좌측
- [공유] 버튼과 나란히 배치

##### 동작
1. 버튼 탭 시 Celeb 입금 주소를 클립보드에 복사
2. 성공 토스트: "복사완료"
3. 복사 실패 시 에러 토스트: "주소 복사에 실패했습니다."

### [공유] 버튼

##### 표시 규칙
- 라벨: "공유" (다국어)
- 위치: CTA 영역 우측
- [복사] 버튼과 나란히 배치

##### 동작
1. 버튼 탭 시 OS 공유 시트(Share Sheet) 호출
2. 공유 대상: Celeb 입금 주소 텍스트 (42자)
3. 사용자가 공유 앱 선택 후 해당 앱으로 주소 전달
4. 공유 완료 후 바텀시트로 복귀

## 4. 규칙/로직

### 지갑 주소 제공 방식

##### 백오피스 연동
- 백오피스 지갑 관리(CEB-BO-GZ-503)에서 등록된 **충전용 대표 지갑**의 주소를 표시
- 대표 지갑이 없는 경우 Error 상태 표시 (아래 "에러 및 예외상황" 참조)

##### 지갑 변경 시
- 백오피스에서 충전용 대표 지갑이 변경되면 앱에서도 즉시 반영
- 바텀시트 오픈 시마다 최신 주소 조회

### QR 코드 생성 로직

##### 생성 시점
- 바텀시트 오픈 시 서버에서 생성된 QR 코드 이미지 URL 전달
- 또는 클라이언트에서 주소를 받아 QR 코드 라이브러리로 생성

##### Celeb 로고 삽입
- QR 코드 중앙에 Celeb 로고(보라색 C) 오버레이
- 오류 정정 레벨 H(30%) 적용하여 로고에 의한 데이터 손실 방지

##### 캐싱 정책
- 동일 주소의 QR 코드는 캐싱하여 재사용
- 주소 변경 시 캐시 무효화

### 주소 복사 로직

##### 복사 대상
- 지갑 주소 전체 (42자)
- 공백, 줄바꿈 없이 순수 주소만 복사

##### 복사 성공 조건
- 클립보드 API 사용 가능
- 권한 허용 상태

##### 토스트 표시
- 성공: "복사완료" (Success 토스트)
- 실패: "주소 복사에 실패했습니다." (Error 토스트)

### 주소 공유 로직

##### 공유 대상
- 지갑 주소 전체 (42자)

##### 공유 방식
- OS 네이티브 공유 시트 (iOS: UIActivityViewController, Android: Intent.ACTION_SEND)
- 텍스트 타입으로 공유

### 다국어 처리

##### 지원 언어
- 한국어 (KO)
- 영어 (EN)
- 일본어 (JP)

##### 적용 대상
- 바텀시트 타이틀: "Celeb 입금 주소"
- 안내사항 (2개 텍스트)
- [복사] / [공유] 버튼 라벨
- 토스트 메시지

##### 주소 및 네트워크 정보
- 지갑 주소: 다국어 처리 불필요 (숫자와 영문자만 사용)
- 네트워크: "BSC (BEP-20)" (고정, 다국어 처리 불필요)

### 입금 확인 처리

##### 입금 모니터링
- 사용자가 Celeb를 입금하면 서버에서 블록체인 트랜잭션 모니터링
- 입금 확인 시 자동으로 GP 가져오기 (CEB-GAM-301의 교환 내역에 표시)
- 사용자에게 푸시 알림 또는 인앱 알림 전송

##### 입금 확인 시간
- 블록체인 네트워크 상태에 따라 1~10분 소요 (BSC 기준)

## 5. 에러 및 예외상황

### 대표 지갑 미등록

##### 상황
- 백오피스에서 충전용 대표 지갑이 등록되지 않은 경우

##### 처리
1. Error 상태 표시
2. 에러 메시지: "현재 Celeb 입금이 불가능합니다. 나중에 다시 시도해주세요."
3. [닫기] 버튼 제공
4. 버튼 탭 시 바텀시트 닫기, GP 교환소(CEB-GAM-301) 화면으로 복귀

### 데이터 로딩 실패

##### 상황
- API 호출 실패 또는 네트워크 오류

##### 처리
1. Error 상태 표시
2. 에러 메시지: "입금 주소를 불러오는 데 실패했습니다."
3. [다시 시도] 버튼 제공
4. 버튼 탭 시 데이터 재로딩

### QR 코드 생성 실패

##### 상황
- QR 코드 이미지 생성 또는 로딩 실패

##### 처리
1. QR 코드 영역에 에러 메시지 표시: "QR 코드를 불러올 수 없습니다."
2. 주소 텍스트 및 복사/공유 기능은 정상 제공
3. [다시 시도] 버튼 제공 (QR 코드 영역 내)

### 주소 복사 실패

##### 상황
- 클립보드 권한 거부 또는 API 오류

##### 처리
1. 에러 토스트: "주소 복사에 실패했습니다."
2. 권한 요청 안내 (iOS/Android 시스템 설정 안내)
3. 사용자가 수동으로 주소 선택 및 복사 가능

### QR 코드 저장 실패

##### 상황
- 갤러리 저장 권한 거부 또는 저장 공간 부족

##### 처리
1. 에러 토스트: "QR 코드 저장에 실패했습니다."
2. 권한 요청 안내 (갤러리 접근 권한)

### Loading 상태

##### 상황
- 데이터 로딩 중

##### 처리
1. Skeleton UI 표시
   - QR 코드: 회색 사각형
   - 네트워크 배지: 회색 박스
   - 주소 텍스트: 회색 박스 (1줄)
   - 안내사항: 회색 박스 (2줄)
2. 로딩 완료 시 실제 데이터로 전환 (페이드인 애니메이션)

### 네트워크 연결 끊김

##### 상황
- 오프라인 상태에서 바텀시트 오픈 또는 데이터 로딩 시도

##### 처리
1. Error 상태 표시
2. 에러 메시지: "네트워크 연결을 확인해주세요."
3. [다시 시도] 버튼 제공
4. 온라인 복구 시 자동 재로딩
