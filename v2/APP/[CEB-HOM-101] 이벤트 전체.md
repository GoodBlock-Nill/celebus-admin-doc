## 1. 화면 개요

##### 목적

홈 캐러셀 배너에 노출되는 래플/서포트/이벤트 등 전체 콘텐츠를 리스트로 확인하는 화면이다. 아티스트 무관 통합 표시.

##### 주요 기능

- [진행중] / [마감됨] 탭 필터로 이벤트 분류
- 카드형 리스트 세로 나열
- 카드 탭 시 딥링크 URL로 이벤트 상세 이동

##### 접근 경로

- CEB-HOM-001 홈 메인 > 캐러셀 배너 > "전체 보기" 탭

##### 레이아웃 참조

> 화면 레이아웃은 프로토타입(https://celebus-app.vercel.app/home)을 참조하세요. 플로팅 디버그 버튼으로 다양한 상태를 확인할 수 있습니다. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

## 2. 영역별 상세

### 2.1 서브페이지 헤더

##### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 좌측 | ← 뒤로가기 (이전 화면 복귀) |
| 타이틀 | "이벤트" |
| 스타일 | 피그마 참조 |

##### 동작

- ← 탭 → 이전 화면(홈) 복귀

### 2.2 필터 탭

##### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 구성 | [진행중] / [마감됨] 2개 탭 |
| 기본 선택 | 진행중 |
| 위치 | 헤더 바로 아래 |
| 스타일 | 피그마 참조 |

##### 동작

- [진행중] 탭: 현재 진행중인 이벤트만 표시
- [마감됨] 탭: 마감된 이벤트만 표시
- 탭 전환 시 리스트 재조회

##### 정렬 규칙

| 필터 | 정렬 |
| ---- | ---- |
| 진행중 | BO 설정 순서 (캐러셀과 동일) |
| 마감됨 | 마감일 최신순 |

### 2.3 이벤트 리스트

##### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 형태 | 카드형 리스트 (세로 나열) |
| 카드 구성 | 썸네일 이미지 + 타이틀 + 서브타이틀 + 상태 뱃지 |
| 상태 뱃지 (진행중) | "D-{N}" (마감까지 남은 일수) |
| 상태 뱃지 (마감됨) | "마감" |
| 스타일 | 피그마 참조 |

##### 동작

- 카드 탭 → 딥링크 URL로 이벤트 상세 이동
- 다른 아티스트 이벤트 탭 시: 아티스트 선택 유지 (전환하지 않음). 뒤로가기 시 기존 아티스트 콘텐츠로 복귀
- 미팔로우 아티스트 이벤트 탭 시: 이벤트 상세 이동 + 하단 인라인 배너 "이 아티스트를 팔로우하시겠습니까?" 표시. [팔로우하기] / [닫기] CTA. 팔로우 완료 시 토스트(Success). 배너 닫기 시 이벤트 상세 유지 (CEB-HOM-001 캐러셀과 동일 패턴)

##### 콘텐츠 범위

| 규칙 | 설명 |
| ---- | ---- |
| 아티스트 | 아티스트 무관 통합 (캐러셀과 동일) |
| 콘텐츠 유형 | 래플, 서포트, 이벤트 등 BO 등록 배너 전체 |
| 로딩 방식 | 무한 스크롤 (20건 단위 배치 로딩). 최하단 도달 시 다음 배치 자동 로드 |
| 리스트 끝 | "모든 이벤트를 확인했어요" 안내 텍스트 |

##### 상태별 표시

| 상태 | 처리 |
| ---- | ---- |
| Loading | Skeleton UI (카드 3~4개 플레이스홀더) |
| Default | 이벤트 리스트 정상 표시 |
| Error | "앗, 불러오지 못했어요" + [다시 불러오기] 버튼 |
| 진행중 0건 | "곧 새로운 이벤트가 찾아올 거예요!" Empty 메시지 + [홈으로 돌아가기] CTA |
| 마감됨 0건 | "마감된 이벤트가 없어요" Empty 메시지 |

##### 비로그인 (공통정책 10.5)

| 상태 | 정책 |
| ---- | ---- |
| 비로그인 | 리스트 전체 열람 가능 |
| 이벤트 카드 탭 | 이벤트 상세 이동 가능 (응모 등 CTA 탭 시 로그인) |

##### 에러/예외
