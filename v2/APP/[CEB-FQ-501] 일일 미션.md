## 1. 화면 목적

아티스트 탭(CEB-ART-101)에서 "일일 미션" 카드를 탭했을 때 진입하는 화면이다. 출석 체크, 오늘의 일일 미션, 스트릭 현황을 한 화면에서 관리하는 데일리 루프 허브 역할을 한다.

##### 주요 기능

- 출석 체크 (버튼 탭, 10pt)
- 오늘의 일일 미션 1건 (자동검증, 페이지 방문형, 20pt)
- 스트릭 현황 (연속일수 + 다음 보너스까지 프로그레스)
- 이번 주 요일별 완료 기록 (도트)
- 스트릭 보너스 달성 시 축하 모달

##### 접근 경로

- CEB-ART-101 > 일일 미션 카드 탭
- CEB-HOM-001 > 오늘의 할 일 > 일일 미션 [하러가기] 탭
- CEB-HOM-001 > 오늘의 할 일 > 출석 체크 [출석하기] — 홈에서 직접 출석 처리 가능 (이중 진입점. CEB-FQ-501 진입 없이 홈에서 완료)

## 3. 화면 내 기능설명

### 3.1 출석 체크

- [출석하기] 탭 → 즉시 완료 처리. 토스트(Success) "출석 완료! 덕력 10pt 획득" 노출
- 완료 후 버튼 비활성화 (당일 재탭 불가)
- 스트릭 카운트 +1 (출석 체크 완료 기준)
- 자정(00:00 KST) 기준 리셋

### 3.2 일일 미션 수행

- [하러가기] 탭 → 해당 서비스 화면으로 이동 (딥링크)
- 해당 화면 진입 기록 → CEB-FQ-501 복귀 시 미션 자동 완료
- 토스트(Success) "미션 완료! 덕력 20pt 획득" 노출
- 미션 대상 화면이 Coming Soon(미오픈)인 경우: 토스트(Info) "해당 서비스는 곧 만나요! 기대해 주세요" + 미션 INCOMPLETE 유지. 이 케이스는 미션 할당 후 서비스가 Coming Soon으로 변경된 레이스 컨디션 대응용 방어 코드이며, 정상 상황에서는 발생하지 않음 (섹션 4.2 미션 풀 제외 규칙 참조)

### 3.3 스트릭 보너스 달성

- 연속일수가 7/14/30/100에 도달하면 축하 모달 자동 표시
- 모달 구성: "🔥 {N}일 연속 달성!" + "{보너스}pt 보너스!" + [받기] CTA
- [받기] 탭 → 보상 즉시 지급 + 토스트(Success) + 모달 닫기
- 1회성 (각 마일스톤 달성 시 1회만 지급)
- Dimmed 영역 탭 → 닫히지 않음 (CTA로만 닫기)

### 3.4 데이터 갱신

| 트리거 | 동작 |
| ---- | ---- |
| 화면 진입 | 출석/미션 상태 + 스트릭 현황 조회 |
| 서브 화면 복귀 | 미션 완료 여부 재조회 (방문 기록 확인) |
| Pull-to-Refresh | 전체 데이터 재조회 (공통정책 11.2 참조) |
| 자정 리셋 | 00:00 KST 기준 출석/미션 초기화. 앱 포그라운드 전환 시 재조회 |

## 4. 규칙/로직

### 4.1 스트릭 규칙

| 규칙 | 설명 |
| ---- | ---- |
| 카운트 기준 | 출석 체크 완료 기준 (일일 미션 완료와 무관) |
| 리셋 조건 | 출석 체크 미완료일이 1일이라도 있으면 리셋 (연속 방식) |
| 보너스 | 7일: 200pt / 14일: 500pt / 30일: 1,000pt / 100일: 5,000pt (BO 설정 가능) |
| 1회성 | 7/14/30/100일 보너스 모두 각 1회만 지급. 추가 마일스톤은 추후 BO 확장 검토 |
| 시즌 리셋 | 스트릭은 시즌 초기화와 무관 (유지됨) |

### 4.2 일일 미션 선택 규칙

| 규칙 | 설명 |
| ---- | ---- |
| 선택 방식 | 시스템이 8종 미션 풀에서 매일 1건 자동 선택 |
| 중복 방지 | 전일과 동일 미션 연속 선택 회피. 미션 풀 1건인 경우 동일 미션 허용 |
| 미오픈 서비스 | 해당 서비스가 Coming Soon이면 미션 풀에서 제외 |
| 미션 0건 | Coming Soon 제외 후 미션 풀이 0건이면 일일 미션 카드 숨김. 출석 체크만 표시 |
| BO 확장 | 관리자가 미션 풀 추가/삭제 가능 (추후 API 연동 미션 확장) |

### 4.3 시간 기준

| 규칙 | 설명 |
| ---- | ---- |
| 하루 기준 | 00:00~23:59 KST |
| 리셋 시점 | 매일 00:00 KST |
| 스트릭 끊김 알림 | 23:00 KST에 출석 미완료 시 푸시 알림 "오늘 출석 체크를 아직 안 했어요! 스트릭 {N}일째 🔥" |

### 4.4 멀티 아티스트 대응

| 항목 | 범위 | 설명 |
| ---- | ---- | ---- |
| 출석 체크 | 앱 전체 1회 | 아티스트 무관. 어떤 아티스트 탭에서 진입해도 동일 |
| 일일 미션 | 앱 전체 1건 | 아티스트 무관. 미션 내용은 현재 선택된 아티스트 맥락이지만, 어떤 탭에서 진입해도 동일 미션 |
