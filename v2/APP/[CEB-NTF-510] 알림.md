## 1. 화면 개요

##### 목적

홈/아티스트 메인 등의 앱 헤더 🔔 알림 아이콘에서 진입하는 알림 센터다. 아티스트 소식·이벤트·게임존·공지 알림을 카테고리/날짜별로 확인하고, 항목 탭 시 관련 화면으로 이동한다.

> ID 정합: 기존 문서가 "CEB-NTF-101 알림 센터(추후 작성)"으로 참조하던 화면의 실제 시안. **CEB-NTF-510으로 일원화** (HOM-001·ART-101 연관화면 참조 갱신).

##### 주요 기능

- 카테고리 탭 (전체 / 아티스트 / 이벤트 / 게임존 / 공지)
- 아티스트 탭 시 아티스트 선택 서브 필터
- 날짜 그룹별 알림 리스트 (오늘 / 어제 / 지난 주)
- 읽음/안읽음 구분 + [모두 읽음]
- 헤더 ⚙️ → 알림 설정(CEB-MY-103)

##### 접근 경로

- 앱 헤더 🔔 알림 아이콘 탭 (CEB-HOM-001 / CEB-ART-101 등)
- 푸시 알림 탭 시 진입

##### 화면 레이아웃

> 시각적 스타일은 Figma 시안을 따릅니다. 변형: 모두 읽음 / 알림 off(미수신).

## 2. 영역별 상세

### 2.1 헤더

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 뒤로가기 | ← 탭 → 이전 화면 복귀 |
| 타이틀 | "알림" |
| 모두 읽음 | [✓ 모두 읽음] 버튼 — 전체 알림 읽음 처리 |
| 설정 | ⚙️ 버튼 → CEB-MY-103 설정(알림) 이동 |
| 스타일 | 피그마 참조 |

### 2.2 카테고리 탭 + 아티스트 필터

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 카테고리 탭 | [전체] / [아티스트] / [이벤트] / [게임존] / [공지] (가로 스크롤) |
| 기본 선택 | 전체 |
| 아티스트 서브 필터 | [아티스트] 탭 활성 시 "{아티스트명} ▼" 드롭다운 노출 (팔로우 아티스트 선택) |
| 스타일 | 피그마 참조 |

#### 동작

- 카테고리 탭 전환 → 해당 카테고리 알림만 표시
- [아티스트] 탭 → 아티스트 드롭다운으로 특정 아티스트 알림만 필터
- [공지] 탭: 서비스(플랫폼) 공지 + 아티스트 공지 통합 (정보 피드 INF-101은 아티스트 공지만, 알림센터는 통합 — [CEB-INF-101] §2.2 참조)

### 2.3 알림 리스트

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 날짜 그룹 | "오늘" / "어제" / "지난 주" 등 그룹 헤더 |
| 항목 구성 | 아이콘 + 제목 + 설명 + 시간 + "・" + 아티스트명 |
| 안읽음 표시 | 안읽음 항목에 ● 점 표시 (읽으면 제거) |
| 정렬 | 최신순 |
| 페이지네이션 | 무한 스크롤 |
| 스타일 | 피그마 참조 |

##### 알림 유형 예시

| 유형 | 제목 / 설명 예시 | 탭 이동 |
| ---- | ---- | ---- |
| 덕력 시즌 시작 | "덕력 랭킹 새로운 시즌이 시작되었어요!" / "🏆 덕력을 쌓고 시즌 보상을 획득해보세요." | CEB-DUK-101 |
| 덕력 시즌 종료 | "덕력 랭킹 시즌이 종료되었어요!" / "🎁 이번 시즌 {N}위 달성! 보상을 확인해보세요." | CEB-DUK-105 |
| 퀘스트 완료 | "퀘스트를 완료했어요!" / "[퀘스트명] {BIVE 에디션명}이 지급 되었어요." | CEB-FQ-101 / CEB-COL-101 |
| 퀘스트 참여 유도 | "아직 참여하지 않은 퀘스트가 있어요" / "지금 참여하고 보상을 받아보세요." | CEB-FQ-101 |
| 이벤트/래플·서포트 | (이벤트 알림) | CEB-EVT-101 / CEB-DUK-201 |
| 공지 | (서비스/아티스트 공지) | 공지 상세 |

#### 동작

- 알림 항목 탭 → 안읽음 → 읽음 처리 + 관련 화면(딥링크)으로 이동
- [모두 읽음] 탭 → 전체 안읽음 → 읽음 처리

#### 상태별 표시

| 상태 | 처리 |
| ---- | ---- |
| Loading | 리스트 Skeleton UI |
| Default | 날짜 그룹 알림 리스트 표시 |
| 모두 읽음 | 안읽음 ● 점 전체 제거 (Figma 모두 읽음 변형) |
| Empty / 알림 off | 수신 알림 없음/알림 미설정: "받은 알림이 없어요" 또는 알림 설정 유도 (Figma 알림off 변형) |
| Error | 조회 실패 시 에러 안내 + 재시도 |

#### 비로그인

- 알림 센터는 로그인 상태에서만 (헤더 🔔은 비로그인 시 미확인 뱃지 미표시)

## 3. 공통 규칙

### 3.1 데이터 갱신

| 트리거 | 동작 |
| ---- | ---- |
| 화면 진입 | 알림 리스트 조회 + 미확인 건수 갱신 |
| 카테고리/아티스트 필터 | 해당 조건 재조회 |
| Pull-to-Refresh | 전체 재조회 (공통정책 11.2 참조) |

### 3.2 알림 수신 정책

- 카테고리별 수신 ON/OFF는 CEB-MY-103 설정에서 제어. OFF 카테고리는 푸시 미발송 (알림 센터 노출 정책은 운영 기준)
- 아티스트별 알림은 정보 피드(CEB-INF-101) 헤더 🔔 토글 / 설정(CEB-MY-103) 아티스트 알림과 연동
