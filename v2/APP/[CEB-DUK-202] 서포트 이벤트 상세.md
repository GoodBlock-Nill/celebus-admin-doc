## 1. 화면 개요

##### 목적

서포트 이벤트 목록(CEB-DUK-201)에서 카드 [참여하기]/탭 시 진입하는 상세 화면이다. 서포트 이벤트의 상세 정보·진행도·내 참여 현황을 확인하고, 보유 덕력으로 응원하는 화면이다.

> v.3 정합: 기존 CEB-DUK-201 인라인 확장으로 제공하던 상세·응원을 **별도 상세 화면으로 분리**.

##### 주요 기능

- 서포트 이벤트 상세 (대표 이미지 + 타이틀 + D-day + 기간 + 설명)
- 진행도 (목표금액 + 달성률 + 누적/목표 DUK)
- 내 참여 현황 (내 누적 응원) + 참여 통계 (참여 유저 수)
- 유의사항 안내
- 하단 고정 응원 바 (보유 덕력 + 수량 입력 + [응원하기])

##### 접근 경로

- CEB-DUK-201 서포트 이벤트 > 카드 [참여하기]/탭

##### 화면 레이아웃

> 시각적 스타일은 Figma 시안을 따릅니다. 변형: 모집중 / 달성 / 완료 / 미달성종료 / 집행취소 등 상태별.

## 2. 영역별 상세

### 2.1 헤더 + 히어로

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 뒤로가기 | ← 탭 → CEB-DUK-201 복귀 |
| 공유 | 우측 공유 아이콘 (OS 기본 공유) |
| 대표 이미지 | 서포트 대표 이미지 (상단 풀 너비) |
| 타이틀 | 아티스트명 + 서포트 타이틀 (예: "V01D 커피차 서포트") |
| 기간·D-day | "D-{N}" + "{YYYY.MM.DD} ~ MM.DD" |
| 설명 | BO에서 등록한 이벤트 설명 텍스트 |
| 스타일 | 피그마 참조 |

### 2.2 진행도

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 목표금액 | "목표금액 {N} DUK" |
| 달성률 | progress bar + "{N}%" |
| 누적/목표 | "{현재} / {목표} DUK" |
| 달성 상태 | 목표 달성 시 100% + "목표 달성!" 표시 (응원 UI 비활성) |
| 스타일 | 피그마 참조 |

### 2.3 내 참여 현황 · 참여 통계

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 내 누적 응원 | "내 누적 응원 ✦{N}" (0이면 미표시 또는 0 표시) |
| 참여 유저 수 | "참여 유저 수 {N}명" |
| 스타일 | 피그마 참조 |

### 2.4 유의사항 안내

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 구성 | 유의사항 리스트 (BO 등록) |
| 스타일 | 피그마 참조 |

### 2.5 응원 바 (하단 고정)

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 보유 표시 | "사용 가능한 덕력 ✦{N} DUK" |
| 수량 입력 | [-] 버튼 + 수량 필드(직접 입력) + [+] 버튼 |
| 증감 단위 | [-]/[+] 탭 시 100DUK 단위 (CEB-DUK-201 §2.4 응원 규칙 참조) |
| 최소/최대 | 최소 1DUK / 최대 min(보유 덕력, 남은 목표량) |
| [응원하기] | 수량 0이면 비활성. 1 이상이면 활성 |
| 달성 상태 | 응원 UI 전체 비활성 + "목표 달성! 감사합니다 💜" |
| 스타일 | 피그마 참조 |

#### 동작

- [응원하기] 탭 → 확인 모달 표시 → [응원] → 서버 처리 → 완료 토스트 "덕력 {N}DUK 응원 완료!" + 진행도·내 누적 응원·보유 덕력 갱신
- 응원 규칙·확인 모달·초과 응원 방지·누적 응원·반환 규칙·상태 6종·에러 케이스는 **[CEB-DUK-201] §2.4·2.5·2.6 영역 정책 준용**

#### 상태별 표시 (CEB-DUK-201 §2.6 상태 정의 준용)

| 상태 | 처리 |
| ---- | ---- |
| 모집중 | 응원 바 활성 |
| 달성 | 응원 바 비활성 + "목표 달성! 서포트 진행 예정" |
| 집행중 | "서포트 진행중..." (응원 불가) |
| 완료 | 결과(사진/영상) 표시 (CEB-DUK-201 §2.3 결과 영역 준용) |
| 미달성종료 / 집행취소 | "응원 덕력 반환 완료" 안내 |

#### 비로그인

- 상세 열람 가능. 내 누적 응원 "-" 표시. [응원하기] 탭 시 로그인 화면 이동 (공통정책 10.5 참조)

## 3. 공통 규칙

### 3.1 데이터 갱신

| 트리거 | 동작 |
| ---- | ---- |
| 화면 진입 | 이벤트 상세 + 진행도 + 내 참여 조회 |
| 응원 완료 | 진행도 + 내 누적 응원 + 보유 덕력 갱신 |
| Pull-to-Refresh | 전체 재조회 (공통정책 11.2 참조) |

### 3.2 멀티 아티스트 대응

- 서포트 이벤트는 아티스트별 분리. 응원 덕력은 해당 서포트 아티스트에 귀속 ([CEB-000] §5.2.2 참조)
