## 1. 화면 목적

아티스트 탭(CEB-ART-101)에서 "{아티스트명} 응원하기" 카드를 탭했을 때 진입하는 화면이다. (유저 대면 명칭: {아티스트명} 응원하기, 예: V01D 응원하기) 아티스트 서포트 이벤트 목록을 표시하고, 유저가 보유 덕력을 응원하여 팬덤 서포트에 참여할 수 있다.

##### 주요 기능

- 서포트 이벤트 카드 리스트 (상태별: 모집중 / 달성 / 집행중 / 완료 / 미달성종료 / 집행취소)
- 카드 탭 시 인라인 확장 (상세 정보 + 덕력 응원 UI + 진행률)
- 덕력 응원: 수량 입력 + 확인 모달 + 응원 완료
- 완료 이벤트 결과 확인 (사진/영상 인라인 표시)

##### 접근 경로

- CEB-ART-101 > 서포트 카드 탭
- CEB-HOM-001 > 바로가기 > {아티스트명} 응원하기 카드 탭

## 3. 화면 내 기능설명

### 3.1 이벤트 카드 탭

- 카드 탭 시 인라인 확장/축소 토글
- 동시에 1개만 확장 (다른 카드 탭 시 이전 카드 자동 축소)
- 화면 진입 시 첫 번째 모집중 이벤트를 자동 확장

### 3.2 덕력 응원 플로우

1. 수량 입력 (직접 입력 또는 [-]/[+] 버튼)
2. [응원하기] 탭 → 확인 모달 표시
3. 확인 모달에서 [응원] 탭 → 서버 처리
4. 성공 → 토스트(Success) "덕력 {N}pt 응원 완료!" + 프로그레스 바 갱신 + "내 응원" 갱신 + 보유 덕력 차감 반영
5. 달성 트리거: 응원으로 누적 >= 목표 시 서버가 즉시 달성 상태로 전환 → 응원 UI 비활성 + 토스트(Success) "🎯 목표 달성!"

### 3.3 누적 응원

- 동일 이벤트에 여러 번 응원 가능
- "내 응원: {N}pt"는 누적 합산 표시
- 매 응원마다 확인 모달 표시

### 3.4 초과 응원 방지

- UI에서 입력량이 남은 목표량을 초과 시 남은 목표량으로 자동 조정 ("남은 목표: {N}pt" 표시)
- 서버에서도 (현재 누적 + 응원량 > 목표) 확인하여 초과분은 보유 덕력에 유지
- 예: 목표 15,000pt, 현재 14,800pt, 500pt 응원 시 → 200pt만 차감, 300pt 유지

### 3.5 데이터 갱신

| 트리거 | 동작 |
| ---- | ---- |
| 화면 진입 | 이벤트 리스트 + 각 이벤트 상태/진행률 조회 |
| 응원 완료 | 해당 이벤트 진행률 + 내 응원 + 보유 덕력 갱신 |
| Pull-to-Refresh | 전체 데이터 재조회 (공통정책 11.2 참조) |
| 복귀 시 | 상태 변경 가능성 재조회 |
