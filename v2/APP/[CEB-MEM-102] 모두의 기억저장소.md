## 1. 화면 개요

[CEB-MEM-101] 기억저장소 헤더에서 [모두의 기억저장소] 탭을 탭했을 때 진입하는 화면이다. 다른 유저가 **저장소 공개 + 게시글 공개** 두 조건을 모두 충족한 게시글이 그리드 형태로 노출된다 (공통정책 §7.1 참조).

### 주요 기능

- 다른 유저의 공개 기억 그리드 조회 (최신순)
- 게시글 좋아요 (공통정책 §7.6 참조)
- 게시글 조회수 자동 카운팅 (브라우저 세션 + 24시간 쿨다운)
- 닉네임 탭 → [CEB-MEM-103] {닉네임} 님의 기억저장소 진입
- 게시글 카드 탭 → [CEB-MEM-202] 기억 상세 진입

### 접근 경로

- [CEB-MEM-101] 기억저장소 헤더 [모두의 기억저장소] 탭 활성화 시 표시
- 비로그인 유저도 열람 가능 (좋아요·조회수 카운트 미적립). 비로그인 시 "내 기억저장소" 탭은 비활성, 본 탭만 활성

### 화면 레이아웃

```
┌────────────────────────────────┐
│ ←   기억저장소        ⚙️   │
│  [내] / [모두의 기억저장소]    │  ← 탭 활성
├────────────────────────────────┤
│ ┌──┐  ┌──┐  ┌──┐               │
│ │📷│  │📷│  │📷│  ← 3열 그리드
│ │👤│  │👤│  │👤│               │
│ │@N│  │@N│  │@N│               │
│ │👁123 ❤45│ │👁87 ❤12│ │...   │

```

## 2. 영역별 상세

### 2.1 서브페이지 헤더

본 화면은 [CEB-MEM-101] 헤더 안에서 탭 활성으로 표시되며, 헤더 자체는 MEM-101의 §2.1을 따른다.

| 항목 | 내용 |
| ---- | ---- |
| 탭 활성화 | "모두의 기억저장소" 활성. "내 기억저장소" 비활성 |
| 뷰 전환 버튼 | **숨김 처리** (모두 탭에서는 캘린더/그리드/지도 전환 미적용) |
| ⚙️ 설정 버튼 | 표시 유지. 본인 저장소 공개 토글 (MEM-101 §2.7) |

### 2.2 그리드 뷰

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 구성 | **3열 그리드**. 정사각형 카드 |
| 프로필 사진 | 작성자 프로필 사진 (없으면 디폴트 회색 아바타) + 감정 이모지 뱃지 (피그마 참조) |
| 닉네임 | 게시글 작성자 닉네임. **탭 시 [CEB-MEM-103] 진입** |
| 멀티 사진 아이콘 🖼️ | 등록 이미지 ≥2장일 때 표시 |
| 메인 사진 | 등록 사진 첫 장 OR **시스템 생성 SVG (글 20자 컷오프)** — 사진 미등록 게시글의 경우 |
| 조회수 | `👁 N` 천 단위 콤마 (예: 1,234) |
| 좋아요 | `❤ N` 1탭 토글 |
| 정렬 | **최신순 단일** (작성일시 내림차순(최신순)). 정렬 토글 없음 |
| 탭 동작 | 카드 본문 탭 → [CEB-MEM-202] 기억 상세 / 닉네임 탭 → [CEB-MEM-103] |

#### 시스템 생성 사진 규격 (사진 미등록 게시글용)

| 항목 | 정책 |
| ---- | ---- |
| 적용 조건 | 게시글에 이미지 0건 + 텍스트 존재 시 |
| 글자수 컷오프 | **20자**. 21자 이상은 "..." 표시 |
| 렌더링 방식 | **클라이언트 사이드 SVG** (서버 부담 없음). 다국어 폰트는 시스템 기본 적용 |
| 배경색 | 게시글 첫 번째 감정 이모지 카테고리에 따른 톤 (피그마 참조) |
| 폴백 | 글이 없는 경우 (편지/메모 필수, 사진 기록만 가능) → "📸" 디폴트 카드 |

#### 좋아요 동작

| 항목 | 정책 |
| ---- | ---- |
| 권한 | 로그인 유저만 |
| 횟수 | 게시글당 1회 (취소·재클릭 가능) |
| 본인 게시글 좋아요 | **허용** (공통정책 §7.6) |
| 카운트 갱신 | 실시간 (옵티미스틱 업데이트 + 서버 확인) |
| 비로그인 탭 시 | 안내 토스트 "로그인이 필요해요" + 로그인 화면 이동 (공통정책 §10.5) |
| 푸시 알림 | **본 단계 미발송** (§7.6 참조) |

#### 조회수 동작

| 항목 | 정책 |
| ---- | ---- |
| 카운트 기준 | **로그인 유저 + 브라우저 세션 + 24시간 쿨다운** |
| 비로그인 | 카운트 미적립 |
| 쿨다운 | 동일 유저가 24시간 내 동일 게시글 재방문 시 미증가 |
| 갱신 시점 | 그리드에서 카드 탭 시 (= MEM-202 진입 시) |
| 표시 | 천 단위 콤마 (예: 1,234) |
| 상한 | 없음 (자연 누적) |
