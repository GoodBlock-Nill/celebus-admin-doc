## 1. 화면 목적

아티스트 탭(CEB-ART-101)에서 "기억 저장소" 카드를 탭했을 때 진입하는 화면이다. 아티스트와의 기억(콘서트, 팬미팅, 일상 덕질)을 기록하고 되돌아볼 수 있는 개인 다이어리 역할을 한다.

> "시간이 흘러도 그날로 돌아갈 수 있어!"

##### 주요 기능

- 캘린더 뷰(기본) / 그리드 뷰 / 지도 뷰 전환
- [+] FAB로 기억 작성 (사진 기록 / 편지 / 메모 3종)
- 감정 이모지(6종) 필수 선택 (1~3개)
- 날짜 탭 시 해당 날짜 기억 확장 (아코디언)
- 기억 상세 보기 / 수정 / 삭제
- MVP: 개인 다이어리 전용 (소셜 기능 추후 도입)

##### 접근 경로

- CEB-ART-101 > 기억 저장소 카드 탭
- CEB-HOM-001 > 추천 카드 [다시 보기] 탭 (1년 전 오늘 기억)
- 푸시 알림 "1년 전 오늘 기억" 탭

## 3. 화면 내 기능설명

### 3.1 뷰 전환

- [📅] [🖼️] [🗺️] 아이콘 탭으로 즉시 전환
- 현재 뷰 아이콘 하이라이트 (보라색)
- 뷰 전환 시 데이터 재조회 없음 (동일 데이터, 표시 방식만 변경)

### 3.2 캘린더 날짜 탭

- 기억이 있는 날짜 탭 → 하단 확장 영역에 해당 날짜 기억 리스트 표시
- 기억이 없는 날짜 탭 → 확장 영역 미표시
- 동시에 1개 날짜만 확장 가능. 다른 날짜 탭 시 이전 확장 영역 자동 축소

### 3.3 기억 작성

- [+] FAB 탭 → 기억 작성 화면으로 이동
- 작성 순서: 타입 선택 → 날짜 → 감정 이모지(필수 1~3개) → 위치(선택) → 미디어(선택) → 텍스트 → 공개 설정
- [저장하기] 탭 → 저장 처리 + 토스트(Success) + 덕력 30pt 즉시 지급 + 캘린더로 복귀. 하단에 "오늘의 퀘스트도 해볼까요?" CTA 표시 (→ CEB-FQ-101)

### 3.4 기억 수정 / 삭제

- 기억 상세에서 [수정] 탭 → 작성 화면(기존 데이터 프리필) → 저장 (덕력 중복 미지급)
- [삭제] 탭 → 확인 모달 → 삭제 처리 (덕력 미회수)

### 3.5 데이터 갱신

| 트리거 | 동작 |
| ---- | ---- |
| 화면 진입 | 현재 월 캘린더 데이터 조회 |
| 작성 / 수정 / 삭제 복귀 | 캘린더 데이터 재조회 |
| 월 전환 (◀ / ▶) | 해당 월 데이터 조회 |
| 뷰 전환 | 재조회 없음 (동일 데이터 재표시) |
| Pull-to-Refresh | 전체 재조회 (공통정책 11.2 참조) |

### 3.6 "1년 전 오늘" 푸시 진입

- 공통정책 17에 정의된 "1년 전 오늘 기억" 푸시 알림 탭 시 본 화면으로 진입
- 진입 시 해당 날짜(1년 전 오늘)의 캘린더로 자동 이동 + 해당 날짜 자동 확장
- 캘린더 뷰로 고정 (그리드/지도 뷰에서 진입해도 캘린더로 전환)

| ---- | ---- |
| 저장소 기본 | 비공개 |
| 게시물 기본 | 비공개 |
| 공개 노출 조건 | 저장소 공개 + 게시물 공개 모두 충족 시에만 타 유저에게 노출 |

### 4.4 멀티 아티스트 대응

- 기억저장소는 아티스트별 분리 (`artist_id` 기반)
- 월간 50건 제한도 아티스트별 독립 적용
- 아티스트 전환 시 CEB-ART-101 재진입 → 해당 아티스트 기억저장소 표시
