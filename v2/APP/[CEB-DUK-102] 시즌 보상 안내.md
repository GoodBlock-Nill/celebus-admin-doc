## 1. 화면 개요

##### 목적

덕력 랭킹(CEB-DUK-101)에서 진입하는 시즌 안내 화면이다. 현재 시즌 기간, 시즌 보상 구간, 덕력을 빠르게 쌓는 방법, 유의사항을 안내한다. (Figma 시안 라벨: "시즌 보상 안내")

##### 주요 기능

- 시즌 정보 (현재 시즌명·기간·종료 D-day)
- 시즌 보상 구간별 안내 (1위 / TOP 10 / TOP 10% 등)
- 덕력 쌓는 방법 안내 + [덕력 쌓으러 가기]
- 유의사항 안내

##### 접근 경로

- CEB-DUK-101 덕력 랭킹 > 시즌 정보/보상 안내 진입

##### 화면 레이아웃

> 시각적 스타일은 Figma 시안을 따릅니다. 변형: 진행중 / 시즌 종료.

## 2. 영역별 상세

### 2.1 헤더

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 타이틀 | "시즌 보상 안내" |
| 우측 | [X] 닫기 (CEB-DUK-101 복귀) |
| 스타일 | 피그마 참조 |

### 2.2 시즌 정보

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 시즌명 | 현재 시즌명 (예: "4월 시즌") + 상태 뱃지 "종료 D-{N}" |
| 기간 | "{YYYY.MM.DD} ~ {YYYY.MM.DD} KST" |
| 안내 | "시즌 종료 후 최종 랭킹이 확정됩니다." |
| 시즌 종료 변형 | 종료된 시즌: "시즌이 종료되었습니다" + 최종 랭킹/보상 확정 안내 (Figma 시즌종료 변형 참조) |
| 스타일 | 피그마 참조 |

### 2.3 시즌 보상

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 안내 | "보상은 최종 달성 랭킹 기준 상위 보상만 지급돼요." |
| 구성 | 보상 구간별 카드 리스트 |
| 1위 달성 시 | 보상 항목 (예: 사인 앨범) |
| TOP 10 진입 시 | 보상 항목 (예: 사인 포토카드) |
| TOP 10% 진입 시 | 보상 항목 (예: 사인 포토카드 + 디지털 포토카드 + 수퍼팬 뱃지) |
| 스타일 | 피그마 참조 |

> 보상 구간(등수/퍼센트)과 보상 상품은 시즌별 BO 설정 ([CEB-DUK-101] §2.5 참조). 위 항목은 기본 예시.

### 2.4 빠르게 덕력 쌓는 방법

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 안내 | "다양한 활동을 통해 덕력을 쌓을 수 있어요" |
| 활동 리스트 | "매일 출석 체크하기" / "일일 미션 모두 완료하기" / "진행 중인 퀘스트 참여하기" |
| CTA | "덕력 쌓으러 가기 >" |

- "덕력 쌓으러 가기 >" 탭 → 덕력 획득 활동(오늘의 할 일/퀘스트 등)으로 이동
