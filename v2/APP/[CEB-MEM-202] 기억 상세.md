## 1. 화면 목적

작성된 기억의 상세 내용을 확인하고, 수정/삭제/신고하는 화면이다. 내 기억과 타인의 공개 기억 모두 이 화면에서 열람한다.

> 핵심 원칙: 팬이 기록한 소중한 순간을 풍부하게 다시 체험할 수 있어야 한다.

##### 주요 기능

- 감정 이모지 + 유형 + 미디어 + 텍스트 + 위치 전체 표시
- 이미지 풀스크린 갤러리 (스와이프 + 핀치 줌)
- 영상 인라인/풀스크린 재생
- 내 기억: 수정/삭제
- 타인 공개 기억: 신고

##### 접근 경로

- CEB-MEM-101 > 기억 항목 탭 (캘린더/그리드/지도 뷰에서)
- 푸시 알림 "1년 전 오늘 기억" 탭 시 진입

## 3. 화면 내 기능설명

### 3.1 ⋯ 메뉴 (바텀시트)

내 기억과 타인의 공개 기억에 따라 메뉴가 다르게 표시된다.

##### 내 기억

| 메뉴 항목 | 동작 |
| ---- | ---- |
| 수정하기 | CEB-MEM-201 (mode=edit)로 이동. 기존 데이터 프리필 |
| 삭제하기 | 삭제 확인 모달 표시 |

##### 타인의 공개 기억

| 메뉴 항목 | 동작 |
| ---- | ---- |
| 신고하기 | 신고 사유 선택 바텀시트 표시 |

### 3.2 수정하기

- [수정하기] 탭 → CEB-MEM-201 (mode=edit, 기존 데이터 프리필)
- 유형은 변경 불가
- 수정 완료 후 본 화면으로 자동 복귀 + 수정된 내용 반영
- 덕력 중복 미지급

### 3.3 삭제하기

- [삭제하기] 탭 → 확인 모달:
  - 텍스트: "삭제한 기억은 복구할 수 없어요. 삭제할까요?"
  - CTA: [취소] / [삭제]
- 삭제 완료 → 토스트(Success) "기억이 삭제되었어요"
- CEB-MEM-101로 자동 복귀 + 캘린더 데이터 재조회
- 덕력 환수 없음

### 3.4 신고하기 (공개 기억, 타인 것만)

- [신고하기] 탭 → 신고 사유 선택 바텀시트:

| 신고 사유 | 코드 |
| ---- | ---- |
| 부적절한 콘텐츠 | INAPPROPRIATE |
| 스팸 | SPAM |
| 욕설·혐오 | HATE_SPEECH |
| 개인정보 노출 | PRIVACY |
| 기타 | OTHER |

- 사유 선택 + [신고] 탭 → 토스트(Success) "신고가 접수되었어요"
- 동일 기억 중복 신고 차단: 토스트(Info) "이미 신고한 기억이에요"
- 신고 접수 후 BO에서 검토 (별도 BO 화면)

### 3.5 풀스크린 미디어 뷰어

- 이미지 탭 → 풀스크린 갤러리:
  - 배경: 검정 (dimmed)
  - 좌우 스와이프로 이미지 넘기기
  - 핀치 줌 (최대 3배)
  - 상단 [X] 닫기 버튼
  - 하단 이미지 카운터 "{현재}/{전체}" (예: "3/7")
- 영상 탭 → 풀스크린 플레이어:
  - 재생/일시정지 버튼
  - 시크바 (드래그로 위치 이동)
  - 음량 조절
  - [X] 닫기 버튼

### 3.6 데이터 갱신

| 트리거 | 동작 |
| ---- | ---- |
| 화면 진입 | 기억 상세 데이터 조회 |
| 수정 완료 복귀 | 수정된 데이터 재조회 |
| Pull-to-Refresh | 상세 데이터 재조회 (공통정책 11.2 참조) |

## 5. 에러 및 예외상황

| 상태 | 처리 |
| ---- | ---- |
| Loading | Skeleton UI (이모지 + 미디어 영역 + 텍스트 플레이스홀더) |
| Default | 기억 상세 정상 표시 |
| Error | "앗, 불러오지 못했어요" + [다시 불러오기] 버튼 |
| 삭제된 기억 접근 | "삭제된 기억이에요" 안내 + [돌아가기] CTA → CEB-MEM-101 |
| 미디어 로드 실패 | 이미지 영역에 "불러오지 못했어요" 플레이스홀더. 탭 시 재시도 |
| 영상 재생 실패 | "영상을 재생할 수 없어요" 안내 + [다시 시도] |
| 네트워크 에러 | 공통정책 11.3 참조 |
| 삭제 실패 | 토스트(Error) "삭제에 실패했어요. 다시 시도해주세요" |
| 신고 실패 | 토스트(Error) "신고 접수에 실패했어요. 다시 시도해주세요" |
| 중복 신고 | 토스트(Info) "이미 신고한 기억이에요" |
| 비로그인 | CEB-MEM-101에서 로그인 유도 처리됨 (공통정책 10.5 참조). 본 화면은 로그인 상태에서만 접근 가능 |
