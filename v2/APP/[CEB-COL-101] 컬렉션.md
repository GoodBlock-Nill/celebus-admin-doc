## 1. 화면 목적

아티스트 탭(CEB-ART-101)에서 "컬렉션" 카드를 탭했을 때 진입하는 화면이다. 유저가 보유한 디지털 굿즈(BIVE)를 카테고리별로 관리하고, 상세 보기/보내기/합성/Pick 등의 기능을 제공하는 컬렉션 허브 역할을 한다.

> 유저 대면 명칭: "컬렉션". 내부 NFT 브랜드명: "BIVE"

##### 주요 기능

- 카테고리 필터([아티스트] [이벤트] [스페셜]) + 2열 그리드 리스트
- BIVE 상세 보기 (이미지 + 등급 + 설명 + 보이스/서사 텍스트)
- BIVE 보내기 (다른 유저에게 전송)
- 합성 (Grade 1~5 전부 보유 시 → 스페셜 에디션 합성)
- Pick (보유 BIVE 1장 소진 → 새 BIVE 뽑기)
- BIVE 히스토리 (획득/전송/삭제 이력)
- BIVE 혜택 (혜택현황 탭 / 지급내역 탭)
- BIVE 삭제 (보유 BIVE 삭제 가능)

##### 접근 경로

- CEB-ART-101 > 컬렉션 카드 탭
- CEB-HOM-001 > 추천 카드 [받으러 가기] 탭

## 3. 화면 내 기능설명

### 3.1 카테고리 필터 전환

- [아티스트] [이벤트] [스페셜] 탭 전환 시 해당 카테고리 BIVE만 그리드에 표시
- 카테고리 내 정렬: 등급순 (낮은 등급 먼저) → 획득일 최신순
- 미보유 BIVE는 보유 BIVE 뒤에 표시

### 3.2 BIVE 상세 보기

> BIVE 상세(CEB-COL-201), 보내기, Pick, 히스토리(CEB-COL-301), 혜택(CEB-COL-401) 화면은 별도 화면 스펙으로 추후 작성합니다.

- 보유 BIVE 카드 탭 시 상세 화면으로 이동
- 상세 내용: 풀사이즈 이미지 + 등급 뱃지 + BIVE 이름 + 서사 텍스트 + 보이스 재생(보이스가 있는 경우에만 노출)
- 상세 화면 내 액션: [보내기] / [삭제]

### 3.3 BIVE 보내기

- 보유 BIVE를 다른 유저에게 전송하는 기능
- 진행 순서: 상세 화면 [보내기] 탭 → 수신자 검색/선택 → 전송 확인 모달 → 전송 완료 토스트(Success)
- 전송 완료 후 해당 BIVE가 내 컬렉션에서 제거됨

### 3.4 합성

- Grade 1~5 서사 굿즈(아티스트 카테고리) 5장 전부 보유 시 [합성] 버튼 활성
- 합성 진행 시 원본 Grade 1~5 유지 + 스페셜 에디션 1장 추가 생성 (공통정책 5.4 참조)
- 이미 합성 완료(스페셜 에디션 보유) 시 [합성] 버튼 비활성
- 조건 미충족 시: [합성] 탭 → "Grade 1~5를 모두 보유해야 합성할 수 있습니다" 토스트(Info)

### 3.5 Pick

- 보유 BIVE 1장을 소진하여 랜덤으로 새 BIVE 1장을 획득하는 가챠 기능
- 진행 순서: [Pick] 탭 → 소진할 BIVE 선택 → 확인 모달 → 뽑기 애니메이션 → 획득 결과 화면
- 보유 BIVE 0장 시 [Pick] 버튼 비활성

### 3.6 BIVE 히스토리

- 내 BIVE 획득/전송/받기/삭제 전체 이력을 시간순(최신 순)으로 표시
- 각 항목: BIVE 이름 + 이벤트 유형(획득 / 보내기 / 받기 / 삭제) + 날짜

### 3.7 BIVE 혜택

- **혜택현황 탭**: 현재 보유 BIVE로 인해 해금된 혜택 목록 표시
- **지급내역 탭**: BIVE 관련 보상 지급 이력 표시

### 3.8 BIVE 삭제

- 상세 화면에서 [삭제] 탭 → 확인 모달 ("삭제한 BIVE는 복구할 수 없습니다") → 삭제 완료 토스트(Success)
- 삭제 완료 후 해당 BIVE가 컬렉션에서 제거됨

### 3.9 데이터 갱신

| 트리거 | 동작 |
| ---- | ---- |
| 화면 진입 | 컬렉션 전체 리스트 + 보유 수 조회 |
| Pull-to-Refresh | 전체 데이터 재조회 (공통정책 11.2 참조) |
| 상세 화면 복귀 | 컬렉션 상태 재조회 (보내기/삭제 등 반영) |
| 합성/Pick 완료 후 | 컬렉션 상태 재조회 |

## 4. 규칙/로직

### 4.1 BIVE 등급 체계 (공통정책 5.4 참조)

| 등급 | 이름 | 획득 방식 |
| ---- | ---- | ---- |
| Grade 1 | 스토리의 시작 | Quest 1장 클리어 보상 |
| Grade 2 | 성장 스토리 | Quest 2장 클리어 보상 |
| Grade 3 | 아티스트의 꿈 | Quest 3장 클리어 보상 |
| Grade 4 | 아티스트의 준비 | Quest 4장 클리어 보상 |
| Grade 5 | 아티스트의 데뷔 | Quest 5장 클리어 보상 |
| Special | 스페셜 에디션 | Grade 1~5 합성 결과 |

### 4.2 합성 규칙

| 규칙 | 설명 |
| ---- | ---- |
| 합성 조건 | 아티스트 카테고리 Grade 1~5 서사 굿즈 5장 전부 보유 |
| 합성 결과 | 원본 Grade 1~5 유지 + 스페셜 에디션 1장 추가 생성 |
| 합성 가능 횟수 | 아티스트당 1회. 이미 합성 완료(스페셜 에디션 보유) 시 재합성 불가 |

### 4.3 그리드 정렬 기준

| 카테고리 내 정렬 | 기준 |
| ---- | ---- |
| 1순위 | 보유 여부 (보유 먼저) |
| 2순위 | 등급순 (낮은 등급 먼저: Grade 1 → Grade 5 → Special) |
| 3순위 | 획득일 최신순 (등급이 같은 경우) |

### 4.4 멀티 아티스트 대응

- 컬렉션은 아티스트별로 분리하여 표시 (artist_id 기반)
- 아티스트 전환 시 CEB-ART-101 재진입 후 해당 아티스트의 컬렉션으로 접근
