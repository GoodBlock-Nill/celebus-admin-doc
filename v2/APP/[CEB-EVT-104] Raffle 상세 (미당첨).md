## 1. 화면 개요

Raffle 추첨 결과 미당첨된 유저에게 결과를 정중히 안내. 부정적 감정을 최소화하기 위해 위로 메시지를 랜덤 노출하며, Raffle 메인으로 복귀를 유도.

##### 주요 기능
- 위로 이미지 (차분한 톤)
- 위로 문구 6종 중 랜덤 노출
- [확인] → Raffle 메인 복귀

##### 접근 경로
- [CEB-EVT-101] Raffle 메인 > 미당첨 리스트 클릭
- 결과 발표 푸시 알림 클릭

## 2. UI 설명

```text
[<] 당첨 결과
─────────────────────────────────
[위로 이미지/감성 일러스트]
(채도 낮은 차분한 톤)

"아쉽게도 당첨되지 않았어요."  (H1, 중앙)
"다음 기회에 다시 도전해주세요!
 더 좋은 이벤트로 찾아올게요."  (Body, 중앙)
─────────────────────────────────
[하단 고정]
[ 확인 ]  (Secondary/Line)
─────────────────────────────────
```

| 영역 | 구성 | 설명 |
| --- | --- | --- |
| 헤더 | [<] / "당첨 결과" | |
| 대표 이미지 | 위로 이미지 | 차분한 톤앤매너, 당첨 화면(EVT-103)의 화려함과 대비 |
| 메시지 | 메인+서브 위로 문구 | 진입 시 6종 중 랜덤 |
| 푸터 | [확인] | Secondary(Line) 또는 Text Button — 과도한 강조 회피 |

> 시각적 스타일은 피그마 참조

## 3. 규칙·로직

### 3.1 진입 시 검증

1. 서버 당첨 여부 재조회
2. 당첨 확인 → [CEB-EVT-103] 당첨 화면 교체 전환 (자연 전환)
3. 미당첨 + 종료 상태 → 본 화면 표시 + 위로 문구 랜덤 선택

### 3.2 [확인] 동작

- [CEB-EVT-101] Raffle 메인으로 이동
- 네비게이션 스택에서 현재 화면 제거

### 3.3 다국어

- 위로 문구·버튼: 다국어 KO/EN/JP (위로 문구 세트 6종 모두 다국어 리소스)
