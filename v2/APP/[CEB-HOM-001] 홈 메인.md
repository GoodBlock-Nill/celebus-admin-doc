## 1. 화면 목적

앱을 열면 가장 먼저 보는 화면이다. "오늘 뭘 하면 되지?"를 즉시 알려주는 허브 역할을 한다.

> 핵심 원칙: "다음에 뭘 해야 하지?"를 유저가 절대 고민하지 않아야 한다.

##### 주요 기능

- 캐러셀 배너로 진행 중인 이벤트를 한눈에 파악
- 아티스트 선택으로 콘텐츠 전환
- 오늘의 할 일(출석+미션+퀘스트) 컴팩트 카드
- 상태 기반 핵심 추천 1개
- 바로가기 그리드로 주요 서비스 빠른 진입

##### 접근 경로

- 하단 탭 > 홈 (기본 탭)

| [+] 추가 | 아티스트 탐색/추가 화면 이동 |

| 구성 | 2줄 컴팩트 카드 |
| 1줄 | "🔥 {N}일째 연속" + 출석 상태(☐/☑) + 미션 상태(☐/☑) |

| 퀘스트 CTA | [이어서→] → CEB-FQ-101 이동 |

##### 5. 핵심 추천 (1개)

| 항목 | 내용 |
| ---- | ---- |
| 아티스트 | 선택된 아티스트 기준 |
| 표시 수 | 1개 (8단계 우선순위 중 최상위만) |
| 형태 | 풀 너비 카드 (아이콘 + 텍스트 + CTA 버튼) |
| 스타일 | 피그마 참조 |

##### 추천 우선순위 (8단계)

| 순위 | 유저 상태 | 추천 카드 | CTA |
| ---- | ---- | ---- | ---- |
| 1 | Quest 5장 미완료 (cleared 아닌 장이 1개 이상) | "V01D 챌린지 {N}장 진행중" | [이어하기] → CEB-FQ-101 |
| 2 | 1년 전 오늘 기억 존재 | "💜 1년 전 오늘의 기억" | [다시 보기] → CEB-MEM-101 |
| 3 | 디지털 굿즈 미수령 | "새 BIVE를 받을 수 있어요!" | [받으러 가기] → CEB-COL-101 |
| 4 | 래플 진행중 + 응모권 보유 | "응모권 {N}장 — 래플 참여하기" | [응모하기] → CEB-EVT-101 |
| 5 | 덕력 시즌 진행중 | "이번 시즌 {순위}위 ({포인트}pt)" | [랭킹 보기] → CEB-DUK-101 |
| 6 | 팬덤 레벨 80%+ | "Lv.{N}까지 {남은}pt!" | [기여하기] → CEB-EVT-201 |
| 7 | 게임존 신규 게임 | "새 PM 오픈!" | [참여하기] → CEB-GAM-101 |
| 8 | 기본 상태 | "오늘의 퀘스트를 완료하세요" | [퀘스트 보기] → CEB-FQ-101 |

> 해당 상태가 없으면 스킵. 남은 것 중 최상위 1개만 표시. 8번(기본)은 항상 존재하므로 추천 카드는 반드시 표시된다.

##### 6. 바로가기

| 항목 | 내용 |
| ---- | ---- |
| 아티스트 | 선택된 아티스트 기준 |
| 구성 | 1열(풀 너비) 카드 4개 세로 나열. 각 카드에 아이콘 + 타이틀 + 상태 정보 + 프로그레스(해당 시) + CTA |
| 탭 동작 | 해당 서브 화면 직접 이동 |

##### 리치 카드 (프로그레스 포함)

| 카드 | 아이콘 | 타이틀 | 상태 정보 | 탭 액션 |
| ---- | ---- | ---- | ---- | ---- |
| 키우기 | 🏆 | {아티스트명} 키우기 | "Lv.{N}" + 프로그레스 바 + "{현재}/{목표}pt · {달성률}%" | → CEB-EVT-201 |
| 응원 | 💜 | 달성 임박 서포트 이벤트명 | "달성 임박!" 뱃지 + 프로그레스 바 + "{현재}/{목표}pt · {달성률}%" | → CEB-DUK-201 |

> 키우기 카드: 따뜻한 그라데이션(amber). 현재 레벨 + 다음 레벨까지 프로그레스를 직관적으로 표시.
> 응원 카드: 바이올렛~핑크 그라데이션. 달성 임박(80%+) 서포트를 우선 표시하여 참여 유도. 서포트 없으면 "곧 열려요!" 표시.

##### 콘텐츠 카드 (오늘의 일정/소식 + Raffle)

| 카드 | 아이콘 | 구성 | 탭 액션 |
| ---- | ---- | ---- | ---- |
| 오늘의 일정/소식 | 📅 | "오늘" 뱃지 + 오늘 첫 번째 일정 (시간+일정명) + 최신 소식 1줄 + "+N건 더보기 →". 오늘 일정 없으면 최신 소식 2줄 표시. 블루 그라데이션 배경 | → CEB-INF-101 |
| Raffle | 🎁 | "D-{N}" 뱃지(마감 임박 래플) + 래플명 + 참여자 수 + "응모하기 →". 래플 없으면 "새 래플 준비 중!" 표시. 핑크 그라데이션 배경 | → CEB-EVT-101 |

> 기존 "정보" 라벨 → 팬에게 의미 있는 실제 콘텐츠(오늘의 일정/소식) 미리보기로 변경. 유저가 "어, 오늘 음악중심 나오네?" 하고 관심을 가지도록 유도.

## 3. 화면 내 기능설명

### 3.1 캐러셀 배너 탭

- 배너 탭 → 딥링크 URL로 이동 (이벤트 상세)
- 다른 아티스트 이벤트 탭 시: 아티스트 선택 유지 (자동 전환 안 함). 이벤트 상세만 이동. 뒤로가기 → 홈 복귀, 기존 아티스트 유지 (공통정책 10.4 참조)
- 미팔로우 아티스트 이벤트 탭 시: 이벤트 상세 이동 + "이 아티스트를 팔로우하시겠습니까?" 안내

### 3.2 아티스트 선택

- 아티스트 로고 탭 → 하단 탭명 변경 + 홈 아래 콘텐츠(오늘의 할 일/추천/바로가기) 전환
- 전환 시 서버 재조회

### 3.3 출석 체크 (홈에서 직접)

- 오늘의 할 일 카드에서 출석 ☐ 아이콘 탭 → 즉시 완료
- 토스트(Success) "출석 완료! 덕력 10pt 획득"
- 스트릭 +1 반영
- CEB-FQ-501에서도 동일하게 처리 가능 (이중 진입점. 공통정책 6.1 참조)

### 3.4 데이터 갱신

| 트리거 | 동작 |
| ---- | ---- |
| 화면 진입 | 전체 영역 데이터 조회 |
| 아티스트 전환 | 아티스트별 데이터 재조회 |
| Pull-to-Refresh | 전체 재조회 (공통정책 11.2 참조) |
| 서브 화면 복귀 | 변경 가능성 있는 데이터 재조회 |

### 3.5 Pull-to-Refresh

- 공통정책 11.2 규격 적용

- `artist_id` 기반으로 오늘의 할 일 / 추천 카드 / 바로가기 데이터 조회
