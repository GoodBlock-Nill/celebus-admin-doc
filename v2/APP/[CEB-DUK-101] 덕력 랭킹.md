## 1. 화면 목적

아티스트 탭(CEB-ART-101)에서 "덕력" 카드를 탭했을 때 진입하는 화면이다. 현재 시즌 덕력 랭킹(리더보드)과 내 덕력 현황을 확인할 수 있다.

##### 주요 기능

- 내 랭킹 정보 (순위/획득 덕력/보유 덕력/시즌 진행 상태)
- TOP 100 리더보드 (내 순위 하이라이트 + 자동 스크롤)
- 월간 시즌 아카이브 (이전 월 랭킹 드롭다운 전환)
- 덕력 이력 바텀시트

##### 접근 경로

- CEB-ART-101 > 덕력 카드 탭
- CEB-HOM-001 > 추천 카드 [랭킹 보기] 탭

## 2. UI 설명

### 2.1 공통 레이아웃

> 화면 레이아웃은 프로토타입(https://celebus-app.vercel.app/home)을 참조하세요. 플로팅 디버그 버튼으로 다양한 상태를 확인할 수 있습니다. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

### 2.2 영역별 구성요소

##### 서브페이지 헤더

> 본 화면은 서브 페이지 헤더(← + 타이틀)를 사용합니다. 앱 공통 헤더(로고+알림+응모권)는 홈/아티스트 메인에서만 노출됩니다.

| 항목 | 내용 |
| ---- | ---- |
| 뒤로가기 | ← 탭 → 이전 화면으로 복귀 (네비게이션 스택 기반) |
| 타이틀 | "덕력 랭킹" |
| 시즌 드롭다운 | 현재 시즌(예: "4월 시즌") + ▼ 드롭다운. 이전 월 선택 시 해당 월 아카이브 표시 |

##### 내 정보 카드

| 항목 | 내용 |
| ---- | ---- |
| 순위 | "시즌 {N}위" (미참여 시 "아직 활동 전이에요!") |
| 획득 덕력 | "획득 덕력: {N}pt" (랭킹 기준 포인트) |
| 보유 덕력 | "보유 덕력: {N}pt" (사용 가능 잔액) |
| 시즌 정보 | "{M}월 (D-{N})" (남은 일수) |
| 덕력 이력 | [덕력 이력 보기 →] 탭 시 바텀시트 표시 |
| 시즌 보상 뱃지 | 이전 시즌 아카이브 열람 시, 해당 시즌에서 1위/TOP 10/TOP 10% 달성한 경우 해당 보상 뱃지 표시 |

##### TOP 100 리더보드

| 항목 | 내용 |
| ---- | ---- |
| 구성 | 순위 + 닉네임 + 획득 덕력(pt) |
| 상위 3위 아이콘 | 🥇(1위) / 🥈(2위) / 🥉(3위) 아이콘 표시 |
| 내 순위 하이라이트 | 보라색 하이라이트 + ⭐ 표시. 진입 시 내 순위 행으로 자동 스크롤 |
| 동순위 | 같은 포인트는 동일 순위 표시 (예: 공동 38위). 다음 순위는 인원수만큼 스킵 (예: 공동 38위 2명 → 다음은 40위) |
| TOP 100 밖 | 리스트 하단에 "── 내 순위 ──" 별도 섹션으로 내 순위만 표시 |

##### 시즌 보상 안내

| 항목 | 내용 |
| ---- | ---- |
| 위치 | 리더보드 하단 |
| 내용 | 1위: 사인 앨범 / TOP 10: 사인 포토카드 / TOP 10%: 슈퍼팬 뱃지 + 디지털 포토카드 |
| 스타일 | 피그마 참조 |

##### 랭킹 안내 텍스트

| 항목 | 내용 |
| ---- | ---- |
| 내용 | "ℹ️ 랭킹은 획득 덕력 기준입니다. 서포트에 응원한 덕력은 랭킹에 영향을 주지 않습니다." |
| 위치 | 화면 최하단 |

##### 덕력 이력 바텀시트

| 항목 | 내용 |
| ---- | ---- |
| 트리거 | [덕력 이력 보기 →] 탭 |
| 구성 | 최근 덕력 이력 리스트 (획득 + 사용 통합. 획득: +50pt 초록 / 사용: -500pt 빨강) |
| 정렬 | 최신순 |
| 항목 예시 | "퀘스트 미션 완료 +50pt (04.14)", "출석 체크 +10pt (04.14)", "서포트 응원 -500pt (04.13)", "독점 콘텐츠 해금 -100pt (04.12)" |
| 최대 표시 | 50건 (더보기 페이지네이션) |
| 닫기 | 드래그 다운 또는 [X] 버튼 |

##### 시즌 드롭다운 (이전 월 아카이브)

| 항목 | 내용 |
| ---- | ---- |
| 트리거 | 헤더 시즌 드롭다운 탭 |
| 구성 | 월 리스트 (현재 월 + 이전 최대 6개월). 현재 시즌에 "현재" 뱃지 표시 |
| 선택 시 | 해당 월의 최종 랭킹 스냅샷 표시 (읽기 전용) |
| 현재 시즌 | 실시간 랭킹 (Pull-to-Refresh 가능) |
| 이전 시즌 | 확정 랭킹 (변경 없음, 읽기 전용) |
| 시즌 보상 이력 | 이전 월 아카이브에서 내가 받은 보상 표시 (예: "이 시즌 보상: 슈퍼팬 뱃지 획득") |

## 3. 화면 내 기능설명

### 3.1 리더보드 조회

- 진입 시 현재 시즌의 TOP 100 + 내 순위 조회
- 내 순위가 TOP 100 내이면 해당 행으로 자동 스크롤 + 하이라이트
- 내 순위가 TOP 100 밖이면 리스트 하단 별도 섹션에 내 순위 표시

### 3.2 동순위 처리

- 같은 획득 덕력 포인트를 가진 유저는 동일 순위 표시 (예: "공동 38위")
- 다음 유저는 동순위 인원수만큼 순위 스킵 (공동 38위 2명 → 다음은 40위)
- 시즌 보상 대상: 동순위 전원 보상 지급 (예: 공동 10위 2명이면 둘 다 TOP 10 보상 수령)

### 3.3 시즌 아카이브 전환

- 헤더 드롭다운에서 이전 월 선택 → 해당 월 최종 랭킹 스냅샷 표시
- 이전 시즌: 읽기 전용 (Pull-to-Refresh 불필요)
- 현재 시즌: 실시간 랭킹 (Pull-to-Refresh 가능)
- 데이터 보존 기간: 최근 6개월 아카이브

##### 시즌 전환 시 내 정보 카드 변경 사항

| 영역 | 현재 시즌 | 이전 시즌 선택 시 |
| ---- | ---- | ---- |
| 내 순위 | 실시간 순위 | 해당 시즌 최종 순위 |
| 획득 덕력 | 실시간 포인트 | 해당 시즌 최종 포인트 |
| 시즌 보상 뱃지 | 미표시 (진행 중) | 해당 시즌 보상 뱃지 표시 (1위/TOP10/TOP10% 달성 시) |
| [덕력 이력 보기] | 표시 | 미표시 (아카이브는 읽기 전용) |
| TOP 100 리더보드 | 실시간 데이터 | 해당 시즌 최종 스냅샷 (고정) |

### 3.4 덕력 이력 바텀시트

- [덕력 이력 보기 →] 탭 시 바텀시트 표시
- 최근 덕력 이력 리스트 (획득+사용 통합, 최신순)
- 각 항목: 활동명 + 포인트(+초록/-빨강) + 날짜
- 최대 50건 표시 (더보기 페이지네이션)

##### 획득 이력 케이스

| 활동 | 표시 예시 | 포인트 |
| ---- | ---- | ---- |
| 출석 체크 | "출석 체크" | +10pt |
| 일일 미션 완료 | "일일 미션 완료" | +20pt |
| Quest 미션 승인 | "퀘스트 미션 완료 (1장 미션3)" | +50pt |
| 기억저장소 업로드 | "기억 저장" | +30pt |
| 스트릭 보너스 | "7일 연속 출석 보너스" | +50pt |
| 시즌 보상 | "4월 시즌 TOP 10 보상" | +500pt |

##### 사용 이력 케이스

| 활동 | 표시 예시 | 포인트 | 탭 액션 |
| ---- | ---- | ---- | ---- |
| 서포트 이벤트 응원 | "서포트 응원: ☕ 커피차 서포트" | -500pt | → CEB-DUK-201 해당 이벤트로 이동 |
| 독점 콘텐츠 해금 | "독점 콘텐츠 해금: 비하인드 사진" | -100pt | → 해당 콘텐츠로 이동 |
| 서포트 미달성 반환 | "서포트 덕력 반환: ☕ 커피차 서포트" | +500pt (반환) | → CEB-DUK-201 해당 이벤트로 이동 |

> 향후 래플 부스팅, 특별 이벤트 등 새로운 사용처 추가 시 이력 유형을 확장합니다. 각 이력 항목을 탭하면 관련 화면으로 이동하여 "어디에 쓰였는지" 확인할 수 있습니다.

### 3.5 시즌 보상 자동 지급

- 월간 시즌 종료 시 서버가 자동 판정 + 보상 분배
- 1위: 사인 앨범 / TOP 10: 사인 포토카드 / TOP 10%: 슈퍼팬 뱃지 + 디지털 포토카드
- 동순위: 전원 보상 지급
- 보상 지급 후 푸시 알림 전송: "이번 시즌 {순위}위! 보상을 확인하세요"
- 보상 수령 이력: 시즌 아카이브에서 해당 월의 보상 이력으로 확인 가능

### 3.6 데이터 갱신

| 트리거 | 동작 |
| ---- | ---- |
| 화면 진입 | 현재 시즌 랭킹 + 내 정보 조회 |
| Pull-to-Refresh | 현재 시즌 전체 재조회 (공통정책 11.2 참조) |
| 시즌 드롭다운 전환 | 선택한 월의 아카이브 데이터 조회 |
| 복귀 시 | 내 정보 재조회 (활동으로 포인트 변동 가능) |

### 3.7 Pull-to-Refresh

- 현재 시즌에서만 동작 (이전 시즌 아카이브는 고정 데이터)
- 공통정책 11.2 규격 적용

## 4. 규칙/로직

### 4.1 랭킹 산정 기준

| 규칙 | 설명 |
| ---- | ---- |
| 기준 | 획득 덕력 (보유 덕력/서포트 응원 덕력 무관) |
| 범위 | 아티스트별 분리 (artist_id 기반) |
| 시즌 | 월간 정산 (매월 1일 00:00 ~ 말일 23:59 KST) |
| 동순위 | 동률 표시 + 다음 순위 스킵. 보상 대상: 동순위 전원 |
| 갱신 | 실시간 (활동 시 즉시 반영) |

### 4.2 시즌 규칙

| 규칙 | 설명 |
| ---- | ---- |
| 월간 시즌 | 매월 1일 00:00 ~ 말일 23:59 KST |
| 월간 보상 | 시즌 종료 시 서버 자동 판정 + 보상 분배 |
| 연간 리셋 | BO에서 시즌 종료일 설정 → 획득 덕력 0 초기화 |
| 보유 덕력 | 시즌 리셋과 무관. 영구 유지 |
| 아카이브 | 최근 6개월 월간 랭킹 스냅샷 보존 |
| 동결 흐름 | 시즌 종료일 도래 → 랭킹 동결 → 보상 처리 → 새 시즌 시작 |

### 4.3 랭킹 동결 기간

| 항목 | 내용 |
| ---- | ---- |
| 동결 시작 | 월 말일 23:59 KST |
| 동결 기간 | 보상 처리 완료까지 (최대 수 시간) |
| 동결 중 화면 | "시즌 정산 중이에요! 새로운 시즌이 곧 시작돼요" 안내 배너 표시. 리더보드 읽기 전용 |
| 동결 중 활동 | 해당 기간 획득 덕력은 새 시즌에 귀속 |
| 새 시즌 시작 | 랭킹 0pt 리셋. 리더보드 초기화 |

### 4.4 멀티 아티스트 대응

- 아티스트별 랭킹 완전 분리
- artist_id 기반으로 모든 데이터 조회
- 아티스트 전환 → CEB-ART-101에서 재진입 시 해당 아티스트 랭킹 표시

## 5. 에러 및 예외상황

| 상태 | 처리 |
| ---- | ---- |
| Loading | 리더보드 영역 Skeleton UI 표시 |
| Default | 내 정보 카드 + TOP 100 리더보드 정상 표시 |
| Error | 전체 조회 실패 시 에러 안내 + 재시도 버튼 |
| Empty (신규 유저) | 획득 0pt, 순위 없음. "활동을 시작하면 랭킹에 참여할 수 있어요!" 안내 표시 |
| 시즌 동결 중 | "시즌 정산 중이에요! 새로운 시즌이 곧 시작돼요" 안내 배너 + 리더보드 읽기 전용 |
| 비로그인 | 비로그인도 열람 가능. TOP 100 리더보드 표시. 내 정보 카드는 "-" 표시. [덕력 이력 보기] 미노출. 참여 유도 CTA 탭 시 → 로그인 화면 이동 (공통정책 10.5 참조) |
| 유저 수 부족 | 참여 유저가 100명 미만일 때: 실제 참여 유저 수만큼만 리스트 표시 |
| 참여자 20명 미만 | 리더보드 상단에 배너 표시: "이번 시즌이 막 시작됐어요! 지금 활동하면 TOP 10 진입 기회 💪" |
| 아카이브 없음 | 첫 월(이전 시즌 없음): 드롭다운에 현재 시즌 1개만 표시. 선택 변경 불가 |
