## 1. 화면 개요

##### 목적

덕력 랭킹(CEB-DUK-101)의 시즌 보상 카드 [보상 확인]에서 진입하는 화면이다. 유저가 시즌별로 달성한 랭킹 등급과 획득 보상(실물·디지털)을 확인하고, 실물 보상의 배송지를 입력하는 화면이다.

##### 주요 기능

- 시즌 선택 (드롭다운)으로 시즌별 내 보상 조회
- 달성 등급 표시 (1위 / TOP 10 / TOP 10% 등)
- 획득 보상 리스트 (실물: 배송지 입력 / 디지털: 자동 지급)
- 보상 받은 시즌 목록

##### 접근 경로

- CEB-DUK-101 덕력 랭킹 > 시즌 보상 카드 [보상 확인] 탭

##### 화면 레이아웃

> 시각적 스타일은 Figma 시안을 따릅니다. 변형: 1위 달성 / TOP 10 / TOP 10% 등 등급별.

## 2. 영역별 상세

### 2.1 헤더 + 시즌 선택

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 뒤로가기 | ← 탭 → CEB-DUK-101 복귀 |
| 타이틀 | "내 보상" |
| 시즌 드롭다운 | 보상받은 시즌 선택 (예: "3월 시즌 ▼") |
| 스타일 | 피그마 참조 |

#### 동작

- 시즌 드롭다운 탭 → 보상받은 시즌 목록에서 선택 → 해당 시즌 보상 표시

### 2.2 달성 결과

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 구성 | 왕관 일러스트 + 달성 등급 + 안내 문구 |
| 달성 등급 | "1위 달성" / "TOP 10 달성" / "TOP 10% 달성" 등 (시즌 보상 구간 기준) |
| 안내 문구 | "시즌 보상을 획득했어요!" |
| 스타일 | 피그마 참조 |

> 보상 대상 구간(등수/퍼센트)은 시즌별 BO 설정 ([CEB-DUK-101] §2.5 보상 대상 기준 참조).

### 2.3 획득한 보상

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 구성 | 보상별 행: 보상 이미지/아이콘 + 보상명 + 상태/안내 |
| 실물 보상 | 보상명(예: "사인 포토카드") + "배송지 정보를 입력해주세요" + [배송지 입력] |
| 디지털 보상 | 보상명(예: "디지털 포토카드") + "보상이 지급되었어요" (자동 지급) |
| 배송 안내 | "실물 보상은 배송지 입력 후 순차적으로 배송됩니다." |
| 입력 기한 | "입력기한 {YYYY.MM.DD} 까지" |
| 스타일 | 피그마 참조 |

#### 동작

- 실물 보상 [배송지 입력] 탭 → 배송지 입력 폼 (이름·연락처·주소). 입력 완료 시 완료 토스트
- 디지털 보상: 시즌 종료 시 자동 지급 완료 (별도 액션 없음)
- 입력 기한 경과 시: 실물 보상 배송지 입력 마감 안내 (수령 불가 또는 운영 문의 안내)

#### 상태별 표시

| 상태 | 처리 |
| ---- | ---- |
| 실물 미입력 | "배송지 정보를 입력해주세요" + [배송지 입력] 활성 |
| 실물 입력 완료 | "배송지 입력 완료 · 배송 준비 중" (비활성) |
| 디지털 지급 완료 | "보상이 지급되었어요" |
| 보상 없음 | 해당 시즌 미수상: "이 시즌에는 받은 보상이 없어요" 안내 |

### 2.4 보상 받은 시즌

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 구성 | 보상받은 시즌 리스트 (하단 영역) |
| 항목 | 시즌명 + 달성 등급 뱃지(예: "1위 달성", "TOP10% 달성") + 기간(YYYY.MM.DD ~ YYYY.MM.DD) + 선택 표시(✓) |
| 정렬 | 최신 시즌순 |
| 탭 동작 | 시즌 선택 → 상단 보상 영역 해당 시즌으로 전환 |
| 스타일 | 피그마 참조 |

#### 비로그인

- 본 화면은 로그인 상태에서만 진입 (CEB-DUK-101에서 비로그인 시 시즌 보상 카드 미노출)
