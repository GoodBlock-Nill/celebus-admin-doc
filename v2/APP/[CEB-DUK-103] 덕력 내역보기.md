## 1. 화면 개요

##### 목적

덕력 랭킹(CEB-DUK-101)의 내 정보 카드 [덕력 내역 보기]에서 진입하는 화면이다. 유저가 획득·사용한 덕력의 전체 내역을 날짜별로 확인하는 화면이다. (Figma 시안 라벨: "덕력 History")

> v.3 정합: 기존 CEB-DUK-101 인앱 바텀시트로 제공하던 덕력 이력을 **별도 화면으로 분리**.

##### 주요 기능

- 전체 / 획득 / 사용 필터로 내역 분류
- 날짜 그룹별 내역 리스트 (획득 +초록 / 사용 -빨강)
- 항목 탭 시 관련 화면으로 이동 (어디에 쓰였는지 확인)

##### 접근 경로

- CEB-DUK-101 덕력 랭킹 > 내 정보 카드 [덕력 내역 보기 >] 탭

##### 화면 레이아웃

> 시각적 스타일은 Figma 시안을 따릅니다.

## 2. 영역별 상세

### 2.1 헤더

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 좌측 | ← 뒤로가기 또는 타이틀 |
| 타이틀 | "덕력 내역" (Figma 시안 라벨 "덕력 History") |
| 우측 | [X] 닫기 (CEB-DUK-101 복귀) |
| 스타일 | 피그마 참조 |

#### 동작

- ← / [X] 탭 → CEB-DUK-101 덕력 랭킹으로 복귀

### 2.2 필터 탭

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 구성 | [전체] / [획득] / [사용] 3개 탭 |
| 기본 선택 | 전체 |
| 전체 | 획득 + 사용 모든 내역 |
| 획득 | 덕력 증가 내역만 (출석·미션·퀘스트·기억·시즌 보상·반환 등) |
| 사용 | 덕력 감소 내역만 (서포트 응원·독점 콘텐츠 해제 등) |
| 탭 전환 | 즉시 리스트 갱신 |
| 스타일 | 피그마 참조 |

### 2.3 내역 리스트

#### UI 구성

| 항목 | 내용 |
| ---- | ---- |
| 그룹 | 날짜별 그룹 헤더 (YYYY. MM. DD) |
| 항목 구성 | 활동명 + 세부 설명(선택) + 증감 수치 + (관련 화면 있을 시 ">") |
| 증감 표기 | 획득: "+{N}" (초록) / 사용: "-{N}" (빨강) |
| 항목 예시 | "서포트 이벤트 / 커피차 서포트·반환 +10 >", "독점 콘텐츠 해제 / 비하인드 사진 -100 >", "출석 체크 +10", "일일 미션 완료 +20", "기억 저장 +30", "시즌 보상 +500" |
| 정렬 | 최신순 (날짜 내림차순) |
| 페이지네이션 | 무한 스크롤 또는 더보기 |
| 스타일 | 피그마 참조 |

#### 동작

| 항목 유형 | 탭 동작 |
| ---- | ---- |
| 서포트 응원/반환 | → CEB-DUK-201 해당 서포트 이벤트로 이동 |
| 독점 콘텐츠 해제 | → 해당 콘텐츠로 이동 |
| 출석·일일 미션·퀘스트·기억·시즌 보상 | 관련 화면 없으면 탭 동작 없음 (정보 표시만) |

#### 내역 유형 (획득/사용)

| 구분 | 활동 | 표시 예시 | 증감 |
| ---- | ---- | ---- | ---- |
| 획득 | 출석 체크 | "출석 체크" | +5DUK |
| 획득 | 일일 미션 완료 | "일일 미션 완료" | +25DUK |
| 획득 | 퀘스트 미션 승인 | "출석 체크 / EP.01·V01D 공식 X 팔로우" | +N DUK |
| 획득 | 기억 저장 | "기억 저장" | +30DUK |
| 획득 | 시즌 보상 | "시즌 보상" | +N DUK |
| 획득 | 서포트 미달성 반환 | "서포트 이벤트 / 커피차 서포트·반환" | +N DUK |
| 사용 | 서포트 응원 | "서포트 이벤트 / 커피차 서포트" | -N DUK |
| 사용 | 독점 콘텐츠 해제 | "독점 콘텐츠 해제 / 비하인드 사진" | -N DUK |

#### 상태별 표시

| 상태 | 처리 |
| ---- | ---- |
| Loading | 리스트 영역 Skeleton UI |
| Default | 날짜 그룹 내역 정상 표시 |
| Empty | 내역 없음(신규 유저): "아직 덕력 내역이 없어요" 안내 (Figma 내역x 변형 참조) |
| Error | 조회 실패 시 에러 안내 + 재시도 버튼 |

#### 비로그인

- 본 화면은 로그인 상태에서만 진입 (CEB-DUK-101에서 비로그인 시 [덕력 내역 보기] 미노출)
