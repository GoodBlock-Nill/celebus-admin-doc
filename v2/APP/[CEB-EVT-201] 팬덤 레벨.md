## 1. 화면 목적

아티스트 탭(CEB-ART-101)에서 "{아티스트명} 키우기" 카드를 탭했을 때 진입하는 화면이다. (유저 대면 명칭: {아티스트명} 키우기, 예: V01D 키우기) 팬덤 전체의 공동 레벨업 진행 상황과 레벨별 보상을 확인하고, 덕력 획득 활동으로 유도하는 화면이다.

##### 주요 기능

- 현재 팬덤 레벨 + 프로그레스 바 + 내 기여도 확인
- 레벨별 보상 리스트 확인 (해금 완료 / 진행중 / 미해금 "???" 실루엣)
- 팬덤 활동 요약 (참여 팬 수 + 이번 달 합계 + 주요 활동 유형)
- 덕력 획득 활동 유도 CTA (출석 체크 / 퀘스트 / 랭킹)
- 레벨업 달성 시 축하 모달 표시

##### 접근 경로

- CEB-ART-101 > 팬덤 레벨 카드 탭
- CEB-HOM-001 > 바로가기 > {아티스트명} 키우기 카드 탭
- CEB-HOM-001 > 추천 카드 [기여하기] 탭

> 참고: 팬덤 레벨은 개인 경쟁이 아닌 팬덤 전체의 공동 달성 목표이다. 개인 순위 경쟁은 CEB-DUK-101 덕력 랭킹에서 제공한다.

## 3. 화면 내 기능설명

### 3.1 레벨 진행 확인

- 화면 진입 시 현재 팬덤 레벨, 프로그레스 바, 내 기여도를 상단 카드에 표시
- 프로그레스 바는 팬덤 전체 획득 덕력 / 현재 레벨 목표 덕력 비율로 렌더링
- 내 기여도는 개인 획득 덕력 / 팬덤 전체 획득 덕력 비율로 계산하여 표시

### 3.2 보상 리스트 확인

- 해금 완료 레벨: 보상명 + ✅ 수령 완료 표시
- 현재 진행 레벨: 보상명 + 레벨 내 진행률 프로그레스 바 표시
- 미해금 레벨: "???" 실루엣 표시 (탭 불가)
- 보상은 레벨 달성 즉시 서버에서 전체 유저에게 자동 지급. 유저 수동 수령 불필요

### 3.3 팬덤 활동 요약 확인

- 참여 팬 수, 이번 달 합계, 주요 활동 유형 표시
- TOP 기여자 개인 리스트는 표시하지 않음 (CEB-DUK-101 덕력 랭킹에서 제공)

### 3.4 활동 CTA 이동

- [출석 체크] 탭 → CEB-FQ-501 일일 미션으로 이동
- [퀘스트] 탭 → CEB-FQ-101 Quest 메인으로 이동
- [랭킹] 탭 → CEB-DUK-101 덕력 랭킹으로 이동

> 활동 CTA로 이동한 서브 화면의 뒤로가기는 네비게이션 스택 기반으로 EVT-201로 복귀합니다.

### 3.5 레벨업 축하 모달

- 레벨업 달성 시 서버에서 팬덤 전체에 푸시 알림 발송
- 앱 진입 시 서버에서 미확인 레벨업 큐 확인 → 해당 아티스트 팬덤 레벨 화면 진입 시 모달 자동 표시
- [보상 확인하기] 탭 → 모달 닫기 + 해당 레벨 보상 항목으로 스크롤
- 1회성 표시 (확인 후 재표시하지 않음)

### 3.6 데이터 갱신

| 트리거 | 동작 |
| ---- | ---- |
| 화면 진입 | 현재 레벨 + 진행률 + 내 기여도 + 보상 상태 + 활동 요약 전체 조회 |
| Pull-to-Refresh | 전체 데이터 재조회 (공통정책 11.2 참조) |
| 복귀 시 | 내 기여도 재조회 (활동 후 변동 반영) |

## 4. 규칙/로직

### 4.1 팬덤 레벨 기본 규칙

| 규칙 | 설명 |
| ---- | ---- |
| 기여 기준 | 팬덤 전체 유저의 **획득 덕력** 합산 (자동 누적. 보유/사용 덕력과 무관) |
| 리셋 | 없음 (상시 누적. 시즌제는 추후 검토) |
| 레벨 수 | 초기 Lv.1~5. BO에서 추가 레벨 자유 오픈 |
| 목표 설정 | BO에서 레벨별 목표 덕력 + 보상 자유 설정 |
| 목표 기간 | 6~12개월에 Lv.5 도달 설계 (충분히 높은 목표값 유지) |
| 아티스트 분리 | 팬덤 레벨은 아티스트별 독립 (artist_id 기반) |

### 4.2 레벨업 판정 규칙

| 규칙 | 설명 |
| ---- | ---- |
| 판정 방식 | 서버 자동 판정 (팬덤 전체 누적 덕력 >= 레벨 목표값 시 즉시 레벨업) |
| 보상 지급 | 레벨업 즉시 서버에서 자동 지급 (전체 유저 대상) |
| 레벨업 알림 | 서버에서 팬덤 전체에 푸시 알림 발송 + 앱 진입 시 축하 모달 표시 |
| 다음 레벨 | 레벨업 즉시 다음 레벨 목표로 프로그레스 전환 |

### 4.3 MAX 상태 규칙 (최고 레벨 달성)

| 규칙 | 설명 |
| ---- | ---- |
| 표시 | 프로그레스 바 100% 고정 + "🏆 최고 레벨 달성!" + "새로운 레벨이 곧 열립니다" 문구 |
| BO 추가 | 관리자가 BO에서 새 레벨 추가 시 화면 자동 갱신 (서버 동적 조회) |
| 활동 CTA | MAX 상태에서도 활동 CTA 섹션 유지 |

### 4.4 멀티 아티스트 대응

- 팬덤 레벨은 아티스트별 분리 (artist_id 기반)
- 아티스트 전환 → CEB-ART-101 재진입 → 해당 아티스트 팬덤 레벨 데이터 표시

### 4.5 프로그레스 수치 계산

| 수치 | 계산 방식 |
| ---- | ---- |
| 레벨 진행률 | 팬덤 전체 현재 누적 덕력 / 현재 레벨 목표 덕력 × 100 (%) |
| 내 기여도 | 개인 누적 획득 덕력 / 팬덤 전체 누적 획득 덕력 × 100 (%) |
| 탈퇴 유저 기여분 | 유지 (차감하지 않음). 공통정책 10.6 참조 |
