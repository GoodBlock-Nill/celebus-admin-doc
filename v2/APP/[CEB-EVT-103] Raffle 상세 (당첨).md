## 1. 화면 개요

Raffle 추첨 결과 당첨된 유저에게 축하 메시지를 전달하고, 수령 방식별(현장/배송/온라인) 가이드와 폼을 제공.

##### 주요 기능
- 컨페티 애니메이션 + 축하 문구
- 당첨 경품 정보 (이미지·경품명·수량)
- 수령 방식 분기: 현장 / 배송 / 온라인
- 배송지 입력 폼 (배송 수령 한정)

##### 접근 경로
- [CEB-EVT-101] Raffle 메인 > 당첨 리스트 클릭
- 당첨 푸시 알림 클릭

```text

[썸네일] {경품명} · 수량 {n}개

[하단 고정 버튼]
  현장/온라인: [확인]
  배송: [배송지 입력 완료]
─────────────────────────────────
```

| 영역 | 구성 | 설명 |
| --- | --- | --- |
| 헤더 | [<] / "당첨 결과" | |
| 대표 이미지 | 축하 이미지 + 컨페티 | 진입 시 1회 재생, 재진입 시 재실행 |

| 안내 | 수령 방식별 텍스트/폼 | 운영자 등록 가이드 줄바꿈 보존 |
| 푸터 | 하단 고정 버튼 | 수령 방식별 텍스트 분기 |

## 3. 규칙·로직

### 3.1 진입 시 검증

1. 서버에서 당첨 여부 재조회
2. 미당첨으로 확인 → [CEB-EVT-104] 미당첨 화면으로 교체 전환 (에러 토스트 없이 자연 전환)
3. 당첨 + 종료 상태 → 본 화면 표시

### 3.2 수령 방식별 동작

| 수령 방식 | 버튼 텍스트 | 동작 |
| --- | --- | --- |
| 현장 | 확인 | [CEB-EVT-101] Raffle 메인으로 이동 |
| 배송 | 배송지 입력 완료 | 배송 정보 저장 → 성공 시 완료 토스트 + 이동 |
| 온라인 | 확인 | 추후 정의 예정 |

### 3.3 배송지 입력 기한

- 운영자 설정 기간 (기본 7일, BO [CEB-BO-FQ-204] 정의)
- 기한 내: 폼 활성화·수정 가능
- 기한 만료: 폼 비활성화 + "입력 기간이 종료되어 당첨이 취소되었어요" 안내 + [확인] 버튼만 활성

### 3.4 미입력 이탈 가드

배송형 당첨자가 배송지 미입력 상태로 [<] 뒤로가기 시 컨펌 모달:
- "기한 내 배송지 정보를 입력해야 상품 수령이 가능해요. 입력하지 않고 나가시겠어요?"
- [나가기] / [계속 입력]

### 3.5 폼 검증

- 필수 필드 미입력 → [배송지 입력 완료] 비활성화
- 모든 필수 입력 → 활성화

### 3.6 다국어

- 축하 문구·수령 가이드·버튼 텍스트: 다국어 KO/EN/JP
- 경품명: 서버 다국어 (BO 설정)
