## 1. 개요

팬덤 레벨(EVT) 영역은 아티스트별 **팬덤 공동 레벨** 시스템을 운영하는 영역이다. 팬덤 전체가 획득한 덕력이 누적되어 레벨업하며, 레벨별로 보상이 지급된다. 운영자는 **팬덤레벨을 아티스트·시즌 단위로 직접 생성(수동)**하고, 레벨 곡선·보상을 설정하며 상태(임시저장→진행중→종료)를 관리한다. 앱 [CEB-EVT-201] "{아티스트} 키우기" 화면에 반영된다.

> 팬덤 레벨(공동 레벨)과 덕력관리(개인 덕력 시즌·랭킹, [아티스트 > 덕력관리])는 별개 영역이다. 팬덤 레벨은 개인 경쟁이 아닌 팬덤 전체의 공동 달성 목표다.

## 4. 영역 고유 정책

### 4.1 팬덤 레벨 기본 규칙

| 항목 | 정책 |
| --- | --- |
| 생성 | **운영자가 [생성]으로 아티스트·시즌 단위 팬덤레벨을 직접 생성**한다. 자동 생성하지 않는다 |
| 기여 기준 | 팬덤 전체 회원의 획득 덕력 합산 (자동 누적. 보유·사용 덕력과 무관) |
| 아티스트 분리 | 팬덤 레벨은 아티스트별 독립 (그룹 단위) |
| 레벨 수 | 최대 10레벨. 운영자가 레벨 수·레벨별 목표 덕력 자유 설정 |
| 판정 | 팬덤 전체 누적 덕력이 현재 레벨 목표 도달 시 서버 자동 레벨업 |
| 보상 대상 | 해당 아티스트에 획득 덕력 1DUK 이상 기여한 회원 |

### 4.1.1 시즌 라이프사이클 (상태)

```text
준비 → 진행중 → 정산중 → 종료
```

| 상태 | 의미 | 앱 노출 | 전환 트리거 |
| --- | --- | --- | --- |
| 준비 | 생성 후 게시 전. 운영자가 레벨 곡선·보상을 설정하는 단계 | 미노출 | 생성 시 자동 진입 |
| 진행중 | 앱 노출·운영 중. 팬덤 누적 덕력에 따라 레벨업 진행 | 노출 | 운영자 [게시] |
| 정산중 | 시즌 종료 직후 보상 지급(자동 지급 + 래플 추첨)을 진행하는 단계 | 미노출 | 시즌 자동 종료(12.31) 또는 운영자 [정산 시작] |
| 종료 | 보상 지급 완료. 시즌 마감 | 미노출 | 운영자 [종료] (정산 완료 후) |

- 시즌은 매년 12월 31일 23:59:59 자동으로 **정산중**으로 전환된다. 운영자는 진행 중 필요 시 [정산 시작]으로 조기 전환할 수 있다.
- 상태 전환 액션(게시·정산 시작·종료)은 [CEB-BO-EVT-201] 상세 상단에서 수행한다.

### 4.1.2 레벨 운영 상태

진행중 시즌의 팬덤 레벨은 누적 덕력 달성에 따라 다음 운영 상태를 가진다.

| 운영 상태 | 의미 |
| --- | --- |
| 운영중 | 최고 레벨 미만. 다음 레벨 목표를 향해 누적 진행 중 |
| 최고레벨(MAX) | 설정된 최고 레벨에 도달. 추가 레벨이 등록되면 자동으로 다음 목표로 전환 |

- 최고레벨(MAX) 도달 후 운영자가 [CEB-BO-EVT-201] 탭1에서 신규 레벨을 추가하면 팬덤 레벨은 다시 운영중으로 전환되어 새 목표를 향해 진행한다.
- **신규 레벨 보상 지급 대상**: 신규 레벨의 보상은 추가 이후 그 레벨을 달성하는 시점에 기여 회원(1DUK 이상)에게 적용된다. 이미 달성·지급 완료된 기존 레벨의 보상·지급 내역은 변경되지 않는다(소급 없음).

### 4.2 레벨 곡선 (카운터 모델)

- 각 레벨은 **0부터 시작**하여 "레벨업 목표 덕력" 도달 시 다음 레벨 진입과 동시에 카운터 0으로 리셋된다. 각 레벨 목표값은 해당 레벨 단독 목표값이다.
- 권장 곡선(운영자 미설정 시 자동 적용): Lv1 10,000 / Lv2 30,000 / Lv3 70,000 / Lv4 150,000 / Lv5 300,000 / Lv6 550,000 / Lv7 1,000,000 / Lv8 1,800,000 / Lv9 3,200,000 / Lv10 6,000,000 (단위 DUK).
- 운영자는 권장값을 자유롭게 조정할 수 있다.

### 4.4 레벨별 보상 타입 (5종)

| 보상 타입 | 예시 | 지급 방식 | 처리 주체 |
| --- | --- | --- | --- |
| 독점 콘텐츠 해금 | 비하인드 사진·미공개 영상·멤버 셀카 | 자동 해금 (앱 열람) | 시스템 |
| 디지털 굿즈 | 디지털 포카·뱃지·특별 에디션 BIVE | 자동 지급 (컬렉션 추가) | 시스템 |
| 다운로드 콘텐츠 | 보이스 메시지·배경화면·영상 메시지 | 자동 지급 (앱 다운로드) | 시스템 |
| 실물 보상 | 사인앨범·사인 포토카드·기념 굿즈 | 래플 추첨 → 배송 | 추첨 시스템, 배송 운영자 |
| 이벤트 참여권 | 팬싸 참가권·콘서트 VIP·영상통화 | 래플 추첨 → 안내 | 추첨 시스템, 안내 운영자 |

- 디지털 보상(독점 콘텐츠·디지털 굿즈·다운로드)은 레벨 달성 시 기여 회원(1DUK 이상)에게 자동 지급.
- 실물·이벤트 보상은 기여 회원 중 래플 추첨(기여도 비례 가중치). 추첨은 레벨 달성 후 운영자가 실행한다.
