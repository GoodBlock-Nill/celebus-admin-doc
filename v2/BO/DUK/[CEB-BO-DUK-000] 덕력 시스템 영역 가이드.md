## 1. 개요

덕력(DUK) 영역은 회원이 Quest·Raffle·서포트·기억 게시 등 모든 활동에서 적립·소비하는 **활동 통합 포인트** 시스템을 관리한다. **시즌제** 운영 (시즌 종료 시 자동 정산 + 보상 지급), **티어 보상**, **캡 조정**, **랭킹 보상**이 핵심 정책. 회원 단위 변동 내역은 USR-201 [활동내역] 탭에서도 통합 노출.

## 6. 영역 고유 정책

### 6.1 시즌 라이프사이클

```
임시저장 (Draft) → 게시대기 (Ready, 시작일 자동 진입) → 진행중 (Active) → 정산중 (Settling) → 종료 (Ended)
```

| 상태 | 의미 | 가능 액션 |
| --- | --- | --- |
| Draft | 정책 작성 중 | 수정·삭제 |
| Ready | 게시 완료, 시작일 대기 | 수정 (제한적) |
| Active | 회원 적립·소비 진행 중 | 캡 조정 (POLICY 사유 입력 필수), 티어 수정 (제한적) |
| Settling | 시즌 종료 후 자동 정산 중 (수 분~수 시간) | (자동) |
| Ended | 정산 완료, 보상 지급 완료 | 조회만 |

### 6.2 시즌 정산 정책

시즌 종료 시점에 자동 트리거:

1. **랭킹 산출**: TOP N명 결정 (POLICY가 시즌별 설정, 기본 100명)
2. **티어 분류**: 회원 잔액 기준 티어 매핑 (Bronze/Silver/Gold/Platinum 등)
3. **티어 보상 지급**: 티어별 응모권·BIVE 자동 발행 — **응모권 지급은 RFT 영역의 정규 발급 출처 "덕력 랭킹 보상"으로 변동 기록** ([CEB-BO-006] §7.6 + [CEB-BO-010] §4-1 정합)
4. **랭킹 보상 지급**: TOP N명에게 추가 보상 (BIVE 한정판·응모권 등) — 응모권 지급은 동일하게 "덕력 랭킹 보상" 출처로 기록
5. **잔액 정산**: 시즌 잔액 일부 이월 (POLICY 결정. 기본 50% 이월, 50% 소멸)
6. **활동 로그 기록** + 회원 푸시 알림

정산 실패 시 운영자 알림 + 자동 재시도 (BIVE 자동 민팅 큐와 동일 패턴 — [CEB-BO-004] §7.3).

> **응모권 출처 정합**: 본 정산 단계에서 발급되는 모든 응모권은 [CEB-BO-RFT-201] 변동 내역에서 출처 "덕력 랭킹 보상"으로 조회 가능. 출처 분포 카드는 [CEB-BO-RFT-101] §2-3에 노출.

### 6.3 캡 (Cap) 정책

회원 한 명이 시즌 내 누적 적립할 수 있는 최대 DUK:
- 기본 캡: 시즌별 POLICY 설정 (예: 100,000 DUK)
- 캡 도달 시: 추가 적립 차단, 회원에게 안내 메시지
- 캡 조정 (Active 중): POLICY가 사유 입력 후 변경 가능. 변경은 신규 적립부터 적용 (기존 잔액 영향 없음)

### 6.4 적립 출처

DUK는 다음 출처에서 자동 적립 (시스템 트리거):

| 출처 영역 | 적립 시점 | 기본 단위 |
| --- | --- | --- |
| Quest (FQ) | 검수 승인 시 | Quest별 POLICY 설정 (기본 50 DUK) |
| Raffle (FQ) | 응모 시 (당첨 무관) | 응모권 1장당 5 DUK (POLICY 설정) |
| 서포트 이벤트 (SUP) | 응원 시 | 응원량별 비례 |
| 기억 게시 (MEM) | 기억 게시 + 좋아요 | POLICY 설정 |
| 일일 출석 | 매일 첫 로그인 | 5 DUK |
| 가입 보너스 | 신규 가입 | 1,000 DUK (POLICY 설정) |

> 적립 출처는 [CEB-BO-FQ-301] 응모권 거래 9종 코드와 유사하게 별도 코드 체계로 관리 (`DUK_QUEST_REWARD`, `DUK_RAFFLE_PARTICIPATION` 등).

### 6.5 소비 출처

- 부스팅 (Quest·Raffle 보상 부스트, POLICY 설정)
- 응모권 추가 구매 (DUK → 응모권 환산, 분석 리포트 §3.6.5 GP 정책 호환 가능)
- 독점 콘텐츠 잠금 해제 (INF 영역)
- BIVE 발행 가속 (BIVE 영역)

### 6.6 회원 통합 노출

분석 리포트 §3.2.2 + USR-201 v1.0 §4.4 정책:

- 회원 USR-201 [활동내역] 탭에 DUK 적립·소비 시계열 노출
- 회원 본인 앱(`[CEB-DUK-101] 덕력 랭킹`)에서 본인 DUK·시즌 랭킹 확인
