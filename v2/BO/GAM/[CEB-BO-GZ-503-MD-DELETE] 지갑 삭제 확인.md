## 1. 화면 개요

비활성 상태의 운영 지갑을 삭제하기 전 최종 확인하는 모달. 삭제된 지갑은 복구할 수 없으므로 신중한 확인이 필요하며, 삭제 후에도 해당 지갑이 처리한 과거 교환 내역([CEB-BO-GZ-501])과 GP 변동 내역([CEB-BO-GZ-601])은 그대로 유지된다. **활성 지갑은 [삭제] 버튼이 부모 화면([CEB-BO-GZ-503])에서 disabled** 상태이므로 본 모달은 비활성 지갑에서만 호출된다.

| 항목 | 값 |
|---|---|
| 화면 경로 | `/gamezone/exchange/wallets` (모달, 부모 화면 내 오픈) |
| Breadcrumb | 홈 › 게임존 › GP 교환소 › 지갑 관리 (모달) |
| 모달 제목 | "지갑을 삭제하시겠습니까?" |
| 진입 경로 | [CEB-BO-GZ-503] 지갑 관리의 비활성 지갑 행 [삭제] 버튼 |
| 접근 권한 | 부모 화면과 동일 ([관리자 > 권한관리] 설정) |
| 용도 | 비활성 지갑 영구 삭제 (조회 전용 + 확정 액션 2종) |

## 2. 화면 구성

### 2-1. 모달 헤더

| 위치 | 요소 | 동작 |
|---|---|---|
| 좌상단 | 모달 제목 (h2) | "지갑을 삭제하시겠습니까?" |
| 우상단 | 닫기 [X] 버튼 | 모달 닫기 (단축키 Esc, 취소와 동일 동작) |

### 2-2. 모달 본문

| 영역 | 내용 |
|---|---|
| 경고 문구 | "삭제된 지갑은 복구할 수 없습니다." (red-600 강조) |
| 삭제 대상 요약 | 라벨 · 값 2컬럼 (지갑 유형 / 지갑 주소) |

**삭제 대상 요약 필드**:

| # | 라벨 | 형식 |
|---|---|---|
| 1 | 지갑 유형 | "충전용" 또는 "출금용" |
| 2 | 지갑 주소 | 축약 표기 `0xXXXX...XXXX` (또는 전체 — 추가 검증 시 보강) |

### 2-3. 모달 푸터

| 버튼 | 위치 | 스타일 | 동작 |
|---|---|---|---|
| [취소] | 우하단 (삭제하기 좌측) | Secondary | 모달 닫기 (상태 변경 없음, 목록 갱신 없음) |
| [삭제하기] | 우하단 | Danger (red 배경) | 지갑 삭제 API 호출 → Toast + 모달 닫기 + 목록 갱신 + 활동 로그 |

**모달 닫기 트리거**: [X] / Esc / [취소] / 모달 외부 클릭(백드롭 — 추가 검증 시 보강)

## 4. 상태별 동작 분기

| 상황 | 동작 |
|---|---|
| 모달 오픈 | 부모 화면의 [삭제] 클릭 → 본 모달 오픈 (비활성 지갑만) |
| 활성 지갑 [삭제] | 부모 화면 [삭제] 버튼이 disabled — 본 모달 호출 불가 ([GZ-503 v1.6] §5-3) |
| [취소] / [X] / Esc / 백드롭 | 모달 닫기 + 상태 변경 없음 |
| [삭제하기] 클릭 | 로딩 표시 + API 호출 + 중복 클릭 방지 |
| 삭제 성공 | Toast "지갑이 삭제되었습니다." + 모달 닫기 + 부모 목록 갱신 (해당 행 제거) + 활동 로그 |
| 삭제 실패 | 에러 토스트 + 모달 유지 (재시도 가능) |
| 권한 없음 | 부모 화면에서 [삭제] 버튼 자체 미노출 |

## 5. 검증 규칙

### 5-1. 삭제 가능 조건

| 조건 | 검증 시점 |
|---|---|
| 지갑이 존재함 | 모달 오픈 전 (부모 화면) |
| 지갑이 **비활성 상태** | 부모 화면 [삭제] 버튼 활성/비활성 분기로 처리 ([GZ-503 v1.6] §5-3) |
| 관리자 삭제 권한 | API 호출 시 (부모 화면 권한 정책에 위임) |

### 5-2. 삭제 시 처리

| 영역 | 처리 |
|---|---|
| 지갑 본체 | 완전 삭제 (지갑 정보·잔액·[대표] 지정·활성 상태 모두 제거) |
| 과거 교환 내역 | 유지 ([CEB-BO-GZ-501]). 교환 내역의 지갑 유형·주소 표기는 그대로 |
| 과거 GP 변동 내역 | 유지 ([CEB-BO-GZ-601]) |

### 5-3. 활동 로그

`[CEB-BO-GZ-000] 공통 정책 §활동 로그` 정합:

| 액션 | 로그 메시지 |
|---|---|
| 지갑 삭제 | 지갑을 삭제했습니다. (유형: {유형}, 주소: {주소}) |

### 5-4. 권한

- 권한은 부모 화면 [CEB-BO-GZ-503]과 동일 ([관리자 > 권한관리] 설정)
- 쓰기 권한 없으면 부모 화면 [삭제] 버튼 미노출 → 본 모달 호출 불가
