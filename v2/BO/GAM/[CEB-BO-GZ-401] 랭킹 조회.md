## 1. 화면 개요

게임존 참여자들의 **누적 GP 기준 랭킹**을 조회하는 화면. 관리자는 TOP 10 카드와 전체 랭킹 테이블로 상위 사용자를 한눈에 파악하고, 닉네임 검색으로 특정 사용자를 빠르게 찾는다. 행 클릭으로 회원 상세 게임존 탭(CEB-BO-USR-203)으로 이동해 개별 사용자의 게임 이력을 확인한다.

| 항목 | 값 |
|---|---|
| 화면 경로 | `/gamezone/ranking` |
| Breadcrumb | 홈 › 게임존 › 랭킹 |
| 페이지 제목 | "랭킹" |
| 진입 경로 | 사이드바 "게임존 › 랭킹" |
| 접근 권한 | [관리자 > 권한관리]에 설정된 권한을 따름 |
| 용도 | 누적 GP 랭킹 조회·검색, 회원 상세 진입, 랭킹 설정 진입 |

> **통합 랭킹**: 본 화면은 Prediction Market과 Survival Trivia를 **통합한 누적 GP** 기준으로 랭킹을 산출한다. 화면 상단 안내 띠에 동일 문구 노출.

## 5. 검증 규칙

### 5-1. 누적 GP 집계

```text
누적 GP = (보상 GP + 환급 GP) − (참여 GP + 부스팅 GP)
```

| 항목 | 설명 | 부호 |
|---|---|---|
| 보상 GP | 정답·생존 게임의 보상 | + |
| 환급 GP | 게임 취소·자동 환급 시 환급액 | + |
| 참여 GP | 게임 참여 시 지불한 GP | − |
| 부스팅 GP | 부스팅으로 지불한 GP | − |

> **게임유형별 구성 요소**
> - **Prediction Market**: 보상·환급·참여·부스팅 4종 모두 적용
> - **Survival Trivia**: 보상·참여 2종만 적용 (부스팅·환급 없음 — [CEB-BO-GZ-000] 공통 정책)
> - 누적 GP는 모든 게임유형을 통합하여 산정

### 5-2. 승률 계산

```text
승률 = (성공 게임 수 ÷ 참여 완료 게임 수) × 100
```

| 게임유형 | 성공 기준 |
|---|---|
| Prediction Market | 사용자가 선택한 선택지가 최종 결과와 일치 |
| Survival Trivia | 10스테이지 모두 완료(최종 생존) |

- 참여 완료 게임 = 결과가 확정된 게임 (대기·진행중·취소 상태 제외)
- 표시 형식: 소수 1자리 (예: `23.0%`) — 운영 정합

### 5-3. 참여 횟수 (운영 정합)

| 항목 | 값 |
|---|---|
| 라벨 | **"참여 횟수"** (v1.2 "참여 게임 수" → 운영 "참여 횟수" 정정) |
| 집계 기준 | 결과가 확정된 게임 (완료 상태) |
| 제외 | 대기·모집중·진행중·취소 상태 |
