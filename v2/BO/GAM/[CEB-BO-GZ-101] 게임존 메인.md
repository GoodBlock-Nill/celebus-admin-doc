### 5-1. 게임 유형별 카드 노출 조건 (보강)

게임 관리 섹션의 카드(PM, ST 등)는 **해당 게임 유형의 서비스 활성화 상태**에 따라 노출. 여기서 Active/Inactive는 개별 게임의 진행 상태가 아니라 **게임 유형 자체가 서비스에 오픈되었는지 여부**를 의미한다.

- Phase 1: PM만 Active → PM 카드 1개만 노출
- Phase 2: ST Active 전환 → PM, ST 2개 가로 배치
- 향후 신규 게임 유형 추가 시 Active 설정된 유형만 그리드에 자동 추가

### 5-2. 항목별 집계 기준

| 항목 | 집계 기준 |
|---|---|
| 진행중 게임 (개) | 상태 Active 전체 게임 수 |
| 결과 입력 필요 (개) | 상태 Pending 전체 게임 수 |
| 오늘 GP 충전 | 오늘 CELB → GP 교환 완료 GP 합계 |
| 오늘 GP 출금 | 오늘 GP → CELB 교환 완료 GP 합계 |
| 진행중 게임 (PM/ST) | 해당 유형의 Active 게임 수 |
| 결과 입력 필요 (PM) | PM의 Pending 게임 수 |
| 게시 대기 (ST) | ST의 Ready 게임 수 (시작 일시가 미래) |
| 오늘 참여자 (PM/ST) | 오늘 해당 유형 게임에 참여한 유저 수 |
| TOP 10 공개 | 랭킹 설정의 TOP 10 공개 여부 (ON/OFF) |
| TOP 1 | 누적 GP 기준 1위 유저 닉네임 (없으면 "-") |
| TOP 1 누적 GP | 누적 GP 기준 1위 유저의 누적 GP |
| 오늘 GP 변동 건수 | 오늘 발생한 모든 GP 증감 내역 건수 |
