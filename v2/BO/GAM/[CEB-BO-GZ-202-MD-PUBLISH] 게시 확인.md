## 1. 모달 개요

게시대기(Ready) 게임을 **즉시 게시**하여 진행중(Active)으로 전이시키는 확인 모달. 게시 즉시 유저에게 노출되고 참여가 시작된다.

| 항목 | 값 |
|---|---|
| 트리거 | [CEB-BO-GZ-202]의 [게시하기] 버튼 (Ready 상태) |
| 표시 조건 | 게임 상태가 Ready(게시대기)일 때 |
| 영향 범위 | 게임 상태 → Active / 투표 시작일시 자동 설정 (게시 시점) / 유저 앱에 노출 |
| 접근 권한 | [관리자 > 권한관리] 설정 (쓰기 권한) |

## 4. 상태별 동작 분기

| 상황 | 동작 |
|---|---|
| [게시하기] 클릭 | Active 전이 + 투표 시작일시 = 현재 서버 시각 + [202] 게임 상세 갱신 (액션 [수정하기] [강제 종료]로 변경) |
| [취소] / ESC / 배경 | 모달 닫기 (Ready 유지) |
| 중복 클릭 | 첫 클릭 후 [게시하기] disabled |
| API 실패 | 토스트 "게시에 실패했습니다." + 모달 유지 |
| 세션 만료 | 로그인 화면 리다이렉트 (운영 실측 — 재로그인 후 다시 진행 필요) |
| 트리거 조건 위반 | [게시하기] 버튼 자체 미노출 (Ready가 아니면 진입 불가) |
