## 2. 화면 구성

### 2-1. 모달 헤더

### 2-2. 모달 본문 (2컬럼 라벨·값)

10개 필드를 라벨(좌, 회색) · 값(우, 검정) 2컬럼 구성으로 표시. 모든 필드는 조회 전용.

| # | 라벨 | 값 형식 | 비고 |
|---|---|---|---|
| 1 | Txid | 전체 hex 텍스트 + 복사 아이콘 | 블록체인 트랜잭션 해시 (`0x` + 64자). 클릭 시 BSCScan 새 탭(`https://bscscan.com/tx/{txid}`) — 운영 BO 동작 추가 검증 시 보강 |
| 2 | 교환일시 | YYYY.MM.DD HH:mm (운영 표기. 명세 v1.1 "HH:mm:ss" 정정) | 거래 완료/실패 시각 |
| 3 | 닉네임 | 텍스트 또는 "-" | 클릭 시 [CEB-BO-USR-203] 회원 상세 게임존 탭 이동 (UID 전달) — 운영 BO 클릭 동작 추가 검증 시 보강 |
| 4 | 지갑주소 | 전체 hex 텍스트 + 복사 아이콘 | BSC 지갑주소(`0x` + 40자). 클릭 시 BSCScan 새 탭(`https://bscscan.com/address/{address}`) — 운영 BO 동작 추가 검증 시 보강 |
| 5 | 교환 방향 | Badge | **"GP로 충전"** (파란 톤) / "GP로 출금"(회색 톤). 부모 테이블 "GP 충전/GP 출금" Badge와 표기 차이 — 모달은 "GP로" 접속사 포함 (운영 실측) |
| 6 | GP 수량 | "{N} GP" | 천 단위 콤마 |
| 7 | CELB 수량 | "{N} CELB" | 천 단위 콤마 |
| 8 | 적용 비율 | "1 CELB = 1 GP" 또는 "1 GP = 1 CELB" | 운영 표기 형태 (명세 v1.1 "1 GP = X CELB" 단일 표기 정정). 거래 시점에 적용된 비율 |
| 9 | 교환 전 GP 잔액 | "{N} GP" | 거래 직전 사용자 GP 잔액 |
| 10 | 교환 후 GP 잔액 | "{N} GP" | 거래 직후 사용자 GP 잔액 (실패 시 = 교환 전 잔액) |
| 11 | 상태 | Badge | **"성공"**(초록) — 실패 발생 시 별도 라벨 (운영 실패 케이스 추가 검증 시 보강. 명세 v1.1 "완료" → "성공" 정정) |
| 12 | 실패 사유 | 텍스트 또는 미노출 | 성공 시 미노출(또는 "-"). 실패 시 사유 메시지 (예: "잔액 부족") |

> 운영 BO 실측 시점(2026-05-26)에는 성공 케이스만 확인됨. 실패 케이스 발생 시 §2-2 #11·#12·§5-2 추가 검증 후 보강.

## 5. 검증 규칙

### 5-1. 교환 전/후 GP 잔액 정합

| 거래 유형 | 정합 공식 |
|---|---|
| GP 충전 (CELB → GP) | 교환 후 GP 잔액 = 교환 전 GP 잔액 + GP 수량 |
| GP 출금 (GP → CELB) | 교환 후 GP 잔액 = 교환 전 GP 잔액 − GP 수량 |
| 실패 | 교환 후 GP 잔액 = 교환 전 GP 잔액 (변동 없음) |

### 5-2. 실패 사유 (운영 BO 실패 케이스 추가 검증 시 보강)

명세 v1.1 시점의 사유 코드 표는 백엔드 식별자로 본 화면 정합 외. 사용자 메시지(한국어)만 운영 정합으로 보강 예정:

| 운영 메시지 (예시) |
|---|
| 잔액 부족 |
| 1일 한도 초과 |
| 최소 교환 수량 미만 |
| 최대 교환 수량 초과 |
| 교환 기능 비활성화 |
| 시스템 오류 |

> 실패 사유 메시지의 정확한 운영 표기는 실패 케이스 발생 시 chrome-devtools 캡처 후 보강.

### 5-3. 적용 비율

- 거래 시점에 적용된 GP ↔ CELB 비율이 기록되며, [CEB-BO-GZ-502] 교환 설정 변경 후에도 과거 거래는 당시 비율 유지 (소급 적용 없음)
- 운영 표기 형태: "1 CELB = X GP"(충전 시 의미 부각) / "1 GP = X CELB"(출금 시 의미 부각)

### 5-4. 권한 및 활동 로그

- 권한은 부모 화면 [CEB-BO-GZ-501]과 동일 ([관리자 > 권한관리] 설정)
- 모달 조회 자체는 활동 로그 미기록 (조회 로그 정책 — [CEB-BO-GZ-000] 정합)
- 모달은 조회 전용 — 수정·취소 액션 없음
