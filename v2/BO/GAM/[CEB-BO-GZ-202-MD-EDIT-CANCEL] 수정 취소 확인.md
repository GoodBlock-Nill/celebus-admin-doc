## 1. 모달 개요

게임 수정 화면에서 [취소] 클릭 시 작성 중 데이터 폐기를 경고하는 **확인 모달**. **변경 유무 무관 항상 표시** (운영 BO 실측 정합).

| 항목 | 값 |
|---|---|
| 트리거 | [CEB-BO-GZ-202-EDIT]의 [취소] 버튼 |
| 표시 조건 | 변경 유무 무관 항상 표시 (운영 실측) |
| 영향 범위 | [취소하기] 선택 시 입력값 폐기 + [202] 게임 상세 이동 |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |
