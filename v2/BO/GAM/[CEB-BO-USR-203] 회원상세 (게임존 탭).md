## 1. 화면 개요

회원상세 페이지 내 **"게임존" 탭**으로, 특정 회원의 게임존 활동 전체(GP 보유·획득·사용, 게임 참여 이력, 보상·환급, GP 변동)를 한눈에 파악하는 **조회 전용** 화면.

| 항목 | 값 |
|---|---|
| 화면 경로 | `/members/{userId}?tab=gamezone` |
| Breadcrumb | 홈 › 회원 › 회원 상세 |
| 페이지 제목 | "회원상세" (공통, h1) + "{유저명}님의 상세 정보를 관리합니다." 안내 |
| 진입 경로 | [CEB-BO-USR-201] 회원상세 페이지에서 "게임존" 탭 클릭, 또는 [CEB-BO-GZ-401] 랭킹·[CEB-BO-GZ-202] 게임 참여내역·[CEB-BO-GZ-601] GP 변동 내역에서 닉네임 클릭 |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 (회원 읽기 + 게임존 읽기) |
| 용도 | 회원별 게임존 활동 모니터링, 분석, CS 대응 |

## 2. 화면 구성

회원상세 페이지의 9개 탭 중 **3번째 탭 "게임존"**. 탭 구조는 [USR-201]에 정의되어 있으며 본 문서는 게임존 탭 본문만 다룬다.

전체 탭 (운영 정합): 기본정보 / 참여내역 / **게임존** / BIVE / 프로젝트 / Fans / 티켓 / 지갑 / 활동내역

### 2-1. GP 요약 (Summary Cards, 가로 4열)

| Card | Label | Value | 보조 텍스트 |
|---|---|---|---|
| 1 | 현재 GP 잔액 | {숫자} GP | — |

| Card | Label | Value | 설명 |
