## 1. 모달 개요

진행중/결과대기 게임을 **강제 종료**하는 확인 모달. 강제 종료 시 참여 GP·부스팅 GP 전액 환급 + 보상 미지급 처리. 이 작업은 되돌릴 수 없다.

> ⚠️ **흐름 분기**: 본 모달은 **[강제 종료]** 액션 전용 (진행중·결과대기 상태에서 트리거). Pending → Closed 전이는 [강제 종료]가 아닌 **[결과 입력]** ([CEB-BO-GZ-202-MD-RESULT]) 액션이며, Closed → Ended 전이는 [보상 지급] ([CEB-BO-GZ-202-MD-REWARD]). 본 모달 확정 시 종료(Ended) 상태로 직접 전이 + 게임 리스트 필터에서 "강제종료"·"환불완료" 별도 옵션으로 노출 (게임 상세 헤더 배지는 "종료" 단일).

| 항목 | 값 |
|---|---|
| 트리거 | [CEB-BO-GZ-202]의 [강제 종료] 버튼 (Active/Pending 상태) |
| 표시 조건 | 게임 상태가 Active 또는 Pending일 때 |
| 영향 범위 | 게임 상태 → Ended / 모든 참여자 환급 / 보상 미지급 |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 (쓰기 권한) |
