## 1. 모달 개요

게임을 **복구 불가능하게 삭제**하는 확인 모달. Draft(임시저장) / Ready(게시대기) 상태에서만 노출.

| 항목 | 값 |
|---|---|
| 트리거 | [CEB-BO-GZ-202]의 [삭제하기] 버튼 (Draft/Ready 상태) |
| 표시 조건 | 게임 상태가 Draft 또는 Ready일 때 |
| 영향 범위 | 게임 영구 삭제 (복구 불가) + 게임 리스트로 이동 |
| 접근 권한 | [관리자 > 권한관리] 설정 (쓰기 권한) |
