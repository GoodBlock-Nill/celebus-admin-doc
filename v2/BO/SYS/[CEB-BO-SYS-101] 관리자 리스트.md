## 1. 화면 목적

CELEBUS 백오피스 운영자(관리자) 통합 조회·초대 화면. **SUPER 전용** 영역. 본 화면에서 관리자 라이프사이클(Active/Pending/Locked/Inactive)을 모니터링하고 신규 관리자를 초대하여 권한 그룹을 부여한다.

### 주요 기능

- 등록된 전체 관리자 리스트 조회 + 상태별 카운터
- 권한 그룹 + 관리자명 검색·필터
- 신규 관리자 초대 진입
- 관리자 상세 진입 (권한 부여·회수·만료 관리)

### 진입 경로

- 사이드 메뉴 > 관리자 > 관리자 리스트

| Locked | 90일 비활성 자동 잠금 ([CEB-BO-000] §3.2) |
| Inactive | 비활성화된 관리자 (제재·만료) |

## 4. 규칙·로직

### 4.1 권한별 분기

[CEB-BO-005] §4 적용:

| 권한 | 화면 진입 | 조회 | [관리자 초대] | Row 클릭 |
| ---- | --------- | ---- | ------------- | -------- |
| SUPER | ✅ | ✅ (전체) | ✅ | ✅ (전체) |
| OPERATOR | ❌ | — | — | — |
| POLICY | ❌ | — | — | — |
| EXTERNAL_ARTIST | ❌ | — | — | — |

> SYS 영역 자체가 SUPER 전용. OPERATOR 이하는 사이드 메뉴 [관리자] 비활성. 자기 정보 수정은 SYS-201 직접 URL로 접근.

### 4.2 정렬·필터

| 기본 정렬 | 등록일시 내림차순 (최신순) |
| --- | --- |

- 상태 + 권한 그룹 + 관리자명 검색 = AND 조합
- 관리자명 검색: 닉네임/이메일 통합 부분 일치

### 4.3 계정 상태 정의 ([CEB-BO-000] §3.2)

| 상태 | 의미 | 진입 가능 |
| --- | --- | --- |
| Active | 활성 (정상 로그인 가능) | ✅ |
| Pending | 초대 메일 발송 후 인증 대기 | ❌ (인증 후 Active 전환) |
| Locked | 90일 비활성 자동 잠금 | ❌ (SUPER 재승인 필요) |
| Inactive | 비활성화 (제재·만료·SUPER 회수) | ❌ |

### 4.4 권한 그룹 표시

[CEB-BO-005] §7 (v2 하이브리드 정책):

- 4단계 기본 프리셋: SUPER / OPERATOR / POLICY / EXTERNAL_ARTIST
- 운영자 정의 그룹: 12영역 × R/W/D 매트릭스로 커스터마이징 (예: "콘텐츠 관리자")
- 한 관리자는 1개 권한 그룹에만 속함 (다중 그룹 불가)
