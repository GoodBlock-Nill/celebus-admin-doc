## 1. 화면 목적

CELEBUS 백오피스 권한 그룹 통합 관리. **v2 하이브리드 정책** (사용자 결정 2026-05-06): 4단계 기본 프리셋(SUPER/OPERATOR/POLICY/EXTERNAL_ARTIST) + 운영자가 12영역 × R/W/D 매트릭스로 커스터마이징하는 추가 그룹 운영. **SUPER 전용** 영역.

### 주요 기능

- 등록된 권한 그룹 리스트 조회 + 상태별 카운터
- 권한 그룹 검색·필터
- 신규 권한 그룹 생성 진입 (커스텀 그룹)
- 권한 그룹 상세 진입 (3탭 — 기본정보/권한설정/관리자)

### 진입 경로

- 사이드 메뉴 > 관리자 > 권한관리

## 2. UI 설명

### 2.2 상태 대시보드 (3카드)

| 카드 | 의미 |
| --- | --- |

| Inactive | 비활성 그룹 (신규 부여 차단, 기존 매핑은 유지) |

> 운영 데이터(2026-05-05): 전체 2 / Active 2 / Inactive 0 — Super Admin (수정불가, 9명) + Viewer (기본값, 0명).

## 4. 규칙·로직

### 4.1 권한별 분기

[CEB-BO-005] §4 적용:

| 권한 | 화면 진입 | 조회 | [권한 생성] | Row 클릭 |
| ---- | --------- | ---- | ----------- | -------- |
| SUPER | ✅ | ✅ | ✅ | ✅ |
| OPERATOR/POLICY/EXTERNAL_ARTIST | ❌ | — | — | — |

### 4.2 그룹 유형 정책 ([CEB-BO-005] §7 하이브리드)

| 유형 | 예시 | 설명 |
| --- | --- | --- |
| **프리셋 (4단계)** | Super Admin / Operator / Policy / External Artist | v2 BO-000 §3 정의. **수정·삭제 불가** (시스템 기본 그룹) |
| **기본값** | Viewer (조회 전용) | 운영팀이 사전 정의한 기본값. **삭제 불가**, 수정 가능 |
| **운영자 정의** | "콘텐츠 관리자" / "BIVE 발행 담당자" 등 | SUPER가 12영역 × R/W/D 매트릭스로 커스터마이징. **수정·삭제 가능** |

### 4.3 정렬·필터

| 기본 정렬 | 그룹 유형 우선 (프리셋 → 기본값 → 운영자 정의) → 등록일 최신순 |
| --- | --- |

- 상태 + 권한명 검색 = AND 조합
- 권한명 검색: 부분 일치

### 4.4 그룹 상태 정의

| 상태 | 의미 |
| --- | --- |
| Active | 활성 (관리자에게 신규 부여 가능) |
| Inactive | 비활성 (신규 부여 차단). 기존 매핑된 관리자는 권한 유지 (운영자가 다른 그룹으로 재매핑 후 완전 비활성화 가능) |

### 4.5 [권한 생성] 가능 범위

- 운영자 정의 그룹만 생성 가능
- 4단계 프리셋·기본값 신규 생성 불가 (시스템 보장)
