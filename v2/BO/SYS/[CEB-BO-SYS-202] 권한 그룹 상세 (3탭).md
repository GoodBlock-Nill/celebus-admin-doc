## 1. 화면 목적

개별 권한 그룹의 상세 정보 조회·수정·관리. **3탭 구조** (v1 유지): 기본정보 / 권한설정 / 관리자. 운영자가 12영역 × R/W/D 매트릭스로 권한을 커스터마이징하고, 해당 그룹에 매핑된 관리자를 통합 관리.

### 주요 기능

- 3탭별 권한 그룹 정보 조회 (기본정보 / 권한설정 / 관리자)
- 권한 매트릭스 수정 (운영자 정의 그룹만)
- 관리자 매핑 추가·제거

### 진입 경로

- [CEB-BO-SYS-102] 권한 그룹 리스트 > Row 클릭

URL: `/admins/permissions/{id}?tab=info` (Default tab=info)

## 2. UI 설명

### 2.1 상단 네비게이션·헤더

- Breadcrumb: `홈 > 관리자 > 권한관리 > {권한명}`
- 뒤로가기 (←) → [CEB-BO-SYS-102]
- 타이틀: `{권한명}` + 그룹 유형 Badge (프리셋·기본값·운영자 정의) + 상태 Badge
- 우측 상단 액션: [수정하기] (Primary, 운영자 정의 그룹만) / [삭제하기] (Danger, 운영자 정의 그룹만 + 매핑 관리자 0명일 때)

### 2.2 탭 메뉴 (3탭, 상호 배타)

| # | 탭명 | URL | 설명 |

## 4. 규칙·로직

### 4.1 권한별 분기

[CEB-BO-005] §4 적용 — 본 화면은 SUPER 전용:

| 권한 | 화면 진입 | 조회 | [수정] | [삭제] | 매트릭스 편집 | 관리자 매핑 |
| ---- | --------- | ---- | ------ | ------ | -------------- | ----------- |
| SUPER | ✅ | ✅ | ✅ (운영자 정의만) | ✅ (운영자 정의 + 매핑 0명) | ✅ (운영자 정의만) | ✅ |
| 그 외 | ❌ | — | — | — | — | — |

### 4.2 그룹 유형별 정책

| 유형 | 명칭 수정 | 매트릭스 수정 | 상태 변경 | 삭제 |
| --- | --- | --- | --- | --- |
| **프리셋 (4단계)** | ❌ | ❌ | ❌ | ❌ |
| **기본값** (Viewer) | ❌ | ✅ (제한적) | ✅ | ❌ |
| **운영자 정의** | ✅ | ✅ | ✅ | ✅ (매핑 관리자 0명일 때) |

### 4.3 매트릭스 수정 시 활동 로그

각 영역 × R/W/D 변경마다 별도 행 기록:

```
권한 그룹 '{권한명}' 매트릭스 수정 — {영역} {R/W/D} {ON/OFF} (사유: {요약})
```

### 4.4 그룹 삭제 가드레일

- 운영자 정의 그룹 + 매핑 관리자 0명일 때만 삭제 가능
- [삭제하기] → [CEB-BO-SYS-202-MD-DELETE] 모달 → 사유 입력 필수 + 2FA 재인증
- 삭제 시 매트릭스 데이터 영구 제거 (활동 로그는 보존)

### 4.5 관리자 매핑 정책

- 한 관리자는 1개 그룹에만 속함 (다중 매핑 불가)
- [그룹 추가] 시 기존 그룹에서 자동 제거 + 신규 그룹 매핑
- 매핑 변경 시 해당 관리자의 활성 세션 갱신 (즉시 새 권한 반영)
