## 1. 화면 목적

운영자가 **회원의 덕력 변동 이력을 적립·사용 영역으로 나누어 조회**하는 화면이다. 그룹·기간·회원 조건을 조합해 누구든 빠르게 변동 기록을 추적할 수 있다.

- **누가**: 덕력 운영·CS 권한이 있는 백오피스 관리자
- **왜**: 회원의 보상 적립 누락·사용 분쟁 등을 추적하고, 운영 의사결정 근거 확보
- **결과**: 조건에 맞는 변동 행을 조회하고 회원 상세로 즉시 진입 가능

---

## 2. UI 설명

### 2.1 화면 레이아웃

```
[탭 메뉴]  [랭킹 시즌 설정] [덕력 랭킹] [덕력 내역]

────────────────────────────────────────────────────────

[내부 탭]  [획득 내역] [사용 내역]      ← 한 번에 한 영역

[필터 바]
그룹: [전체 ▾]   기간 시작: [2026-04-26]   기간 종료: [2026-05-26]   회원 검색: [...]   [초기화]

(기간 시작이 종료보다 늦으면 안내 메시지 표시)

[요약 카드]
| 필터 결과 | 획득(또는 사용) 합계  |
|---|---|
| 1,234건 | +12,345 DUK |

[내역 테이블]
| 일시 | 회원 | 그룹 | 출처 | 변동 | 잔액 | 시즌 |
|---|---|---|---|---|---|---|
| 2026.05.26 17:30 | 닉네임A | V01D | Quest 완료 | +75 DUK | 1,250 DUK | V01D 2026 시즌 |
| ... |

[페이지네이션]
```

### 2.2 구성요소

| 영역 | 요소 | 설명 |
| --- | --- | --- |
| 내부 탭 | [획득 내역] / [사용 내역] | 한 번에 한 영역만 노출. 첫 진입 시 [획득 내역]. 탭 전환 시 각 탭의 필터·페이지 상태는 별도 보존 |
| 필터 | 그룹 선택 메뉴 | "전체" + 활성 그룹 5종. 기본 "전체" |
| 필터 | 기간 시작·종료 | 날짜 선택. 기본 최근 30일 (오늘 기준) |
| 필터 | 회원 검색 | 닉네임 부분 일치 |
| 필터 | [초기화] 버튼 | 현재 탭의 필터만 기본값 복원 (다른 탭에 영향 없음) |
| 요약 | 필터 결과 카드 | 조회 결과 건수 |
| 요약 | 합계 카드 | 획득 탭 = + 합계 / 사용 탭 = − 합계 (DUK 단위) |
| 리스트 | 내역 테이블 | 일시·회원·그룹·출처·변동·잔액·시즌 7컬럼 |
| 하단 | 페이지네이션 | 20건/페이지 |

### 2.3 내역 테이블 컬럼

| 컬럼 | 표시 |
| --- | --- |
| 일시 | 변동 발생 일시 (YYYY.MM.DD HH:mm) |
| 회원 | 회원 닉네임 — 클릭 시 회원 상세 [CEB-BO-USR-201]로 새 창 이동 |
| 그룹 | 변동이 귀속된 아티스트 그룹 |
| 출처 | 변동 발생 활동 — 획득 탭: 출석 체크·응모권 사용·일일 미션·게임 참여·SNS 공유·Quest 완료·기억저장소 업로드·디지털 굿즈 획득 / 사용 탭: 서포트 응원·독점 콘텐츠 해금 |
| 변동 | 획득은 +N DUK (초록) / 사용은 −N DUK (분홍) |
| 잔액 | 해당 그룹 기준 변동 후 잔액 |
| 시즌 | 해당 시점 진행 중 시즌명 (시즌이 없으면 "—") |

### 2.4 시각적 스타일

- 획득 탭: + 변동 초록 / 사용 탭: − 변동 분홍
- 회원 닉네임 hover 시 강조 + 새 창 이동 안내
- 빈 상태: "조건에 일치하는 내역이 없습니다."

## 3. 기능 설명

운영자가 이 화면에서 할 수 있는 동작:

1. **획득/사용 탭 전환** — 한 번에 한 영역
2. **그룹·기간·회원 검색 필터링** — 필터 조합으로 빠른 조회
3. **필터 초기화** — 현재 탭의 필터만 복원
4. **회원 상세 진입** — 닉네임 클릭 시 새 창

---

## 4. 로직·플로우

### 4.1 화면 진입 → 기본 표시

1. 첫 진입 시 [획득 내역] 탭 활성화
2. 필터: 그룹 "전체" + 기간 최근 30일 + 회원 검색 빈 값
3. 일시 내림차순(최신순)으로 변동 행 표시

### 4.2 내부 탭 전환

1. [사용 내역] 클릭 시 [획득 내역] 화면을 숨기고 [사용 내역] 표시
2. 두 탭의 필터·페이지·검색어 상태는 각각 보존 (전환 후 복귀 시 직전 상태 유지)

### 4.3 필터 변경

1. 그룹·기간·회원 검색 중 하나라도 변경 시 즉시 결과 갱신
2. 페이지는 1페이지로 복귀

### 4.4 회원 닉네임 클릭

1. 새 창으로 회원 상세 [CEB-BO-USR-201] 진입
2. 현재 내역 화면 상태는 유지

## 5. 검증·예외 상황

### 5.1 빈 상태

| 상황 | 표시 |
| --- | --- |
| 조회 결과 0건 | "조건에 일치하는 내역이 없습니다." + 페이지네이션 숨김 |

### 5.2 기간 검증

| 상황 | 안내 |
| --- | --- |
| 기간 시작이 종료보다 늦음 | 필터 바 하단에 안내 문구 "기간 시작은 종료보다 이전이어야 합니다." 표시. 결과 테이블은 0건으로 표시 |

### 5.3 권한별 동작

- 본 화면은 조회 전용 — 모든 운영자 권한에 R 권한 부여
- 외부 협력사 권한자는 본인 매핑 소속사 산하 그룹의 변동만 조회 가능

### 5.4 활동 로그

조회 화면 — 활동 로그 대상 아님.
