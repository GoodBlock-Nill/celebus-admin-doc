## 1. 화면 목적

CELEBUS에 등록된 K-pop 아티스트 그룹을 통합 조회·관리하는 화면. 운영자는 그룹 상태(활성/비활성)를 모니터링하고 신규 그룹 등록·포지션 설정으로 진입한다. 개별 그룹은 [CEB-BO-ART-301] 그룹 상세로 이동한다.

### 주요 기능

- 등록된 전체 그룹 리스트 조회 + 상태 카운터(전체/Active/Inactive)
- 상태 필터·검색(그룹명·멤버명)
- 신규 그룹 생성 진입
- 포지션 설정 진입
- 그룹 상세 진입

### 진입 경로

- 사이드 메뉴 > 아티스트 > 그룹 리스트

## 2. UI 설명

### 2.1 상단 영역

- Breadcrumb: `홈 > 아티스트 > 그룹 리스트`
- 화면 타이틀: 그룹 리스트
- 우측 상단 액션: [포지션 설정] (Secondary) / [그룹 생성] (Primary)

### 2.2 상태 카운터 (3개)

| 카운터 | 의미 |
| --- | --- |
| 전체 | 등록된 전체 그룹 수 |
| Active | 활성 그룹 수 (앱 노출) |
| Inactive | 비활성 그룹 수 (앱 미노출) |

- 각 카운터 클릭 시 해당 상태로 필터링된다.

### 2.3 검색·필터 영역

| 구분 | 요소 | 타입 | 설명 |
| ---- | ------ | ---- | ---- |
| 필터 | 상태 | Dropdown | 상태(전체) / Active / Inactive |
| 검색 | 검색 대상 | Dropdown | 그룹명 / 멤버명 |
| 입력 | 검색어 | Text Input | 선택한 검색 대상으로 부분 일치 검색 (입력 안내: "그룹명 입력") |
| 버튼 | 초기화 | Dark Button | 필터·검색어 리셋 |

> 별도 정렬 Dropdown은 없으며, 정렬 기준은 업데이트 일시 내림차순 고정이다(§4.2).

### 2.4 목록 영역 (테이블)

| 컬럼 | 데이터 예시 | 설명 |
| ------ | ----------- | ---- |
| 상태 | Active / Inactive | 상태 뱃지 |
| 그룹명 | "밀짚모자 일당" | 한국어 우선 표시 (다국어 KO/EN/JA) |
| 멤버 수 | 10 | 그룹에 소속된 멤버 수 |
| 팔로워 수 | 128,430 | 앱에서 해당 그룹을 팔로우 중인 회원 수 (천 단위 콤마) |
| 탐색 노출 | 토글(켬/끔) | 앱 아티스트 탐색 화면 노출 여부 — 토글 즉시 변경(§4.4) |
| 업데이트 일시 | 2026.05.27 17:21 | YYYY.MM.DD HH:mm (KST) |

- Row 클릭 → [CEB-BO-ART-301] 그룹 상세 (기본정보 탭) 이동
- 탐색 노출 토글 클릭은 행 이동과 분리된다(토글만 동작, 상세 미진입).
- 페이지네이션: 하단 중앙

> 시각적 스타일은 피그마 참조

## 4. 규칙·로직

### 4.1 권한

- 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.
- 그룹 생성·포지션 설정은 쓰기 권한 보유 그룹만 노출된다. 탐색 노출 토글 변경도 쓰기 권한 보유 그룹만 가능하다.

### 4.2 정렬·필터

- 기본 정렬: 업데이트 일시 내림차순 (최신순). 사용자 변경 불가.
- 상태 필터 + 검색어 = AND 조합.
- 검색: 검색 대상(그룹명/멤버명) 선택 후 해당 항목으로 부분 일치. 그룹명 검색은 한국어·영어·일본어 통합.

### 4.3 상태 정의

| 상태 | 의미 | 앱 노출 |
| --- | --- | --- |
| Active | 활성 그룹 | 노출 |
| Inactive | 비활성 그룹 | 미노출 |

### 4.4 탐색 노출 정책

목록 [탐색 노출] 컬럼 토글로 앱 [CEB-ART-201] 아티스트 탐색 화면 노출 여부를 그룹 단위로 제어한다.

| 항목 | 정책 |
| --- | --- |
| 노출 토글 | 그룹 단위 켬/끔. 토글 즉시 저장·반영 + 완료 토스트 (별도 확인 모달 없음) |
| 기본값 | 신규 그룹은 탐색 노출 "켬"으로 등록 |
| 앱 노출 조건 | **상태 Active + 탐색 노출 켬** 인 그룹만 앱 아티스트 탐색(추천·전체)에 노출 |
| Inactive 그룹 | 탐색 노출 켬이어도 앱 미노출 (상태가 우선) |
| 활성 상태와의 관계 | 탐색 노출은 활성 상태와 별개 플래그. 활성 그룹이라도 탐색에서 숨길 수 있음 |

## 6. 활동 로그

목록 조회 자체는 로그를 기록하지 않는다. 단, 탐색 노출 토글 변경은 로그를 기록한다. 그 외 그룹 단위 액션은 [CEB-BO-ART-301]·[CEB-BO-ART-201-Create]에서 발생한다.

| 액션 | 로그 메시지 (한국어) |
| --- | --- |
| 탐색 노출 변경 | 그룹 '{그룹명}'의 탐색 노출을 {켬/끔}(으)로 변경했습니다. |
