## 1. 화면 목적

운영자가 **아티스트 그룹별 랭킹 시즌을 등록하고 관리**하는 화면이다. 한 화면에서 모든 그룹의 시즌 현황을 확인하고, 새 시즌을 생성하거나 진행 중인 시즌의 상세를 확인할 수 있다.

- **누가**: 덕력 운영 권한이 있는 백오피스 관리자
- **왜**: 그룹별로 1년 단위 랭킹 시즌을 운영해야 회원 덕력 적립·정산·보상 지급 흐름을 시작할 수 있음
- **결과**: 시즌이 등록되면 회원 앱의 덕력 랭킹에 노출되고, 매월 말일 자동 정산이 시작됨

### 2.2 구성요소

| 영역 | 요소 | 설명 |
| --- | --- | --- |
| 상단 안내 | 안내 문구 | "그룹별 랭킹 시즌을 생성합니다. 시즌을 클릭하면 상세 페이지로 이동하여 월별 보상을 설정할 수 있습니다. 같은 그룹에 진행 중 시즌은 1개만 운영됩니다." |
| 상단 액션 | [+ 신규 시즌] 버튼 (Primary) | 시즌 생성 모달 [CEB-BO-ART-401-MD-SEASON] 열기 |
| 필터 | 그룹 선택 메뉴 | "전체" + 활성 그룹 5종(V01D·iKON·MADEIN·CELEBUS·언더라이트). 기본 "전체" |
| 필터 | 결과 건수 | 현재 필터 조건의 시즌 개수 표시 |
| 리스트 | 시즌 테이블 | 그룹·시즌명·기간·상태 4컬럼. 시즌 행 클릭 시 시즌 상세 페이지로 이동 |
| 하단 | 페이지네이션 | 20건/페이지 |

### 2.3 시즌 테이블 컬럼

| 컬럼 | 표시 |
| --- | --- |
| 그룹 | 그룹 이름 (한국어 표시명) |
| 시즌명 | 운영자가 입력한 시즌 이름 (1~50자) |
| 기간 | 시작일시 ~ 종료일시 (YYYY.MM.DD HH:mm) |
| 상태 | 시즌 상태 배지 — 예정(회색) / 진행중(초록) / 종료(검정) |

> 시즌 행 자체가 클릭 가능 영역. 별도 [수정]·[종료] 버튼은 행에 노출하지 않으며, 시즌 상세 페이지 상단에서 제공한다.

### 2.4 시각적 스타일

- 상태 배지: 예정(회색 배경) / 진행중(초록 배경) / 종료(검정 배경)
- 행 hover 시 강조 + 마우스 커서 변화 (클릭 가능 안내)
- 빈 상태: "등록된 시즌이 없습니다."

## 3. 기능 설명

운영자가 이 화면에서 할 수 있는 동작:

1. **시즌 목록 조회** — 모든 그룹의 시즌을 한 화면에서 확인
2. **그룹별 필터링** — 특정 그룹의 시즌만 추려서 확인
3. **신규 시즌 생성** — [+ 신규 시즌] 버튼으로 모달 진입
4. **시즌 상세 이동** — 시즌 행 클릭으로 월별 보상 설정 화면 진입
5. **시즌 수정·종료** — 시즌 상세 화면에서 진행 (본 화면에서는 진입만)

## 4. 로직·플로우

### 4.1 화면 진입 → 리스트 표시

1. 사이드바 [아티스트] > [덕력관리] 클릭 또는 다른 탭에서 [랭킹 시즌 설정] 탭으로 전환
2. 그룹 필터 기본값 "전체"로 모든 시즌을 시작일시 내림차순 정렬해 표시
3. 시즌이 0건이면 빈 안내 메시지 표시

### 4.2 그룹 필터 변경

1. 그룹 선택 메뉴에서 특정 그룹 선택
2. 해당 그룹의 시즌만 리스트에 표시되며, 결과 건수도 즉시 갱신
3. 페이지는 1페이지로 복귀

### 4.3 신규 시즌 생성

1. [+ 신규 시즌] 버튼 클릭 → 시즌 생성·수정 모달 [CEB-BO-ART-401-MD-SEASON] 열기
2. 모달에서 그룹·시즌명·시작일시 입력 후 [저장]
3. 저장 성공 시 안내 메시지 "시즌이 저장되었습니다." + 리스트 자동 갱신

### 4.4 시즌 상세 이동

1. 시즌 행 클릭 (행 어디든 클릭 가능)
2. 시즌 상세 페이지 [CEB-BO-ART-401-DETAIL]로 이동
3. 상세에서 월별 보상 설정·시즌 수정·시즌 종료 가능

## 5. 검증·예외 상황

### 5.1 빈 상태

| 상황 | 표시 |
| --- | --- |
| 시즌 0건 (전체) | "등록된 시즌이 없습니다." 안내 + [+ 신규 시즌] 버튼으로 시작 유도 |
| 필터 결과 0건 | "등록된 시즌이 없습니다." 동일 안내 |

### 5.2 권한별 동작

- 권한은 [CEB-BO-SYS-102] 권한 관리 그룹 설정에 따른다
- 쓰기 권한 없으면 [+ 신규 시즌] 버튼 숨김 (조회만 가능)
- 외부 협력사 권한자는 본인 매핑 소속사 산하 그룹의 시즌만 조회 가능 (쓰기 불가)

### 5.3 활동 로그

본 화면 자체는 조회·진입만 — 활동 로그 대상 아님. 시즌 생성·수정·종료는 각 모달에서 기록.
