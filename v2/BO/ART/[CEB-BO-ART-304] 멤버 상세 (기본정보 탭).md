## 1. 화면 목적

개별 멤버의 기본 프로필을 조회하고 상태(활성/비활성)를 관리하는 화면. 멤버 상세는 **2탭 구조**(기본정보 / 소속그룹)이며, 본 문서는 [기본정보] 탭을 다룬다.

### 진입 경로

- [CEB-BO-ART-202] 멤버 리스트 > Row 클릭
- [CEB-BO-ART-303] 그룹 상세 (멤버 탭)에서 멤버 연결
- URL: `/artists/members/{id}?tab=info` (기본 탭)

---

## 2. UI 설명

### 2.1 상단 헤더

- Breadcrumb: `홈 > 아티스트 > 멤버 리스트 > {멤버명(한국어)}`
- 상단 표기: "아티스트 멤버" + 타이틀 `{멤버명(한국어)}`
- 우측 상단 액션: [수정하기]

### 2.2 탭 메뉴 (2탭, 상호 배타)

| # | 탭명 | URL | 문서 |
| --- | --- | --- | --- |
| 1 | **기본정보** | `?tab=info` | 본 문서 |
| 2 | **소속그룹** | `?tab=groups` | [CEB-BO-ART-305] |

### 2.3 멤버 상태 (우측 사이드)

| 항목 | 표시 |
| --- | --- |
| 멤버 상태 | Active / Inactive 뱃지 |
| 프로필 | 프로필 이미지 |
| 운영 상태 | 노출 중 / 비노출 |
| 활성 상태 토글 | 스위치 — Active↔Inactive 즉시 변경(§4.3) |
| 생성 관리자 | 등록한 관리자 |
| 생성 일시 | YYYY.MM.DD (KST) |
| 최근 수정자 | 마지막 수정 관리자 |
| 최근 수정 일시 | YYYY.MM.DD (KST) |

### 2.4 기본정보 본문

#### A. 멤버명 (다국어)

| 필드 | 한국어 | 영어 | 일본어 |
| --- | --- | --- | --- |
| 멤버명 | "징베" | "Jinbe" | "ジンベエ" |

- 각 언어 100자.

#### B. 프로필 속성

| 항목 | 표시 |
| --- | --- |
| 소속사 | 소속사명 (없으면 "소속사 없음") |
| 생년월일 | YYYY.MM.DD |
| 성별 | 남성 / 여성 |
| 별자리 | 생년월일로 자동 도출 (읽기 전용, 수동 변경 불가) |
| 키 | {n}cm |
| 몸무게 | {n}kg (미입력 시 "-") |

#### C. 다국어 소개 항목

| 필드 | 언어 | 글자수 |
| --- | --- | --- |
| 멤버 소개 | 한국어·영어·일본어 | 각 255자 |
| 인사말 | 한국어·영어·일본어 | 각 100자 |
| 취미 | 한국어·영어·일본어 | 각 100자 |
| 좌우명 | 한국어·영어·일본어 | 각 100자 |

- 미입력 항목은 "-"로 표시된다.

> 시각적 스타일은 피그마 참조

## 4. 규칙·로직

### 4.1 권한

- 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.

### 4.2 별자리 자동 도출

- 별자리는 생년월일 입력 시 자동 계산되어 표시되며, 운영자가 직접 변경할 수 없다.

### 4.3 상태 토글 정책

| 변경 | 동작 |
| --- | --- |
| Active → Inactive / Inactive → Active | 토글 클릭 시 즉시 반영 + 완료 토스트 ("멤버가 비활성화되었습니다." / "멤버가 활성화되었습니다."). 별도 확인 모달 없음 |
