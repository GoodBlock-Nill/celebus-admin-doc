## 1. 화면 목적

신규 포지션을 등록하는 입력 모달. 운영자 노출명과 유저 노출명(한국어·영어·일본어)을 입력한다.

---

## 2. UI 설명

### 2.1 레이아웃

| 요소 | 내용 |
| --- | --- |
| 제목 | 새 포지션 추가 |
| 닫기 | 우측 상단 [X] |
| 버튼 | [취소하기] / [추가하기] |

### 2.2 입력 항목

| 항목 | 필수 | 형식 |
| --- | --- | --- |
| 운영자 노출명 | 필수 | 50자 |
| 한국어 (유저 노출명) | 필수 | 50자 |
| 영어 (유저 노출명) | 필수 | 50자 |
| 일본어 (유저 노출명) | 필수 | 50자 |

- 신규 추가 포지션의 상태는 "사용"으로 등록된다.

> 시각적 스타일은 피그마 참조

---

## 3. 기능 설명

| 버튼 | 동작 |
| --- | --- |
| 추가하기 | 4개 항목 모두 입력 시 활성. 클릭 시 포지션 등록 + 모달 닫힘 |
| 취소하기 | 입력 폐기 + 모달 닫힘 (별도 확인 없음) |
| X | 모달 닫힘 |

---

## 4. 로직·플로우

## 5. 검증·예외 상황

- 운영자 노출명·유저 노출명(한국어·영어·일본어) 4개를 모두 입력해야 [추가하기]가 활성화된다.
- 미입력 항목이 있으면 [추가하기] 비활성.
