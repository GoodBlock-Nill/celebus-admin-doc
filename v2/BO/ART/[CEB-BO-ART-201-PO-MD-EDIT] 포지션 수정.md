## 1. 화면 목적

기존 포지션의 노출명과 상태(사용/미사용)를 수정하는 모달.

---

## 2. UI 설명

### 2.1 레이아웃

| 요소 | 내용 |
| --- | --- |
| 제목 | 포지션 수정 |
| 닫기 | 우측 상단 [X] |
| 버튼 | [취소하기] / [저장하기] |

### 2.2 입력 항목

| 항목 | 필수 | 형식 |
| --- | --- | --- |
| 운영자 노출명 | 필수 | 50자 (기존 값 채움) |
| 한국어 (유저 노출명) | 필수 | 50자 |
| 영어 (유저 노출명) | 필수 | 50자 |
| 일본어 (유저 노출명) | 필수 | 50자 |
| 상태설정 | 필수 | Dropdown — 사용 / 미사용 |

> 추가 모달과의 차이: 수정 모달에는 **상태설정(사용/미사용)** Dropdown이 있다.

> 시각적 스타일은 피그마 참조

---

## 3. 기능 설명

| 버튼 | 동작 |
| --- | --- |
| 저장하기 | 변경 사항 저장 + 모달 닫힘 |
| 취소하기 | 변경 폐기 + 모달 닫힘 |
| X | 모달 닫힘 |

---

## 4. 로직·플로우

[CEB-BO-ART-201-PO] 행 클릭 → 본 모달(기존 값 채움) → 수정 → [저장하기].

- 상태를 "미사용"으로 변경하면 멤버 관리 화면의 포지션 선택지에서 제외된다.

---

## 5. 검증·예외 상황
