## 1. 화면 목적

신규 아티스트 멤버를 등록하는 화면. 멤버명(다국어)·프로필 이미지·신체 정보·소속사·다국어 소개 항목을 입력해 생성한다.

### 진입 경로

- [CEB-BO-ART-202] 멤버 리스트 > [멤버 생성]
- URL: `/artists/members/create`

---

## 2. UI 설명

### 2.1 상단 헤더

- 타이틀: 멤버 생성
- 우측 상단 액션: [취소하기] / [생성하기]
- [생성하기]는 필수 항목 미충족 시 비활성 상태이다.

### 2.2 입력 항목

| 항목 | 필수 | 형식 |
| --- | --- | --- |
| 멤버명 | 필수 | 한국어·영어·일본어, 각 100자 (3언어 모두 입력해야 생성 가능) |
| 프로필이미지 | 필수 | 1:1, 최대 5MB (jpeg·png·jpg·webp·gif·svg) |
| 생년월일 | 필수 | 날짜 선택 (YYYY.MM.DD) |
| 별자리 | 자동 | 생년월일 입력 시 자동 도출 (읽기 전용, 직접 선택 불가) |
| 키(cm) | 필수 | 숫자 |
| 소속사 | 선택 | Dropdown — "소속사 없음" + [CEB-BO-ART-202-CP] 등록 소속사 |
| 성별 | 필수 | Dropdown — 남자 / 여자 |
| 멤버소개 | 선택 | 한국어·영어·일본어, 각 255자 |
| 인사말 | 선택 | 한국어·영어·일본어, 각 100자 |
| 취미 | 선택 | 한국어·영어·일본어, 각 100자 |
| 좌우명 | 선택 | 한국어·영어·일본어, 각 100자 |
| 몸무게(kg) | 선택 | 숫자 |

> 시각적 스타일은 피그마 참조

## 3. 화면 내 기능설명

| 요소 | 동작 |
| --- | --- |
| [생성하기] | 입력 검증 후 확인 모달([CEB-BO-ART-202-MD-COMPLE]) |
| [취소하기] | 확인 모달([CEB-BO-ART-202-MD-CANCEL]) |
| 생년월일 선택 | 날짜 선택 시 별자리 자동 표시 |
| 이미지 업로드 | 클릭 또는 드래그로 파일 등록 |

---

## 4. 규칙·로직

### 4.1 권한

- 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.

### 4.2 입력 검증

- 멤버명은 한국어·영어·일본어를 모두 입력해야 한다(셋 중 하나라도 비면 생성 불가).
- 프로필이미지·생년월일·키·성별은 필수이다.
- 별자리는 생년월일로 자동 계산되어 표시되며 직접 선택할 수 없다.
- 필수 항목이 모두 충족되어야 [생성하기]가 활성화된다.
