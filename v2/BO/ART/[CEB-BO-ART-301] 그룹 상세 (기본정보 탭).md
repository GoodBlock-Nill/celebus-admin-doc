## 1. 화면 목적

개별 그룹의 기본 정보를 조회하고 상태(활성/비활성)를 관리하는 화면. 그룹 상세는 **2탭 구조**(기본정보 / 멤버)이며, 본 문서는 [기본정보] 탭을 다룬다. SNS·공식 링크는 별도 탭이 아니라 본 탭 내 "대표 이미지 및 링크" 섹션으로 통합되어 있다.

### 주요 기능

- 그룹 기본 정보 조회 (그룹명·소개·이미지·SNS 링크, 다국어 KO/EN/JA)
- 그룹 관리 정보 조회 (상태·운영 상태·생성/수정 이력)
- 그룹 활성/비활성 상태 토글
- 그룹 정보 수정 진입

### 진입 경로

- [CEB-BO-ART-201] 그룹 리스트 > Row 클릭
- URL: `/artists/groups/{id}?tab=info` (기본 탭)

## 4. 규칙·로직

### 4.1 권한

- 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.
- [수정하기]·상태 토글은 쓰기 권한 보유 그룹만 사용 가능하다.

### 4.2 다국어 정책

- 그룹명·그룹 소개는 한국어·영어·일본어 모두 입력한다. 글자수 등 상세 기준은 [CEB-BO-003] 영역 가이드를 따른다.

### 4.3 상태 토글 정책

| 변경 | 동작 |
| --- | --- |
| Active → Inactive / Inactive → Active | 토글 클릭 시 즉시 반영 + 완료 토스트. 별도 확인 모달 없음 |

- Active: 앱 노출 / Inactive: 앱 미노출

### 4.4 탐색 노출 정책

- 탐색 노출은 앱 [CEB-ART-201] 아티스트 탐색 화면 노출 여부를 켜고 끄는 별개 플래그다. 토글 즉시 반영 + 완료 토스트(확인 모달 없음).
- 앱 노출 조건 = **상태 Active + 탐색 노출 켬**. Inactive 그룹은 탐색 노출 켬이어도 앱 미노출(상태 우선).
- 동일 정책을 [CEB-BO-ART-201] 그룹 리스트 [탐색 노출] 컬럼에서도 제어하며 두 화면은 동기화된다.
