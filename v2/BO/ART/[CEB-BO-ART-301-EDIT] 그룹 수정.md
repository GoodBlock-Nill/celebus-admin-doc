## 1. 화면 목적

기존 그룹의 정보를 수정하는 화면. 입력 항목·구성은 [CEB-BO-ART-201-Create] 그룹 생성과 동일하며, 진입 시 기존 값이 채워진다.

### 진입 경로

- [CEB-BO-ART-301] 그룹 상세 (기본정보 탭) > [수정하기]
- URL: `/artists/groups/{id}/edit`

---

## 2. UI 설명

### 2.1 상단 헤더

- 타이틀: 그룹 수정
- 우측 상단 액션: [취소하기] / [수정하기]
- [수정하기]는 변경 사항이 없으면 비활성 상태이다.

### 2.2 입력 항목

| 항목 | 필수 | 형식 |
| --- | --- | --- |
| 그룹명 | 필수 | 한국어·영어·일본어, 각 100자 |
| 그룹 소개 | 필수 | 한국어·영어·일본어, 각 500자 |
| 로고(심볼) | 필수 | 1:1 권장 512×512px, 최대 5MB (jpeg·png·jpg·webp·gif·svg). 기존 이미지 표시 + [변경]/[제거] |
| 대표이미지(배너) | 필수 | 16:9 권장 1920×1080px, 최대 5MB. 기존 이미지 표시 + [변경]/[제거] |
| Instagram | 선택 | URL |
| X | 선택 | URL |
| YouTube | 선택 | URL |
| TikTok | 선택 | URL |
| Homepage | 선택 | URL |

> 시각적 스타일은 피그마 참조

---

## 4. 규칙·로직

### 4.1 권한

- 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.

### 4.2 입력 검증

- 그룹명·그룹 소개는 한국어·영어·일본어 모두 입력해야 한다.
- 로고·대표이미지는 필수이다.
- 변경 사항이 없으면 [수정하기]가 비활성화된다.
