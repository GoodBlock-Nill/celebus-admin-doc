## 1. 화면 목적

운영자가 **아티스트 그룹별·기간별 회원 덕력 누적 순위**를 조회하는 화면이다. 시즌과 무관하게 년 단위 또는 월 단위로 자유롭게 기간을 지정해 순위를 확인할 수 있다.

- **누가**: 덕력 운영·CS·마케팅 권한이 있는 백오피스 관리자
- **왜**: 어떤 회원이 어떤 그룹에 얼마나 활동하는지 확인하고, 보상·이벤트 의사결정에 활용
- **결과**: 회원·기간·그룹 조건에 맞는 순위 목록을 보고 회원 상세로 이동 가능

---

## 2. UI 설명

### 2.1 화면 레이아웃

```
[탭 메뉴]  [랭킹 시즌 설정] [덕력 랭킹] [덕력 내역]

────────────────────────────────────────────────────────

[필터 바]
그룹: [V01D ▾]   단위: [년 단위 | 월 단위]   년: [2026 ▾]   회원 검색: [...]

[안내 카드]
[2026년] V01D — 활동 회원 N명 · 누적 N,NNN DUK

[랭킹 테이블]
| 순위 | 회원 | 누적 덕력 | 최근 변동일 |
|---|---|---|---|
| ① | 닉네임A | 12,345 DUK | 2026.05.26 17:30 |
| ② | 닉네임B | 11,234 DUK | 2026.05.25 14:05 |
| ③ | 닉네임C | 10,123 DUK | 2026.05.24 21:50 |
| 4 | ...    | ...       | ...               |

[페이지네이션]
```

### 2.2 구성요소

| 영역 | 요소 | 설명 |
| --- | --- | --- |
| 필터 | 그룹 선택 메뉴 | 활성 그룹 5종 중 1종. 기본 첫 번째 그룹 |
| 필터 | 단위 토글 | "년 단위" / "월 단위" 두 가지. 기본 "년 단위" |
| 필터 | 기간 선택 메뉴 | 년 단위 선택 시 "년" 메뉴 / 월 단위 선택 시 "년-월" 두 메뉴 |
| 필터 | 회원 검색 | 닉네임 부분 일치 |
| 안내 | 컨텍스트 카드 | 현재 조회 중인 기간·그룹·활동 회원 수·누적 덕력 합계 한 줄 표시 |
| 리스트 | 랭킹 테이블 | 순위·회원·누적 덕력·최근 변동일 4컬럼 |
| 하단 | 페이지네이션 | 20건/페이지 |

### 2.3 랭킹 테이블 컬럼

| 컬럼 | 표시 |
| --- | --- |
| 순위 | 1·2·3위는 메달 색상 강조 (금·은·동) / 4위 이하는 회색 숫자 |
| 회원 | 회원 닉네임 — 클릭 시 회원 상세 [CEB-BO-USR-201]로 새 창 이동 |
| 누적 덕력 | 해당 기간 누적 (획득 − 사용) 합계, 천 단위 콤마 + "DUK" 단위 |
| 최근 변동일 | 해당 기간 내 마지막 적립·사용 일시 (YYYY.MM.DD HH:mm) |

### 2.4 시각적 스타일

- 순위 메달: 1위 금색·2위 은색·3위 동색
- 누적 덕력: 보라색 강조 텍스트
- 빈 상태: "해당 필터 조건에 일치하는 회원이 없습니다."

## 3. 기능 설명

운영자가 이 화면에서 할 수 있는 동작:

1. **그룹별 순위 조회** — 5개 그룹 중 1개 선택
2. **기간 단위 전환** — 년 단위 ↔ 월 단위 토글
3. **년·월 선택** — 활동 데이터가 있는 년·월에서 자유 선택
4. **회원 닉네임 검색** — 부분 일치 검색
5. **회원 상세 진입** — 닉네임 클릭 시 새 창으로 이동

---

## 4. 로직·플로우

### 4.1 화면 진입 → 기본 순위 표시

1. 첫 번째 그룹 + 년 단위 + 현재 년 기본 선택
2. 안내 카드에 현재 조회 기간 컨텍스트 표시
3. 회원별 누적 덕력 내림차순으로 정렬 후 노출

### 4.2 그룹 변경

1. 그룹 메뉴에서 다른 그룹 선택
2. 해당 그룹의 활동 데이터가 있는 년 중 가장 최근 년이 자동 선택
3. 순위 테이블·안내 카드 갱신 + 페이지 1로 복귀

### 4.3 단위 토글

1. "월 단위" 선택 시 "년-월" 두 메뉴로 교체 (현재 년·월 기본)
2. "년 단위" 선택 시 "년" 한 메뉴로 복귀

### 4.4 회원 닉네임 클릭

1. 새 창으로 회원 상세 [CEB-BO-USR-201] 진입 (기본 탭은 "기본 정보")
2. 현재 랭킹 화면은 그대로 유지

## 5. 검증·예외 상황

### 5.1 빈 상태

| 상황 | 표시 |
| --- | --- |
| 활동 회원 0명 | "해당 필터 조건에 일치하는 회원이 없습니다." + 페이지네이션 숨김 |
| 검색어 일치 0건 | 동일 안내 |

### 5.2 동일 누적 덕력 처리

[CEB-000] §5.2.1 SSOT 정합 — 같은 누적 덕력일 경우 **먼저 도달한 회원이 상위 순위**. 동순위 없음, 모두 고유 순위.

### 5.3 권한별 동작

- 본 화면은 조회 전용 — 모든 운영자 권한에 R 권한 부여
- 외부 협력사 권한자는 본인 매핑 소속사 산하 그룹만 조회 가능

### 5.4 활동 로그

조회 화면 — 활동 로그 대상 아님.
