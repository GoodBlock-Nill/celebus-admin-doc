## 1. 화면 목적

그룹에 소속될 멤버를 추가·제거하고, 소속 멤버의 그룹 내 포지션을 지정하는 화면. **좌우 2패널**로 구성되며, 좌측의 전체 멤버 풀에서 추가하고 우측의 소속 멤버에서 포지션 지정·제거한다. 모든 변경은 화면 하단 [수정하기]로 일괄 저장된다.

### 진입 경로

- [CEB-BO-ART-303] 그룹 상세 (멤버 탭) > [멤버 관리]
- URL: `/artists/groups/{id}/members/manage`

---

## 2. UI 설명

### 2.1 상단 헤더

- Breadcrumb: `홈 > 아티스트 > 그룹리스트 > 그룹 상세 > 멤버 관리`
- 타이틀: `{그룹명(한국어)} | 멤버 관리`
- 우측 상단 액션: [취소하기] / [수정하기]

### 2.2 좌측 패널 — 전체 멤버 풀 (추가 후보)

| 구분 | 내용 |
| --- | --- |
| 검색 | 멤버명 입력 + [초기화] |
| 컬럼 | 멤버명 / 성별 / 생년월일 / 관리 |
| 관리 버튼 | [추가] — 클릭 시 해당 멤버를 우측 소속 멤버로 이동(임시 반영) |
| 페이지네이션 | 하단 중앙 |

- 이미 소속된 멤버는 좌측 풀에 표시되지 않는다.

### 2.3 우측 패널 — 현재 그룹 소속 멤버

| 구분 | 내용 |
| --- | --- |
| 컬럼 | 멤버명 / 포지션 / 성별 / 생년월일 / 관리 |
| 포지션 | Dropdown — "미선택" + [CEB-BO-ART-201-PO]에 등록된 포지션 목록 (멤버당 1개 선택) |
| 관리 버튼 | [제거] — 클릭 시 해당 멤버를 그룹에서 제외(임시 반영) |
| 신규 표시 | 좌측에서 새로 추가한 멤버는 "NEW" 뱃지로 구분 |

### 2.4 하단 액션

| 버튼 | 동작 |
| --- | --- |
| [수정하기] | 추가·제거·포지션 변경을 일괄 저장 → 확인 모달([CEB-BO-ART-303-MD-EDIT-COMPLE]) |
| [취소하기] | 변경 폐기 → 확인 모달([CEB-BO-ART-303-MD-EDIT-CANCEL]) |

> 좌·우 패널의 [추가]·[제거]·포지션 변경은 화면 내 임시 반영이며, [수정하기] 전까지 실제 저장되지 않는다.

> 시각적 스타일은 피그마 참조

## 3. 화면 내 기능설명

| 요소 | 동작 |
| --- | --- |
| 좌측 [추가] | 멤버를 우측 소속 멤버로 이동 (NEW 표시) |
| 우측 [제거] | 멤버를 그룹에서 제외 |
| 우측 포지션 Dropdown | 그룹 내 포지션 지정 (미선택 가능) |
| [수정하기] | 변경 일괄 저장 (확인 모달) |
| [취소하기] | 변경 폐기 (확인 모달) |

---

## 4. 규칙·로직

### 4.1 권한

- 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다. 쓰기 권한 보유 그룹만 진입·저장 가능.

### 4.2 포지션 지정

- 포지션 선택지는 [CEB-BO-ART-201-PO] 포지션 설정에 등록되어 "사용" 상태인 포지션이다.
- 멤버당 포지션 1개를 지정하며, "미선택" 시 포지션 없음으로 저장된다.

### 4.3 저장 방식

- 좌·우 변경 사항은 임시 반영 후 [수정하기]로 일괄 커밋된다. [취소하기] 또는 이탈 시 변경은 폐기된다.
