## 1. 화면 목적

CELEBUS에 등록된 아티스트 멤버를 통합 조회·관리하는 화면. 그룹과 별도로 멤버 단위 검색·관리하며, 신규 멤버 생성·소속사 설정으로 진입한다. 개별 멤버는 [CEB-BO-ART-304] 멤버 상세로 이동한다.

### 주요 기능

- 등록된 전체 멤버 리스트 조회 + 상태 카운터(전체/Active/Inactive)
- 상태 필터·검색(멤버명·그룹명)
- 신규 멤버 생성 진입
- 소속사 설정 진입
- 멤버 상세 진입

### 진입 경로

- 사이드 메뉴 > 아티스트 > 멤버 리스트

## 2. UI 설명

### 2.1 상단 영역

- Breadcrumb: `홈 > 아티스트 > 멤버 리스트`
- 화면 타이틀: 멤버 리스트
- 우측 상단 액션: [소속사 설정] (Secondary) / [멤버 생성] (Primary)

### 2.2 상태 카운터 (3개)

| 카운터 | 의미 |
| --- | --- |
| 전체 | 등록된 전체 멤버 수 |
| Active | 활성 멤버 수 |
| Inactive | 비활성 멤버 수 |

- 각 카운터 클릭 시 해당 상태로 필터링된다.

### 2.3 검색·필터 영역

| 구분 | 요소 | 타입 | 설명 |
| ---- | ------ | ---- | ---- |
| 필터 | 상태 | Dropdown | 상태(전체) / Active / Inactive |
| 검색 | 검색 대상 | Dropdown | 멤버명 / 그룹명 |
| 입력 | 검색어 | Text Input | 선택한 검색 대상으로 부분 일치 (placeholder: "멤버명 입력") |
| 버튼 | 초기화 | Dark Button | 필터·검색어 리셋 |

> 별도 정렬 Dropdown은 없으며, 정렬 기준은 업데이트 일시 내림차순 고정이다(§4.2).

### 2.4 목록 영역 (테이블)

| 컬럼 | 데이터 예시 | 설명 |
| ------ | ----------- | ---- |
| 상태 | Active / Inactive | 상태 뱃지 |
| 멤버명 | "징베" | 한국어 우선 표시 (다국어 KO/EN/JA) |
| 소속 그룹 수 | 1 | 멤버가 소속된 그룹 수 |
| 생년월일 | 1976.04.02 | YYYY.MM.DD |
| 성별 | 남자 / 여자 | |
| 업데이트 일시 | 2026.05.27 18:45 | YYYY.MM.DD HH:mm (KST) |

- Row 클릭 → [CEB-BO-ART-304] 멤버 상세 (기본정보 탭) 이동
- 페이지네이션: 하단 중앙, 페이지당 15건

> 시각적 스타일은 피그마 참조

## 4. 규칙·로직

### 4.1 권한

- 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.
- 멤버 생성·소속사 설정은 쓰기 권한 보유 그룹만 노출된다.

### 4.2 정렬·필터

- 기본 정렬: 업데이트 일시 내림차순 (최신순). 사용자 변경 불가.
- 상태 필터 + 검색어 = AND 조합.
- 검색: 검색 대상(멤버명/그룹명) 선택 후 해당 항목으로 부분 일치.

### 4.3 상태 정의

| 상태 | 의미 |
| --- | --- |
| Active | 활성 멤버 |
| Inactive | 비활성 멤버 |
