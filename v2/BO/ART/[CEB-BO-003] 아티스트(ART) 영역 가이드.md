## 1. 개요

아티스트(ART) 영역은 그룹·멤버·소속사·포지션 등 K-pop 아티스트 마스터 데이터를 관리한다. **v1.1 (사용자 결정 2026-05-06)**: 실제 운영 그룹 상세 3탭(기본정보·SNS&Link·멤버) + 멤버 상세 2탭(기본·소속그룹) 그대로 정렬. v2에서 신규 정의했던 "활동 통계" 탭은 별도 영역에서 조회 (BIVE-102 그룹별 BIVE 발행, FQ-202 Quest 참여 통계 등 이미 영역별로 분산 노출 — 회원 상세 §7.1 v2 신규 4탭과 동일 정책). **소속사 ↔ EXTERNAL_ARTIST 권한자 매핑**은 후속 ART-103에서 정의.

### 7.3 소속사 설정 ↔ EXTERNAL_ARTIST 매핑

EXTERNAL_ARTIST(외부 협력사 권한자)는 본인이 매핑된 **소속사** 산하의 모든 그룹·멤버에 R/W 권한을 가진다. 1:N 관계.

#### 매핑 모델

```
소속사 (예: IX Entertainment)
  ├── 그룹 1 (예: V01D)
  │     └── 멤버 N명
  └── 그룹 2 (예: 다른 그룹)
        └── 멤버 N명
        
EXTERNAL_ARTIST 권한자 (예: ix_admin@ix-ent.kr)
  └── 소속사 'IX Entertainment'에 매핑됨
       → IX Entertainment 산하 모든 그룹·멤버 R/W
       → 다른 소속사 그룹·멤버 접근 차단
```

#### 매핑 정책

| 항목 | 정책 |
| --- | --- |
| 매핑 권한 | SUPER 전용 |
| 매핑 화면 | [CEB-BO-ART-103-MD-MAPPING] |
| 매핑 입력 | EXTERNAL_ARTIST 이메일(또는 사용자 ID) ↔ 소속사 ID 1:N |
| 매핑 검증 | EXTERNAL_ARTIST 권한이 부여된 사용자만 매핑 대상으로 선택 가능 |
| 매핑 변경 | SUPER만 가능. 변경 즉시 EXTERNAL_ARTIST 세션 권한 갱신 |
| 매핑 해제 | SUPER만 가능. 해제 시 즉시 해당 외부 권한자의 세션 강제 종료 + 데이터 접근 차단 |
| 매핑 만료 | EXTERNAL_ARTIST 권한 만료([CEB-BO-000] §3.2) 시 자동 매핑 해제 |
| 활동 로그 | 매핑 추가·해제 모두 [CEB-BO-000] §6 표준 로그 |

#### 소속사 변경 시 권한 재할당 흐름

소속사 ↔ 그룹 매핑 변경(예: 그룹 'V01D'가 소속사 'A' → 'B' 이전)이 발생할 경우:

```
SUPER가 [CEB-BO-ART-103]에서 그룹의 소속사 변경
    ↓
시스템 자동: 기존 소속사의 EXTERNAL_ARTIST → 해당 그룹 접근 즉시 차단
신규 소속사의 EXTERNAL_ARTIST → 해당 그룹 접근 자동 부여
    ↓
관련 EXTERNAL_ARTIST 세션 권한 갱신 + 활동 로그 기록
```

### 7.4 포지션 설정

v1 유지. 보컬·메인보컬·리드보컬·리드댄서·메인댄서·랩퍼·메인랩퍼·리더·서브 등.

| 항목 | 정책 |
| --- | --- |
| 포지션 수 | 무제한 (운영자 정의) |
| 포지션명 | KO/EN/JP 3언어 필수 |
| 포지션 ↔ 멤버 매핑 | N:N (1명이 여러 포지션 가능, 예: 메인보컬 + 리더) |
| 정렬 | 운영자 지정 표시 순서 (드래그 정렬) |
| 삭제 | 매핑된 멤버가 0명일 때만 삭제 가능. 매핑 있으면 차단 + 토스트 |

### 7.5 EXTERNAL_ARTIST 권한 노출 범위

[CEB-BO-000] §3.1·§3.2 기반.

| 영역 | 접근 범위 |
| --- | --- |
| ART-101·201 (그룹) | 자기 소속사 산하 그룹만 R/W (정보 수정만, 생성·삭제 불가) |
| ART-102·202 (멤버) | 자기 소속사 산하 멤버만 R/W (정보 수정·SNS·BIVE 연결만) |
| ART-103 (소속사) | 자기 소속사 정보 R (수정 권한 없음 — SUPER 영역) |
| ART-103-MD-MAPPING | 접근 차단 (SUPER 전용) |
| ART-301 (포지션) | R only (전체 조회만, 수정 불가) |

화면 진입 시 EXTERNAL_ARTIST의 매핑 소속사가 아닌 데이터는 검색·조회 결과에서 자동 제외 (URL 직접 접근도 차단).
