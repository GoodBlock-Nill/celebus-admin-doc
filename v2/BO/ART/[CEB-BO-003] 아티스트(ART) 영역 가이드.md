## 1. 개요

아티스트(ART) 영역은 그룹·멤버·소속사·포지션 등 K-pop 아티스트 마스터 데이터를 관리한다. 사이드 메뉴 [아티스트] 하위는 **그룹 리스트**와 **멤버 리스트** 두 진입점으로 구성된다(단일 "아티스트 메인" 화면은 폐기).

운영 BO 실측 기준 화면 구조:

- **그룹 상세 = 2탭** (기본정보 / 멤버). SNS·공식 링크는 [기본정보] 탭 내 섹션으로 통합.
- **멤버 상세 = 2탭** (기본정보 / 소속그룹).
- **멤버 관리** = 그룹에 멤버를 추가·제거하고 포지션을 지정하는 통합 화면(좌우 2패널).
- **포지션·소속사** = 운영자 노출명 1개 + 유저 노출명(한국어·영어·일본어) 구성의 마스터 목록.

| 항목 | 위치 |
| --- | --- |
| 마스터 가이드 | [CEB-BO-001] 마이그레이션·재구성 마스터 가이드 |
| 권한 | [CEB-BO-SYS-102] 권한 관리 그룹 설정 |

## 3. 화면 ID 체계 (ART 영역)

운영 실측 메뉴 계층에 맞춘 ART 영역 번호 체계.

| 구분 | 번호 | 화면 |
| --- | --- | --- |
| 메인 (폐기) | 101 | 아티스트 메인 (폐기) |
| 리스트 | 201 / 202 | 그룹 리스트 / 멤버 리스트 |
| 상세 탭 | 301·303 / 304·305 | 그룹 상세(기본정보·멤버) / 멤버 상세(기본정보·소속그룹) |
| 덕력 관리 | 401 | 덕력 관리 (별도 메뉴) |

부속·하위 화면은 상위 번호에 접미사를 붙인다.

| 접미사 | 의미 | 예시 |
| --- | --- | --- |
| -Create | 생성 페이지 | 201-Create, 202-Create |
| -EDIT | 수정 페이지 | 301-EDIT, 304-EDIT, 303-EDIT(멤버 관리) |
| -PO | 포지션 설정 (그룹 리스트 하위) | 201-PO |
| -CP | 소속사 설정 (멤버 리스트 하위) | 202-CP |
| -MD-{동작} | 모달 | 201-MD-COMPLE, 202-CP-MD-ADD |

## 4. 권한

- 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.
- 그룹·멤버·소속사·포지션의 생성·수정·삭제는 쓰기 권한 보유 그룹만 수행한다.
- 외부 아티스트 협력사 권한자는 자기 소속사 산하 그룹·멤버에 한해 접근하며, 그룹·멤버 신규 생성·삭제는 불가하고 정보 수정 범위만 가진다(세부 범위는 [CEB-BO-SYS-102] 정의).

## 7. 영역 고유 정책

### 7.1 그룹 상세 2탭 구조

| # | 탭명 | 핵심 표시 항목 |
| --- | --- | --- |
| 1 | 기본정보 | 그룹명·그룹 소개(3언어)·로고·대표 이미지·SNS&Link(Instagram·X·YouTube·TikTok·Homepage)·관리 정보(상태·운영 상태·생성/수정 이력) |
| 2 | 멤버 | 소속 멤버 목록(상태·멤버명·포지션·생년월일·성별·등록 일시). [멤버 관리]로 추가·포지션·제거 |

- SNS&Link는 별도 탭이 아니라 [기본정보] 탭 내 섹션이다.
- 멤버 추가·포지션 지정·제거는 [CEB-BO-ART-303-EDIT] 멤버 관리(좌우 2패널)에서 일괄 처리한다.

### 7.2 멤버 상세 2탭 구조

| # | 탭명 | 핵심 표시 항목 |
| --- | --- | --- |
| 1 | 기본정보 | 멤버명(3언어)·프로필 이미지·소속사·생년월일·별자리(자동)·키·몸무게·멤버 소개·인사말·취미·좌우명(각 3언어)·관리 정보 |
| 2 | 소속그룹 | 소속 그룹 목록(상태·소속 그룹명·그룹내 포지션·멤버 수·설명·업데이트 일시) |

- 별자리는 생년월일 입력 시 자동 도출되며 직접 변경할 수 없다.
- 한 멤버는 여러 그룹에 소속될 수 있다.

### 7.3 포지션 설정

- 그룹 멤버에게 지정할 포지션의 마스터 목록. 운영자 노출명 + 유저 노출명(3언어) + 상태(사용/미사용)로 구성된다.
- "사용" 포지션만 멤버 관리 화면의 포지션 선택지로 노출된다.

### 7.4 소속사 설정

- 멤버에게 지정할 소속사의 마스터 목록. 구성은 포지션과 동일(운영자 노출명 + 유저 노출명 3언어 + 상태).
- "사용" 소속사만 멤버 생성·수정의 소속사 선택지로 노출된다.

### 7.5 탐색 노출 설정

- 앱 [CEB-ART-201] 아티스트 탐색 화면에 노출할 그룹을 그룹 단위로 켜고 끈다. [CEB-BO-ART-201] 그룹 리스트의 [탐색 노출] 컬럼 토글과 [CEB-BO-ART-301] 그룹 상세에서 제어하며, 두 화면은 동기화된다.
- 탐색 노출은 그룹 활성 상태(Active/Inactive)와 **별개 플래그**다. 활성 그룹이라도 탐색에서 숨길 수 있다.
- 앱 노출 조건 = **상태 Active + 탐색 노출 켬**. Inactive 그룹은 탐색 노출 켬이어도 앱에 노출되지 않는다(상태 우선).
- 신규 그룹은 탐색 노출 "켬"으로 등록된다.
