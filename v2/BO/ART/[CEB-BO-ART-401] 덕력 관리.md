## 1. 화면 목적

운영자가 **덕력 운영을 시작하는 진입 페이지**다. 시즌 설정·랭킹 조회·내역 조회 3개 업무를 탭으로 묶어 한 메뉴 안에서 전환할 수 있게 한다.

- **누가**: 덕력 운영·CS·마케팅 권한이 있는 백오피스 관리자
- **왜**: 시즌·랭킹·내역은 같은 도메인의 다른 시점이므로 한 화면에서 빠르게 전환 가능해야 함
- **결과**: 사이드바 진입 시 시즌 설정 탭이 기본 활성화되며, 3개 탭을 자유롭게 전환

---

| 탭 | 라우트 쿼리 | 본문 화면 |

- 각 탭의 본문은 별도 화면 명세에 정의되어 있으며, 본 화면은 탭 메뉴와 본문 슬롯만 제공

## 3. 기능 설명

운영자가 이 화면에서 할 수 있는 동작:

1. **3개 탭 전환** — 랭킹 시즌 설정 / 덕력 랭킹 / 덕력 내역
2. **각 탭의 본문 조작** — 본문은 별도 명세 참조

---

## 4. 로직·플로우

### 4.1 화면 진입 → 기본 탭

1. 사이드바 [아티스트] > [덕력관리] 클릭
2. URL `/artists/duk` 진입 시 [랭킹 시즌 설정] 탭이 자동 활성화 (쿼리 `?tab=season`)
3. URL에 `?tab=ranking` 또는 `?tab=history`가 명시되어 있으면 해당 탭이 활성화된 상태로 진입

### 4.2 탭 전환

1. 탭 메뉴에서 다른 탭 클릭
2. URL 쿼리 `?tab=...`가 갱신되며 본문이 해당 탭의 화면으로 교체
3. 각 탭의 필터·페이지 상태는 탭 내부 정책에 따름 (분할 화면 명세 참조)

---

## 5. 검증·예외 상황

### 5.1 잘못된 탭 쿼리

| 상황 | 동작 |
| --- | --- |
| `?tab=` 쿼리 값이 정의된 3종(`season`·`ranking`·`history`)이 아닌 경우 | 기본 탭(`season`)으로 자동 복귀 |
| 쿼리 누락 | 기본 탭(`season`)으로 자동 진입 |

### 5.2 권한별 동작

- 권한은 [CEB-BO-SYS-102] 권한 관리 그룹 설정에 따른다
- 덕력 관리 메뉴 자체에 접근 권한이 없으면 사이드바 메뉴 미노출 + 직접 URL 진입 차단
- 각 탭 본문의 쓰기·삭제 권한 분기는 분할 화면 명세에서 정의

### 5.3 활동 로그

본 화면은 탭 컨테이너 — 활동 로그 대상 아님. 각 탭 본문의 액션(시즌 생성·수정·종료·월별 보상 저장)에서 기록.
