## 1. 화면 목적

신규 아티스트 그룹을 등록하는 화면. 그룹명·소개(다국어)·이미지·SNS 링크를 입력해 생성한다.

### 진입 경로

- [CEB-BO-ART-201] 그룹 리스트 > [그룹 생성]
- URL: `/artists/groups/create`

---

## 2. UI 설명

### 2.1 상단 헤더

- 타이틀: 그룹 생성
- 우측 상단 액션: [취소하기] / [생성하기]
- [생성하기]는 필수 항목 미충족 시 비활성 상태이다.

### 2.2 입력 항목

| 항목 | 필수 | 형식 |
| --- | --- | --- |
| 그룹명 | 필수 | 한국어·영어·일본어, 각 100자 |
| 그룹 소개 | 필수 | 한국어·영어·일본어, 각 500자 |
| 로고(심볼) | 필수 | 1:1 권장 512×512px, 최대 5MB (jpeg·png·jpg·webp·gif·svg) |
| 대표이미지(배너) | 필수 | 16:9 권장 1920×1080px, 최대 5MB |
| Instagram | 선택 | URL |
| X | 선택 | URL |
| YouTube | 선택 | URL |
| TikTok | 선택 | URL |
| Homepage | 선택 | URL |

> 시각적 스타일은 피그마 참조

---

## 3. 화면 내 기능설명

| 요소 | 동작 |
| --- | --- |
| [생성하기] | 입력 검증 후 확인 모달([CEB-BO-ART-201-MD-COMPLE]) |
| [취소하기] | 확인 모달([CEB-BO-ART-201-MD-CANCEL]) |
| 이미지 업로드 | 클릭 또는 드래그로 파일 등록 |

---

## 4. 규칙·로직

### 4.1 권한

- 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.

### 4.2 입력 검증

- 그룹명·그룹 소개는 한국어·영어·일본어 모두 입력해야 한다.
- 로고·대표이미지는 필수이다.
- 필수 항목이 모두 충족되어야 [생성하기]가 활성화된다.
