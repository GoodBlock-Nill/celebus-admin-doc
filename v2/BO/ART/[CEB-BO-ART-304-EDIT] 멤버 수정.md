## 1. 화면 목적

기존 멤버의 프로필 정보를 수정하는 화면. 입력 항목·구성은 [CEB-BO-ART-202-Create] 멤버 생성과 동일하며, 진입 시 기존 값이 채워진다.

### 진입 경로

- [CEB-BO-ART-304] 멤버 상세 (기본정보 탭) > [수정하기]
- URL: `/artists/members/{id}/edit`

---

## 2. UI 설명

### 2.1 상단 헤더

- 타이틀: 멤버 수정
- 우측 상단 액션: [취소하기] / [수정하기]
- [수정하기]는 변경 사항이 없으면 비활성 상태이다.

### 2.2 입력 항목

| 항목 | 필수 | 형식 |
| --- | --- | --- |
| 멤버명 | 필수 | 한국어·영어·일본어, 각 100자 |
| 프로필이미지 | 필수 | 1:1, 최대 5MB. 기존 이미지 표시 + [변경]/[제거] |
| 생년월일 | 필수 | 날짜 선택 (YYYY.MM.DD) |
| 별자리 | 자동 | 생년월일로 자동 도출 (읽기 전용) |
| 키(cm) | 필수 | 숫자 |
| 소속사 | 선택 | Dropdown — [CEB-BO-ART-202-CP] 등록 소속사 / "소속사 없음" |
| 성별 | 필수 | Dropdown — 남자 / 여자 |
| 멤버소개 | 선택 | 한국어·영어·일본어, 각 255자 |
| 인사말 | 선택 | 한국어·영어·일본어, 각 100자 |
| 취미 | 선택 | 한국어·영어·일본어, 각 100자 |
| 좌우명 | 선택 | 한국어·영어·일본어, 각 100자 |
| 몸무게(kg) | 선택 | 숫자 |

> 시각적 스타일은 피그마 참조

### 2.3 확인 모달

| 트리거 | 제목 | 본문 | 버튼 |
| --- | --- | --- | --- |
| [수정하기] | 멤버 수정 | "확인 버튼을 누르면 수정사항이 적용됩니다" / "계속 진행하시겠습니까?" | [다시 돌아가기] / [확인] |
| [취소하기] | 멤버 수정 취소 | "수정된 데이터는 저장되지 않습니다" / "계속 진행하시겠습니까?" | [다시 돌아가기] / [확인] |

| --- | --- |

---

## 4. 규칙·로직

### 4.1 권한

- 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.

### 4.2 입력 검증

- 멤버명은 한국어·영어·일본어 모두 입력해야 한다.
- 멤버명·프로필이미지·생년월일·키·성별은 필수이다.
- 별자리는 생년월일로 자동 계산되며 직접 변경할 수 없다.
- 변경 사항이 없으면 [수정하기]가 비활성화된다.
