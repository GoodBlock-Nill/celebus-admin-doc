## 1. 개요

디지털 굿즈(COL) 영역은 **BIVE(NFT) 외 비-NFT 디지털 자산** (포토카드 이미지·배지·스티커·이모티콘 등)을 관리. BIVE는 온체인 NFT 발행이 필요한 자산, COL은 온체인 발행 없이 회원 컬렉션에 디지털 자산으로 추가하는 영역.

### COL vs BIVE

| 항목 | BIVE | COL |
| --- | --- | --- |
| 유형 | NFT (온체인) | 비-NFT (디지털 자산) |
| 발행 가스비 | 운영 지갑 부담 | 없음 |
| 회원 보유 | 외부 지갑 | CELEBUS 내부 컬렉션 |
| 거래·전송 | 가능 | 불가 (회원 본인 컬렉션만) |
| 발행 트리거 | Quest/Raffle 보상 등 | 동일 + 운영자 수동 발급 |

| 항목 | 위치 |
| --- | --- |
| 마스터 가이드 | [CEB-BO-001] §2 (#8 COL) |

## 4. 영역 고유 정책

### 4.1 굿즈 타입

| 타입 | 설명 | 예시 |
| --- | --- | --- |
| 포토카드 | 이미지 컬렉션 | "V01D 송유찬 콘서트 포토카드" |
| 배지 | 활동 인증 | "Quest 100건 달성 배지" |
| 스티커 | 앱 내 사용 가능 | "iKON 응원 스티커" |
| 이모티콘 | 앱 채팅·기억저장소에서 사용 | |
| 디지털 카드 | 게임 카드형 | "MADEIN 시즌 1 카드 #001" |

### 4.2 발행 트리거

| 트리거 | 설명 |
| --- | --- |
| 운영자 수동 발급 | OPERATOR가 회원 선택 후 발급 |
| Quest 보상 (자동) | Quest 검수 승인 시 자동 발급 (Quest 생성 시 [COL 자동 발급] 옵션) |
| Raffle 당첨 보상 (자동) | Raffle 추첨 시 |
| EVT 레벨업 보상 (자동) | 레벨 도달 시 |
| 가입 보너스 | 신규 가입 시 (선택) |

### 4.3 회원 컬렉션

회원 앱(`v2 APP [CEB-COL-101]`)에서 보유 굿즈 통합 조회:
- 타입별·아티스트별 필터
- 보유 일자
- 같은 굿즈 다중 발급 시 수량 표시

### 4.4 EXTERNAL_ARTIST 정합

EXTERNAL_ARTIST는 자기 아티스트 굿즈 **등록만** 가능. 회원 발급·회수는 OPERATOR가 처리 (POLICY 검수 가능).

### 4.5 굿즈 회수 정책

OPERATOR (어뷰징 적발) 또는 SUPER (자산 회수, USR §7.4) 시 회수:
- 활동 로그 + 회원 알림
- DUK·EVT 영향 X (별도 자산)
