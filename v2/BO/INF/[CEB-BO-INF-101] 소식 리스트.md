## 1. 화면 목적

아티스트의 소식·일정·공지를 통합 조회·관리하는 화면. 운영자는 타입 탭으로 종류를 전환하며 상태를 모니터링하고, 신규 콘텐츠 작성으로 진입한다. 개별 콘텐츠는 [CEB-BO-INF-201] 작성·상세로 이동한다.

### 진입 경로

- 사이드 메뉴 > 아티스트 > 소식/일정

---

## 2. UI 설명

### 2.1 상단 영역

- Breadcrumb: `홈 > 아티스트 > 소식/일정`
- 화면 타이틀: 소식/일정
- 우측 상단 액션: [작성하기] (Primary)

### 2.2 타입 탭

| 탭 | 내용 |
| --- | --- |
| 소식 | 아티스트 뉴스·소식 |
| 일정 | 콘서트·팬미팅·방송 등 일정 |
| 공지 | 운영 공지·중요 안내 |

- 탭 전환 시 해당 타입 목록으로 필터링된다. 기본 탭은 소식.

### 2.3 상태 카운터 (3개)

| 카운터 | 의미 |
| --- | --- |
| 게시 | 앱 노출 중 콘텐츠 수 |
| 임시저장 | 작성 중(게시 전) 수 |
| 보관 | 노출 종료 수 |

- 각 카운터 클릭 시 해당 상태로 필터링된다.

### 2.4 검색·필터 영역

| 구분 | 요소 | 타입 | 설명 |
| --- | --- | --- | --- |
| 필터 | 상태 | Dropdown | 상태(전체) / 게시 / 임시저장 / 보관 |
| 필터 | 그룹 | Dropdown | 전체 / 연결 아티스트(그룹) |
| 입력 | 제목 | Text Input | 부분 일치 검색 (입력 안내: "제목 입력") |
| 버튼 | 초기화 | Dark Button | 탭 외 필터·검색어 리셋 |

### 2.5 목록 영역 (테이블)

| 컬럼 | 데이터 예시 | 설명 |
| --- | --- | --- |
| 상태 | 게시 / 임시저장 / 보관 | 상태 뱃지 |
| 타입 | 소식 / 일정 / 공지 | 타입 뱃지 |
| 제목 | "V01D 신곡 Tug of War MV 공개" | 한국어 우선 표시 |
| 그룹 | V01D | 연결 아티스트 |
| 공식 | 공식 / — | 공식 출처 플래그 여부 (공지는 해당 없음) |
| 일시 | 일정: 2026.05.18 12:00 잠실 MBC / 소식·공지: 게시일 | 일정은 일시·장소, 소식·공지는 게시일 |

- 정렬: 게시일(일정은 일시) 내림차순 고정.
- Row 클릭 → [CEB-BO-INF-201] 작성·상세 이동.
- 페이지네이션: 하단 중앙.

> 시각적 스타일은 피그마 참조

## 4. 규칙·로직

### 4.1 권한

- 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.
- [작성하기]는 쓰기 권한 보유 그룹만 노출된다.

### 4.2 정렬·필터

- 기본 정렬: 게시일(일정은 일시) 내림차순. 사용자 변경 불가.
- 타입 탭 + 상태 필터 + 그룹 필터 + 검색어 = AND 조합.
- 제목 검색은 한국어·영어·일본어 통합 부분 일치.
