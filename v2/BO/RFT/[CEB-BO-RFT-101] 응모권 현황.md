## 2. 화면 구성

### 2-1. 헤더

페이지 제목 우측에 **기간 필터 5종** 토글 버튼:

| 옵션 | 의미 |
|---|---|
| 오늘 | 오늘 0시 ~ 현재 |
| 이번 주 | 이번 주 월요일 0시 ~ 현재 |
| 이번 달 | 이번 달 1일 0시 ~ 현재 (기본값) |
| 올해 | 올해 1월 1일 0시 ~ 현재 |
| 전체 | 누적 전체 (기간 제한 없음) |

기간 변경 시 통계 카드 + 분포 패널의 발급·사용 수치가 해당 기간 기준으로 재집계된다. **보유 카드는 시점 스냅샷이므로 기간 필터 영향을 받지 않는다**.

### 2-2. 통계 카드 (가로 3개)

| 카드 | 의미 |
|---|---|
| 총 발급 | 기간 내 발급된 응모권 누적 수량 (에메랄드 톤) |
| 총 보유 | **현재 시점** 회원들이 보유 중인 응모권 잔액 합계. 기간 필터 영향 없음 |
| 총 사용 | 기간 내 사용된 응모권 누적 수량 (회색 톤) |

### 2-3. 출처별 분포 패널 (풀폭, grid-cols-1)

가로 풀폭 단일 패널에 발급·사용 분포를 상하로 분리하여 노출. 정규 발급 출처 5종 + 사용 출처 1종 = 정규 6종.

#### 상단: 정규 발급 출처 분포 (에메랄드 톤 배지 "발급 5종")

- 막대 5개: **퀘스트 보상 / PM·ST 보상 / BIVE 보유혜택 / 덕력 랭킹 보상 / GP로 응모권 구매**
- 각 항목에 "+{N}장 ({비중%})" 표시
- **막대 클릭 시 [CEB-BO-RFT-201]로 이동하여 해당 출처 필터 자동 적용**

#### 하단: 출처별 사용 분포 (rose 톤 배지 "사용 1종")

- 막대 1개: 래플 응모 사용
- 항목에 "-{N}장 (100%)" 표시
- **막대 클릭 시 [CEB-BO-RFT-201]로 이동하여 사용 출처 필터 자동 적용**

> ⚠️ **일일 미션 카드 제외** — 일일 미션은 응모권을 발급하지 않음 ([CEB-BO-FQ-501] 덕력 25만 적립).

### 2-4. 최근 변동 — 좌우 분할 (각 5건 고정)

응모권 발급·사용은 본질이 다른 데이터(+/-)이므로 한 테이블에 섞어 표시하면 운영자 정신 모델이 혼란스러워진다. 좌우 분할로 동시 비교 가능하도록 배치.

`grid-cols-2 gap-4` 좌우 분할:

| 패널 | 좌 | 우 |
|---|---|---|
| 제목 | 최근 발급 (5건 배지, 에메랄드 톤) | 최근 사용 (5건 배지, rose 톤) |
| 데이터 | 변동 > 0인 최신 5건 | 변동 < 0인 최신 5건 |
| 더 보기 | "더 보기 →" 클릭 시 [CEB-BO-RFT-201]?type=ISSUED 진입 (변동 유형 발급 사전 선택) | "더 보기 →" 클릭 시 [CEB-BO-RFT-201]?type=USED 진입 (변동 유형 사용 사전 선택) |
| 빈 상태 | "발급 이력이 없습니다." | "사용 이력이 없습니다." |

각 패널의 테이블 컬럼:

| 컬럼 | 표시 내용 |
|---|---|
| 일시 | 변동 발생 시각 (YYYY.MM.DD HH:mm) |
| 회원 | 회원 닉네임 — 클릭 시 `target=_blank`로 [CEB-BO-USR-201] 회원 상세 새 탭 진입 |
| 변동 | +N장 (좌 패널, 에메랄드) / -N장 (우 패널, rose) |
| 출처 | 한국어 출처명 (좁아진 너비를 고려해 호버 시 풀 텍스트 툴팁) |

> 아티스트 컨텍스트·참조 컬럼은 좁아진 너비로 인해 제거. 풀 정보는 RFT-201 "더 보기" 진입으로 확인.

## 4. 상태별 동작 분기

본 화면은 **상태 변경 액션이 없는 조회 전용 화면**이므로 상태별 분기 대신 데이터 가용성 분기만 존재.

| 데이터 조건 | 동작 |
|---|---|
| 기간 내 변동 0건 | 통계 카드 3개 모두 "0", 분포 패널 막대 모두 0% (회색), 최근 변동 좌·우 패널 각각 "발급 이력이 없습니다." / "사용 이력이 없습니다." 빈 상태 |
| 분포 0% 카운트 | 막대 길이 0%, 텍스트만 회색 처리 |
| 권한 없음 진입 | [관리자 > 권한관리] 설정에 따라 화면 자체 미노출 또는 접근 차단 |

## 5. 검증 규칙

| 규칙 | 내용 |
|---|---|
| 데이터 캐시 | 통계·분포 패널 = 5분 캐시 (실시간 대시보드 아님). 최근 변동 = 0분 캐시 (요청 시점 최신) |
| 보유 카드 시점 | **현재 시점 스냅샷**. 기간 필터 변경에도 값 고정 |
| 막대 클릭 → 필터 이동 | 분포 막대 클릭 시 RFT-201의 출처 드롭다운에 해당 출처 자동 선택. 기간 필터는 RFT-101 선택값과 연계되지 않음 (RFT-201은 자체 기간 정책) |
| 변동 데이터 | append-only 영구 보존. 만료·환수 없음 ([CEB-BO-010] §5 정합) |
