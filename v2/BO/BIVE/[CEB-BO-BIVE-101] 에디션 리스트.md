## 1. 화면 개요

BIVE NFT의 묶음 단위인 **에디션**을 생성·조회·수정·삭제하는 운영자용 리스트 화면. 각 에디션별로 등록된 BIVE 관리 페이지([CEB-BO-BIVE-102])로 진입하는 허브 역할.

| 항목 | 값 |
|---|---|
| 화면 경로 | `/bive/editions` |
| Breadcrumb | 홈 / BIVE / 에디션 관리 |
| 페이지 제목 | 에디션 관리 |
| 진입 경로 | 사이드바 > BIVE > 에디션 관리 |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |

## 2. 화면 구성

### 2-1. 상단 컨트롤 바

- **정렬 드롭다운**: 정렬 기준(생성일시) + 방향(오름차순/내림차순)
  - 기본: 생성일시 내림차순
- **검색 입력**: 에디션명 부분 일치 (한국어/영어 모두 가능), placeholder "에디션명 입력"
- **[조회] 버튼**: 현재 조건으로 테이블 재조회
- **[+ 에디션 생성] 버튼** (우측 고정): 클릭 시 [CEB-BO-BIVE-201-CREATE] 모달 오픈

### 2-2. 데이터 테이블

| 컬럼 | 설명 |
|---|---|
| ID | 내부 일련번호 |
| 에디션명 (한국어) | 한국어 명칭 표시 |
| 등록된 BIVE | 본 에디션에 등록된 BIVE 수 |
| 총 발행 | 본 에디션에 등록된 BIVE의 총 민팅 수 (누적) |
| 생성일시 | `YYYY.MM.DD hh:mm` |
| 생성 관리자 | 최초 생성한 관리자 이름 |
| 관리 | 톱니 아이콘 — 클릭 시 [CEB-BO-BIVE-201-EDIT] 수정/삭제 모달 오픈 |

**행 인터랙션**:

- 행(ID~생성 관리자 컬럼) 클릭 → [CEB-BO-BIVE-102] BIVE 관리 페이지로 이동
- [관리] 컬럼 톱니 아이콘 클릭 → 수정/삭제 모달 오픈

### 2-3. 페이지네이션

- 하단 페이지네이션: `Previous / 1, 2, 3 … / Next`
- 기본 페이지 크기: 20행

## 3. 화면에 노출되는 정보

| 위치 | 정보 | 의미 |
|---|---|---|
| 상단 컨트롤 바 | 정렬·검색·조회·[+ 에디션 생성] | 운영자 액션 진입 |
| 테이블 | ID·에디션명·등록 BIVE·총 발행·생성일시·생성 관리자·관리 | 에디션 핵심 메타 + 운영 지표 |
| 페이지네이션 | 페이지 이동 | 대량 데이터 탐색 |

## 4. 상태별 동작 분기

| 상황 | 동작 |
|---|---|
| 데이터 로딩 중 | Skeleton 테이블 |
| 데이터 없음 | "생성된 에디션이 없습니다." 안내 |
| 검색·필터 결과 없음 | "조건에 맞는 에디션이 없습니다." 안내 |
| 행 클릭 | [CEB-BO-BIVE-102] BIVE 관리 페이지로 이동 |
| [관리] 톱니 아이콘 클릭 | [CEB-BO-BIVE-201-EDIT] 수정/삭제 모달 오픈 |
| [+ 에디션 생성] 클릭 | [CEB-BO-BIVE-201-CREATE] 생성 모달 오픈 |
| 권한 없음 진입 | 화면 미노출 또는 액션 버튼 비활성 |
| 네트워크 오류 | 토스트 "목록을 불러오지 못했습니다. 다시 시도해 주세요." |

## 5. 검증 규칙

| 검증 | 내용 |
|---|---|
| 에디션 수정·삭제 가능 조건 | 등록된 BIVE 0건 + 민팅 기록 0건일 때만 수정·삭제 가능. 한 건이라도 있으면 입력 필드·삭제 버튼 모두 비활성 |
| 검색 | 에디션명 부분 일치, 대소문자 무시 |
| 정렬 | 상단 드롭다운 기준 (생성일시 + 방향) |
| 권한 | [관리자 > 권한관리] BIVE 쓰기 권한 보유자에게만 [+ 에디션 생성]·[관리] 버튼 노출 |

## 6. 외부 영역 연동

| 영역 | 운영 동작 | 사용처 |
|---|---|---|
| 권한 관리 | BIVE 대메뉴 읽기/쓰기 권한 조회 | 액션 버튼 노출 분기 |
| BIVE 관리 (하위) | 에디션별 BIVE 수·민팅 수 집계 | 테이블 컬럼 "등록된 BIVE" + "총 발행" 값 |
