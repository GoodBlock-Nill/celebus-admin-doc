## 1. 화면 개요

특정 **에디션**에 속한 BIVE 목록을 조회·검색·필터링하고, 신규 BIVE 등록 진입 및 개별 BIVE 상세로 이동하는 인덱스 역할. 운영 중인 상태(Active/Inactive/Draft) 항목을 한눈에 파악하고, 민팅 캠페인 연결 수 + 발행 수 등 핵심 지표를 확인한다.

| 항목 | 값 |
|---|---|
| 화면 경로 | `/bive/editions/{editionId}` |
| Breadcrumb | 홈 / BIVE / 에디션 관리 / {에디션명} BIVE 관리 |
| 페이지 제목 | "{에디션명} BIVE 관리" |
| 진입 경로 | [CEB-BO-BIVE-101] 에디션 리스트의 행 클릭 |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |

## 2. 화면 구성

### 2-1. 상단 바

- **페이지 제목**: "{에디션명} BIVE 관리"
- **[+ BIVE 등록] 버튼** (우측 상단): 클릭 시 [CEB-BO-BIVE-202-CREATE] BIVE 등록 페이지로 이동

### 2-2. 필터·검색 영역

- **상태 필터** (드롭다운): `전체(기본) / Active / Inactive / Draft`
- **등급 필터** (드롭다운): `전체(기본) / Event / Normal / Special / Wicked / Next / Final`
- **검색 대상 드롭다운**: `명칭(기본) / 아티스트 / 멤버`
- **검색 입력**: placeholder "검색어 입력"
- **[조회] 버튼**: 현재 조건으로 테이블 재조회

### 2-3. 데이터 테이블

| 컬럼 | 설명 |
|---|---|
| 상태 | 배지 — Active(녹색) / Inactive(빨강) / Draft(노랑) |
| 명칭 | `{멤버명} {등급}-{등급번호}` 자동 생성 (예: "하시모토 칸나 Normal-001") |
| 아티스트 | 아티스트 그룹명 |
| 멤버 | 멤버명 |
| 등급 | Event / Normal / Special / Wicked / Next / Final |
| 등급번호 | 최대 5자리 숫자 |
| 민팅 캠페인 | 본 BIVE에 연결된 민팅 캠페인 수 (없으면 "-") |
| 발행 수 | 누적 발행량 (없으면 "-") |
| 등록일시 | `YYYY.MM.DD` |

**행 인터랙션**:

- 행 전체 클릭 → [CEB-BO-BIVE-202] BIVE 상세로 이동 (기본정보 탭)

**정렬**:

- 기본: 등록일시 내림차순 (최신순)

### 2-4. 페이지네이션

- 하단 페이지네이션: `Previous / 페이지 번호 / Next`
- 기본 페이지 크기: 20행

## 3. 화면에 노출되는 정보

| 위치 | 정보 | 의미 |
|---|---|---|
| 상단 바 | 페이지 제목·[+ BIVE 등록] | 에디션 컨텍스트 + 운영 액션 진입 |
| 필터·검색 | 상태·등급·검색 대상·검색어·조회 | 다양한 조건 탐색 |
| 테이블 | 상태·명칭·아티스트·멤버·등급·등급번호·민팅 캠페인·발행 수·등록일시 | BIVE 핵심 메타 + 운영 지표 |
| 페이지네이션 | 페이지 이동 | 대량 데이터 탐색 |

## 4. 상태별 동작 분기

| 상황 | 동작 |
|---|---|
| 데이터 로딩 중 | Skeleton 테이블 |
| 데이터 없음 | "등록된 BIVE가 없습니다." 안내 |
| 검색·필터 결과 없음 | "조건에 맞는 BIVE가 없습니다." 안내 |
| 행 클릭 | [CEB-BO-BIVE-202] BIVE 상세 (기본정보 탭) 진입 |
| [+ BIVE 등록] 클릭 | [CEB-BO-BIVE-202-CREATE] BIVE 등록 페이지 진입 |
| 권한 없음 진입 | 화면 미노출 또는 [+ BIVE 등록] 비활성 |

## 5. 검증 규칙

| 검증 | 내용 |
|---|---|
| 검색 — 명칭 | `{멤버명} {등급}-{등급번호}` 전체 텍스트 부분 일치 |
| 검색 — 아티스트 | 아티스트명 부분 일치 |
| 검색 — 멤버 | 멤버명 부분 일치 |
| 필터 결합 | 상태 × 등급 × 검색 = AND 조건 |
| 정렬 | 등록일시 내림차순 (최신순) |
| 권한 | [관리자 > 권한관리] BIVE 쓰기 권한 보유자에게만 [+ BIVE 등록] 노출. 읽기 권한만 있으면 행 클릭(상세 이동)만 가능 |

## 6. 외부 영역 연동

| 영역 | 운영 동작 | 사용처 |
|---|---|---|
| 아티스트 관리 | 아티스트 활성 목록 조회 | BIVE 등록 시 드롭다운 |
| 멤버 관리 | 멤버 활성 목록 조회 | BIVE 등록 시 드롭다운 |
| 민팅 캠페인 (BIVE-103) | BIVE별 연결 캠페인 수 집계 | 테이블 컬럼 "민팅 캠페인" |
