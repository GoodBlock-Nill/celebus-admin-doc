## 1. 모달 개요

신규 **에디션**(BIVE 묶음 단위)을 생성한다. 다국어 명칭 3종(한국어/영어/일본어)만 입력하면 생성되며, 생성 직후 [CEB-BO-BIVE-101] 리스트에 추가된다. 에디션 자체는 명칭 외 다른 메타 데이터가 없으므로 별도 상세 페이지 없이 본 모달과 [CEB-BO-BIVE-201-EDIT] 수정 모달로 완결된다.

| 항목 | 값 |
|---|---|
| 트리거 | [CEB-BO-BIVE-101] 에디션 리스트 우측 상단 [+ 에디션 생성] 버튼 |
| 차단 조건 | [관리자 > 권한관리] BIVE 쓰기 권한 없을 때 트리거 버튼 자체 미노출 |
| 영향 범위 | [CEB-BO-BIVE-101] 리스트에 신규 행 추가 |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |

## 2. 화면 구성

### 2-1. 모달 헤더

- 타이틀: "에디션 생성"
- 우측 상단: 닫기 [X] 아이콘 (ESC·오버레이 클릭으로도 닫기)

### 2-2. 안내 문구

> "에디션에 BIVE를 등록 후 통합 관리합니다."
> "에디션 명칭은 등록된 BIVE가 없을 때 수정이 가능합니다."

### 2-3. 입력 폼 (다국어 명칭 3종 모두 필수)

| 필드 | 입력 방식 | 검증·옵션 | 필수 |
|---|---|---|---|
| 한국어 | Text Input | placeholder "에디션명 입력" | ★ |
| 영어 | Text Input | placeholder "에디션명 입력" | ★ |
| 일본어 | Text Input | placeholder "에디션명 입력" | ★ |

### 2-4. 하단 액션

- 우측 하단 [생성하기] (Primary, 인디고)

## 3. 화면에 노출되는 정보

| 위치 | 정보 | 의미 |
|---|---|---|
| 모달 헤더 | "에디션 생성" + 닫기 [X] | 모달 식별 + 종료 |
| 안내 문구 | 에디션 정의 + 명칭 수정 가능 시점 안내 | 운영자 사전 안내 |
| 입력 폼 | 한국어·영어·일본어 명칭 3종 | 다국어 입력 |
| 하단 액션 | [생성하기] | 제출 |

## 4. 상태별 동작 분기

| 상황 | 동작 |
|---|---|
| 모달 오픈 직후 | 한국어 입력 필드에 자동 포커스 |
| 입력 필드 모두 유효 입력 | [생성하기] 활성화 (인디고) |
| 입력 필드 1개라도 비어 있음 | [생성하기] 비활성화 (회색) |
| [생성하기] 클릭 (정상) | 토스트 "에디션이 생성되었습니다." → 모달 닫힘 → [CEB-BO-BIVE-101] 리스트에 신규 행 추가 |
| [생성하기] 클릭 (네트워크 오류) | 토스트 "생성에 실패했습니다. 다시 시도해주세요." 모달 유지 |
| 닫기 [X] / ESC / 오버레이 클릭 | 모달 닫힘 (입력값 폐기) |
| 권한 없음 | 트리거 버튼 자체 미노출이라 본 모달 진입 불가 |

## 5. 검증 규칙

| 검증 | 내용 |
|---|---|
| 다국어 3종 모두 입력 | 한국어·영어·일본어 모두 공백 제외 1자 이상이어야 [생성하기] 활성 ([CEB-BO-011] §5 다국어 필수 정합) |
| 명칭 중복 허용 | 동일 명칭 에디션 생성 가능 (운영자 책임) |
| 권한 | [관리자 > 권한관리] BIVE 쓰기 권한 보유자만 접근 |
