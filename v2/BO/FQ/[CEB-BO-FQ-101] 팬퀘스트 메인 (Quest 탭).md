## 1. 화면 목적

팬퀘스트 서비스의 Quest 목록을 조회·관리하는 어드민 메인 화면. 운영자는 본 화면에서 등록된 모든 Quest의 상태·검수 대기 건수를 한눈에 파악하고, 신규 Quest 생성·반려사유 설정·개별 Quest 상세로 진입한다.

### 주요 기능

- 등록된 전체 Quest 목록 조회 및 검색
- 상태별 Quest 필터링 (진행중 / 임시저장 / 종료)
- 신규 Quest 생성 진입
- 반려사유 데이터베이스 진입
- Quest 상세 화면 진입

### 진입 경로

- 사이드 메뉴 > 팬퀘스트 > Fan Quest 관리 진입 시 기본 노출 (Quest 탭 활성)

## 2. UI 설명

### 2.1 상단 영역 (Top Area)

- Breadcrumb: `홈 > 팬퀘스트 > Fan Quest 관리`
- Tab Menu (상호 배타):
  - **Quest** (현재 화면)
  - **Raffle** → [CEB-BO-FQ-102] 이동

### 2.2 검색·필터 영역

| 구분 | 요소명 | 타입 | 설명 |
| ---- | ------ | ---- | ---- |
| 필터 | 상태 | Dropdown | 전체 / 진행중 / 임시저장 / 종료 (Default: 전체) |
| 필터 | 검색 대상 | Dropdown | 타이틀 / 아티스트 (Default: 타이틀) |
| 입력 | 키워드 | Text Input | 부분 일치 검색. Enter 또는 검색 아이콘으로 실행. Placeholder는 검색 대상 따라 변경 |
| 버튼 | 초기화 | Dark Button | 검색 조건 초기 상태 리셋 |
| 버튼 | 반려사유 설정 | Secondary Button | [CEB-BO-FQ-101-REJETSET] 이동 |
| 버튼 | 생성하기 | Primary Button | [CEB-BO-FQ-101-CREATE] 이동 |

> 시각적 스타일은 피그마 참조

### 2.3 목록 영역 (List Table)

| 컬럼명 | 데이터 타입 | 설명 |
| ------ | ----------- | ---- |
| 상태 | Badge | Quest 상태 표시 ([CEB-BO-006] §7.1 라이프사이클) |
| 타이틀 | Text | Quest 타이틀 (KO 우선 노출, 다국어 — [CEB-BO-006] §5) |
| 아티스트 | Text | 연결된 아티스트명 |
| 기간 | Date Range | 시작(게시)일 ~ 마감(종료)일 (YYYY.MM.DD ~ YYYY.MM.DD) |
| 검토 필요 | Number | 검수 대기 중인 유저 제출물 수량 (실시간) |

- Row 클릭 → [CEB-BO-FQ-201] 퀘스트 상세 (기본정보 탭) 이동
- 마감일 전 수동 종료 시 기간 종료일은 종료 처리 일자로 변경

### 2.4 페이지네이션

## 4. 규칙·로직

### 4.1 권한별 기능 분기

[CEB-BO-006] §4 권한 매트릭스 적용. 본 화면 한정 표:

| 권한 | 화면 접근 | 조회 | [반려사유 설정] | [생성하기] | Row 클릭 |
| ---- | --------- | ---- | --------------- | ---------- | -------- |
| SUPER (슈퍼관리자) | ✅ | ✅ | ✅ | ✅ | ✅ |
| OPERATOR (운영자) | ✅ | ✅ | ✅ | ✅ | ✅ |
| POLICY (정책관리자) | ✅ | ✅ (조회 전용) | ❌ | ❌ | ✅ |
| EXTERNAL_ARTIST (아티스트사) | ✅ | ✅ (자기 아티스트만) | ❌ | ✅ (자기 아티스트만) | ✅ (자기 아티스트만) |

> EXTERNAL_ARTIST 가 본 화면에 진입하면 목록은 자기 아티스트가 매핑된 Quest 만 노출 (서버 측 필터링).

### 4.2 정렬 기준

| 정렬 기준 | 정렬 방식 | 비고 |
| --------- | --------- | ---- |
| 마감(종료)일 | 오름차순 (임박순) | 기본 정렬 |

### 4.3 검토 필요 카운트

- 상태가 **임시저장(Draft)** 인 경우 → 0 (참여 불가, 고정)
- 상태가 **진행중(Active)** 또는 **종료(Ended)** 인 경우 → 검수 미처리(Pending) 건수 실시간 표시
- 카운트 0 표기: "0" (대시 사용 안 함)

### 4.4 상태 정의

[CEB-BO-006] §7.1 Quest 라이프사이클 참조. 본 화면 노출 라벨:

| 상태 코드 | 라벨 (KO) | Badge 색상 | 검토 필요 컬럼 |
| ---- | ---- | ---- | ---- |
| Draft | 임시저장 | Gray | 0 (고정) |
| Active | 진행중 | Green | 실시간 카운트 |
| Ended | 종료 | Black | 실시간 카운트 (검수 잔여건) |

### 4.5 검색 조건

- 상태 필터 + 키워드 검색은 AND 조건
- 키워드 검색은 부분 일치(LIKE) 지원
- 검색어 미입력 시 해당 조건 무시 (상태 필터만 적용)
- 검색 대상이 "아티스트"일 때: 아티스트 그룹·멤버명 통합 검색
