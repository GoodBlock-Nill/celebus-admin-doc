## 1. 화면 목적

이미 생성된 Quest의 정보를 변경. Quest 상태(Draft / Active)에 따라 수정 가능 항목 범위를 엄격히 제어 — 운영 유연성과 데이터 무결성 동시 확보. [CEB-BO-006] §7.1 라이프사이클 적용.

### 주요 기능

- Draft 상태: 모든 항목 수정 가능
- Active 상태: 기본 정보(타이틀/설명/유저 가이드/대표 이미지/연관 링크/마감 일시 연장만) 수정 가능. 핵심 정보(아티스트/타입/보상) 비활성화
- 다국어 입력 패널(KO/EN/JP)을 통한 콘텐츠 수정

### 진입 경로

- [CEB-BO-FQ-201] 퀘스트 상세 (기본정보 탭) > [수정하기] 버튼 클릭

### 2.2 화면 레이아웃 (3-Column)

생성 화면([CEB-BO-FQ-101-CREATE])과 동일 레이아웃이지만 상태에 따라 비활성화 처리.

```
+----------------------------+----------------------------+----------------------------+
|        기본 정보           |        퀘스트 설정          |        보상 설정            |
|                            |                            |                            |
|  타이틀  *      [입력완료] |  아티스트  *    [Dropdown]  |  보상 종류  *   [Dropdown]  |
|  설명    *      [입력완료] |  퀘스트 타입 * [Dropdown]   |  응모권 수량    [Dropdown]  |
|  유저 가이드 * [입력완료]  |  마감 일시  *  [DatePicker] |  NFT 이벤트    [Dropdown]  |
|  대표 이미지 *  [썸네일]   |  연관 링크     [+ 링크 추가]|                            |

```

### 2.3 Active 상태 안내 배너

Active 상태로 진입 시 헤더 하단에 노란색 배너:

> "진행 중 상태에서는 기본 정보(타이틀/설명/유저 가이드/대표 이미지/연관 링크)와 마감 일시 연장만 수정 가능합니다."

### 4.6 마감 일시 수정 로직

- **Draft**: 현재 시점 +1일 이후 자유 수정
- **Active 연장**: 기존 마감일 이후로만 수정 가능 (단축 불가). 연장 시 활동 로그 기록

### 4.7 연관 링크 유효성

[CEB-BO-FQ-101-CREATE] §4.5 동일 정책. 미입력 시 [수정하기] 활성화에 영향 없음.

### 4.8 [수정하기] 활성화 조건

| 상태 | 조건 |
| ---- | ---- |

| Active | 수정 가능 항목 내 변경 사항 1건 이상 + 유효성 통과 |
