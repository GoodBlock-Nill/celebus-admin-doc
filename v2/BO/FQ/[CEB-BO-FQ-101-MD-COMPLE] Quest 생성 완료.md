## 1. 화면 목적

Quest 생성 화면([CEB-BO-FQ-101-CREATE])에서 모든 필수 입력 완료 후 [생성하기] 클릭 시 최종 생성 의사 확인. 의도하지 않은 즉시 생성 방지 가드레일.

### 진입 경로

- [CEB-BO-FQ-101-CREATE] > 모든 필수값 입력 + 유효성 통과 + [생성하기] 버튼 클릭

## 4. 규칙·로직

### 4.1 권한 검증

호출자([CEB-BO-FQ-101-CREATE])의 [생성하기] 버튼 활성화 조건이 권한을 보장. 본 모달에서 추가 검증 없음.

### 4.2 생성 프로세스

- [확인] 클릭 → 서버 API 호출 (Quest 생성)
- 신규 Quest는 **임시저장(Draft)** 상태로 저장 ([CEB-BO-006] §7.1 라이프사이클)
- 게시는 별도 프로세스 ([CEB-BO-FQ-201-MD-OPEN])
- 다국어 입력은 임시저장 1언어 허용 ([CEB-BO-006] §5.2)

[CEB-BO-000] §7 + §9 적용:

| 상태 | 표시 |
| --- | --- |
| Loading (서버 전송 중) | [확인] 버튼에 스피너, 비활성화 |
