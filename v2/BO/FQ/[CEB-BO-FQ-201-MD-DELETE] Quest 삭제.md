## 1. 화면 목적

Draft 상태 Quest를 영구 삭제. 게시된 Quest는 삭제 불가 (종료만 가능). [CEB-BO-006] §7.1 라이프사이클 정책 준수.

### 진입 경로

- [CEB-BO-FQ-201] (Draft 상태) > [삭제하기] 클릭

## 4. 규칙·로직

### 4.1 권한별 분기

| 권한 | 사용 |
| --- | --- |
| SUPER | ✅ (Draft 한정) |
| OPERATOR | ✅ (Draft 한정) |
| POLICY | ❌ |
| EXTERNAL_ARTIST | ❌ |

### 4.2 삭제 가능 조건

- **Draft 상태만 가능** ([CEB-BO-006] §7.1)
- Active/Ended 상태는 [삭제하기] 버튼 미노출 (FQ-201 §2.2)
- 삭제 시점에 상태가 Active로 변경된 경우(동시성) → 서버 측에서 차단

### 4.3 삭제 프로세스

- [확인] → DELETE API 호출
- 데이터 영구 제거 (Soft delete X, Hard delete)
- 활동 로그는 보존 ([CEB-BO-000] §6 append-only — 로그에는 삭제 기록만 남음)
