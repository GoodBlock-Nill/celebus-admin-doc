Quest 수정 화면([CEB-BO-FQ-201-EDIT])에서 [수정하기] 클릭 시 변경 사항 최종 확인 후 서버에 반영.

- [CEB-BO-FQ-201-EDIT] > 변경 사항 1건 이상 + 유효성 통과 + [수정하기] 버튼 클릭

## 4. 규칙·로직

### 4.1 권한 검증

호출자([CEB-BO-FQ-201-EDIT])의 [수정하기] 활성화 조건이 권한 보장. 본 모달 추가 검증 없음.

### 4.2 수정 프로세스

- [확인] → PUT API 호출
- 변경 항목만 서버 전송 (delta)
- 활동 로그 기록 ([CEB-BO-006] §6)
- 마감 일시 연장 시 별도 로그 행
