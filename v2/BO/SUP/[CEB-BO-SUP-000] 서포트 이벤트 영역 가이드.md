## 1. 개요

서포트 이벤트는 팬덤이 **보유 덕력을 응원(차감)** 하여 아티스트 응원(생일 카페·콘서트 응원·차트 인증 등)의 목표 응원량 달성에 함께 참여하는 이벤트다. 운영자는 이벤트를 생성·게시하고, 목표 달성 후 집행·결과물 등록까지 한 화면 흐름에서 관리한다.

| 항목 | 내용 |
| --- | --- |
| 사이드 메뉴 | **아티스트 > 서포트 관리** (덕력관리·팬덤레벨과 같은 아티스트 메뉴 묶음) |
| 응원 방식 | 회원이 **보유 덕력을 차감**하여 응원 (1 덕력 = 1 응원, 별도 단가 없음) |
| 덕력·랭킹 영향 | 응원 덕력은 **랭킹·팬덤 레벨에 영향 없음** (보유 덕력에서만 차감, [CEB-DUK-101]·[CEB-DUK-201] 정합) |
| 멀티 아티스트 | 이벤트는 그룹(아티스트)별 분리, 응원 덕력은 해당 아티스트에 귀속 ([CEB-000] §5.2.2) |
| 회원 화면 | 앱 [CEB-DUK-201] 서포트 이벤트 / [CEB-DUK-202] 서포트 이벤트 상세 |
| 회원 통합 노출 | [CEB-BO-USR-201] 회원 상세 활동내역 |
| 권한 | [CEB-BO-SYS-102] 권한 관리 그룹 설정에 따른다 |
| 활동 로그 | [CEB-BO-000] 공통 정책 활동 로그 규격 |

## 4. 영역 고유 정책

### 4.1 서포트 이벤트 라이프사이클 (7상태)

운영자 관점의 전체 상태는 **임시저장**과 회원 앱에 노출되는 공개 6상태(모집중·달성·집행중·완료·미달성종료·집행취소)로 구성된다.

```
임시저장 ─[게시]→ 모집중 ┬─(자동: 누적 응원 ≥ 목표)──→ 달성 ┬─[집행 시작]→ 집행중 ─[결과물 등록·완료]→ 완료
                          │                                  └─[집행 취소]──→ 집행취소 (전액 반환·자동)
                          ├─(자동: 마감 + 미달성)──────→ 미달성종료 (전액 반환·자동)
                          └─[강제 종료]──────────────→ 미달성종료 (전액 반환·자동)
                                                          집행중 ─[집행 취소]→ 집행취소 (전액 반환·자동)
```

| 상태 | 의미 | 앱 노출 | 전이 주체 |
| --- | --- | --- | --- |
| 임시저장 | 작성 중, 미게시 | 노출 안 됨 | 운영자 (생성·수정) |
| 모집중 | 응원 진행 중 | 응원 가능 | 운영자 [게시] |
| 달성 | 누적 응원이 목표 도달, 집행 대기 | 응원 차단·집행 대기 표시 | 서버 자동 (누적 ≥ 목표) |
| 집행중 | 운영자가 서포트 집행 시작 | "서포트 진행중" 표시 | 운영자 [집행 시작] |
| 완료 | 집행 완료, 결과물 공개 | 결과물(사진·영상) 확인 | 운영자 [결과물 등록·완료] |
| 미달성종료 | 마감까지 미달성 또는 강제 종료 | "미달성·반환 완료" 표시 | 서버 자동(마감+미달성) / 운영자 [강제 종료] |
| 집행취소 | 달성 후 집행 불가로 취소 | "취소·반환 완료" 표시 | 운영자 [집행 취소] |

### 4.2 응원(차감) 정책

회원이 앱에서 서포트 이벤트에 응원하면:

- 보유 덕력에서 응원량만큼 **차감** (1 덕력 = 1 응원)
- 누적 응원이 목표 응원량에 도달하면 서버가 자동으로 **달성** 상태로 전환
- 동일 이벤트 여러 번 응원 가능 (누적 합산), 응원 후 취소 불가
- 남은 목표량을 초과하는 응원은 남은 목표량까지만 차감 (초과분은 보유 덕력 유지)
- 응원 덕력은 **랭킹·팬덤 레벨에 가산되지 않는다** (보유 덕력에서만 차감)

### 4.3 결과물 등록 정책

운영자가 집행 완료 후 결과물을 등록한다:

- 다국어 결과 메시지 (한국어/영어/일본어, 각 200자)
- 결과 사진·영상 (최대 10장, 회원 앱에서 갤러리로 확인)
- 결과물 등록 시 응원 회원에게 결과물 알림 발송 → 앱 [CEB-DUK-201]에서 확인

### 4.4 반환 정책 (전액·자동)

미달성종료·집행취소·강제 종료 시 응원 회원 전원에게 응원 덕력을 **전액 자동 반환**한다.

- 반환 시점: 상태 전이 즉시 (서버 자동 처리, 운영자 수동 반환·추가 인증 절차 없음)
- 반환 덕력은 회원 보유 덕력으로 즉시 복구되어 다른 응원에 재사용 가능
- 반환 알림 발송 + 앱 진입 시 안내
- 달성 후 정상 집행(완료)된 이벤트는 반환 없음 (응원 덕력은 서포트 집행에 사용)

### 4.5 변동 유형

응원·반환·결과물 알림 내역은 [CEB-BO-SUP-401] 응원·반환 변동 내역에서 통합 조회한다.

| 변동 유형 | 의미 | 발생 |
| --- | --- | --- |
| 응원 | 회원이 보유 덕력 차감하여 응원 | 앱 [CEB-DUK-201] 응원 |
| 반환 | 미달성·취소 시 응원 덕력 복구 | 서버 자동 (상태 전이 즉시) |
| 결과물 알림 | 결과물 등록 시 회원 알림 발송 | 운영자 [결과물 등록·완료] |
