## 1. 화면 목적

서포트 이벤트를 통합 조회하고 신규 이벤트를 생성하는 진입 화면. 상태별 현황을 한눈에 파악하고, 그룹·상태·이벤트명으로 검색해 개별 이벤트 상세([CEB-BO-SUP-201])로 진입한다.

| 항목 | 값 |
| --- | --- |
| 화면 경로 | `/artists/support` |
| 진입 경로 | 사이드 메뉴 아티스트 > 서포트 관리 |
| 용도 | 이벤트 조회·검색·생성 진입, 응원 현황 모니터링 |

---

## 2. 화면 구성

### 2.1 헤더

| 요소 | 설명 |
| --- | --- |
| 타이틀 | "서포트 관리" + Breadcrumb (홈 / 아티스트 / 서포트 관리) |
| [변동 내역] | 우측 보조 버튼 → [CEB-BO-SUP-401] 응원·반환 변동 내역 (`/artists/support/history`) |
| [이벤트 생성] | 우측 주요 버튼 → [CEB-BO-SUP-101-CREATE] 생성 화면 (`/artists/support/new`) |

### 2.2 상태 카운터

상태별 이벤트 수를 카드로 표시하며, 카드 클릭 시 해당 상태로 필터 적용(재클릭 시 해제).

| 카드 | 의미 |
| --- | --- |
| 모집중 | 응원 진행 중 |
| 달성 | 목표 도달, 집행 대기 |
| 집행중 | 서포트 집행 진행 중 |
| 완료 | 집행 완료, 결과물 공개 |
| 종료 | 미달성종료 + 집행취소 (반환 완료 묶음) |

> 임시저장은 카운터에서 제외하고 상태 필터 드롭다운으로만 조회 (작성 중 항목은 운영 현황 집계에서 분리).

### 2.3 검색·필터

| 구분 | 요소 | 설명 |
| --- | --- | --- |
| 필터 | 상태 | 전체 / 임시저장 / 모집중 / 달성 / 집행중 / 완료 / 미달성종료 / 집행취소 |
| 필터 | 그룹 | 전체 / 그룹(아티스트) 목록 |
| 입력 | 이벤트명 | 부분 일치 (한국어 기준) |
| 버튼 | 초기화 | 모든 필터·검색어 해제 |

### 2.4 목록 테이블

| 컬럼 | 설명 |
| --- | --- |
| 상태 | 상태 Badge |
| 이벤트명 | 한국어 이벤트명 |
| 그룹 | 응원 대상 아티스트 |
| 기간 | 시작 ~ 마감 (KST) |
| 응원자 수 | 1 덕력 이상 응원한 회원 수 |
| 누적 응원 | 누적 응원 덕력 |
| 달성률 | 누적 응원 / 목표 응원량 (%) |

- Row 클릭 → [CEB-BO-SUP-201] 상세
- 페이지네이션 10건 단위

## 3. 규칙·로직

### 3.1 정렬 기준

기본 정렬: **마감일 임박순 → 생성일 최신순**. 종료(미달성종료·집행취소·완료) 이벤트는 진행 이벤트 하단에 배치.

### 3.2 권한

권한은 [CEB-BO-SYS-102] 권한 관리 그룹 설정에 따른다. 외부 아티스트 계정은 자기 아티스트 이벤트만 조회된다. 목록 조회는 활동 로그를 남기지 않는다.
