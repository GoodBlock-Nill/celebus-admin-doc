## 1. 화면 목적

전체 회원의 서포트 응원·반환·결과물 알림 내역을 통합 조회한다. 고객 응대(CS)와 운영 효과 측정에 사용하는 읽기 전용 화면이다.

| 항목 | 값 |
| --- | --- |
| 화면 경로 | `/artists/support/history` |
| 진입 경로 | [CEB-BO-SUP-101] 리스트 [변동 내역] |

---

## 2. 화면 구성

### 2.1 검색·필터

| 구분 | 요소 | 설명 |
| --- | --- | --- |
| 필터 | 변동 유형 | 전체 / 응원 / 반환 / 결과물 알림 |
| 필터 | 이벤트 | 전체 / 특정 이벤트 |
| 필터 | 그룹 | 전체 / 특정 그룹 |
| 필터 | 기간 | 시작·종료일 (KST) |
| 입력 | 닉네임/회원 | 부분 일치 |
| 버튼 | 초기화 | 필터·검색어 해제 |

### 2.2 목록 테이블

| 컬럼 | 설명 |
| --- | --- |
| 변동 유형 | 응원 / 반환 / 결과물 알림 |
| 회원 | 닉네임 |
| 이벤트명 | 응원 대상 이벤트 |
| 그룹 | 응원 대상 아티스트 |
| 변동 내용 | "+{N} 덕력 응원" / "−{N} 덕력 반환" / "결과물 알림 발송" |
| 일시 | KST |

- 읽기 전용, 최신순 정렬, 페이지네이션 10건 단위
- 동일 회원이 여러 번 응원한 경우 **각 응원 건이 1행**으로 기록된다 (회차별 추적, [CEB-DUK-201] 누적 응원 정합)

---

## 3. 규칙·로직

### 3.1 변동 유형

| 변동 유형 | 의미 | 발생 |
| --- | --- | --- |
| 응원 | 회원이 보유 덕력 차감하여 응원 | 앱 [CEB-DUK-201] 응원 |
| 반환 | 미달성·취소 시 응원 덕력 복구 | 서버 자동 (상태 전이 즉시) |
| 결과물 알림 | 결과물 등록 시 회원 알림 발송 | [CEB-BO-SUP-201] [결과물 등록·완료] |

### 3.2 권한

권한은 [CEB-BO-SYS-102] 권한 관리 그룹 설정에 따른다. 외부 아티스트 계정은 자기 아티스트 내역만 조회된다. 본 화면은 조회 전용으로 액션이 없다.
