## 1. 화면 목적

서포트 이벤트 1건을 신규 작성한다. 작성 완료 시 **임시저장** 상태로 생성되며, 게시는 상세([CEB-BO-SUP-201])에서 별도로 수행한다(래플·게임존과 동일한 2단계 흐름).

| 항목 | 값 |
| --- | --- |
| 화면 경로 | `/artists/support/new` |
| 진입 경로 | [CEB-BO-SUP-101] 리스트 [이벤트 생성] |
| 생성 결과 | 임시저장 상태로 저장 → 리스트로 복귀 |

---

## 2. 입력 폼

### A. 기본 정보 (다국어)

| 필드 | 한국어 | 영어 | 일본어 | 글자수 |
| --- | --- | --- | --- | --- |
| 이벤트명 | 필수 | 필수 | 필수 | 50자 |
| 설명 | 필수 | 필수 | 필수 | 500자 |

### B. 대상·목표

| 필드 | 입력 방식 | 규격 |
| --- | --- | --- |
| 그룹(아티스트) | 드롭다운 | 활성 아티스트 목록에서 선택. 필수 |
| 목표 응원량 | 숫자 입력 | 1 이상 정수 (덕력 단위). 필수 |

> 응원은 보유 덕력 1:1 차감이므로 별도 응원 단가는 없다. 목표 응원량(덕력)만 설정한다.

### C. 일정

| 필드 | 입력 방식 | 규격 |
| --- | --- | --- |
| 시작일시 | 날짜+시간 (KST) | 필수. 게시 후 도달 시 모집 시작 |
| 마감일시 | 날짜+시간 (KST) | 필수. 시작일시 이후. 도달 시 미달성이면 자동 미달성종료(전액 반환) |

### D. 대표 이미지 (필수)

| 항목 | 값 |
| --- | --- |
| 필수 여부 | **[생성하기] 시 필수** (미업로드 시 생성 차단) |
| 권장 비율 | 16:9 (가로) |
| 허용 형식 | JPG / PNG / WEBP |
| 최대 용량 | [CEB-BO-000] 이미지 정책 |
| 미리보기 | 업로드 즉시 미리보기 + 파일명·[제거]로 교체 |

### CTA 버튼

[취소] / [임시저장] / [생성하기]

## 4. 검증 규칙

| 검증 시점 | 검증 항목 |
| --- | --- |
| [임시저장] | 이벤트명 한국어 최소 입력 (그 외 항목 추후 보강 가능) |
| [생성하기] | **기본 정보 다국어 필수 필드 전체 + 그룹 + 목표 응원량 + 시작·마감일시 + 대표 이미지 필수** — 이벤트명·설명 한국어/영어/일본어 모두 입력 + 그룹 선택 + 목표 응원량 1 이상 + 시작·마감일시 입력(마감 > 시작) + **대표 이미지 업로드**. 미충족 시 [생성하기] 비활성 + 누락 항목 안내 |

---

## 5. 시점별 동작

| 상황 | 동작 |
| --- | --- |
| [임시저장] 클릭 | 이벤트명 한국어 검증 → 임시저장 저장 + 리스트 복귀 + 토스트 "임시저장되었습니다" |
| [생성하기] 클릭 | 필수 검증 통과 시에만 활성 → 임시저장 상태로 저장 + 리스트 복귀 + 토스트 "서포트 이벤트가 생성되었습니다 (게시 전 임시저장 상태)" |
| [취소] 클릭 | 입력값 폐기 + 리스트 복귀 (입력값 있으면 확인) |
| 권한 없음 | [CEB-BO-SYS-102] 설정에 따라 화면 미노출 또는 버튼 비활성 |
