## 1. 모달 개요

임시저장 Quest를 진행중 상태로 전환하기 위한 확인 모달.

| 항목 | 값 |
|---|---|
| 트리거 | Quest 상세의 [게시하기] (임시저장 상태에서만 노출) |
| 진입 조건 | Quest 상태가 "임시저장" |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |
| 영향 범위 | Quest 상태 임시저장 → 진행중. 회원 앱 노출 시작 |

## 2. 화면 구성

### 2-1. 헤더

- 모달 제목: "Quest 게시"

### 2-2. 본문

> 확인 버튼을 누르면 Quest가 진행중 상태로 게시됩니다.
> 계속 진행하시겠습니까?

### 2-3. 하단 액션

| 버튼 | 색상 | 동작 |
|---|---|---|
| 취소 | 회색 외곽선 | 모달 닫기 |
| 확인 | 인디고 | Quest 상태 임시저장 → 진행중 전환 + "Quest가 게시되었습니다." 토스트 + 상세 페이지로 복귀 |
