## 1. 화면 개요

기존 에피소드의 마스터 메타데이터를 수정한다. 종료 상태에서는 read-only 안내 후 저장 불가.

| 항목 | 값 |
|---|---|
| 화면 경로 | `/sq/{id}/edit` |
| Breadcrumb | 홈 / 에피소드 / {그룹명} / {에피소드명} / 수정 |
| 페이지 제목 | "에피소드 수정" + 유형 배지 |
| 진입 경로 | [CEB-BO-SQ-202] 에피소드 상세의 [수정] 버튼 |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |

## 2. 화면 구성

### 2-1. 헤더

- 좌측: ← 뒤로가기 → 에피소드 상세 복귀 (취소 확인 다이얼로그)
- 페이지 제목 + 유형 배지 (메인 / 반복)
- 보조: 아티스트 배지 + 그룹 타이틀

### 2-2. 상태별 알림 배너 (조건부)

| 상태 | 톤 | 메시지 |
|---|---|---|
| 종료 | 로즈 | "종료된 에피소드는 수정할 수 없습니다. 새 에피소드 생성을 권장합니다." (모든 필드 read-only) |
| 진행중 | 노랑 | "진행중 에피소드의 수정 사항은 즉시 회원 앱에 반영됩니다." |
| 임시저장 | (배너 없음) | — |

### 2-3. 입력 폼 — 기본 정보 (2열 그리드, v2.2부터)

| 필드 | 입력 방식 | 변경 가능 |
|---|---|---|
| 아티스트 | read-only 배지 (그룹 상속) | ❌ |
| 표시 순서 | **read-only 표시** (`#{표시 순서}` 배지 + "그룹 상세에서 ↑↓로 변경" 안내) | ❌ |

> ⚠️ **v2.2 변경**: 표시 순서 입력 필드 제거. 순서는 **그룹 상세 화면의 ↑↓ 버튼**(임시저장 상태에서만 활성)으로 변경한다. 상태(임시저장/진행중/종료) 표시도 헤더에서 유형 배지만 노출되므로 본 화면에서는 별도 표시하지 않는다.

### 2-4. 다국어 필드

| 필드 | 입력 방식 | 글자수 |
|---|---|---|
| 타이틀 | 한국어/영어/일본어 탭 | 각 100자 |
| 에피소드 설명 | 한국어/영어/일본어 탭 (4행) | 각 200자 |

> 모든 언어 입력 필수. 한 언어라도 비면 저장 비활성.

### 2-5. 메인 이미지

- 좌측: 3:4 비율 드래그앤드롭 존 (현재 이미지 미리보기)
- 우측: URL 직접 입력 + 변경 / 제거 버튼
- 안내: "3:4 비율 권장, 5MB 이하, PNG/JPG/WebP"

### 2-6. 상위 그룹 기간 (read-only)

- 그룹의 시작일자·종료일자 표시 (에피소드 자체 기간 없음, 그룹 SSOT)
- 안내: "기간은 상위 에피소드 그룹의 설정을 따릅니다. 변경하려면 그룹 수정에서 진행하세요."

### 2-7. 에피소드 완료 보상 (메인 에피소드 전용)

> 에피소드 안의 **모든 미션이 완료된 회원에게 자동 지급**되는 보상. 메인 에피소드에서만 노출되며, 반복 에피소드에서는 본 섹션이 숨겨집니다. 진행중 상태 진입 시 회원 기대권 보호 차원에서 본 섹션 전체가 **read-only로 잠금**됩니다.

| 필드 | 입력 방식 | 진행중 잠금 |
|---|---|---|
| 응모권 보상 | 정수 입력 (단위: 장) | ★ |
| 팬덤 포인트 보상 | 정수 입력 | ★ |
| BIVE 보상 | 토글 ON/OFF | ★ |
| 민팅 캠페인 | 드롭다운 (BIVE 토글 ON일 때만 노출) | ★ |

> 진행중 상태에서 보상을 변경하려면 종료 후 새 에피소드 재게시 또는 운영팀 별도 절차로 진행. 임시저장 상태에서는 자유롭게 변경 가능.

### 2-8. 관리 정보 (read-only)

생성 관리자 / 생성 일시 / 최근 수정자 / 최근 수정 일시 — 4컬럼 표시.

### 2-9. 하단 액션 (Sticky 푸터)

- [취소하기] (빨강) — 변경사항 확인 다이얼로그
- [변경사항 저장] (인디고) — 모든 필수 항목 충족 + 변경사항 있을 시 활성

## 5. 검증 규칙

| 검증 | 동작 |
|---|---|
| 상태 = 종료 | 모든 입력 read-only, 저장 불가 |
| 다국어 미입력 (한 언어라도 비어있음) | [저장] 비활성 |
| 메인 이미지 미등록 | [저장] 비활성 |
| 변경사항 없음 + 저장 시도 | "변경사항이 없습니다" 토스트 |
| 완료 보상 응모권·팬덤 포인트 | 0 이상 정수만 허용. 음수·소수·문자 입력 시 인라인 에러 |
| 완료 보상 BIVE 토글 ON | 민팅 캠페인 드롭다운 선택 필수. 미선택 시 [저장] 비활성 |
| 진행중 상태 + 완료 보상 수정 시도 | 모든 보상 필드 read-only. 변경 불가 (회원 기대권 보호) |
| 반복 에피소드 진입 | 완료 보상 섹션 자체 미노출 |
