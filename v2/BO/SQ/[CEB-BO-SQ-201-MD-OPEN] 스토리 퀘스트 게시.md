```text
┌─── 스토리 퀘스트 게시 ──────────────────────────┐

│                                                 │
│ 검증 항목:                                       │
│  ✅ 에피소드 N개 정의됨                                │

│                                                 │
│ 게시 후:                                          │
│  - 상태: DRAFT → ACTIVE                          │

│  - 첫 에피소드 자동 ACTIVE, 이후 에피소드는 LOCK            │
│                                                 │
│ 공개 일시 [2026.05.10 10:00] (수정 가능)           │
│                                                 │
│        [취소] [게시하기]                          │
```

## 3. 검증 룰

| 검증 | 통과 조건 |
| --- | --- |
| 에피소드 1개 이상 | 에피소드 0개면 차단 |
| 에피소드별 미션 1개 이상 | 빈 에피소드 차단 |
| 타이틀 KO 입력 | 필수 |
| ⚠️ 종료 일시 미설정 | 경고만 (진행 가능) |
