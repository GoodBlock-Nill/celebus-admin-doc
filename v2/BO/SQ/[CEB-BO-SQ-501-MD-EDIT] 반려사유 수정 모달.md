> 모달 트리거: [CEB-BO-SQ-501] 반려사유 설정의 테이블 행 클릭

## 1. 모달 개요

기존 반려사유의 운영자 노출명·다국어 메시지·상태(사용/미사용)를 수정하는 모달. [CEB-BO-SQ-501-MD-ADD]와 동일 구성에 **상태설정 필드 추가**.

| 항목 | 값 |
|---|---|
| 트리거 | 반려사유 설정의 테이블 행 클릭 |
| 진입 조건 | 기존 사유 존재 |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |
| 영향 범위 | 사유 1건 수정 (운영자 노출명·다국어 메시지·상태) |

## 2. 화면 구성

### 2-1. 헤더

- 모달 제목: "반려사유 수정"
- 우상단: X 닫기 버튼

### 2-2. 입력 폼

| 필드 | 입력 방식 | 글자수·검증 | 필수 |
|---|---|---|---|
| 운영자 노출명 | 텍스트 + 글자수 카운터 (기존 값 prefill) | 최대 100자 | ★ (한국어 1자 이상) |
| 유저 노출 메시지 | 다국어 라디오 + 텍스트 영역 (4행) + 진행률 표시 (기존 값 prefill) | 각 200자 | ★ (한국어 1자 이상) |
| 상태설정 | 드롭다운 | "사용" / "미사용" (기존 값 prefill) | — |

#### 다국어 라디오 그룹

[CEB-BO-SQ-501-MD-ADD] §2-2와 동일 (KO/EN/JA).

#### 상태 안내 텍스트

상태설정 드롭다운 아래 회색 텍스트:
> 미사용 처리된 반려사유는 검수 화면에서 노출되지 않습니다.

### 2-3. 하단 액션

| 버튼 | 색상 | 동작 |
|---|---|---|
| 취소하기 | 회색 외곽선 | 모달 닫기 (수정값 폐기) |
| 저장하기 | 인디고 | 사유 수정 + 모달 닫기 |

> [저장하기]는 운영자 노출명 + 한국어 메시지 모두 입력 시 활성.

## 4. 상태별 동작 분기

| 상황 | 동작 |
|---|---|
| 운영자 노출명 미입력 | [저장하기] 비활성 |
| 한국어 메시지 미입력 | [저장하기] 비활성 |
| 상태 "미사용"으로 변경 | 저장 시 본 사유는 [CEB-BO-SQ-304-MD-REJECT] 반려 모달에서 노출되지 않음 |
| [저장하기] 클릭 | 사유 수정 + 모달 닫기 + 리스트 갱신 |
| [취소하기] / X / 외부 클릭 | 모달 닫기 (수정값 폐기) |
| 권한 없음 | 트리거(행 클릭) 자체 비활성 |

## 5. 검증 규칙

| 규칙 | 내용 |
|---|---|
| 운영자 노출명 | 한국어 1자 이상 필수, 100자 초과 입력 차단 |
| 유저 노출 메시지 | 한국어 1자 이상 필수, 영어·일본어 선택 |
| 상태 변경 영향 | 사용 ↔ 미사용 토글은 즉시 반영되며 검수 운영에 영향 |
| 사유 식별값 | 수정 시 변경되지 않음 (기존 식별값 유지) |
| 삭제 | 본 화면에서 직접 삭제 불가. 상태를 "미사용"으로 전환만 가능 |
