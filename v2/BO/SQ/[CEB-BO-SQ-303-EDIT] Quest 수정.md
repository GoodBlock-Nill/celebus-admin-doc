## 1. 화면 개요

기존 Quest 1건의 정보를 수정하는 폼 화면. 생성 화면([CEB-BO-SQ-302-CREATE])과 동일한 3섹션 폼을 재사용하되, **진행중 상태일 때 일부 필드 자동 잠금**.

| 항목 | 값 |
|---|---|
| 화면 경로 | `/fanquest/{id}/edit` (목표 `/sq/quests/{id}/edit`) |
| Breadcrumb | 홈 / 에피소드 / 팬퀘스트 / Quest 수정 |
| 페이지 제목 | "Quest 수정" + 상태 배지 |
| 진입 경로 | [CEB-BO-SQ-303] Quest 상세의 [수정하기] 버튼 (임시저장·진행중에서만 노출) |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |

## 2. 화면 구성

### 2-1. 헤더

- 페이지 제목 "Quest 수정" + 상태 배지 (진행중=에메랄드 / 임시저장=회색)
- 우측 액션 2개:
  - [취소하기] (빨강) — 변경사항 확인 후 상세 화면으로 복귀
  - [수정하기] (인디고) — 검증 통과 시 활성

> 종료 상태 진입 시: "종료된 Quest는 수정할 수 없습니다. [상세로 돌아가기]" 안내 화면만 노출 (폼 미렌더링)

### 2-2. 진행중 상태 배너 (조건부 노출)

진행중 Quest 진입 시 상단에 앰버 톤 안내 박스:

> 진행중 상태에서는 기본 정보(타이틀, 설명, 가이드, 연관 링크)만 수정 가능합니다.

### 2-3. 에피소드 사용 현황 배너

진행중 Quest 진입 시 본 Quest를 사용 중인 에피소드 정보 표시 (변경 영향도 안내).

### 2-4. 입력 폼 (3섹션, [CEB-BO-SQ-302-CREATE] §2-2와 동일 구조)

| 섹션 | 임시저장 수정 가능 | 진행중 수정 가능 |
|---|---|---|
| A. 기본 정보 — 타이틀·설명·유저 가이드 (다국어) | ✅ | ✅ |
| A. 기본 정보 — 대표 이미지 | ✅ | ❌ (잠금) |
| B. 퀘스트 설정 — 아티스트·타입 | ✅ | ❌ (잠금) |
| B. 퀘스트 설정 — 참여 방식·반복 주기·기간 | ✅ | ❌ (잠금) |
| B. 퀘스트 설정 — 연관 링크 | ✅ | ✅ |
| C. 보상 설정 | ✅ | ❌ (잠금) |

> 진행중 잠금 필드는 시각적으로 비활성 처리 + opacity 50% + 커서 not-allowed.

## 3. 화면에 노출되는 정보 / 입력되는 정보

| 영역 | 정보 | 의미 |
|---|---|---|
| 헤더 | 페이지 제목·상태 배지·취소·수정 버튼 | 화면 식별 + 액션 |
| 진행중 배너 | 수정 제한 안내 (조건부) | 운영자 영향도 안내 |
| 에피소드 사용 현황 | 본 Quest 사용 미션 목록 | 수정 영향 범위 |
| A 기본 정보 | 타이틀·설명·가이드·이미지 | 회원 노출 콘텐츠 (가이드 500자) |
| B 퀘스트 설정 | 아티스트·타입·참여 방식·기간·링크 | 운영 메타 |
| C 보상 설정 | 응모권·덕력·NFT | 회원 보상 |

## 4. 상태별 동작 분기

| Quest 상태 | 동작 |
|---|---|
| 임시저장 | 모든 필드 수정 가능. 검증 통과 시 [수정하기] 활성 |
| 진행중 | 기본 정보 4종(타이틀·설명·가이드·연관 링크)만 수정 가능. 다국어 KO/EN/JA 모두 입력 + 이미지 존재 검증만 필요 |
| 종료 | 안내 화면만 노출, 폼 미렌더링 |
| [취소하기] / ← 뒤로가기 | [CEB-BO-SQ-303-MD-CANCEL] Quest 수정 취소 모달 |
| [수정하기] 성공 | [CEB-BO-SQ-303-MD-COMPLE] Quest 수정 완료 모달 |
| 권한 없음 진입 | 화면 미노출 또는 [수정하기] 비활성 |

### 5-1. 임시저장 상태 ([수정하기] 활성 조건)

### 5-2. 진행중 상태 ([수정하기] 활성 조건)

| 항목 | 조건 |
|---|---|
| 타이틀·설명·유저 가이드 | 한국어/영어/일본어 3개 언어 모두 입력 |
| 대표 이미지 | 등록 |

> 보상·기간·아티스트·타입·참여 방식 검증은 진행중 수정에서 제외 (잠금 필드).
