## 4. 반려 처리 결과

| 영역 | 처리 |
| --- | --- |
| `user_story_quest_chapter_mission` | `status = REJECTED` + `rejected_dt` 기록 |
| 회원 푸시 | 다국어 메시지 (회원 언어 기준 KO/EN/JP) |
| 활동 로그 | [CEB-BO-SYS-601] 자동 기록 (`artistContext` 포함) |
