## 1. 모달 개요

Quest 수정 폼에서 작성 중인 내용을 폐기하고 상세 화면으로 복귀하기 위한 확인 모달. 의도치 않은 작업 손실 방지.

| 항목 | 값 |
|---|---|
| 트리거 | Quest 수정 화면의 [취소하기] |
| 진입 조건 | 항상 가능 |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |
| 영향 범위 | 수정 중인 입력값 모두 폐기 (저장 없음) |

## 2. 화면 구성

### 2-1. 헤더

- 모달 제목: "Quest 수정 취소"

### 2-2. 본문

> 작성 중인 내용이 사라집니다. 취소하시겠습니까?

### 2-3. 하단 액션

| 버튼 | 색상 | 동작 |
|---|---|---|
| 계속 작성 | 회색 외곽선 | 모달 닫기 (수정 폼 유지) |
| 확인 | 인디고 | 수정 내용 폐기 + Quest 상세 화면으로 복귀 |
