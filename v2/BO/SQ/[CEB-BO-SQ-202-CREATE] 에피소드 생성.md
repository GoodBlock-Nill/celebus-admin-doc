## 1. 화면 개요

기존 에피소드 그룹에 새 에피소드를 추가하는 폼 화면. 그룹 상세에서 [새 메인 에피소드] 또는 [반복 에피소드 추가] 버튼으로 진입하며, URL 쿼리로 그룹 식별값과 유형(메인/반복)이 전달된다.

| 항목 | 값 |
|---|---|
| 화면 경로 | `/sq/create?groupId={gid}&kind={메인/반복}` |
| Breadcrumb | 홈 / 에피소드 / 그룹 리스트 / {그룹명} / 새 에피소드 생성 |
| 페이지 제목 | "새 에피소드 생성" + 유형 배지 |
| 보조 안내 | "{그룹 타이틀} 그룹에 추가 (메인 N/5)" |
| 진입 경로 | [CEB-BO-SQ-201] 그룹 상세의 [새 메인 에피소드] / [반복 에피소드 추가] |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |

## 2. 화면 구성

### 2-1. 헤더

- 좌측: ← 뒤로가기 (취소 확인 모달 후 그룹 리스트로)
- 페이지 제목 "새 에피소드 생성" + 유형 배지 (메인=인디고 / 반복=바이올렛)
- 보조 텍스트: "{그룹 타이틀} 그룹에 추가 (메인 N/5)"

### 2-2. 만석 차단 박스 (조건부 노출)

| 유형 | 조건 | 메시지 |
|---|---|---|
| 메인 | 그룹에 이미 메인 5개 등록됨 | "본 그룹은 이미 메인 5개 최대치에 도달했습니다. 추가 등록이 불가능합니다." |
| 반복 | 그룹에 이미 반복 1개 등록됨 | "본 그룹은 이미 반복 1개 최대치에 도달했습니다." |

### 2-3. 정책 안내 박스 (인디고 톤)

> 본 화면에서는 마스터 메타데이터(아티스트·다국어 타이틀·다국어 설명·메인 이미지·표시 순서)만 입력합니다.
> **기간은 상위 에피소드 그룹에서 관리**하므로 별도 기간 입력 필드가 없습니다.
> 다국어 입력은 **한국어/영어/일본어 3개 언어 모두 필수**입니다.

### 2-4. 입력 폼 — 기본 정보 (3열 그리드)

| 필드 | 입력 방식 | 옵션·검증 | 필수 |
|---|---|---|---|
| 아티스트 | 그룹 진입 시 read-only 배지 (자동 상속) / 드롭다운 | V01D / iKON / CELEBUS | ★ |
| 표시 순서 | 숫자 입력 | 메인: 1~5 / 반복: 6 (read-only 고정) | ★ |
| 상태 | read-only | "임시저장 (자동)" | — |

### 2-5. 다국어 필드

| 필드 | 입력 방식 | 글자수 | 필수 |
|---|---|---|---|
| 타이틀 | 한국어/영어/일본어 탭 전환 입력 | 각 100자 | ★ |
| 에피소드 설명 | 한국어/영어/일본어 탭 전환 입력 (4행) | 각 200자 | ★ |

> 게시 검증 시: 한국어·영어·일본어 모두 입력되어 있어야 통과.

### 2-6. 메인 이미지

- 좌측: 200px × 3:4 비율 드래그앤드롭 존
- 우측: URL 직접 입력 또는 파일 업로드 버튼
- 안내: "3:4 비율 권장 (예: 750×1000px), 5MB 이하, PNG/JPG/WebP"
- 드롭 시: 파일 타입 검증 (이미지 외 차단) → 자동 업로드 URL 생성

### 2-7. 다음 단계 안내 (앰버 박스)

> 에피소드 생성 후 상세 화면에서 **미션(1~10개)** 을 등록하세요.

### 2-8. 하단 액션 (Sticky 푸터)

- [취소하기] (빨강) → 그룹 리스트
- [임시저장] (회색) — 한국어 타이틀 + 메인 이미지만 있으면 가능
- [생성하기] (인디고) — 모든 필수 항목 충족 시 활성
