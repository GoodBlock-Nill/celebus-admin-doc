## 1. 모달 개요

에피소드 그룹과 그 안의 모든 하위 항목(에피소드 / 미션)을 영구 삭제하는 확인 모달. **임시저장 상태에서만 가능**하며, 진행중·종료 그룹은 액션 자체가 노출되지 않는다.

| 항목 | 값 |
|---|---|
| 트리거 | 그룹 상세의 [삭제] 액션 (임시저장 상태에서만 노출) |
| 진입 조건 | 그룹 상태가 "임시저장" |
| 차단 조건 | 진행중 / 종료 그룹은 액션 자체 미노출 |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |
| 영향 범위 | 그룹 + 하위 모든 에피소드 + 모든 미션 (cascade) |

## 4. 상태별 동작 분기

| 상황 | 동작 |
|---|---|
| 임시저장 그룹 진입 | 모달 정상 노출 |
| 진행중·종료 그룹 | 트리거 액션 자체 미노출 → 모달 진입 불가 |
| 삭제 성공 | 그룹 리스트로 이동 + "에피소드 그룹이 삭제되었습니다." 토스트 |
| 삭제 실패 (서버 오류) | 모달 유지 + 에러 안내 |
| 취소 | 모달 닫힘 |
| 권한 없음 | 트리거 액션 미노출 또는 [삭제] 비활성 |

## 5. 검증 규칙

| 검증 | 동작 |
|---|---|
| 그룹 상태 ≠ 임시저장 | 모달 자체 진입 불가 (액션 미노출) |
| 그룹에 진행중 에피소드 존재 (이론상 임시저장 그룹에선 발생 불가) | 삭제 차단 + 에러 안내 |

> 진행중·종료 그룹은 별도의 상태 전환(종료 전환)을 통해 라이프사이클을 마무리해야 함. 삭제는 임시저장 한정.

## 6. 삭제 영향 (cascade)

```
에피소드 그룹 (삭제 대상)
  └─ 하위 에피소드 N개   ← 함께 삭제
       └─ 하위 미션 N개    ← 함께 삭제
            └─ 팬퀘스트 매핑   ← 함께 해제 (팬퀘스트 자체는 보존)
```

> 팬퀘스트 Quest 자체는 삭제되지 않음. 매핑 해제만 발생 → 팬퀘스트 상세의 "사용 중인 에피소드" 패널에서 자동 제외.
