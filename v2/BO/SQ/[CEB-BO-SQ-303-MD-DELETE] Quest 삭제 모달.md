## 1. 모달 개요

임시저장 Quest를 영구 삭제하는 확인 모달. 위험 액션이므로 빨강 톤. 복구 불가.

| 항목 | 값 |
|---|---|
| 트리거 | Quest 상세의 [삭제하기] (임시저장 상태에서만 노출) |
| 진입 조건 | Quest 상태가 "임시저장" |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |
| 영향 범위 | Quest 영구 삭제 (복구 불가) |

## 2. 화면 구성

### 2-1. 헤더

- 모달 제목: "Quest 삭제"

### 2-2. 본문

> 삭제한 Quest는 복구할 수 없습니다.
> 계속 진행하시겠습니까?

### 2-3. 에피소드 사용 현황 패널 (조건부 추가 노출)

본 Quest를 사용 중인 에피소드 미션이 있으면 본문 아래 "사용 중인 에피소드" 패널 추가 노출. 모달 사이즈는 md로 확대. 삭제 시 미션 매핑 자동 해제 안내 포함.

### 2-4. 하단 액션

| 버튼 | 색상 | 동작 |
|---|---|---|
| 취소 | 회색 외곽선 | 모달 닫기 |
| 삭제 | **빨강 (danger)** | Quest 영구 삭제 + 팬퀘스트 리스트로 복귀 |

| 상황 | 동작 |

## 5. 검증 규칙

| 규칙 | 내용 |
|---|---|
| Quest 상태 ≠ 임시저장 | 모달 진입 불가 |
| 영구 삭제 | 복구 불가. git history 등 별도 백업 없음 |
| 미션 매핑 자동 해제 | 본 Quest가 매핑된 미션이 있으면 삭제 시 미션의 팬퀘스트 매핑이 자동 해제 (미션 자체는 보존) |
