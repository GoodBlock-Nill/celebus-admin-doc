## 1. 화면 개요

아티스트별 에피소드 그룹(시즌·큐레이션)을 한 화면에서 조회·필터·정렬하고 신규 생성을 시작하는 진입점.

| 항목 | 값 |
|---|---|
| 화면 경로 | `/sq/groups/list` |
| Breadcrumb | 홈 / 에피소드 / 그룹 리스트 |
| 페이지 제목 | "에피소드 그룹" |
| 진입 경로 | 사이드바 "에피소드 > 그룹 리스트" |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |

## 2. 화면 구성

### 2-1. 헤더

- 페이지 제목 + 우상단 [새 그룹 생성] 버튼 → `/sq/groups/create`

### 2-2. 통계 카드 (가로 4개)

| 카드 | 값 |
|---|---|
| 전체 | 전체 그룹 수 |
| 임시저장 | 임시저장 상태 그룹 수 (회색 톤) |
| 진행중 | 진행중 상태 그룹 수 (에메랄드 톤) |
| 종료 | 종료 상태 그룹 수 (진한 회색 톤) |

### 2-3. 필터·검색 바

| 위치 | 요소 | 옵션 |
|---|---|---|
| 좌 | 상태 드롭다운 | 전체 / 임시저장 / 진행중 / 종료 |
| 좌 | 아티스트 드롭다운 | 전체 / V01D / iKON / CELEBUS |
| 우 | 키워드 입력 | 타이틀 부분 일치 검색 |
| 우 | [초기화] 버튼 | 모든 필터·검색·페이지 초기화 |

### 2-4. 테이블 컬럼

| 컬럼 | 너비 | 표시 내용 |
|---|---|---|
| 상태 | 100px | 상태 배지 (임시저장 회색 / 진행중 에메랄드 / 종료 진한 회색) |
| 아티스트 | 100px | V01D / iKON / CELEBUS |
| 타이틀 | flex | 한국어 타이틀 |
| 에피소드 수 | 100px | 그룹 내 에피소드 합계 (메인 + 반복) |
| 시작일 | 140px | 그룹 시작일 |
| 종료일 | 140px | 그룹 종료일 |
| 최근 수정자 | 100px | 마지막 수정자 닉네임 |

행 클릭 → 그룹 상세 화면([CEB-BO-SQ-201], `/sq/groups/{gid}`) 이동.

### 2-5. 페이지네이션

- 페이지당 20건 (필터·검색 결과 기준)
- 항상 노출 (1페이지일 때도 비활성 형태로 노출, [CEB-BO-011] §5 공통 페이지네이션 정합)
- 윈도잉 규칙: 총 페이지 ≤ 7이면 전체 페이지 번호, 8 이상이면 윈도우 + 양 끝 `...` 표시
- 필터·검색·키워드 변경 시 자동으로 1페이지로 리셋

### 2-6. 빈 상태

검색 결과 0건 시: "등록된 에피소드 그룹이 없습니다." 메시지 노출.

## 3. 화면에 노출되는 정보

| 위치 | 정보 | 의미 |
|---|---|---|
| 통계 카드 1~4 | 전체 / 임시저장 / 진행중 / 종료 카운트 | 그룹 상태별 분포 |
| 필터 바 | 상태·아티스트·키워드 | 그룹 조회 조건 |
| 테이블 컬럼 1 | 상태 | 현재 운영 단계 배지 |
| 테이블 컬럼 2 | 아티스트 | 그룹이 속한 아티스트 |
| 테이블 컬럼 3 | 타이틀 | 한국어 타이틀 |
| 테이블 컬럼 4 | 에피소드 수 | 메인 + 반복 에피소드 합계 |
| 테이블 컬럼 5~6 | 시작일·종료일 | 그룹 기간 (그룹 = 에피소드 SSOT) |
| 테이블 컬럼 7 | 최근 수정자 | 마지막 운영자 닉네임 |

## 4. 상태별 동작 분기

본 화면은 **상태 변경 액션이 없는 조회·진입 전용 화면**이므로 데이터 가용성 분기만 존재.

| 데이터 조건 | 동작 |
|---|---|
| 검색 결과 0건 | "등록된 에피소드 그룹이 없습니다." 빈 상태 메시지 |
| 데이터 로딩 중 | Skeleton 테이블 |
| 데이터 조회 실패 | 에러 메시지 + [재시도] 버튼 |
| 권한 없음 진입 | [관리자 > 권한관리] 설정에 따라 화면 미노출 또는 접근 차단 |
| 기본 정렬 | **최근 수정 일시 내림차순** |
