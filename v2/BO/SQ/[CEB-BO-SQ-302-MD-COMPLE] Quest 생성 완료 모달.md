## 1. 모달 개요

Quest 생성 폼의 모든 필수 항목 검증 통과 후 임시저장 상태로 저장되었음을 알리는 단일 액션(정보 안내) 모달.

| 항목 | 값 |
|---|---|
| 트리거 | Quest 생성 화면의 [생성하기] 검증 통과 후 |
| 진입 조건 | 모든 필수 항목 입력 완료 (SQ-302-CREATE §5 검증 규칙 통과) |
| 접근 권한 | [관리자 > 권한관리] 설정에 따른다 |
| 영향 범위 | 단일 액션 — 외부 클릭으로 닫히지 않음, [확인] 클릭 시 리스트로 복귀 |
