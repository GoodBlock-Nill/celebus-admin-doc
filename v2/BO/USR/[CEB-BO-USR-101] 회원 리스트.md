## 1. 화면 목적

CELEBUS 가입자(회원) 통합 조회·검색·필터 화면. 운영자는 본 화면에서 회원 라이프사이클 전반을 모니터링하고 개별 회원 상세로 진입하여 제재·자산 회수·운영자 메모 관리.

### 주요 기능

- 전체 회원 리스트 조회 + 계정 상태별 카운터 대시보드
- 다중 조건 검색·필터 (계정 상태·유저명·전화번호·기간 등)
- 회원 상세 진입 ([CEB-BO-USR-201])
- 정지사유 설정 진입 ([CEB-BO-USR-501])

### 진입 경로

- 사이드 메뉴 > 회원

## 4. 규칙·로직

### 4.1 권한별 분기

[CEB-BO-002] §4 적용:

| 권한 | 화면 진입 | 조회 | [정지사유 설정] | Row 클릭 |
| ---- | --------- | ---- | -------------- | -------- |
| SUPER | ✅ | ✅ | ✅ | ✅ |
| OPERATOR | ✅ | ✅ (조회 전용) | ✅ | ✅ |
| POLICY | ✅ | ✅ (조회 전용) | ❌ | ✅ |
| EXTERNAL_ARTIST | ❌ | — | — | — |

> EXTERNAL_ARTIST 차단: 회원 정보는 본사 운영 영역.

### 4.2 정렬·필터

| 기본 정렬 | 가입일 내림차순 (최신순) |
| --- | --- |

- 계정 상태 필터 + 정렬 + 유저명 검색 = AND 조합
- 유저명 검색은 부분 일치(LIKE)
- 검색어 미입력 시 해당 조건 무시

### 4.3 계정 상태 정의

[CEB-BO-002] §7.3 제재 정책 적용:

| 상태 | 의미 | Badge |
| --- | --- | --- |
| 정상 | 일반 활동 가능 | Green |
| 정지 | 임시·영구 정지 (1일/7일/30일/90일/영구) | Red |
| 탈퇴대기 | 본인 탈퇴 신청 후 30일 유예 ([CEB-BO-000] §13) | Orange |
| 탈퇴완료 | 30일 경과 후 익명화 처리 (개인정보 마스킹) | Gray |

> 정지 상세 단계는 회원 상세 [기본정보] 탭에서 확인 (경고/부분정지/임시정지/영구정지 등).

### 4.4 데이터 표시 규칙

- 지갑주소: 미연동 회원은 "—" 표시
- 전화번호: 국가코드 포함 표시 (회원 가입 시 입력 형식 그대로)
- 가입일시: KST 기준
