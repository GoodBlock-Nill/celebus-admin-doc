## 1. 화면 개요

현재 자동 생성된 슬롯 인벤토리를 노출 위치별 탭으로 한눈에 파악하는 화면. 어떤 슬롯이 채워져 있고 비어 있는지 / 최근 어떤 슬롯을 누가 손댔는지 확인. 슬롯 Row 클릭으로 [CEB-BO-APP-202] 슬롯 상세에 진입해 그 슬롯의 배너 추가·수정·정렬을 수행한다.

| 항목 | 값 |
|---|---|
| 화면 경로 | `/home/banners` (탭 쿼리 `?placement=home` / `?placement=artist`) |
| Breadcrumb | 홈 / 앱 / 배너 관리 |
| 진입 경로 | 사이드바 "앱 > 배너 관리" |
| 용도 | 자동 생성 슬롯 인벤토리 조회 + 슬롯 상세 진입 |

## 2. 화면 구성

### 2-1. 헤더

- 페이지 제목 "배너 관리"
- 신규 등록 버튼은 **없음** — 배너 추가는 슬롯 상세 안에서만

### 2-2. 노출 위치 탭

탭 카운트 = 자동 생성 슬롯 수.

| 탭 | 슬롯 종류 (타겟 모드) | 자동 생성 슬롯 수 (활성 아티스트 5명 기준) |
|---|---|---|
| 홈 (기본) | 메인(전역만) + 오늘의 할일(아티스트만) | **6**(메인 1 + 오늘의 할일 5) |
| 아티스트 메인 | 다함께(아티스트만) + 미션(아티스트만) | **10**(다함께 5 + 미션 5) |

탭 전환 시 URL 쿼리(`?placement=home` / `?placement=artist`) 동기화 — 북마크·새로고침 유지.

### 2-3. 안내 띠

페이지 상단에 현재 탭의 슬롯 개수 + "슬롯은 자동 생성되며, 배너는 슬롯 상세에서 관리합니다" 안내.

### 2-4. 통계 카드 (4종 상태값, 현재 탭 컨텍스트)

상태 분포만 간결하게 표시. 카드 클릭 시 본 탭에서 해당 조건으로 필터 적용.

| 카드 | 의미 | 클릭 동작 |
|---|---|---|
| 전체 배너 | 현재 탭의 슬롯 안 배너 총 개수 | 필터 초기화 |
| 노출중 | 현재 탭의 노출중 배너 수 (초록 강조) | 상태=노출중 필터 적용 |
| 임시저장 | 현재 탭의 임시저장 배너 수 (회색) | 상태=임시저장 필터 적용 |
| 노출 종료 | 현재 탭의 노출 종료 배너 수 | 상태=노출 종료 필터 적용 |

### 2-5. 필터 바

| 필터 | 옵션 |
|---|---|
| 아티스트 | 전체 / (홈 탭만) 전역 / {활성 아티스트 동적 목록} |
| 위치 | 전체 / (홈 탭) 메인·오늘의 할일 / (아티스트 메인 탭) 다함께·미션 |
| 키워드 | 아티스트·위치 명칭 부분 일치 |
| [초기화] | 모든 필터 해제 |

### 2-6. 슬롯 인벤토리 테이블

| 컬럼 | 표시 내용 |
|---|---|
| 아티스트 | "전역" 또는 그룹명 Badge |
| 배너 위치 | "메인 · 캐러셀 · 최대 N개" / "오늘의 할일 · 단일" / "다함께 · 단일" / "미션 · 단일" 형태 Badge + 보조 설명 (캐러셀은 **메인 한정**, 그 외는 모두 단일) |
| 배너 수 | "N건 · 노출중 X · 대기 Y" / 비어 있으면 "비어 있음" |
| 최근 수정일 | 슬롯 내 가장 최근 배너의 수정 시각 / 없으면 "—" |
| 최근 수정자 | 마지막 수정자 / 없으면 "—" |

- Row 클릭 → [CEB-BO-APP-202] 슬롯 상세 (`/home/banners/slot/{slotKind}/{artist}`) 이동
- 슬롯 수가 적으므로 기본은 페이지네이션 없이 전체 표시 (활성 아티스트 다수 시 추가 검토)

## 3. 화면에 노출되는 정보

| 위치 | 정보 | 의미 |
|---|---|---|
| 노출 위치 탭 | 홈 / 아티스트 메인 + 탭 카운트 | 현재 보고 있는 탭과 슬롯 총수 |
| 안내 띠 | "{탭명} 슬롯 N개" | 현재 탭의 슬롯 자동 생성 결과 |
| 통계 카드 | 전체 / 노출중 / 임시저장 / 노출 종료 | 슬롯 내 배너 상태별 합산 |
| 필터 바 | 아티스트 / 위치 / 키워드 | 슬롯 인벤토리 조회 조건 |
| 인벤토리 테이블 | 아티스트 · 배너 위치 · 배너 수 · 최근 수정일 · 최근 수정자 | 슬롯별 메타 |

## 4. 상태별 동작 분기

본 화면은 상태 변경 액션이 없는 조회·진입 전용 화면이므로 데이터 가용성 분기만 존재.

| 데이터 조건 | 동작 |
|---|---|
| 활성 아티스트 0명 | 홈 탭은 메인 1슬롯만, 아티스트 메인 탭은 0슬롯 (빈 안내) |
| 슬롯 N개 모두 배너 0건 | 인벤토리 테이블은 정상 노출 (각 행에 "비어 있음" 표기) |
| 필터 결과 0건 | "조건에 맞는 슬롯이 없습니다." + [필터 초기화] |
| 데이터 로딩 중 | Skeleton 테이블 |
| 데이터 조회 실패 | 에러 메시지 + [재시도] |
| 권한 없음 진입 | [관리자 > 권한관리] 설정에 따라 화면 자체 미노출 또는 접근 차단 |

## 5. 검증 규칙

| 규칙 | 내용 |
|---|---|
| 슬롯 자동 생성 | (slotKind × artistGroup) 조합으로 derived. 운영자 추가·삭제 불가 |
| 활성 아티스트 동기화 | [아티스트 관리]의 활성 그룹 변경 시 슬롯 인벤토리도 자동 갱신 |
