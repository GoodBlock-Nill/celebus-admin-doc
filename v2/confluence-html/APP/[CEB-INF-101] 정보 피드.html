1. 화면 목적

아티스트 탭(CEB-ART-101)에서 "정보" 카드를 탭했을 때 진입하는 화면이다. 아티스트의 소식(뉴스)과 일정을 시간순 통합 타임라인으로 보여주는 정보 허브 역할을 한다.

주요 기능
  • 소식(뉴스)과 일정을 시간순 통합 타임라인으로 표시
  • "오늘 / 이번 주 / 지난 주 / 이전" 날짜 그룹핑
  • 일정 항목에 🔔 알림 설정 기능 (D-1 + 당일 푸시)
  • IX 선공개 일정/소식에 "CELEBUS 단독" 뱃지 표시
  • 공지는 상단 고정 배너 (등록된 공지가 있을 때만 표시)
접근 경로
  • CEB-ART-101 > 정보 카드 탭
  • CEB-HOM-001 > 바로가기 > 오늘의 일정/소식 카드 탭

3. 화면 내 기능설명

3.1 통합 타임라인

  • 소식과 일정을 날짜 기준으로 통합 정렬하여 하나의 타임라인으로 표시
  • 일정: 해당 날짜/시간 기준으로 배치
  • 소식: 게시일 기준으로 배치
  • 동일 날짜 내 정렬 기준: 일정(시간순 오름차순) → 소식(최신순)

3.2 일정 알림 설정

  • 각 일정 항목의 🔔 버튼 탭 시 알림 ON/OFF 토글
  • 알림 ON 전환 시: 토스트(Success) "알림이 설정되었습니다. D-1과 당일에 알려드릴게요"
  • 알림 OFF 전환 시: 토스트(Info) "알림이 해제되었습니다"
  • 기기 알림 권한 미허용 상태에서 알림 ON 시도 시: "알림 권한을 허용해주세요" 안내 → OS 설정 이동
  • 이미 지나간 일정: 🔔 버튼 미표시

3.3 공지 배너

  • 공지 배너는 스크롤해도 화면 상단에 고정(sticky)
  • 탭 시 공지 상세 바텀시트 표시
  • BO에서 공지 등록/삭제로 관리
  • 아티스트 공지와 플랫폼 공지를 통합하여 최신 1건만 배너에 표시

3.4 소식 상세

소식 상세 화면(CEB-INF-201)은 별도 화면 스펙으로 추후 작성합니다.

  • 소식 항목 탭 시 소식 상세 화면으로 이동
  • 상세 구성: 전체 텍스트 + 이미지 갤러리 + 게시일 + 공유 버튼
  • 공유 버튼 탭 시 OS 기본 공유 시트 표시 (X, Instagram, 카카오톡 등)
