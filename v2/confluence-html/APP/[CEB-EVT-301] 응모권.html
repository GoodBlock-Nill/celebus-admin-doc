1. 화면 개요

유저가 보유한 응모권(티켓) 수량을 확인하고, 적립·사용 거래 이력을 조회하는 화면. 응모권 구매(GP 사용) 진입점.

주요 기능
  • 보유 응모권 실시간 표시
  • 응모권 구매 진입 (GP 사용, 비율은 정책 결정)
  • 적립 이력: Quest 보상, BIVE 보유 보상, ST 탈락 보상, PM 보상, GP 구매, 환불·회수
  • 사용 이력: Raffle 응모, 만료 소멸
접근 경로
  • [CEB-HOM-001] 홈 메인 > 응모권 카드/배너 진입
  • [CEB-EVT-101] Raffle 메인 > 응모권 영역

3.2 탭 동작

  • 탭 전환 시 해당 부호의 이력만 필터링
  • 전환 시 리스트 최상단으로 스크롤 초기화

3.3 [구매하기] 동작

  • 탭 시 응모권 구매 화면(Buy-RF-Ticket-001, [CEB-BO-RFT-301] §5에서 정의)으로 이동
  • GP 사용 → 응모권 적립 (변동 유형: GP로 응모권 구매)
  • 교환 비율과 1일 한도는 [CEB-BO-RFT-301] 정책 화면에서 운영자가 조정
  • 앱내 구매 운영 토글이 OFF인 경우: [구매하기] 버튼이 [점검중]으로 비활성. 탭 시 토스트 안내 ("현재 응모권 구매가 일시적으로 중단되었습니다.")
