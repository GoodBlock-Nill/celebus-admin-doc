1. 화면 개요

작성된 기억의 상세 내용을 확인하고, 수정/삭제/신고하는 화면이다. 내 기억과 타인의 공개 기억 모두 이 화면에서 열람한다.

핵심 원칙: 팬이 기록한 소중한 순간을 풍부하게 다시 체험할 수 있어야 한다.

주요 기능
  • 감정 이모지 + 유형 + 미디어 + 텍스트 + 위치 전체 표시
  • 이미지 풀스크린 갤러리 (스와이프 + 핀치 줌)
  • 이미지 최대 3장 표시
  • 내 기억: 수정/삭제
  • 타인 공개 기억: 신고
  • 공개 기억 외부 공유 (Web Share API / 클립보드 복사)
접근 경로
  • CEB-MEM-101 > 기억 항목 탭 (캘린더/그리드/지도 뷰에서)
  • 푸시 알림 "1년 전 오늘 기억" 탭 시 진입
  • 외부 공유 링크 접속 (app.celebus.xyz/memory/{memoryId}) — 비로그인도 공개 기억 읽기 전용 열람 가능
화면 레이아웃

화면 레이아웃은 프로토타입(https://celebus-app.vercel.app/home)을 참조하세요. 플로팅 디버그 버튼으로 다양한 상태를 확인할 수 있습니다. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

동작

수정하기
  • [수정하기] 탭 → CEB-MEM-201 (mode=edit, 기존 데이터 프리필)
  • 유형은 변경 불가
  • 수정 완료 후 본 화면으로 자동 복귀 + 수정된 내용 반영
  • 덕력 중복 미지급
삭제하기
  • [삭제하기] 탭 → 확인 모달:
  • 텍스트: "삭제한 기억은 복구할 수 없어요. 삭제할까요?"
  • CTA: [취소] / [삭제]
  • 삭제 완료 → 토스트(Success) "기억이 삭제되었어요"
  • CEB-MEM-101로 자동 복귀 + 캘린더 데이터 재조회
  • 덕력 환수 없음

3.2 비로그인 공통

아래는 화면 전체에 적용되는 공통 원칙입니다. 각 영역(2.1~2.7)의 "비로그인" 섹션에는 해당 영역 고유의 구체적 동작만 기술합니다.

  • 앱 내 접근 (MEM-101 → MEM-202): 로그인 상태에서만 접근 가능 (기존 유지). CEB-ART-101 카드 탭 시 로그인 유도 (공통정책 10.5 참조)
  • 공유 링크 접근 (외부 URL): 비로그인도 공개 기억 읽기 전용 열람 가능. ⋯ 메뉴 미노출. 하단 가입 유도 배너 표시
  • 비로그인 CTA: [시작하기] → app.celebus.xyz/home 이동
