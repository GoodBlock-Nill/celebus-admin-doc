1. 화면 개요

목적

앱 사용 환경(언어)과 알림 수신을 관리하는 설정 화면이다. 하단 My 탭에서 진입한다.

주요 기능
  • 언어 선택 (한국어 / English / 日本語)
  • 알림 설정 (아티스트 알림 / 이벤트 / 게임존 / 공지)
접근 경로
  • 하단 탭 > My > 설정
화면 레이아웃

시각적 스타일은 Figma 시안을 따릅니다. 변형: 알림 on/off 등.

UI 구성

항목내용

동작

  • 서버 콘텐츠(다국어 지원 항목)도 선택 언어 기준으로 표시

2.3 알림 설정

UI 구성

항목내용
섹션 타이틀🔔 "알림 설정"
아티스트"아티스트 / 팔로우한 아티스트 소식" + > (하위 화면 — 아티스트별 알림 ON/OFF 관리)
이벤트"이벤트 / 이벤트, 프로모션 알림" + 토글
게임존"게임존 / 게임존 알림" + 토글
공지"공지 / 보안 알림 및 중요한 업데이트" + 토글
스타일피그마 참조

동작

  • 아티스트 행 [>] 탭 → 아티스트별 알림 ON/OFF 관리 화면 (팔로우 아티스트별 개별 제어). 정보 피드(CEB-INF-101) 헤더 🔔 토글과 동일 상태 연동
  • 이벤트/게임존/공지 토글 → 해당 카테고리 푸시 수신 ON/OFF. 변경 즉시 반영 + 토스트(필요 시)
  • OS 알림 권한 거부 시: 토글 ON 시도 → OS 설정 화면으로 이동 유도

상태별 표시

상태처리
알림 권한 거부토글 ON 불가 안내 + OS 설정 이동 유도
전체 OFF각 토글 OFF 상태 표시 (Figma 알림off 변형)

비로그인

  • 언어 설정은 비로그인도 가능. 알림 설정(아티스트/이벤트/게임존/공지)은 로그인 상태에서만 (비로그인 시 로그인 유도)
