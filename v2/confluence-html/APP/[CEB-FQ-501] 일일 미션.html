1. 화면 개요

목적

아티스트 탭(CEB-ART-101)에서 "일일 미션" 카드를 탭했을 때 진입하는 화면이다. 출석 체크, 오늘의 일일 미션, 스트릭 현황을 한 화면에서 관리하는 데일리 루프 허브 역할을 한다.

주요 기능
  • 출석 체크 (버튼 탭, 5pt — T1 즉시·습관)
  • 오늘의 일일 미션 1건 (자동검증, 페이지 방문형, 25pt — T2 저관여)
  • 스트릭 현황 (연속일수 + 다음 보너스까지 프로그레스)
  • 이번 주 요일별 완료 기록 (도트)
  • 스트릭 보너스 달성 시 축하 모달
접근 경로
  • CEB-ART-101 > 일일 미션 카드 탭
  • CEB-HOM-001 > 오늘의 할 일 > 미션 아이콘/텍스트 영역 탭
  • CEB-HOM-001 > 오늘의 할 일 > 출석 ☐ 아이콘 탭으로 홈에서 직접 출석 가능 (이중 진입점. CEB-FQ-501 진입 없이 홈에서 완료)
레이아웃 참조

화면 레이아웃은 프로토타입(https://celebus-app.vercel.app/home)을 참조하세요. 플로팅 디버그 버튼으로 다양한 상태를 확인할 수 있습니다. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

2.3 일일 미션

UI 구성

항목내용
미완료 상태☐ 미션명(예: "V01D 페이지 방문") + 설명 + [하러가기 →]
완료 상태✅ 미션명 + "완료" (비활성) — 개별 보상 표시 없음
보상 지급 방식모든 일일 미션 완료 시 25pt 일괄 지급 (Fan Power / 推し力 — T2 저관여 / 일일·주간 캡 적용. BO 자유 변경 가능)
보상 표시 위치일일 미션 영역 하단에 "오늘의 미션 완료 보상: +25pt" 표시. 모두 완료 전까지 회색 비활성, 모두 완료 시 즉시 지급 + 상태 강조
하루 미션 수시스템이 미션 풀에서 자동 선택 (현 1건/일, 향후 BO 다건 확장 가능)

동작

  • [하러가기] 탭 → 해당 서비스 화면으로 이동 (딥링크)
  • 해당 화면 진입 기록 → CEB-FQ-501 복귀 시 미션 자동 완료
  • 모든 미션 완료 시점에 한 번: 토스트(Success) "오늘의 일일 미션을 모두 완료했어요! 덕력 25pt 획득" 노출 + 25pt 일괄 적립
  • 개별 미션 완료 시에는 토스트 없음 (체크 표시만 갱신)
