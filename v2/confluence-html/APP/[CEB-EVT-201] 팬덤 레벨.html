1. 화면 개요

아티스트 탭(CEB-ART-101)에서 "{아티스트명} 키우기" 카드를 탭했을 때 진입하는 화면이다. (유저 대면 명칭: {아티스트명} 키우기, 예: V01D 키우기) 팬덤 전체의 공동 레벨업 진행 상황과 레벨별 보상을 확인하고, 덕력 획득 활동으로 유도하는 화면이다.

참고: 팬덤 레벨은 개인 경쟁이 아닌 팬덤 전체의 공동 달성 목표이다. 개인 순위 경쟁은 CEB-DUK-101 덕력 랭킹에서 제공한다.

주요 기능
  • 현재 팬덤 레벨 + 프로그레스 바 + 내 기여도 확인
  • 레벨별 보상 리스트 확인 (해금 완료 / 진행중 / 미해금 "???" 실루엣)
  • 팬덤 활동 요약 (참여 팬 수 + 이번 달 합계 + 주요 활동 유형)
  • 덕력 획득 활동 유도 CTA (출석 체크 / 퀘스트 / 랭킹)
  • 레벨업 달성 시 축하 모달 표시
접근 경로
  • CEB-ART-101 > 팬덤 레벨 카드 탭
  • CEB-HOM-001 > 바로가기 > {아티스트명} 키우기 카드 탭
  • CEB-HOM-001 > 추천 카드 [기여하기] 탭
화면 레이아웃

화면 레이아웃은 프로토타입(https://celebus-app.vercel.app/home)을 참조하세요. 플로팅 디버그 버튼으로 다양한 상태를 확인할 수 있습니다. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

MAX 상태 레이아웃(Lv.5 달성 + 추가 레벨 미등록)도 프로토타입에서 확인할 수 있습니다.

2.2.1 권장 레벨링 곡선

카운터 모델: 각 레벨에서 0부터 시작하여 아래 "레벨업 목표 덕력" 도달 시 다음 레벨 진입과 동시에 카운터 0으로 리셋. 즉, 각 레벨의 목표값은 이전 레벨 누적이 아니라 해당 레벨 단독 목표값. 진행률은 현재 레벨 카운터 / 해당 레벨 목표 덕력 × 100 (%).

권장 곡선 설계 원칙: 초기 빠른 레벨업 → 후반 점진 증가. 동일 활성도 기준 레벨업 간격이 점차 길어져 도전감 유지.

BO 자유 변경: 운영자(BO 관리자)는 위 권장값을 자유롭게 조정 가능. 미설정 시 권장값 자동 적용.
