1. 화면 개요

정보 피드(CEB-INF-101)에서 소식(뉴스) 항목을 탭했을 때 진입하는 상세 화면이다. 아티스트 관련 소식의 본문, 이미지, 관련 링크를 풍부하게 보여주고 공유할 수 있다.

주요 기능
  • 소식 본문 전체 표시 (제목 + 본문 텍스트 + 이미지)
  • 공식 제공 뱃지 표시 (공식 출처 표기된 콘텐츠)
  • 관련 링크 바로가기
  • 소식 공유 (Web Share API)
  • 이전/다음 소식 네비게이션
접근 경로
  • CEB-INF-101 > 소식(📰) 항목 탭
화면 레이아웃

화면 레이아웃은 프로토타입을 참조하세요. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

3. 공통 규칙

3.1 데이터 갱신

트리거동작
화면 진입소식 상세 데이터 조회
좋아요 토글서버 반영 + 카운트 즉시 갱신
Pull-to-Refresh소식 데이터 재조회 (공통정책 11.2 참조)

3.2 멀티 아티스트 대응

  • 소식은 아티스트별 분리 (artist_id 기반)
  • 이전/다음 네비게이션도 동일 아티스트 내에서만 이동

3.3 비로그인 공통

  • 비로그인도 소식 상세 전체 열람 가능
  • 좋아요 CTA 탭 시 → 로그인 화면 이동

3.4 공통 에러 상태

상태처리
LoadingSkeleton UI (헤더 + 본문 영역)
Default소식 상세 정상 표시
Error에러 안내 + [다시 불러오기] 버튼
삭제된 소식"삭제된 소식이에요" + [돌아가기] CTA
