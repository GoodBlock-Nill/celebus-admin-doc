1. 화면 목적

앱을 열면 가장 먼저 보는 화면이다. "오늘 뭘 하면 되지?"를 즉시 알려주는 허브 역할을 한다.

핵심 원칙: "다음에 뭘 해야 하지?"를 유저가 절대 고민하지 않아야 한다.

주요 기능
  • 캐러셀 배너로 진행 중인 이벤트를 한눈에 파악
  • 아티스트 선택으로 콘텐츠 전환
  • 오늘의 할 일(출석+미션+퀘스트) 컴팩트 카드
  • 상태 기반 핵심 추천 1개
  • 바로가기 그리드로 주요 서비스 빠른 진입
접근 경로
  • 하단 탭 > 홈 (기본 탭)

3. 화면 내 기능설명

3.1 캐러셀 배너 탭

  • [전체 보기 >] 탭 → CEB-HOM-101 이벤트 전체 화면 이동
  • 배너 탭 → 딥링크 URL로 이동 (이벤트 상세)
  • 다른 아티스트 이벤트 탭 시: 아티스트 선택 유지 (자동 전환 안 함). 이벤트 상세만 이동. 뒤로가기 → 홈 복귀, 기존 아티스트 유지 (공통정책 10.4 참조)
  • 미팔로우 아티스트 이벤트 탭 시: 이벤트 상세 이동 + 하단 인라인 배너 '이 아티스트를 팔로우하시겠습니까?' 표시. [팔로우하기] / [닫기] CTA. 팔로우 완료 시 토스트(Success). 배너 닫기 시 이벤트 상세 유지

3.2 아티스트 선택

  • 아티스트 로고 탭 → 하단 탭명 변경 + 홈 아래 콘텐츠(오늘의 할 일/추천/바로가기) 전환
  • 전환 시 서버 재조회

3.3 출석 체크 (홈에서 직접)

  • 오늘의 할 일 카드에서 출석 ☐ 아이콘 탭 → 즉시 완료
  • 토스트(Success) "출석 완료! 덕력 10pt 획득"
  • 스트릭 +1 반영
  • CEB-FQ-501에서도 동일하게 처리 가능 (이중 진입점. 공통정책 6.1 참조)
