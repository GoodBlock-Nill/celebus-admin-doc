1. 화면 개요

아티스트 탭(CEB-ART-101)에서 "{아티스트명} 응원하기" 카드를 탭했을 때 진입하는 화면이다. (유저 대면 명칭: {아티스트명} 응원하기, 예: V01D 응원하기) 아티스트 서포트 이벤트 목록을 표시하고, 유저가 보유 덕력을 응원하여 팬덤 서포트에 참여할 수 있다.

주요 기능
  • [전체] / [참여가능] / [종료] 필터 탭 + "내 참여만 보기" 토글
  • 서포트 이벤트 카드 리스트 (풀 너비 이미지 + 목표금액 + 달성률 + 인라인 [참여하기])
  • 카드의 [참여하기]로 응원 진입 (덕력 응원 UI: 수량 입력 + 확인 모달 + 응원 완료)
  • 완료 이벤트 결과 확인 (사진/영상 인라인 표시)
  • 이벤트 상태: 모집중 / 달성 / 진행중 / 완료 / 미달성종료 / 취소
접근 경로
  • CEB-ART-101 > 서포트 카드 탭
  • CEB-HOM-001 > 바로가기 > {아티스트명} 응원하기 카드 탭
화면 레이아웃

화면 레이아웃은 프로토타입(https://celebus-app.vercel.app/home)을 참조하세요. 플로팅 디버그 버튼으로 다양한 상태를 확인할 수 있습니다. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

동작

덕력 응원 플로우
  1. 수량 입력 (직접 입력 또는 [-]/[+] 버튼)
  2. [응원하기] 탭 → 확인 모달 표시
  3. 확인 모달에서 [응원] 탭 → 서버 처리
  4. 성공 → 완료 토스트 "덕력 {N}DUK 응원 완료!" + 프로그레스 바 갱신 + "내 응원" 갱신 + 보유 덕력 차감 반영
  5. 달성 트리거: 응원으로 누적 >= 목표 시 서버가 즉시 달성 상태로 전환 → 응원 UI 비활성 + 완료 토스트 "🎯 목표 달성!"
누적 응원
  • 동일 이벤트에 여러 번 응원 가능
  • "내 응원: {N}DUK"는 누적 합산 표시
  • 매 응원마다 확인 모달 표시
초과 응원 방지
  • UI에서 입력량이 남은 목표량을 초과 시 남은 목표량으로 자동 조정 ("남은 목표: {N}DUK" 표시)
  • 서버에서도 (현재 누적 + 응원량 > 목표) 확인하여 초과분은 보유 덕력에 유지
  • 예: 목표 15,000DUK, 현재 14,800DUK, 500DUK 응원 시 → 200DUK만 차감, 300DUK 유지
