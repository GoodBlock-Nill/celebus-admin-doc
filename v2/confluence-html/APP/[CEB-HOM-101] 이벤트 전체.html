1. 화면 개요

목적

홈 캐러셀 배너에 노출되는 래플/서포트/이벤트 등 전체 콘텐츠를 리스트로 확인하는 화면이다. 아티스트 무관 통합 표시.

주요 기능
  • [진행중] / [마감됨] 탭 필터로 이벤트 분류
  • 카드형 리스트 세로 나열
  • 카드 탭 시 딥링크 URL로 이벤트 상세 이동
접근 경로
  • CEB-HOM-001 홈 메인 > 캐러셀 배너 > "전체 보기" 탭
레이아웃 참조

화면 레이아웃은 프로토타입(https://celebus-app.vercel.app/home)을 참조하세요. 플로팅 디버그 버튼으로 다양한 상태를 확인할 수 있습니다. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

2.3 이벤트 리스트

동작

  • 카드 탭 → 딥링크 URL로 이벤트 상세 이동
  • 다른 아티스트 이벤트 탭 시: 아티스트 선택 유지 (전환하지 않음). 뒤로가기 시 기존 아티스트 콘텐츠로 복귀
  • 미팔로우 아티스트 이벤트 탭 시: 이벤트 상세 이동 + 하단 인라인 배너 "이 아티스트를 팔로우하시겠습니까?" 표시. [팔로우하기] / [닫기] CTA. 팔로우 완료 시 토스트(Success). 배너 닫기 시 이벤트 상세 유지 (CEB-HOM-001 캐러셀과 동일 패턴)
