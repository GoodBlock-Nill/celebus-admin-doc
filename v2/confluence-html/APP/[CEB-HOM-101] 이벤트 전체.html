1. 화면 개요

목적

홈 화면 상단 EVENTS 탭에서 노출되는 래플/서포트 전체 콘텐츠를 리스트로 확인하는 화면이다. 홈(HOME)과 동일 화면 내 탭 전환으로 진입하며(별도 페이지 아님), 홈 헤더를 그대로 유지한다. 아티스트 무관 통합 표시. 퀘스트는 미포함.

주요 기능
  • [전체] / [참여가능] / [종료] 3개 탭 필터로 이벤트 분류
  • "내 참여만 보기" 토글로 본인 참여 이벤트만 필터
  • 카드형 리스트 세로 나열 (래플/서포트 타입별 지표)
  • 카드 탭 시 딥링크 URL로 이벤트 상세 이동
접근 경로
  • CEB-HOM-001 홈 메인 > 상단 [EVENTS] 탭
  • CEB-HOM-001 홈 메인 > 캐러셀 배너 > "전체 보기" (EVENTS 탭 전환)
레이아웃 참조

화면 레이아웃은 프로토타입(https://celebus-app.vercel.app/home)을 참조하세요. 플로팅 디버그 버튼으로 다양한 상태를 확인할 수 있습니다. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

동작

  • [전체] 탭: 진행중 + 종료 모든 이벤트 표시
  • [참여가능] 탭: 현재 응모/응원 가능한 이벤트만 표시 (진행중)
  • [종료] 탭: 마감된 이벤트 표시 (집계중 + 추첨/집행 완료 포함). 집계중은 카드 "집계 중" 뱃지로 구분
  • "내 참여만 보기" 체크 → 현재 탭 결과에서 본인 참여 이벤트만 추가 필터
  • 탭 전환·토글 시 리스트 재조회

정렬 규칙

필터정렬
전체진행중(참여가능) 먼저 → 종료. 그룹 내 마감 임박/최신순
참여가능D-day 임박순 (마감이 가까운 이벤트 먼저)
종료마감일 최신순 (집계중 → 완료 순, 각 그룹 최신순)

2.3 이벤트 리스트

UI 구성

항목내용
형태카드형 리스트 (세로 나열). 카드 상단 풀 너비 이미지
공통 구성타입 뱃지(응모/서포트) + 아티스트명 + 타이틀 + 기간(YYYY.MM.DD ~ MM.DD) + 상태 뱃지 + CTA
타입 뱃지"응모"(래플) / "서포트"(서포트 이벤트) — 좌상단
래플 카드 지표"응모 {N}" + "당첨인원 {N}명" + D-day
서포트 카드 지표"목표금액 {N} DUK" + 달성률 progress bar({N}%) + D-day
상태 뱃지 (참여가능)"D-{N}" (마감까지 남은 일수)
상태 뱃지 (집계중)"집계 중" (추첨형 이벤트 전용)
상태 뱃지 (종료)래플: "당첨/미당첨/마감" / 서포트: "달성/미달성" + CLOSED 스탬프
카드 CTA참여가능: [응모하기](래플) / [참여하기](서포트) 인라인 버튼. 종료: 버튼 비활성 또는 결과 표시
스타일피그마 참조
서브타이틀 관리 정책

서브타이틀은 BO에서 이벤트별로 1개 입력 + 상태별 자동 전환 방식으로 관리한다.

항목정책
입력 방식BO에서 이벤트 등록 시 서브타이틀 텍스트 1개 입력 (필수)
진행중 표시BO에서 입력한 서브타이틀 그대로 표시 (예: "지금 응모하세요!", "목표 70% 달성!")
집계중 표시"추첨 결과를 기다려주세요"로 자동 전환 (BO 입력값 무시). 추첨형 이벤트 전용
마감됨 표시"종료"로 자동 전환 (BO 입력값 무시). 모든 마감 이벤트에 동일하게 "종료" 표시
최대 길이30자. 초과 시 말줄임(...)
다국어KO/EN/JP 각각 입력 (공통정책 14.1 참조)

설계 근거: 진행중일 때는 이벤트 특성에 맞는 개별 문구가 참여율을 높이고 (예: 래플은 "지금 응모하세요!", 서포트는 "목표 70% 달성!"), 집계중(추첨 대기)·마감 후에는 일관된 자동 문구로 전환해 유저 혼란을 방지한다. 서포트의 경우 달성률은 서버에서 자동 계산하여 서브타이틀에 반영할 수 있다.

동작

  • 카드 탭 → 딥링크 URL로 이벤트 상세 이동
  • 다른 아티스트 이벤트 탭 시: 아티스트 선택 유지 (전환하지 않음). 뒤로가기 시 기존 아티스트 콘텐츠로 복귀
  • 미팔로우 아티스트 이벤트 탭 시: 이벤트 상세 이동 + 하단 인라인 배너 "이 아티스트를 팔로우하시겠습니까?" 표시. [팔로우하기] / [닫기] CTA. 팔로우 완료 시 완료 토스트. 배너 닫기 시 이벤트 상세 유지 (CEB-HOM-001 캐러셀과 동일 패턴)
