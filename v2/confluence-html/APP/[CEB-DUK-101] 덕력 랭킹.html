1. 화면 개요

아티스트 탭(CEB-ART-101)에서 "덕력" 카드를 탭했을 때 진입하는 화면이다. 현재 시즌 덕력 랭킹(리더보드)과 내 덕력 현황을 확인할 수 있다.

주요 기능
  • 내 랭킹 정보 (순위/획득 덕력/보유 덕력/시즌 진행 상태)
  • TOP 100 리더보드 (내 순위 하이라이트 + 자동 스크롤)
  • 시즌 아카이브 (이전 시즌 랭킹 드롭다운 전환)
  • 덕력 이력 바텀시트
접근 경로
  • CEB-ART-101 > 덕력 카드 탭
  • CEB-HOM-001 > 추천 카드 [랭킹 보기] 탭
화면 레이아웃

화면 레이아웃은 프로토타입(https://celebus-app.vercel.app/home)을 참조하세요. 플로팅 디버그 버튼으로 다양한 상태를 확인할 수 있습니다. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

2.4 TOP 100 리더보드

UI 구성

항목내용
구성순위 + 닉네임 + 획득 덕력(pt)
상위 3위 아이콘🥇(1위) / 🥈(2위) / 🥉(3위) 아이콘 표시
내 순위 하이라이트보라색 하이라이트 + ⭐ 표시. 진입 시 내 순위 행으로 자동 스크롤
동일 포인트먼저 도달한 유저가 상위 순위 (동순위 없음, 모든 유저 고유 순위). 예: A(100pt, 4/10 도달) = 38위, B(100pt, 4/15 도달) = 39위
TOP 100 밖리스트 하단에 "── 내 순위 ──" 별도 섹션으로 내 순위만 표시

동작

  • 진입 시 현재 시즌의 TOP 100 + 내 순위 조회
  • 내 순위가 TOP 100 내이면 해당 행으로 자동 스크롤 + 하이라이트
  • 내 순위가 TOP 100 밖이면 리스트 하단 별도 섹션에 내 순위 표시
동일 포인트 처리
  • 같은 획득 덕력 포인트를 가진 유저는 해당 포인트에 먼저 도달한 유저가 상위 순위 (동순위 없음)
  • 모든 유저는 고유 순위를 가짐 (순위 스킵 없음, 연속 순위)
  • 예: A(100pt, 4/10 도달) = 38위, B(100pt, 4/15 도달) = 39위
  • 시즌 보상: 동일 포인트여도 도달 시점에 따라 순위가 다르므로 순위별 보상 적용 (보상 상이 가능)

3. 공통 규칙

3.1 데이터 갱신

트리거동작
화면 진입현재 시즌 랭킹 + 내 정보 조회
Pull-to-Refresh현재 시즌 전체 재조회 (공통정책 11.2 참조)
시즌 드롭다운 전환선택한 월의 아카이브 데이터 조회
복귀 시내 정보 재조회 (활동으로 포인트 변동 가능)

3.2 Pull-to-Refresh

  • 현재 시즌에서만 동작 (이전 시즌 아카이브는 고정 데이터)
  • 공통정책 11.2 규격 적용

3.3 멀티 아티스트 대응

  • 아티스트별 랭킹 완전 분리
  • artist_id 기반으로 모든 데이터 조회
