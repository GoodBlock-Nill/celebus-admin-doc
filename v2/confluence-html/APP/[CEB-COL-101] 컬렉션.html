1. 화면 개요

아티스트 탭(CEB-ART-101)에서 "컬렉션" 카드를 탭했을 때 진입하는 화면이다. 유저가 보유한 디지털 굿즈(BIVE)를 카테고리별로 관리하고, 상세 보기/보내기/합성/Pick 등의 기능을 제공하는 컬렉션 허브 역할을 한다.

유저 대면 명칭: "컬렉션". 내부 NFT 브랜드명: "BIVE". (Figma v.3 시안 라벨은 "BIVE 도감"·"바이브"·"도감 보기" — 대면 표준명은 "컬렉션"으로 유지하며, 디자인팀에 라벨 정합 권고)

주요 기능
  • 카테고리 필터([아티스트] [이벤트] [스페셜]) + 2열 그리드 리스트
  • BIVE 상세 보기 (이미지 + 등급 + 설명 + 보이스/서사 텍스트)
  • BIVE 보내기 (다른 유저에게 전송)
  • 합성 (Grade 1~5 전부 보유 시 → 스페셜 에디션 합성)
  • Pick (보유 BIVE 1장 소진 → 새 BIVE 뽑기)
  • BIVE 히스토리 (획득/전송/삭제 이력)
  • BIVE 혜택 (혜택현황 탭 / 지급내역 탭)
  • BIVE 삭제 (보유 BIVE 삭제 가능)
접근 경로
  • CEB-ART-101 > 컬렉션 카드 탭
  • CEB-HOM-001 > 추천 카드 [받으러 가기] 탭
화면 레이아웃

화면 레이아웃은 프로토타입(https://celebus-app.vercel.app/home)을 참조하세요. 플로팅 디버그 버튼으로 다양한 상태를 확인할 수 있습니다. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

2.4 BIVE 상세 / 보내기 / Pick / 합성 / 히스토리 / 혜택 / 삭제

아래 기능들은 추후 별도 화면 스펙으로 분리될 수 있습니다. 현재는 CEB-COL-101 내 요약으로 기술합니다.

>

BIVE 상세(CEB-COL-201), 보내기(CEB-COL-101-MD-SEND), Pick(CEB-COL-101-MD-PICK), 히스토리(CEB-COL-301), 혜택(CEB-COL-401) 화면은 별도 화면 스펙으로 추후 작성합니다.

BIVE 상세 보기

  • 보유 BIVE 카드 탭 시 상세 화면으로 이동
  • 상세 내용: 풀사이즈 이미지 + 등급 뱃지 + BIVE 이름 + 서사 텍스트 + 보이스 재생(보이스가 있는 경우에만 노출)
  • BIVE 파일 타입: 이미지 Only / 이미지+음원 / 이미지+영상
  • 음원이 포함된 BIVE: 재생 버튼 표시
  • 영상이 포함된 BIVE: 영상 미리보기 표시
  • 상세 화면 내 액션: [보내기] / [삭제]

BIVE 보내기

  • 보유 BIVE를 다른 유저에게 전송하는 기능
  • 진행 순서: 상세 화면 [보내기] 탭 → 수신자 검색/선택 → 전송 확인 모달(CEB-COL-101-MD-SEND) → 전송 완료 토스트(Success)
  • 전송 완료 후 해당 BIVE가 내 컬렉션에서 제거됨

합성

  • Grade 1~5 서사 굿즈(아티스트 카테고리) 5장 전부 보유 시 [합성] 버튼 활성
  • 합성 진행 시 원본 Grade 1~5 유지 + 스페셜 에디션 1장 추가 생성 (공통정책 5.4 참조)
  • 이미 합성 완료(스페셜 에디션 보유) 시 [합성] 버튼 비활성
  • 조건 미충족 시: [합성] 탭 → "Grade 1~5를 모두 보유해야 합성할 수 있습니다" 토스트(Info)
합성 규칙
규칙설명
합성 조건아티스트 카테고리 Grade 1~5 서사 굿즈 5장 전부 보유
합성 결과원본 Grade 1~5 유지 + 스페셜 에디션 1장 추가 생성
합성 가능 횟수아티스트당 1회. 이미 합성 완료(스페셜 에디션 보유) 시 재합성 불가

Pick

  • 보유 BIVE 1장을 소진하여 랜덤으로 새 BIVE 1장을 획득하는 가챠 기능
  • 진행 순서: [Pick] 탭 → 소진할 BIVE 선택 → 확인 모달(CEB-COL-101-MD-PICK) → 뽑기 애니메이션 → 획득 결과 화면
  • 보유 BIVE 0장 시 [Pick] 버튼 비활성

BIVE 히스토리

  • 내 BIVE 획득/전송/받기/삭제 전체 이력을 시간순(최신 순)으로 표시
  • 각 항목: BIVE 이름 + 이벤트 유형(획득 / 보내기 / 받기 / 삭제) + 날짜

BIVE 혜택

  • 혜택현황 탭: 현재 보유 BIVE로 인해 해금된 혜택 목록 표시
  • 지급내역 탭: BIVE 관련 보상 지급 이력 표시

BIVE 삭제

  • 상세 화면에서 [삭제] 탭 → 확인 모달(CEB-COL-101-MD-DELETE, "삭제한 BIVE는 복구할 수 없습니다") → 삭제 완료 토스트(Success)
  • 삭제 완료 후 해당 BIVE가 컬렉션에서 제거됨

  • 컬렉션은 아티스트별로 분리하여 표시 (artist_id 기반)
  • 아티스트 전환 시 CEB-ART-101 재진입 후 해당 아티스트의 컬렉션으로 접근

3.3 비로그인 공통

  • 이 화면은 로그인 상태에서만 접근 가능
  • CEB-ART-101 컬렉션 카드 탭 시 로그인 유도 처리 (공통정책 10.5 참조)
