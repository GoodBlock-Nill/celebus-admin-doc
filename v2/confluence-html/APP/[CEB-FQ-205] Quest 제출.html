1. 화면 개요

목적

유저가 퀘스트의 구체적인 수행 방법(가이드)을 확인하고, 요구사항에 맞는 이미지(인증샷)를 촬영하거나 업로드하여, 운영진에게 심사를 요청(제출)하는 화면이다.

주요 기능
  • 퀘스트 기본정보 조회 (타이틀, 설명)
  • 이미지 촬영 또는 갤러리 업로드
  • 업로드 사진 가이드 확인 (아코디언)
  • 제출하기 플로우 (심사 요청)
접근 경로
  • CEB-FQ-101 Quest 메인 > 활성 장 > 미션 [참여하기] 탭 시 이동 (캡처 제출형 미션)
  • CEB-FQ-101 Quest 메인 > 타임라인 > 반려 미션 → CEB-FQ-201-MD 반려 사유 모달에서 [재도전] 탭 시 본 화면으로 이동 (기존 이미지 클리어 상태)
  • 진입 가능 조건: 해당 미션의 상태가 INCOMPLETE(미완료) 또는 REJECTED(반려)인 경우에만 진입
레이아웃 참조

화면 레이아웃은 프로토타입(https://celebus-app.vercel.app/home)을 참조하세요. 플로팅 디버그 버튼으로 다양한 상태를 확인할 수 있습니다. 프로토타입은 구조/플로우 확인용이며, 시각적 스타일은 Figma 시안을 따릅니다.

UI 구성

항목내용

성공"제출 완료! 결과는 빠르면 하루 안에 알려드릴게요" Success 토스트 노출 → 화면을 닫고 CEB-FQ-101 Quest 메인(타임라인)으로 복귀 → 해당 미션의 로컬 상태를 SUBMITTED(심사중)으로 업데이트
실패에러 유형별 Error 토스트 노출

제출 즉시 해당 건은 Admin의 검토 대기 목록([CEB-BO-FQ-202])에 등록된다.

상황처리

3. 공통 규칙

3.1 반려 후 재진입

  • 기존에 반려된 이미지는 초기화(삭제)된 상태로 진입
  • 유저는 새 이미지를 업로드하여 다시 제출 가능
  • 진입 경로: CEB-FQ-101 타임라인 > 반려 미션 [재도전] 탭 (CEB-FQ-101 3.2 참조)

3.2 보상 지급 정책

  • 보상(덕력, 응모권 등)은 심사 승인 시 즉시 지급
  • 보상 지급 상세 정책은 공통정책 8.5 참조

3.3 심사 SLA

  • 제출 완료 토스트에 "결과는 빠르면 하루 안에 알려드릴게요" 심사 완료 기준 안내
  • 심사 SLA 상세 기준은 CEB-FQ-101 4.6 참조

3.4 다국어 처리

항목다국어 범위
퀘스트 타이틀서버 데이터 (다국어 지원 여부 어드민 설정에 따름)
퀘스트 설명서버 데이터 (다국어 지원 여부 어드민 설정에 따름)
가이드 텍스트서버 데이터 (다국어 지원 여부 어드민 설정에 따름)
