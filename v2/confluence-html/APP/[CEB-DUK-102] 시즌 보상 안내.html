1. 화면 개요

목적

덕력 랭킹(CEB-DUK-101)에서 진입하는 시즌 안내 화면이다. 현재 시즌 기간, 시즌 보상 구간, 덕력을 빠르게 쌓는 방법, 유의사항을 안내한다. (Figma 시안 라벨: "시즌 보상 안내")

주요 기능
  • 시즌 정보 (현재 시즌명·기간·종료 D-day)
  • 시즌 보상 구간별 안내 (1위 / TOP 10 / TOP 10% 등)
  • 덕력 쌓는 방법 안내 + [덕력 쌓으러 가기]
  • 유의사항 안내
접근 경로
  • CEB-DUK-101 덕력 랭킹 > 시즌 정보/보상 안내 진입
화면 레이아웃

시각적 스타일은 Figma 시안을 따릅니다. 변형: 진행중 / 시즌 종료.

2.3 시즌 보상

UI 구성

항목내용
안내"보상은 최종 달성 랭킹 기준 상위 보상만 지급돼요."

TOP 10% 진입 시보상 항목 (예: 사인 포토카드 + 디지털 포토카드 + 수퍼팬 뱃지)

보상 구간(등수/퍼센트)과 보상 상품은 시즌별 BO 설정 ([CEB-DUK-101] §2.5 참조). 위 항목은 기본 예시.
