3.2 덕력 이원화 원칙

덕력은 획득 덕력과 보유 덕력으로 이원 관리된다. 유저 화면에서는 통합 표시하고, 상세 페이지에서만 구분한다.

구분설명용도초기화
획득 덕력활동할 때마다 누적개인 랭킹 산출 기준연 1회 초기화 (BO에서 시즌 종료일 설정, 종료 시 자동 초기화)
보유 덕력획득 - 사용서포트 이벤트 응원, 독점 콘텐츠 해금소멸 없음, 영구 유지
사용 덕력서포트 이벤트 + 독점 콘텐츠에 응원한 총량이력 조회용-
  • 서포트 이벤트에 덕력을 응원해도 랭킹에는 영향 없음 (랭킹은 획득 덕력 기준)
  • 유저 화면: 통합 표시 ("내 덕력 1,234pt"), 상세에서만 획득/보유/사용 구분

연간 시즌 초기화 절차

순서절차
1BO에서 시즌 종료일 도래 → 랭킹 동결
2관리자가 TOP 순위 확인 후 보상 수동 지급
3관리자가 "시즌 초기화" 실행 → 획득 덕력 0으로 리셋
4새 시즌 자동 시작

보유 덕력은 초기화되지 않음. 진행 중인 서포트 이벤트 응원분도 영향 없음.

시즌 간 공백: 종료~초기화 사이 획득 덕력은 새 시즌에 귀속.
