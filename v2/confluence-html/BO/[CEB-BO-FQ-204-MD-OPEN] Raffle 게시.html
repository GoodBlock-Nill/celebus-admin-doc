4.2 게시 전 검증

[CEB-BO-006] §5.2 다국어 게시 검증:

  • KO/EN/JP 3언어 모두 비어있지 않아야 함
  • 1개라도 빈 칸 → 게시 차단 + Toast (Error)
  • 백엔드 추가 검증: 필수 항목 누락 시 차단

4.3 게시 프로세스

  • [확인] → 서버 API 호출
  • 시작 일시: 서버 시간(KST UTC+9) 자동 기록
  • 상태: Draft → Active
  • 앱 즉시 노출
