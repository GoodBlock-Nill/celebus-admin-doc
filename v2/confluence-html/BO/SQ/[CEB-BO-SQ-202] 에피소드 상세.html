2-5. 미션 그리드

  • 섹션 헤더: "미션 (N/10)" + 우측 [미션 추가] 버튼
  • 10개 도달 시 [미션 추가] 비활성 + 안내 툴팁
  • 카드 그리드 — 표시 순서 오름차순 정렬
  • 각 카드 (v2.2부터 "완료 조건 수" 행 제거):
  • 미션 썸네일
  • 유형 배지 (팬퀘스트 / 예측 마켓 / 서바이벌 트리비아)
  • 미션 한국어 타이틀
  • 보상 행: 응모권 · 팬덤 포인트 · BIVE 보상 ON/OFF
  • 팬퀘스트 유형일 때 매핑된 팬퀘스트 타이틀 표시
  • 우상단 ↑↓ 화살표 버튼 — 순서 변경, 임시저장 상태에서만 활성
  • 카드 본문 클릭 → 미션 상세 화면([CEB-BO-SQ-204], /sq/{id}/quests/{qid}) 이동

4-1. 게시 (임시저장 → 진행중) 흐름

  1. 부모 그룹이 진행중 상태인지 검증 (임시저장 그룹에서는 에피소드 게시 불가)
  2. 다국어 한국어/영어/일본어 3개 언어 입력 완료 검증
  3. 메인 이미지 등록 검증
  4. 확인 모달 → 상태 = 진행중

4-2. 종료 (진행중 → 종료) 흐름

  1. 확인 모달 "에피소드를 종료합니다. 모든 하위 미션도 종료됩니다."
  2. 확인 → 상태 = 종료

4-3. 삭제 (임시저장만)

  1. 확인 모달 "에피소드를 삭제합니다. 복구할 수 없습니다."
  2. 확인 → 그룹 상세로 복귀, 토스트
