1. 모달 개요

특정 미션(ERD story_quest_chapter) 안에 새 완료 조건을 추가하는 확인성 모달. 부모 미션의 type(FAN_QUEST / PREDICTION_MARKET / SURVIVAL_TRIVIA)에 따라 입력 가능한 판정 유형 옵션이 분기된다.

항목값
트리거미션 상세의 완료 조건 추가 버튼
진입 조건미션의 완료 조건 수 < MAX_MISSIONS_PER_EPISODE (10)
차단 조건만석(10개) 도달 시 버튼 자체 disabled + 툴팁 안내
접근 권한OPERATOR, EXTERNAL_ARTIST(자기 아티스트)
데이터 모델StoryMission ([prototype/src/mock/sq.ts](prototype/src/mock/sq.ts) lines 99~109)

  • REPEAT 에피소드: 반복 에피소드의 미션이므로 주기(주간/월간)를 설정할 수 있습니다.
