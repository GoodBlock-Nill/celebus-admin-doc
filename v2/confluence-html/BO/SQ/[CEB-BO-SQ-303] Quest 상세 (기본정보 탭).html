1. 화면 개요

특정 Quest 1건의 모든 운영 정보·통계·다국어 콘텐츠·연관 에피소드 사용 현황을 한 화면에 노출하는 운영자용 허브. 상단 3탭(기본정보 / 대기내역 / 처리내역) 중 기본정보 탭.

항목값
화면 경로/fanquest/{id} (기본 탭 = info), 목표 라우트 /sq/quests/{id}
Breadcrumb홈 / 에피소드 / 팬퀘스트 / Quest 상세
페이지 제목Quest의 한국어 타이틀
진입 경로[CEB-BO-SQ-302] 팬퀘스트 리스트의 행 클릭
접근 권한[관리자 > 권한관리] 설정에 따른다
자매 탭[CEB-BO-SQ-304] 대기내역 / [CEB-BO-SQ-305] 처리내역

2. 화면 구성

2-1. 헤더

  • 좌측: ← 뒤로가기 → 리스트로 복귀
  • 페이지 제목: Quest 한국어 타이틀 (최대 너비 640px, 길면 줄바꿈)
  • 우측: 상태 배지 (진행중=에메랄드 / 임시저장=회색 / 종료=진한 회색) + 상태별 액션 버튼

2-2. 상태별 헤더 액션 (CTA)

상태우측 버튼

진행중[수정하기] (회색) · [종료하기] (빨강)
종료(액션 없음 — read-only)

2-6. 에피소드 사용 현황 패널 (카드)

본 Quest를 1:1로 사용 중인 에피소드의 미션 매핑 현황 표시 (SQ 영역 컨텍스트):

  • 매핑된 미션 0건: "이 팬퀘스트는 아직 어떤 에피소드 미션에도 매핑되지 않았습니다." 안내
  • 매핑된 미션 N건: 각 미션의 부모 에피소드 그룹·에피소드명·미션명·순서 표시. 행 클릭 시 [CEB-BO-SQ-204] 미션 상세로 이동

[CEB-BO-011] §7 외부 영역 연동 정합. 팬퀘스트 ↔ 미션은 1:1 매핑이므로 본 팬퀘스트는 최대 1개 미션에만 연결됨.

2-7. 다국어 콘텐츠 섹션 (각 카드 3행)

다음 3개 항목을 각각 카드 1개로 노출. 한국어/영어/일본어 3행 (각 행에 언어 배지 + 본문 또는 "—"):

  • Quest 타이틀 (KO/EN/JA)
  • Quest 설명 (KO/EN/JA)
  • Quest 유저 가이드 (KO/EN/JA)
