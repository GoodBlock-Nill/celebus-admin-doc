1. 화면 개요

기존 Quest 1건의 정보를 수정하는 폼 화면. 생성 화면([CEB-BO-SQ-302-CREATE])과 동일한 3섹션 폼을 재사용하되, 진행중 상태일 때 일부 필드 자동 잠금.

항목값
화면 경로/fanquest/{id}/edit (목표 /sq/quests/{id}/edit)
Breadcrumb홈 / 에피소드 / 팬퀘스트 / Quest 수정
페이지 제목"Quest 수정" + 상태 배지
진입 경로[CEB-BO-SQ-303] Quest 상세의 [수정하기] 버튼 (임시저장·진행중에서만 노출)
접근 권한[관리자 > 권한관리] 설정에 따른다

2. 화면 구성

2-1. 헤더

  • 페이지 제목 "Quest 수정" + 상태 배지 (진행중=에메랄드 / 임시저장=회색)
  • 우측 액션 2개:
  • [취소하기] (빨강) — 변경사항 확인 후 상세 화면으로 복귀
  • [수정하기] (인디고) — 검증 통과 시 활성

종료 상태 진입 시: "종료된 Quest는 수정할 수 없습니다. [상세로 돌아가기]" 안내 화면만 노출 (폼 미렌더링)

2-2. 진행중 상태 배너 (조건부 노출)

진행중 Quest 진입 시 상단에 앰버 톤 안내 박스:

진행중 상태에서는 기본 정보(타이틀, 설명, 가이드, 연관 링크)만 수정 가능합니다.

2-3. 에피소드 사용 현황 배너

진행중 Quest 진입 시 본 Quest를 사용 중인 에피소드 정보 표시 (변경 영향도 안내).

2-4. 입력 폼 (3섹션, [CEB-BO-SQ-302-CREATE] §2-2와 동일 구조)

섹션임시저장 수정 가능진행중 수정 가능
A. 기본 정보 — 타이틀·설명·유저 가이드 (다국어)✅✅
A. 기본 정보 — 대표 이미지✅❌ (잠금)
B. 퀘스트 설정 — 아티스트·타입✅❌ (잠금)
B. 퀘스트 설정 — 참여 방식·반복 주기·기간✅❌ (잠금)
B. 퀘스트 설정 — 연관 링크✅✅
C. 보상 설정✅❌ (잠금)

진행중 잠금 필드는 시각적으로 비활성 처리 + opacity 50% + 커서 not-allowed.
