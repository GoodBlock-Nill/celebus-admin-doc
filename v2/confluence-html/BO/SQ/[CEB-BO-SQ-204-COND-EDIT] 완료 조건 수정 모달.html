1. 모달 개요

기존 완료 조건 정보를 수정. 부모 미션의 type은 변경 불가이고, 그 결과 sourceType도 변경 불가. 부모 에피소드/그룹이 CLOSED 상태이면 수정 자체 차단.

항목값
트리거미션 상세 완료 조건 테이블 행 클릭
진입 조건부모 에피소드/그룹의 status ≠ CLOSED
차단 조건CLOSED 시 read-only 또는 클릭 비활성
접근 권한OPERATOR, EXTERNAL_ARTIST(자기 아티스트)
