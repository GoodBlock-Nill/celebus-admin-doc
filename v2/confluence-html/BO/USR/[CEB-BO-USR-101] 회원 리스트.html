1. 화면 목적

CELEBUS 가입자(회원) 통합 조회·검색·필터 화면. 운영자는 본 화면에서 회원 라이프사이클 전반을 모니터링하고 개별 회원 상세로 진입하여 제재·자산 회수·운영자 메모 관리.

주요 기능

  • 전체 회원 리스트 조회 + 계정 상태별 카운터 대시보드
  • 다중 조건 검색·필터 (계정 상태·유저명·전화번호·기간 등)
  • 회원 상세 진입 ([CEB-BO-USR-201])
  • 정지사유 설정 진입 ([CEB-BO-USR-501])

진입 경로

  • 사이드 메뉴 > 회원

EXTERNAL_ARTIST 차단: 회원 정보는 본사 운영 영역.

4.2 정렬·필터

기본 정렬가입일 내림차순 (최신순)
  • 계정 상태 필터 + 정렬 + 유저명 검색 = AND 조합
  • 유저명 검색은 부분 일치(LIKE)
  • 검색어 미입력 시 해당 조건 무시
