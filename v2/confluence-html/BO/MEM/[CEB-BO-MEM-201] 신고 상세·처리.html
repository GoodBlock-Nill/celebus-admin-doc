3.2 처리 프로세스

신고 인정 ([CEB-BO-MEM-000] §4.3)

  1. [신고 인정] 클릭 → 처리 사유 입력 모달
  2. 게시물 즉시 차단 (앱 미노출)
  3. 신고자 푸시 알림 ("신고하신 게시물이 차단되었습니다")
  4. 작성자 푸시 알림 ("작성하신 게시물이 차단되었습니다")
  5. 활동 로그 기록
  6. (선택) [회원 제재] 클릭 → USR-201-MD-SANCTION

신고 반려

  1. [신고 반려] 클릭 → 반려 사유 입력 모달
  2. 게시물 유지
  3. 신고자 푸시 알림 ("신고가 반려되었습니다 — 사유: {요약}")
  4. 활동 로그 기록

3.3 자동 차단 (다중 신고 누적)

동일 게시물에 신고 5건 이상 누적 시:

  • 시스템 자동 임시 차단 (Pending → 즉시 앱 미노출)
  • 운영자 푸시 알림
  • 운영자 검토 후 [신고 인정] 또는 [신고 반려] 처리
