1. 화면 개요

회원상세 페이지 내 "게임존" 탭으로, 특정 회원의 게임존 활동 전체(GP 보유·획득·사용, 게임 참여 이력, 보상·환급, GP 변동)를 한눈에 파악하는 조회 전용 화면.

2. 화면 구성

회원상세 페이지의 9개 탭 중 3번째 탭 "게임존". 탭 구조는 [USR-201]에 정의되어 있으며 본 문서는 게임존 탭 본문만 다룬다.

전체 탭 (운영 정합): 기본정보 / 참여내역 / 게임존 / BIVE / 프로젝트 / Fans / 티켓 / 지갑 / 활동내역
