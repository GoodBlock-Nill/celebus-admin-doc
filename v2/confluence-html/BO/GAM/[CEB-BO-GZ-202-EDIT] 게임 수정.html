2-4. 폼 본문

[201-CREATE] §2-3 (PM) / §2-4 (ST)와 동일한 폼 구조를 가지며, 차이점은:

  • 각 필드에 기존 값이 prefilled
  • §2-2 / §2-3 매트릭스에 따라 일부 필드가 disabled 처리
  • 게임유형은 항상 텍스트로만 표시 (라디오 잠금)
  • 이미지 영역은 미리보기 + [변경] [제거] 버튼 형태 (Pending에서는 모두 disabled)
  • 다국어 탭(KO/EN/JP)도 해당 필드 잠금 시 함께 disabled

폼 항목 상세는 [CEB-BO-GZ-201-CREATE] §2-3·§2-4 참조.
