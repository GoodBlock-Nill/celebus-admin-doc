1. 모달 개요

게임 생성 화면에서 [취소] 버튼 클릭 시, 작성 중인 데이터가 저장되지 않음을 경고하고 최종 확인을 받는 확인 모달. 실수로 인한 데이터 손실 방지가 목적.

항목값
트리거[CEB-BO-GZ-201-CREATE]의 [취소] 버튼 클릭
차단 조건없음 (항상 모달 표시) — 운영 BO 실측: 입력 유무와 무관하게 모달 노출
영향 범위[취소하기] 선택 시 입력 데이터 폐기 + 게임 리스트로 이동
접근 권한[관리자 > 권한관리] 설정에 따른다
