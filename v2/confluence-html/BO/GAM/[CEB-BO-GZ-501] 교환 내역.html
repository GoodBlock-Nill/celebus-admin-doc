1. 화면 개요

게임존 GP ↔ CELB 교환 내역을 조회하고 운영 지갑 현황·당일 거래 통계를 한눈에 파악하는 화면. 관리자는 운영 지갑(충전·출금) 잔액·BNB·상태를 모니터링하고, 교환 방향·기간·닉네임 필터로 거래 내역을 탐색하며, 행 클릭으로 [CEB-BO-GZ-501-MD-DETAIL] 교환 상세 모달을 열어 교환 전후 GP 잔액과 처리 상태를 확인한다.

5. 검증 규칙

5-1. 통계 카드 집계 (오늘 = KST 00:00 ~ 현재)

카드집계 기준

탈퇴 유저 GP탈퇴 유저 보유 GP 총합 (회수·소각 정책 별도)

5-2. 교환 비율 표기

  • 운영 표기 형태: "1 CELB = 1 GP" (CELB → GP 충전 시 의미) 또는 "1 GP = 1 CELB" (GP → CELB 출금 시 의미)
  • 교환 시점의 비율이 기록되며, [CEB-BO-GZ-502] 교환 설정 변경 후에도 과거 거래는 당시 비율 유지 (소급 적용 없음)

5-3. 운영 지갑 정책

  • 운영 지갑은 충전용·출금용 각 1개의 [대표] 지갑이 활성 상태로 동작
  • [대표] 미지정 시 첫 번째 활성 지갑의 잔액 표시
  • 출금 지갑 BNB 잔고 < 0.01 BNB 시 가스비 부족 경고 (텍스트 강조 + 아이콘)
  • 지갑 등록·수정·삭제는 [CEB-BO-GZ-503] 지갑 관리에서 처리

5-4. 권한 및 활동 로그

  • 권한은 [관리자 > 권한관리]에 설정된 그룹별 권한을 따름 ([CEB-BO-SYS-102])
  • 활동 로그는 [CEB-BO-GZ-000] 공통 정책 §활동 로그 정합
  • 조회 자체는 활동 로그 미기록 (조회 로그 정책)
  • 거래 내역은 삭제·수정 불가 (이력 보존)
