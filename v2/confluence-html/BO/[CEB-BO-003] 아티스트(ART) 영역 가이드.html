1. 개요

아티스트(ART) 영역은 그룹·멤버·소속사·포지션 등 K-pop 아티스트 마스터 데이터를 관리한다. v1에서 그룹 상세 3탭(기본·SNS&LINK·멤버) + 멤버 상세 2탭(기본·소속그룹)을 운영 중이며, v2에서 그룹 상세에 "활동 통계" 탭을 추가하고, 소속사 ↔ EXTERNAL_ARTIST 권한자 매핑을 신규 정의한다.

7.3 소속사 설정 ↔ EXTERNAL_ARTIST 매핑

EXTERNAL_ARTIST(외부 협력사 권한자)는 본인이 매핑된 소속사 산하의 모든 그룹·멤버에 R/W 권한을 가진다. 1:N 관계.

매핑 모델

소속사 (예: IX Entertainment)
  ├── 그룹 1 (예: V01D)
  │     └── 멤버 N명
  └── 그룹 2 (예: 다른 그룹)
        └── 멤버 N명
        
EXTERNAL_ARTIST 권한자 (예: ix_admin@ix-ent.kr)
  └── 소속사 'IX Entertainment'에 매핑됨
       → IX Entertainment 산하 모든 그룹·멤버 R/W
       → 다른 소속사 그룹·멤버 접근 차단

매핑 정책

항목정책
매핑 권한SUPER 전용
매핑 화면[CEB-BO-ART-103-MD-MAPPING]
매핑 입력EXTERNAL_ARTIST 이메일(또는 사용자 ID) ↔ 소속사 ID 1:N
매핑 검증EXTERNAL_ARTIST 권한이 부여된 사용자만 매핑 대상으로 선택 가능
매핑 변경SUPER만 가능. 변경 즉시 EXTERNAL_ARTIST 세션 권한 갱신
매핑 해제SUPER만 가능. 해제 시 즉시 해당 외부 권한자의 세션 강제 종료 + 데이터 접근 차단
매핑 만료EXTERNAL_ARTIST 권한 만료([CEB-BO-000] §3.2) 시 자동 매핑 해제
활동 로그매핑 추가·해제 모두 [CEB-BO-000] §6 표준 로그

소속사 변경 시 권한 재할당 흐름

소속사 ↔ 그룹 매핑 변경(예: 그룹 'V01D'가 소속사 'A' → 'B' 이전)이 발생할 경우:

SUPER가 [CEB-BO-ART-103]에서 그룹의 소속사 변경
    ↓
시스템 자동: 기존 소속사의 EXTERNAL_ARTIST → 해당 그룹 접근 즉시 차단
신규 소속사의 EXTERNAL_ARTIST → 해당 그룹 접근 자동 부여
    ↓
관련 EXTERNAL_ARTIST 세션 권한 갱신 + 활동 로그 기록
