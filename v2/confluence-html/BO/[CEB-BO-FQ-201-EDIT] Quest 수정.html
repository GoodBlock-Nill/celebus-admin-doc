1. 화면 목적

이미 생성된 Quest의 정보를 변경. Quest 상태(Draft / Active)에 따라 수정 가능 항목 범위를 엄격히 제어 — 운영 유연성과 데이터 무결성 동시 확보. [CEB-BO-006] §7.1 라이프사이클 적용.

주요 기능

  • Draft 상태: 모든 항목 수정 가능
  • Active 상태: 기본 정보(타이틀/설명/유저 가이드/대표 이미지/연관 링크/마감 일시 연장만) 수정 가능. 핵심 정보(아티스트/타입/보상) 비활성화
  • 다국어 입력 패널(KO/EN/JP)을 통한 콘텐츠 수정

진입 경로

  • [CEB-BO-FQ-201] 퀘스트 상세 (기본정보 탭) > [수정하기] 버튼 클릭
