1. 화면 목적

추첨 완료(Ended) Raffle의 당첨자(Win)에 한해 운영자가 비고사항(배송 정보·CS 기록 등) 수동 기록. 낙첨자(Lose) Row 클릭 시에는 모달 호출되지 않음.

진입 경로

  • [CEB-BO-FQ-206] 추첨내역 탭 (Ended 상태) > Win Row 클릭

4.1 모달 호출 조건

  • 화면: [CEB-BO-FQ-206] 추첨내역 탭
  • 조건: Row 상태 = Win
  • 예외: Lose Row 클릭 시 동작 없음

4.3 데이터 바인딩

  • 클릭한 당첨자 ID 기준 비고 데이터 조회 (있으면 바인딩)
  • 저장 시 UPDATE (덮어쓰기). 이력 보존은 활동 로그가 담당

4.4 유효성

  • 빈 비고 저장 가능 (기존 비고 삭제 용도)
  • 200자 초과 시 인라인 에러: "200자 이내로 입력해주세요."
