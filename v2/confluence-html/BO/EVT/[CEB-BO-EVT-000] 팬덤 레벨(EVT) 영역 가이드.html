1. 개요

팬덤 레벨(EVT) 영역은 아티스트별 팬덤 공동 레벨 시스템을 운영하는 영역이다. 팬덤 전체가 획득한 덕력이 누적되어 레벨업하며, 레벨별로 보상이 지급된다. 운영자는 팬덤레벨을 아티스트·시즌 단위로 직접 생성(수동)하고, 레벨 곡선·보상을 설정하며 상태(임시저장→진행중→종료)를 관리한다. 앱 [CEB-EVT-201] "{아티스트} 키우기" 화면에 반영된다.

팬덤 레벨(공동 레벨)과 덕력관리(개인 덕력 시즌·랭킹, [아티스트 > 덕력관리])는 별개 영역이다. 팬덤 레벨은 개인 경쟁이 아닌 팬덤 전체의 공동 달성 목표다.

진행중앱 노출·운영 중. 팬덤 누적 덕력에 따라 레벨업 진행, 디지털 굿즈(BIVE) 실시간 자동 민팅노출운영자 [게시]

  • 운영자 수동 상태 전환은 [게시](준비→진행중) 만 수행한다.
  • 시즌은 매년 12월 31일 23:59:59 자동 종료된다. 종료 시 시스템이 최종 레벨 확정·통계 마감(내부 정산 처리)을 거쳐 종료 상태가 되며, 운영자 정산 작업은 없다 (디지털 굿즈는 진행중 실시간 자동 민팅, 래플·서포트는 예고 표시 전용).
  • 예외적 조기 종료가 필요하면 운영자 [강제 종료](진행중→종료) 를 사용한다. 상태 전환은 [CEB-BO-EVT-201] 상세 상단에서 수행한다.

4.1.2 레벨 운영 상태

진행중 시즌의 팬덤 레벨은 누적 덕력 달성에 따라 다음 운영 상태를 가진다.

운영 상태의미
운영중최고 레벨 미만. 다음 레벨 목표를 향해 누적 진행 중. 미달성 레벨에 한해 운영자가 신규 레벨 추가·곡선·보상 수정 가능
최고레벨(MAX)설정된 최고 레벨에 도달. 시즌 곡선·보상이 확정되어 수정·신규 레벨 추가 모두 불가
  • 운영자는 팬덤이 최고 레벨에 도달하기 전(운영중)에 의도한 레벨·보상을 모두 설정한다. 최고레벨(MAX) 도달 후에는 해당 시즌의 곡선·보상을 변경할 수 없으며, 추가 확장은 차기 시즌에서 진행한다.
  • 운영중 시즌에서 추가한 신규 레벨의 보상은 추가 이후 그 레벨을 달성하는 시점에 기여 회원(1DUK 이상)에게 적용되며, 이미 달성·지급 완료된 기존 레벨의 보상·지급 내역은 변경되지 않는다(소급 없음).

4.2 레벨 곡선 (카운터 모델)

  • 각 레벨은 0부터 시작하여 "레벨업 목표 덕력" 도달 시 다음 레벨 진입과 동시에 카운터 0으로 리셋된다. 각 레벨 목표값은 해당 레벨 단독 목표값이다.
  • 권장 곡선(운영자 미설정 시 자동 적용): Lv1 10,000 / Lv2 30,000 / Lv3 70,000 / Lv4 150,000 / Lv5 300,000 / Lv6 550,000 / Lv7 1,000,000 / Lv8 1,800,000 / Lv9 3,200,000 / Lv10 6,000,000 (단위 DUK).
  • 운영자는 권장값을 자유롭게 조정할 수 있다.

항목정책

4.4 레벨별 보상 (활성 3종)

운영자는 레벨별로 다음 3종 보상을 설정한다. 실제 지급(자동 민팅)은 디지털 굿즈만 EVT에서 발생하며, 래플·서포트는 앱 예고 표시 전용이다.

보상 종류운영자 설정지급 동작
디지털 굿즈 (BIVE)보상 타이틀(다국어 한·영·일) + "아티스트 키우기 보상" 연결 기능을 가진 활성 BIVE 민팅 캠페인 선택 ([CEB-BO-BIVE-203] 연계)레벨 달성 시 BIVE 영역이 기여 회원(1DUK 이상)에게 자동 민팅. 보상 타이틀은 앱 보상 리스트 표시명, 지급 현황은 BIVE 영역에서 추적
래플 예고안내 텍스트(다국어) 입력 — 예: "Lv.5 달성 시 사인 굿즈 래플 진행 예정"앱에 예고만 표시. 실제 래플은 [CEB-BO-RFL-101] 래플 영역에서 별도 운영
서포트 예고안내 텍스트(다국어) 입력 — 예: "Lv.10 달성 시 서포트 이벤트 예정"앱에 예고만 표시. 실제 서포트는 [CEB-BO-SUP-101] 서포트 영역에서 별도 운영
  • 디지털 굿즈 보상을 사용하려면 운영자가 [CEB-BO-BIVE-203-CREATE]에서 "아티스트 키우기 보상" 연결 기능 캠페인을 먼저 생성·활성화한 뒤, 본 영역 상세에서 선택한다. (BIVE가 자동 민팅 SSOT — [CEB-BO-004])
  • 래플·서포트는 "예고"이므로 EVT에서 추첨·지급·배송을 수행하지 않는다. 실제 운영은 각 영역(RFL·SUP)에서 별도 진행한다.
  • 레벨별 보상 설정은 선택이다. 미설정 레벨은 레벨업만 발생하고 지급 보상은 없다.

보류 (향후 확장): 독점 콘텐츠 해금 · 다운로드 콘텐츠 · 실물 보상(EVT 직접 배송) · 이벤트 참여권은 현 단계 보류한다.

4.5 보상 지급 현황

  • 디지털 굿즈(BIVE): 레벨 달성 시 BIVE 영역이 기여 회원(1DUK 이상)에게 자동 민팅한다. 지급 현황(대상 회원·발행 BIVE·일시)은 BIVE 영역([CEB-BO-BIVE-203] 민팅 이벤트 상세)에서 추적한다.
  • 래플·서포트 예고: 앱 안내 표시 전용이므로 EVT에 지급 현황을 두지 않는다 (실제 지급은 RFL·SUP 영역).
  • EVT 영역은 레벨업 이력(달성 레벨·일시·기여 회원 수)과 BIVE 지급 연계 현황을 [CEB-BO-EVT-401]·[CEB-BO-EVT-201] 탭3에서 조회한다. 회원 덕력 변동 자체는 [CEB-BO-ART-401-HISTORY] 덕력 내역에서 조회한다.
