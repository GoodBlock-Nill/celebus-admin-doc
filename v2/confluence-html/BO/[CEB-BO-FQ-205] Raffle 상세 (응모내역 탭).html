1. 화면 목적

특정 Raffle에 응모한 모든 참여자(Participant) 정보와 응모권 사용 누적 내역을 조회. 추첨은 [CEB-BO-FQ-206]에서 진행되며 본 화면은 순수 모니터링·CS 대응 용도.

진입 경로

  • [CEB-BO-FQ-204] Raffle 상세 (기본정보 탭) > [응모내역] 탭 클릭

2.1 상단 탭·통계

  • 탭 메뉴 (3탭): 기본정보 / 응모내역 (현재) / 추첨내역
  • 통계 대시보드:

4.2 정렬·필터 로직

  • 정렬 조건 + 검색 조건 AND 조합
  • 검색어 미입력 시 정렬만 적용
  • 응모일시는 KST 기준 (서버 시간)

4.3 데이터 집계

  • 유저 단위 Grouping: 한 유저가 N번 나누어 응모해도 리스트에는 1행. 누적 사용 응모권에 합산
  • 실시간성: Active 상태 Raffle은 응모가 계속 발생 → 새로고침/재진입 시 통계 갱신

4.4 응모권 거래 코드 ([CEB-BO-006] §7.6)

응모 시 응모권 차감은 TICKET_RAFFLE_USE 코드로 기록. 본 화면 데이터는 해당 코드 기반 집계.
