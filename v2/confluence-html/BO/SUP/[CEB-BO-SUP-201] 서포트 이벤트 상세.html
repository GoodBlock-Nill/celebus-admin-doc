3. 규칙·로직

3.1 상태별 수정 범위

상태수정 가능
임시저장전 항목 (이벤트명·설명·그룹 제외·목표 응원량·기간·이미지)
모집중이벤트명·설명·대표 이미지만 (목표 응원량·기간·그룹 잠금 — 응원 진행 중 기준 변경 방지)
달성·집행중·완료·미달성종료·집행취소수정 불가 (읽기 전용)

3.2 라이프사이클 전환

[CEB-BO-SUP-000] §4.1 적용. 각 전환은 §2.5 액션 버튼 + 확인 모달. 모집중→달성, 모집중→미달성종료(마감+미달성)는 서버 자동 전이로 운영자 버튼 없이 발생한다.

3.3 반환 (전액·자동)

미달성종료·집행취소·강제 종료 시 응원 회원 전원에게 응원 덕력을 즉시 전액 자동 반환한다([CEB-BO-SUP-000] §4.4). 운영자 수동 반환·추가 인증 절차는 없다.

3.4 권한

권한은 [CEB-BO-SYS-102] 권한 관리 그룹 설정에 따른다. 외부 아티스트 계정은 자기 아티스트 이벤트만 조회 가능하며 상태 전환·수정은 불가하다.

4. UI 상태·에러

[CEB-BO-000] UI 상태 규격. 집행중인데 결과물 미등록 시 탭3에 "결과물 등록 대기" 안내. 게시 시 시작·마감일시·목표 미설정이면 차단 토스트.

5. 활동 로그

[CEB-BO-SUP-000] §5 정합. 생성·게시·집행 시작·결과물 등록·완료·집행 취소·강제 종료 6종 기록.
