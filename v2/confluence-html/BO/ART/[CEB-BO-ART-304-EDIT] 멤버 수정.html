1. 화면 목적

기존 멤버의 프로필 정보를 수정하는 화면. 입력 항목·구성은 [CEB-BO-ART-202-Create] 멤버 생성과 동일하며, 진입 시 기존 값이 채워진다.

진입 경로

  • [CEB-BO-ART-304] 멤버 상세 (기본정보 탭) > [수정하기]
  • URL: /artists/members/{id}/edit

2. UI 설명

2.1 상단 헤더

  • 타이틀: 멤버 수정
  • 우측 상단 액션: [취소하기] / [수정하기]
  • [수정하기]는 변경 사항이 없으면 비활성 상태이다.

4. 규칙·로직

4.1 권한

  • 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.

4.2 입력 검증

  • 멤버명은 한국어·영어·일본어 모두 입력해야 한다.
  • 멤버명·프로필이미지·생년월일·키·성별은 필수이다.
  • 별자리는 생년월일로 자동 계산되며 직접 변경할 수 없다.
  • 변경 사항이 없으면 [수정하기]가 비활성화된다.
