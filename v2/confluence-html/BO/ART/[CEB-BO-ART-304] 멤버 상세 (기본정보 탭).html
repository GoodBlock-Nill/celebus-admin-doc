1. 화면 목적

개별 멤버의 기본 프로필을 조회하고 상태(활성/비활성)를 관리하는 화면. 멤버 상세는 2탭 구조(기본정보 / 소속그룹)이며, 본 문서는 [기본정보] 탭을 다룬다.

진입 경로

  • [CEB-BO-ART-202] 멤버 리스트 > Row 클릭
  • [CEB-BO-ART-303] 그룹 상세 (멤버 탭)에서 멤버 연결
  • URL: /artists/members/{id}?tab=info (기본 탭)

4. 규칙·로직

4.1 권한

  • 권한 그룹별 읽기·쓰기·삭제 범위는 [CEB-BO-SYS-102] 권한 관리 그룹 설정을 따른다.

4.2 별자리 자동 도출

  • 별자리는 생년월일 입력 시 자동 계산되어 표시되며, 운영자가 직접 변경할 수 없다.

4.3 상태 토글 정책
