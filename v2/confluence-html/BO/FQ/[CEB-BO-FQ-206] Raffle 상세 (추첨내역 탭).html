4.2 추첨 진입 조건

  • Raffle 상태 = Closed (모집 마감 상태)
  • Draft / Active / Ended 에서는 [추첨하기] 미노출

4.3 SUPER 추첨 가드레일 ([CEB-BO-006] §4.2)

다음 조건 중 하나라도 만족하면 OPERATOR 단독 추첨 차단 → SUPER 2FA 재인증 필수:

  • 응모권 사용 누적 ≥ 10,000장
  • 보상 BIVE 발행 예정 ≥ 100장

조건 발동 시 [추첨하기] 버튼 옆에 안내 배지: "SUPER 인증 필요" + 추첨 모달 진입 시 SUPER 자격 검증.

4.4 보상 지급 로직

추첨 확정 시 자동 트리거:

  1. Win 회원에게 보상 자동 지급 요청 (현장/배송/온라인 타입별 별도)
  2. 보상 지급 결과 → 리스트 "보상 지급여부" 컬럼 실시간 갱신
  3. 보상 지급 실패 시 운영자 알림 + 미지급 상태로 보존 (재시도 큐)

4.5 데이터 표시 규칙

  • 정렬: 기본은 상태 (Win 우선) → 추첨 일시 내림차순
  • 날짜: YYYY.MM.DD HH:mm KST
  • 전화번호: [국가코드][휴대번호] 표준 포맷 ([CEB-BO-006] §7.5)
