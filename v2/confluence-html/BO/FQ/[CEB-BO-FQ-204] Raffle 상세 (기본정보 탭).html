1. 화면 목적

등록된 Raffle의 전체 설정 정보·생애 주기·실시간 응모 현황을 한눈에 파악하고 라이프사이클 액션 (게시/수정/종료/삭제)을 수행. 응모자 관리·추첨 실행은 형제 탭으로 이동.

주요 기능

  • Raffle 핵심 정보 조회 (기본 / 관리 / 다국어 콘텐츠)
  • 실시간 응모·당첨 대시보드 (4카드)
  • Raffle 라이프사이클 관리 ([CEB-BO-006] §7.2)
  • 형제 탭 진입 (응모내역 / 추첨내역)

진입 경로

  • [CEB-BO-FQ-102] 팬퀘스트 메인 (Raffle 탭) > Raffle 리스트 Row 클릭

4.2 상태 전환 로직 ([CEB-BO-006] §7.2)

  • 게시 (Draft → Active): [게시하기] → [MD-OPEN] 확인 → 시작 일시 자동 기록 + 앱 노출
  • 종료 (Active → Closed): [종료하기] → [MD-CLOSE] 확인 → 응모 마감 + 추첨 대기. 마감일 도래 시 시스템 자동 종료도 동일 결과
  • 추첨 (Closed → Ended): [CEB-BO-FQ-206-MD-DRAW]에서 추첨 실행 시 자동 전환
  • 삭제 (Draft 한정): [삭제하기] → [MD-DELETE] 확인 → 영구 삭제

4.3 데이터 표시 규칙

  • 날짜 포맷: YYYY.MM.DD HH:mm KST
  • 응모자 카운트: 중복 응모 시 1명으로 집계 (총 응모자) / 응모권 사용 누적은 별도 카운트 (총 응모권)
  • 다국어 콘텐츠: 임시저장 일부 빈 칸 가능. 게시 시 3언어 검증 ([CEB-BO-006] §5.2)
