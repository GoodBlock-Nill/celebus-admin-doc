1. 화면 목적

전체 회원의 응모권(티켓) 거래 이력을 관리자가 통합 조회. Fan Quest 보상, Game Zone 보상, BIVE 보유 보상, GP 구매, Raffle 응모/반환, 만료 소멸, 운영자 환불·회수 등 모든 응모권 변동 내역을 단일 화면에서 모니터링.

주요 기능

  • 전체 응모권 변동 내역 통합 조회 (read-only)
  • 거래 코드별 필터링 ([CEB-BO-006] §7.6 6종)
  • 적립경로별 필터링 (Fan Quest / Game Zone / BIVE / GP 구매)
  • 기간·유저(닉네임/유저ID) 검색
  • 어뷰징 분석·CS 대응 활용

진입 경로

  • 사이드 메뉴 > 팬퀘스트 > 응모권 거래 내역

3.2 필터 동작

  • 거래 유형 + 적립경로 + 기간 + 검색어 = AND 조건
