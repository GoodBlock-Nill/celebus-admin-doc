4.2 삭제 가능 조건

  • Draft 상태만 가능 ([CEB-BO-006] §7.2)
  • Active/Closed/Ended 상태는 [삭제하기] 미노출 (응모자/당첨자 보호)
  • 삭제 시점에 상태가 변경된 경우(동시성) → 서버 측 차단

4.3 삭제 프로세스

  • [확인] → DELETE API 호출
  • 데이터 영구 제거 (Hard delete)
  • 활동 로그는 보존 (append-only)
