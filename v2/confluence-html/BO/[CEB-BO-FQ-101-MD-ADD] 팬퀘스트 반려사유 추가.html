4.2 초기값 및 상태 설정

  • 신규 등록되는 모든 반려사유는 기본 Active(사용) 상태로 등록 ([CEB-BO-006] §7.4)
  • 기본 언어: KO 라디오 선택 상태로 진입
  • 코드 형식: 자동 채번 REJ_{n} (4자리)

4.3 다국어 입력 로직

[CEB-BO-000] §5 적용:

  • 라디오 전환 시 각 언어별 입력 데이터 보존 (서버 저장 전 클라이언트 메모리 보존)
  • KO/EN/JP 3언어 모두 필수 (반려사유 메시지는 회원 알림용 — 임시저장 개념 없음)
  • 1개 이상 빈 칸 → 인라인 에러 + [추가하기] 비활성화

4.4 [추가하기] 활성화 조건

  • 운영자 노출명: 입력 완료 (KO, 30자 이내)
  • 유저 노출 메시지: KO + EN + JP 모두 입력 완료 (각 200자 이내)
