1. 화면 개요

배너 1건을 신규 작성·조회·수정·노출 제어하는 단일 화면. 신규(/home/banners/new)와 상세(/home/banners/{id})를 동일 폼으로 통합한다. 슬롯 컨텍스트(위치 + 아티스트)는 [CEB-BO-APP-202] 슬롯 상세에서 [+ 배너 추가]로 진입할 때 자동 prefill되며, 본 화면에서는 read-only로 표기한다.

일반 신규 작성 모드 (복제가 아닌 경우)는 슬롯 상세에서 prefill된 슬롯 컨텍스트가 read-only로 고정. 변경하려면 다른 슬롯 상세에서 새로 [+ 배너 추가] 진입.

종료일 과거 처리: 재개 시 기존 종료일이 과거인 경우 모달에서 운영자에게 안내 + [수정 후 재개] / [그대로 재개] 분기 선택. "그대로 재개" 선택 시 즉시 다시 종료되므로 경고 강조.

>

슬롯 한도 검증: 노출 재개도 신규 노출 시작과 동일한 슬롯 한도 검증 적용. 캐러셀 8개 초과 → 모달 안에서 차단 토스트 + 모달 유지 (운영자가 다른 배너를 먼저 종료할 수 있도록).

>

단일 슬롯 자동 교체: 노출 재개 시 동일 슬롯에 기존 ACTIVE 1건이 있으면 모달 내부에 "기존 배너 '{기존 제목}'이(가) 자동 종료됩니다" 안내 + 확인 시 자동 처리.

>

CLOSED [수정] 허용 (v6.8 신규): 종료 후 콘텐츠 수정 → 재개 흐름 자연스러움. 다국어 타이틀·이미지·딥링크 모두 수정 가능. 단, 슬롯 컨텍스트는 read-only (변경하려면 [복제] 사용).

5-7. CLOSED [수정] 후 [노출 재개] 통합 흐름 (v6.9 신규)

CLOSED 상태에서 콘텐츠 정정 후 재개하는 흐름을 운영자가 끊김 없이 수행할 수 있도록 정책 명시.
