본 문서는 회원 앱 운영 영역(APP)의 전체 범위·정책·서브 영역을 정의한다. 사이드바 "앱" 메뉴 하위의 모든 서브 화면 SSOT.

1. 영역 개요

회원 앱 운영(APP)은 운영자가 회원 앱의 노출 콘텐츠와 알림을 관리하는 통합 영역. 현재 2개 서브 영역으로 구성된다.

항목값
영역 범위배너 관리(슬롯 인벤토리) + 알림 관리(인앱 + 푸시)
사이드바 위치운영 사이트 "앱" 메뉴 하위
서브 영역 수2종 (배너 / 알림) — 추후 확장 가능
영역 ID 체계CEB-BO-APP-{번호} — 배너 1xx·2xx, 알림 3xx

3. 권한

본 영역의 모든 화면·액션 권한은 [관리자 > 권한관리]에서 설정된 권한을 따른다. 권한이 없는 운영자가 진입한 경우 해당 화면·버튼은 미노출 또는 비활성 상태로 처리한다.

9. 딥링크 공통 정책 (배너·알림 공유)

배너(APP-201)·알림(APP-302)이 동일한 딥링크 입력 체계를 사용한다 — 소스 타입 + URL 2개 자유 텍스트박스. 사전 정의된 드롭다운·ID 검색은 사용하지 않으며, 운영자가 분류 라벨과 도달 URL을 직접 입력한다.

필드의미입력

알림 서브 영역 정책

11. 알림 채널 정의

회원 앱은 PWA(Progressive Web App)로 운영되며, 푸시알림은 Web Push API 기반으로 발송된다.

채널정의회원 앱 노출 형태
기본알림인앱 알림센터 (회원 앱 안의 알림 아이콘·리스트)회원 앱 알림 화면, 영구 보관, 회원이 읽음/삭제 처리 가능
푸시알림Web Push (PWA · VAPID 기반)회원 디바이스의 알림 배너·잠금화면. 회원 브라우저 알림 권한(Notification.permission), 푸시 구독(PushSubscription) 등록, iOS의 경우 PWA 홈 화면 추가 여부 영향

한 메시지는 기본 + 푸시 동시 발송을 기본으로 한다. 운영자는 폼에서 푸시 발송 토글로 푸시만 끌 수 있다 (기본알림은 항상 생성).

11-1. PWA 푸시 발송 제약

회원 앱이 PWA이기 때문에 네이티브 앱과 다음 점이 다르다. 운영자는 발송 규모·도달 예측 시 이 제약을 인지해야 한다.

환경푸시 수신 조건
Android (Chrome/Edge 등)브라우저 탭만 있어도 수신 가능. 알림 권한 허용 + 푸시 구독 등록 시
iOS SafariPWA가 홈 화면에 추가되어 있어야만 푸시 수신 가능 (iOS 16.4+). Safari 탭만 사용하는 회원은 푸시 미수신
Desktop (Chrome/Edge/Firefox)알림 권한 허용 + 푸시 구독 등록 시 수신 가능
브라우저 푸시 쿼터일부 브라우저는 자체 발송 빈도 제한(예: Chrome budget). 우리 일일 한도(§17) 미만에서도 잘릴 수 있음

기본알림(인앱 알림센터)은 위 제약과 무관하게 항상 생성된다.

12. 알림 발송 방식

방식동작
운영자 수동운영자가 [CEB-BO-APP-302]에서 메시지를 직접 작성·예약·발송. 즉시 또는 예약 시점 발송. 모든 메시지·대상·발송 시점을 운영자가 자유롭게 정의
시스템 자동 트리거시스템 이벤트(래플 당첨·퀘스트 제출 승인/반려·BIVE 구매 완료·회원 가입 환영 등)에 백엔드가 자동 발송. 메시지·대상·발송 조건은 모두 백엔드 코드/설정에 정의되어 있어 백오피스에서 신규 등록·수정 불가. 변경은 개발팀 요청 채널을 통해서만 가능. 일정 기반 캠페인(BIVE 컬렉션 출시 D-1·응모권 만료 임박 등)은 자동 트리거가 아니라 수동 알림으로 운영자가 직접 작성·예약한다

12-1. 자동 트리거 변경 절차

운영자가 자동 트리거의 메시지·대상·활성 상태를 변경하고 싶을 경우:

  1. 백오피스 [CEB-BO-APP-301] 자동 트리거 탭에서 현재 정책 목록 확인 (조회 전용)
  2. 변경 요청을 개발팀에 전달 (요청 채널은 별도 합의)
  3. 개발팀이 백엔드 코드/설정 변경 후 배포
  4. 배포 후 백오피스에서 갱신된 정책 카탈로그 확인

수동 + 자동은 [CEB-BO-APP-301]에서 별도 탭으로 분리 조회된다 (수동 알림 인벤토리 / 자동 트리거 정책 카탈로그).

13. 알림 대상(타겟팅)

대상정의
전역모든 회원 (글로벌 공지·점검 안내 등)
아티스트 팬덤특정 아티스트 그룹 팬덤(예: V01D 팬). 회원 팬덤 그룹 매핑 기준 ([CEB-BO-USR-201])

14. 알림 상태 정의
