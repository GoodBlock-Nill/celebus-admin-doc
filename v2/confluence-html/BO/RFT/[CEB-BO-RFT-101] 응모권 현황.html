1. 화면 개요

응모권의 발급·보유·사용 총량과 출처별 발급·사용 분포를 한 화면에서 모니터링하는 대시보드. 분석·CS 대응 용도. 순수 조회 전용 — 운영자가 데이터를 직접 수정할 수 없다.

항목값
화면 경로/rft/current

진입 경로사이드바 "래플 > 응모권 현황"
용도응모권 발급·보유·사용 추이 모니터링, 출처별 비중 분석

2. 화면 구성

2-1. 헤더

페이지 제목 우측에 기간 필터 5종 토글 버튼:

옵션의미
오늘오늘 0시 ~ 현재
이번 주이번 주 월요일 0시 ~ 현재
이번 달이번 달 1일 0시 ~ 현재 (기본값)
올해올해 1월 1일 0시 ~ 현재
전체누적 전체 (기간 제한 없음)

기간 변경 시 통계 카드 + 분포 패널의 발급·사용 수치가 해당 기간 기준으로 재집계된다. 보유 카드는 시점 스냅샷이므로 기간 필터 영향을 받지 않는다.

2-2. 통계 카드 (가로 3개)

카드의미
총 발급기간 내 발급된 응모권 누적 수량 (에메랄드 톤)
총 보유현재 시점 회원들이 보유 중인 응모권 잔액 합계. 기간 필터 영향 없음
총 사용기간 내 사용된 응모권 누적 수량 (회색 톤)

2-3. 출처별 분포 패널 (풀폭, grid-cols-1)

가로 풀폭 단일 패널에 발급·사용 분포를 상하로 분리하여 노출. 정규 발급 출처 5종 + 사용 출처 1종 = 정규 6종.

상단: 정규 발급 출처 분포 (에메랄드 톤 배지 "발급 5종")

  • 막대 5개: 퀘스트 보상 / PM·ST 보상 / BIVE 보유혜택 / 덕력 랭킹 보상 / GP로 응모권 구매
  • 각 항목에 "+{N}장 ({비중%})" 표시
  • 막대 클릭 시 [CEB-BO-RFT-201]로 이동하여 해당 출처 필터 자동 적용

하단: 출처별 사용 분포 (rose 톤 배지 "사용 1종")

  • 막대 1개: 래플 응모 사용
  • 항목에 "-{N}장 (100%)" 표시
  • 막대 클릭 시 [CEB-BO-RFT-201]로 이동하여 사용 출처 필터 자동 적용

운영 카테고리(래플 취소 환불 / 운영자 보정 적립 / 만료 소멸 / 운영자 회수)는 본 분포 패널에 노출하지 않는다 — 정규 회원 노출 출처가 아니므로 [CEB-BO-RFT-201] 변동 내역에서만 조회 가능.

⚠️ 일일 미션 카드 제외 — 일일 미션은 응모권을 발급하지 않음 ([CEB-BO-FQ-501] 덕력 25만 적립).

2-4. 최근 변동 (10건 고정 테이블)

컬럼정렬표시 내용
일시좌측변동 발생 시각 (YYYY.MM.DD HH:mm)
회원좌측회원 닉네임 — 클릭 시 target=_blank로 [CEB-BO-USR-201] 회원 상세 새 탭 진입 ([CEB-BO-RFT-201] §6 정합)
변동좌측발급(+N장, 에메랄드) / 사용(-N장, rose)
출처좌측한국어 출처명 (퀘스트 보상 / PM·ST 보상 / BIVE 보유혜택 / 덕력 랭킹 보상 / GP로 응모권 구매 / 래플 응모. 운영 카테고리는 "(운영) {유형}")
아티스트 컨텍스트좌측V01D/iKON/CELEBUS 배지 또는 "전역"
참조좌측관련 엔티티 식별값
  • 10건 고정 — 페이지네이션 없음
  • 우상단 "더 보기 →" 버튼 클릭 시 [CEB-BO-RFT-201] 응모권 변동 내역으로 이동 (필터 없이 전체)

3. 화면에 노출되는 정보

위치정보의미

분포 패널 하단출처별 사용 1종막대 클릭 → RFT-201 필터 이동

4. 상태별 동작 분기

본 화면은 상태 변경 액션이 없는 조회 전용 화면이므로 상태별 분기 대신 데이터 가용성 분기만 존재.

데이터 조건동작
기간 내 변동 0건통계 카드 3개 모두 "0", 분포 패널 막대 모두 0% (회색), 최근 변동 테이블 "응모권 발급·사용 이력이 없습니다." 빈 상태
분포 0% 카운트막대 길이 0%, 텍스트만 회색 처리
권한 없음 진입[관리자 > 권한관리] 설정에 따라 화면 자체 미노출 또는 접근 차단

5. 검증 규칙

규칙내용
데이터 캐시통계·분포 패널 = 5분 캐시 (실시간 대시보드 아님). 최근 변동 = 0분 캐시 (요청 시점 최신)
보유 카드 시점현재 시점 스냅샷. 기간 필터 변경에도 값 고정
막대 클릭 → 필터 이동분포 막대 클릭 시 RFT-201의 출처 드롭다운에 해당 출처 자동 선택. 기간 필터는 RFT-101 선택값과 연계되지 않음 (RFT-201은 자체 기간 정책)
변동 데이터append-only 영구 보존. 만료·환수 없음 ([CEB-BO-010] §5 정합)
