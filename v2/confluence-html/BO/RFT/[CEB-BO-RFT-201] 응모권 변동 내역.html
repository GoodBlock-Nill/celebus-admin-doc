1. 화면 개요

응모권의 발급·사용 시계열 로그 전체를 회원·출처·아티스트 컨텍스트별로 필터링·검색하여 조회하는 화면. CS 대응·어뷰징 분석 용도. 순수 조회 전용 — 변동 데이터는 append-only로 영구 보존되며 운영자가 직접 수정·삭제할 수 없다.

항목값
화면 경로/rft/history

진입 경로사이드바 "래플 > 변동 내역", [CEB-BO-RFT-101] 분포 막대 클릭, "더 보기 →" 버튼
용도회원별 변동 추적, 출처별 발급 검증, 아티스트 컨텍스트 분석

2. 화면 구성

2-1. 필터 바

가로 한 줄에 5개 컨트롤:

위치요소옵션
좌변동 유형 드롭다운전체 (기본) / 발급 / 사용 / 운영
좌출처 드롭다운전체 (기본) / 정규 발급 5종 (퀘스트 보상 · PM·ST 보상 · BIVE 보유혜택 · 덕력 랭킹 보상 · GP로 응모권 구매) / 사용 1종 (래플 응모 사용) / 운영 카테고리 4종 (래플 취소 환불 · 운영자 보정 적립 · 만료 소멸 · 운영자 회수)
좌아티스트 컨텍스트 드롭다운전체 (기본) / V01D / iKON / CELEBUS / 전역
우닉네임 검색키워드 입력 (부분 일치)
우[초기화] 버튼모든 필터·검색 초기화

분포 막대 진입: [CEB-BO-RFT-101]의 분포 막대 클릭으로 진입 시 출처 드롭다운이 해당 출처로 사전 선택된 상태로 표시된다.

⚠️ 일일 미션은 출처 옵션에서 제외 — 일일 미션은 응모권을 발급하지 않으므로 출처 드롭다운에 노출되지 않는다. [CEB-BO-FQ-501] 일일 미션 관리는 덕력 25 적립만 발생.

2-3. 변동 내역 테이블 (8개 컬럼)

컬럼정렬표시 내용
일시좌측변동 발생 시각 (YYYY.MM.DD HH:mm)
회원좌측회원 닉네임 — 클릭 시 [CEB-BO-USR-201] 회원 상세 화면이 새 탭으로 열림. 호버 시 밑줄 표시. 회원이 삭제된 경우 클릭 비활성 + "(삭제됨)" 라벨 표시
상태중앙"발급" (에메랄드 배지) / "사용" (rose 배지) / "운영" (회색 배지)
변동우측"+N장" (에메랄드) / "-N장" (rose)
잔액우측변동 직후 회원 응모권 잔액
출처좌측한국어 출처명 (퀘스트 보상 · PM·ST 보상 · BIVE 보유혜택 · 덕력 랭킹 보상 · GP로 응모권 구매 · 래플 응모 사용 · 운영 카테고리는 "(운영) {유형}")
아티스트 컨텍스트좌측V01D/iKON/CELEBUS 배지 또는 "전역"
참조좌측관련 엔티티 식별값 (Quest 식별값, 래플 식별값, 게임 식별값, 시즌 식별값 등)

페이지네이션:

  • 페이지당 20건
  • 항상 노출 — 결과가 20건 미만이라 1페이지일 때도 ‹ 1 › 비활성 형태로 노출
  • 윈도잉 규칙 — 총 페이지 ≤ 7이면 전체 페이지 번호 노출, 8 이상이면 1 ... cur-1 cur cur+1 ... last 윈도우 + ... 표시
  • 필터·검색·드롭다운 변경 시 자동으로 1페이지로 리셋
