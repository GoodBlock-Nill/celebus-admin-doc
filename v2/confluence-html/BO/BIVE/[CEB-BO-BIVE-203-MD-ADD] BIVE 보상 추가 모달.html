1. 모달 개요

캠페인의 보상으로 지급할 BIVE를 다중 선택하여 추가한다. 에디션·아티스트·아티스트·등급 필터로 BIVE 목록을 좁혀가며, 이미 추가된 BIVE는 중복 방지된다.

항목값
트리거[CEB-BO-BIVE-203] / [CEB-BO-BIVE-203-CREATE]의 BIVE 보상 탭 [+ 추가하기] 버튼
차단 조건캠페인 상태가 활성·종료이면 트리거 버튼 자체 비활성
영향 범위호출 화면의 BIVE 보상 테이블에 선택한 BIVE 행 추가 (가중치는 미입력 상태)
접근 권한[관리자 > 권한관리] 설정에 따른다
