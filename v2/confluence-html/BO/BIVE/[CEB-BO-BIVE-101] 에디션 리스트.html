화면 경로: /bive/editions · Breadcrumb: 홈 › BIVE › 에디션 관리

1. 화면 개요

BIVE NFT의 묶음 단위인 에디션을 생성·조회·수정·삭제하는 운영자용 리스트 화면. 각 에디션별로 등록된 BIVE 관리 페이지([CEB-BO-BIVE-102])로 진입하는 허브 역할.

항목값
화면 경로/bive/editions
Breadcrumb홈 / BIVE / 에디션 관리
페이지 제목에디션 관리
진입 경로사이드바 > BIVE > 에디션 관리
접근 권한[관리자 > 권한관리] 설정에 따른다

2. 화면 구성

2-1. 상단 컨트롤 바

  • 정렬 드롭다운: 정렬 기준(생성일시) + 방향(오름차순/내림차순)
  • 기본: 생성일시 내림차순
  • 검색 입력: 에디션명 부분 일치 (한국어/영어 모두 가능), placeholder "에디션명 입력"
  • [조회] 버튼: 현재 조건으로 테이블 재조회
  • [+ 에디션 생성] 버튼 (우측 고정): 클릭 시 [CEB-BO-BIVE-201-CREATE] 모달 오픈

2-2. 데이터 테이블

컬럼설명
ID내부 일련번호
에디션명 (한국어)한국어 명칭 표시
등록된 BIVE본 에디션에 등록된 BIVE 수
총 발행본 에디션에 등록된 BIVE의 총 민팅 수 (누적)
생성일시YYYY.MM.DD hh:mm
생성 관리자최초 생성한 관리자 이름
관리톱니 아이콘 — 클릭 시 [CEB-BO-BIVE-201-EDIT] 수정/삭제 모달 오픈

행 인터랙션:

  • 행(ID~생성 관리자 컬럼) 클릭 → [CEB-BO-BIVE-102] BIVE 관리 페이지로 이동
  • [관리] 컬럼 톱니 아이콘 클릭 → 수정/삭제 모달 오픈

5. 검증 규칙

검증내용
에디션 수정·삭제 가능 조건등록된 BIVE 0건 + 민팅 기록 0건일 때만 수정·삭제 가능. 한 건이라도 있으면 입력 필드·삭제 버튼 모두 비활성
검색에디션명 부분 일치, 대소문자 무시
정렬상단 드롭다운 기준 (생성일시 + 방향)
권한[관리자 > 권한관리] BIVE 쓰기 권한 보유자에게만 [+ 에디션 생성]·[관리] 버튼 노출
