화면 경로: /bive/minting · Breadcrumb: 홈 › BIVE › 민팅 관리

1. 화면 개요

회원에게 BIVE NFT를 자동 민팅하는 민팅 이벤트을 유형별로 조회·생성·상태 관리하는 SSOT 화면. SQ/RFL/FQ 등 외부 영역은 본 화면에 등록된 "활성" 상태 캠페인을 드롭다운으로 참조한다.

2. 화면 구성

2-1. 상단 탭

4종 캠페인 유형별 탭으로 화면 분리. 본 명세는 공통 구조 정의 (각 탭별 차이는 §4-2에 명시).

탭의미연결 기능 예시
Event (기본)일반 이벤트형 자동 민팅운영 BO 실제 4종 — 회원가입 보상 / 출석체크 보상 / 래플 보상 / 팬퀘스트 보상
Ticket티켓 검증 기반 민팅공연·이벤트 티켓 검증 후 자동 에어드랍 (운영 프리셋 별도)
Mix회원이 BIVE를 합성(Mix)할 때 자동 민팅(개발 프리셋)
Pick회원이 BIVE를 픽(Pick)할 때 자동 민팅(개발 프리셋)

2-2. 필터·검색 영역

  • 정렬 드롭다운: 생성일(빠른순) / 생성일(오래된순) (운영 BO 실제 표기, 기본값 "생성일(빠른순)")
  • 상태 필터 (드롭다운): 상태(전체) / 초안 / 활성 / 중지 / 종료 (운영 BO 실제 표기, 기본값 "상태(전체)")
  • 검색 입력: placeholder "캠페인명 입력" — 키워드 부분 일치
  • [조회] 버튼: 운영 BO는 별도 [조회] 버튼 없이 필터·검색 입력 즉시 자동 갱신
  • [+ 캠페인 생성] 버튼 (우측 상단): 현재 탭의 캠페인 생성 페이지로 이동
  • Event 탭 → [CEB-BO-BIVE-203-CREATE] (Event 변형)
  • Ticket 탭 → [CEB-BO-BIVE-203-CREATE] (Ticket 변형)
