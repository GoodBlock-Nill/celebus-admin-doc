1. 모달 개요

기존 에디션의 다국어 명칭(한국어/영어/일본어)을 수정하거나 에디션 자체를 삭제한다. 등록된 BIVE가 0건이고 민팅 기록이 0건일 때만 수정·삭제 가능.

항목값
트리거[CEB-BO-BIVE-101] 에디션 리스트의 [관리] 컬럼 톱니 아이콘 클릭
차단 조건등록 BIVE > 0 또는 민팅 > 0이면 입력 필드 + [수정하기] + [삭제] 모두 비활성
영향 범위[CEB-BO-BIVE-101] 리스트 갱신 (수정 시 명칭 변경, 삭제 시 행 제거)
접근 권한[관리자 > 권한관리] 설정에 따른다

2. 화면 구성

2-1. 모달 헤더

  • 타이틀: "에디션 수정"
  • 우측 상단: 닫기 [X] 아이콘

2-2. 안내 문구

"에디션에 BIVE가 등록되었거나 민팅되면 에디션 명칭은 수정할 수 없습니다."

2-3. 입력 폼 (다국어 명칭 3종, 기존 값 prefill)

필드입력 방식검증·옵션잠금 조건
한국어Text Input (기존 값 prefill)placeholder "에디션명 입력"등록 BIVE > 0 또는 민팅 > 0이면 read-only
영어Text Inputplaceholder "Edition name"(동일)
일본어Text Inputplaceholder "エディション名"(동일)

2-4. 하단 액션 (좌측 [삭제] + 우측 [수정하기])

  • 좌측 [삭제] (빨강): 등록 BIVE > 0 또는 민팅 > 0이면 비활성
  • 우측 [수정하기] (인디고, 기본 비활성)
  • 입력 필드 1개라도 변경되면 활성화

4. 상태별 동작 분기

상황동작
모달 오픈 (등록 BIVE 0 + 민팅 0)입력 필드 활성, [삭제]·[수정하기] 활성 가능 상태
모달 오픈 (등록 BIVE > 0 또는 민팅 > 0)입력 필드 read-only, [삭제]·[수정하기] 모두 비활성
입력 필드 변경 (모두 유효)[수정하기] 활성화
입력 필드 변경 없음[수정하기] 비활성 유지
[수정하기] 클릭 (정상)토스트 "에디션이 수정되었습니다." → 모달 닫힘 → 리스트 갱신
[수정하기] 클릭 (네트워크 오류)토스트 "수정에 실패했습니다. 다시 시도해주세요." 모달 유지
[삭제] 클릭즉시 삭제 처리 → 토스트 "에디션이 삭제되었습니다." → 모달 닫힘 → 리스트 행 제거
닫기 [X] / ESC / 오버레이 클릭모달 닫힘 (변경 사항 폐기)

5. 검증 규칙

검증내용
수정·삭제 가능 조건등록된 BIVE 0건 + 민팅 기록 0건일 때만 가능. 한 건이라도 있으면 모두 비활성
다국어 3종 모두 유효한국어·영어·일본어 모두 공백 제외 1자 이상이어야 [수정하기] 활성
변경 사항 존재입력값 1개라도 변경되어야 [수정하기] 활성
권한[관리자 > 권한관리] BIVE 쓰기 권한 보유자만 트리거 아이콘 노출
