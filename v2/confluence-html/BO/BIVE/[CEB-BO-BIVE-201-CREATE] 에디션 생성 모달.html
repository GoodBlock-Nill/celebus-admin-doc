1. 모달 개요

신규 에디션(BIVE 묶음 단위)을 생성한다. 다국어 명칭 3종(한국어/영어/일본어)만 입력하면 생성되며, 생성 직후 [CEB-BO-BIVE-101] 리스트에 추가된다. 에디션 자체는 명칭 외 다른 메타 데이터가 없으므로 별도 상세 페이지 없이 본 모달과 [CEB-BO-BIVE-201-EDIT] 수정 모달로 완결된다.

항목값
트리거[CEB-BO-BIVE-101] 에디션 리스트 우측 상단 [+ 에디션 생성] 버튼
차단 조건[관리자 > 권한관리] BIVE 쓰기 권한 없을 때 트리거 버튼 자체 미노출

접근 권한[관리자 > 권한관리] 설정에 따른다
