1. 화면 목적

개별 관리자 정보 조회·권한 부여·만료 연장·세션 회수. 2FA 정책 명시 (구현은 운영팀 결정).

주요 기능

  • 관리자 기본 정보 조회 (닉네임·이메일·등록일·최근 로그인)
  • 권한 그룹 부여·변경 (SUPER 전용)
  • 만료일 연장 (EXTERNAL_ARTIST 6개월 만료 정책)
  • IP 화이트리스트 관리 (SUPER + EXTERNAL_ARTIST)
  • 세션 강제 종료·계정 잠금·비활성화

진입 경로

  • [CEB-BO-SYS-101] 관리자 리스트 > Row 클릭
  • (자기 정보 수정) 관리자 본인 프로필 메뉴

URL: /admins/list/{id}

3. 화면 내 기능설명

3.1 권한 그룹 변경

권한 그룹 변경 모달 (Phase 8 후속):

  • 현재 그룹 → 신규 그룹 선택 (Active 그룹만 노출)
  • 변경 사유 입력 (필수, 200자 KO)
  • 즉시 반영 + 활동 로그 + 회원 푸시 알림 (자기 권한 변경 안내)

3.2 만료 연장 (EXTERNAL_ARTIST)

[CEB-BO-000] §3.2: EXTERNAL_ARTIST는 만료일 필수. 기본 6개월, [만료 연장] 클릭 시 모달 → 신규 만료일 선택 + 사유 입력.

3.3 세션 강제 종료

  • [세션 강제 종료] 클릭 → 확인 모달
  • 즉시 세션 무효화 (해당 관리자는 다음 요청 시 로그아웃)
  • 활동 로그 기록 (사유 입력 필수)

3.4 계정 잠금·비활성화

액션동작
[계정 잠금]Active → Locked. 재로그인 차단. SUPER 재승인 후 Active 복구 가능
[비활성화]Active → Inactive. 영구 차단. 활동 로그·권한 매핑은 보존 (감사용)

4.1 권한별 분기

SUPER✅ (전체)✅✅✅✅✅

자기 정보 수정 범위: 닉네임·휴대전화·2FA 설정·비밀번호 변경. 권한·만료·IP 화이트리스트는 SUPER만.

4.2 2FA 정책 정의 (구현은 후속)

권한2FA 정책
SUPER필수 + IP 화이트리스트 + 90일 비밀번호 변경 강제
OPERATOR양의 권장 (운영팀 결정 시 강제 가능)
POLICY권장
EXTERNAL_ARTIST필수 + 시간대 제한 + 6개월 만료

구현 시점은 운영팀 결정. 본 v1.0 명세는 정책만 정의 (사용자 결정 2026-05-06).

4.3 90일 비활성 자동 잠금 ([CEB-BO-000] §3.2)

  • 90일 미접속 → Active → Locked 자동 전환
  • 활동 로그 기록 (시스템 자동)
  • SUPER 재승인 시 Active 복구

4.4 EXTERNAL_ARTIST 만료 정책

  • 등록 시 만료일 필수 (기본 6개월)
  • 만료 7일 전 → SUPER + 본인에게 푸시 알림
  • 만료 시 자동 Inactive 전환
  • [만료 연장]으로 갱신 가능
