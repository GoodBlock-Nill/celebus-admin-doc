1. 화면 목적

DUK 시즌 통합 조회·관리. POLICY가 시즌을 생성·수정하고 시즌별 캡·기간을 모니터링. 개별 시즌의 티어 보상·정책은 [CEB-BO-DUK-201] 진입.

주요 기능

  • 등록된 전체 시즌 리스트 조회 (Draft/Ready/Active/Settling/Ended)
  • 시즌별 캡·기간·참여 회원 수 모니터링
  • 신규 시즌 생성 진입
  • 시즌 상세 진입

진입 경로

  • 사이드 메뉴 > 덕력 > 시즌 관리

4. 규칙·로직

4.1 권한별 분기

4.2 정렬

기본 정렬Active 우선 → 시작일 내림차순 (최신 시즌 상단)

4.3 시즌 동시 운영 제한

  • Active 시즌은 동시 1개만 (기본 정책)
  • 신규 시즌 [게시] 시 기존 Active 시즌이 있으면 차단
  • 정산 완료(Ended) 후 신규 시즌 시작 가능
  • 운영팀 결정 시 다중 Active 허용 가능 (DUK-201에서 별도 시즌으로 분리, 회원 잔액 통합 정책 별도 정의)

4.4 캡 표시

캡은 시즌별로 다를 수 있음. Active 중 변경 가능 (DUK-201에서 사유 입력 필수).
