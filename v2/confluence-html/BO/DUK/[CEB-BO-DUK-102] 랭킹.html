1. 화면 목적

DUK 시즌별 회원 랭킹 통합 조회. 시즌 종료 시 랭킹 보상 자동 지급 결과 확인 + 수동 재지급. 회원 앱([CEB-DUK-101])에서 보는 랭킹과 동일 데이터, BO는 운영자 시각.

주요 기능

  • 시즌별 회원 랭킹 조회 (TOP 100 + 검색)
  • 시즌 종료 후 랭킹 보상 지급 결과 모니터링
  • 수동 재지급 (지급 실패 회원 대상)

진입 경로

  • 사이드 메뉴 > 덕력 > 랭킹

3.2 정렬

기본잔액 내림차순 → 동률 시 시즌 마지막 적립 시점 빠른 순

3.3 보상 지급 로직 ([CEB-BO-DUK-000] §6.2)

시즌 종료(Settling) → 자동 트리거:

  1. 잔액 기준 순위 산출
  2. POLICY 설정 TOP N 선발
  3. 등수별 보상 지급 (응모권·BIVE)
  4. 활동 로그 기록 + 회원 푸시

지급 실패 (가스비·지갑·블록체인 오류) → 본 화면 [수동 재지급] 버튼.

3.4 잔액 산출 정책

  • 기본: 시즌 내 순 적립 잔액 (적립 누적)
  • POLICY 결정 시: 순잔액 (적립 − 소비) 기준 가능

DUK-201 [정산 정책] 섹션에서 시즌별 설정.
