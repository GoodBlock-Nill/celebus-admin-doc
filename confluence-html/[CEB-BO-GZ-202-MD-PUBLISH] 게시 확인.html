1. 화면 목적

게임 상태가 Ready(게시대기)일 때 게임을 게시(Active 상태로 전환)하기 전 최종 확인을 받는 모달입니다. 게시하면 즉시 게임이 시작되어 참여가 가능해집니다. 게시 후에는 일부 항목만 수정 가능함을 안내합니다.

3. 화면 내 기능설명

모달 트리거

트리거 조건 동작
게임 상세(CEB-BO-GZ-202)에서 상태가 Ready "게시하기" 버튼 활성화
"게시하기" 버튼 클릭 본 모달 오픈

버튼 동작

취소 버튼

  1. 모달 닫기
  2. 게임 상세 화면으로 복귀
  3. 상태 변경 없음

게시하기 버튼

  1. 게임 게시 요청 실행
  2. 성공 시:
    • 게임 상태: Ready → Active
    • 투표 시작일시 = 현재 시각으로 자동 설정
    • 토스트 메시지: "게임이 게시되었습니다."
    • 모달 닫기
    • 게임 상세 화면 갱신 (상태 및 액션 버튼 변경)
    • 실패 시:
      • 에러 토스트 표시
      • 모달 유지 (재시도 가능)

닫기(X) 버튼

  • 취소 버튼과 동일한 동작

4. 규칙/로직

게시 가능 조건

조건 검증 시점 실패 시 처리
게임 상태 = Ready 모달 오픈 전 버튼 비활성화
필수 정보 입력 완료 모달 오픈 전 상세 화면에서 검증

게시 후 동작

  1. 게임 상태가 Active로 변경됨
  2. 투표 시작일시가 현재 시각으로 자동 설정됨
  3. 게시 후에도 일부 항목 수정 가능 (상세는 CEB-BO-GZ-202-EDIT 수정 화면 참조)

5. 에러 및 예외상황

클라이언트 에러

에러 상황 에러 메시지 처리 방법
게임 상태가 Ready가 아님 "게시할 수 없는 상태의 게임입니다." 모달 닫기, 상세 화면 갱신
필수 정보 누락 "게임 정보가 완전하지 않습니다." 모달 닫기, 누락 필드 안내

서버 에러

HTTP 상태 에러 메시지 처리 방법
403 "게시 권한이 없습니다." 에러 토스트, 모달 닫기
404 "게임을 찾을 수 없습니다." 에러 토스트, 리스트로 이동
409 "이미 게시된 게임입니다." 에러 토스트, 상세 화면 갱신
500 "서버 오류가 발생했습니다." 에러 토스트, 모달 유지

네트워크 에러

에러 상황 처리 방법
타임아웃 "요청 시간이 초과되었습니다. 다시 시도해주세요."
연결 실패 "네트워크 연결을 확인해주세요."
