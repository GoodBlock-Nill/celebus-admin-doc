1. 화면 목적

유저가 현재 보유한 사용 가능한 응모권(Ticket)의 총 수량을 명확하게 확인합니다. 응모권의 획득(In)과 사용(Out) 내역을 투명하게 제공하여 재화 관리에 대한 신뢰를 줍니다. 특히, 퀘스트 심사 현황(승인/반려/대기)과 래플 당첨 결과를 한곳에서 조회하고 관리하는 '마이페이지' 역할을 수행하는 화면입니다.

주요 기능
  • QUEST 탭: 퀘스트 참여 이력 및 심사 현황 조회 (지급완료/검토중/반려)
  • RAFFLE 탭: 래플 응모 이력 및 당첨 결과 조회 (진행중/집계중/당첨/미당첨)
  • 반려 항목 탭 시 반려 사유 모달(CEB-FQ-201-MD) 호출
접근 경로
  • 홈 메인(CEB-HOM-501-FQ) > 환영 배너 영역에서 진입

3. 화면 내 기능설명

3.1 QUEST 탭 상호작용

상태 탭 동작
반려 탭 시 [CEB-FQ-201-MD] 반려 사유 모달 호출
지급완료 탭하여도 별도 동작 없음
검토중 "현재 검토 진행 중입니다." 토스트 노출

3.2 RAFFLE 탭 상호작용

상태 탭 동작
진행중 탭 시 [CEB-FQ-202] Raffle 상세(진행중) 화면으로 이동
집계중 "현재 집계중입니다" 토스트 노출
당첨 탭 시 [CEB-FQ-203] Raffle 상세(당첨) 화면으로 이동
미당첨 탭 시 [CEB-FQ-204] Raffle 상세(미당첨) 화면으로 이동

3.3 반려 사유 코드 정의

코드명 (Code) 운영자 노출명 (Admin Label) 유저 노출 메시지 (Client Message) 설명 및 가이드
IMG_BLUR 식별 불가 (흐림/어두움) 이미지가 흐릿하거나 너무 어두워서 내용을 확인할 수 없습니다. 선명한 사진으로 다시 올려주세요! 초점이 나갔거나 조명이 부족해 검증이 어려운 경우
WRONG_OBJ 대상 불일치 (다른 사물) 퀘스트와 관련 없는 사진입니다. 미션 가이드를 다시 확인하고 올바른 대상을 촬영해 주세요. 영수증 미션에 풍경 사진을 올리는 등 엉뚱한 이미지인 경우
DUP_ENTRY 중복 참여 의심 이미 제출되었거나 다른 유저가 사용한 이미지와 동일한 것으로 확인됩니다. 직접 촬영한 새로운 사진으로 참여해 주세요. 어뷰징(도용, 재사용) 방지용
DATE_EXP 날짜/조건 미달 인증 기준(날짜, 시간, 필수 정보 등)이 충족되지 않았습니다. 유효 기간 내의 사진인지 확인해 주세요. 영수증 날짜가 지났거나 스트리밍 시간이 부족한 경우
ETC_INPUT 기타 (직접 입력) {운영자가 입력한 사유 텍스트 노출} 위 4가지 프리셋에 해당하지 않는 특수 상황용

4. 규칙/로직

4.1 정렬 기준

  • 날짜별(최신순) 정렬
  • 동일 날짜 내에서는 최신 처리 시각 기준 내림차순

4.2 페이징(Paging) 처리

  • 스크롤 바닥 도달 시 추가 로딩 (Infinite Scroll)
  • 1회 로딩 단위: 20개

4.3 연관 링크 참조

  • Quest 카드에서 연관 링크 정보는 직접 표시하지 않음. Quest 제출 화면(CEB-FQ-205)에서 라벨 버튼으로 표시됨.

5. 에러 및 예외상황

5.1 내역 없음 (Empty Case)

탭 안내 문구 액션
QUEST "참여한 퀘스트 내역이 없습니다." 하단에 '퀘스트 하러 가기' 버튼 (메인 화면으로 유도)
RAFFLE "응모한 래플 내역이 없습니다." 하단에 '래플 보러 가기' 버튼 (메인 화면으로 유도)

각 탭 또는 필터 조건에 맞는 데이터가 0건일 경우, 리스트 영역 중앙에 아이콘 + 안내 문구 노출
