Breadcrumb 홈 > 응모권

1. 화면 목적

유저가 보유한 응모권(티켓) 수량을 확인하고, 적립/사용 거래 이력을 조회하는 화면입니다.

주요 기능
  • 보유 응모권 수량 실시간 확인
  • 응모권 구매 기능 (GP 사용)
  • 적립 이력 조회 (Fan Quest 보상, Survival Trivia 탈락 보상, Prediction Market 보상, 응모권 구매)
  • 사용 이력 조회 (Raffle 응모)
접근 경로
  • 홈 메인(CEB-HOM-501-FQ) > 환영 배너 영역에서 진입

3. 화면 내 기능설명

3.1 [<] 뒤로가기

  • 이전 화면(홈 메인)으로 복귀

3.2 구매하기 버튼

  • 보유 응모권 카드 내 [구매하기] 버튼 탭 시 응모권 구매 화면(Buy_RF_Ticket)으로 이동
  • GP를 사용하여 응모권 구매 (교환 비율은 추후 결정)

3.3 탭 전환

  • 적립 / 사용 탭 전환 시 해당 조건에 맞는 이력만 필터링하여 표시
  • 탭 전환 시 리스트 최상단으로 스크롤 초기화

3.4 리스트 스크롤

  • 무한 스크롤 (20건씩 추가 로드)
  • 스크롤 바닥 도달 시 추가 데이터 로딩

4. 규칙/로직

4.1 탭별 필터

탭 표시 내용
적립 + 변동만 (Quest 보상, ST 탈락 보상, PM 보상, 응모권 구매)
사용 - 변동만 (Raffle 응모)

4.2 적립 탭 이력 항목 유형

유형 서브 타이틀 메인 타이틀 날짜/시간 금액
Quest [{아티스트}] QUEST {Quest title} YYYY.MM.DD hh:mm:ss +{N} (보라색)
Survival Trivia Survival Trivia {Game title} YYYY.MM.DD hh:mm:ss +{N} (보라색)
Prediction Market Prediction Market {Game title} YYYY.MM.DD hh:mm:ss +{N} (보라색)
응모권 구매 {N} Game point 사용 응모권 구매 YYYY.MM.DD hh:mm:ss +{N} (보라색)

- Quest 유형의 서브 타이틀에는 아티스트명 포함
- 응모권 구매 유형: GP를 사용하여 응모권을 구매한 경우. 서브 타이틀에 사용된 GP 수량 표시

4.3 사용 탭 이력 항목 유형

유형 서브 타이틀 메인 타이틀 날짜/시간 금액
Raffle 응모 [{아티스트}] RAFFLE {Raffle title} YYYY.MM.DD hh:mm:ss -{N} (보라색)

- 현재 Raffle 응모 시 차감되는 케이스만 존재

4.4 정렬 기준

  • 최신 일시순 (desc)

4.5 검색

  • 없음

4.6 페이지네이션

  • 무한 스크롤 (20건씩 추가 로드)
