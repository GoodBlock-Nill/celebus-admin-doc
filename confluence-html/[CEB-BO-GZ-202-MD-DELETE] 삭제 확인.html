1. 화면 목적

임시저장(Draft) 또는 게시대기(Ready) 상태의 게임을 완전히 삭제하는 확인 모달입니다. 삭제된 게임은 복구할 수 없으므로 신중한 확인이 필요합니다. Draft 또는 Ready 상태에서만 삭제가 가능하며, 진행 중이거나 종료된 게임은 삭제할 수 없습니다.

3. 화면 내 기능설명

모달 트리거

트리거 조건 동작
게임 상세(CEB-BO-GZ-202)에서 상태가 Draft 또는 Ready "삭제하기" 버튼 활성화
"삭제하기" 버튼 클릭 본 모달 오픈

버튼 동작

취소 버튼

  1. 모달 닫기
  2. 게임 상세 화면으로 복귀
  3. 상태 변경 없음

삭제하기 버튼

  1. 게임 삭제 요청 실행
  2. 성공 시:
    • 게임 데이터 완전 삭제
    • 토스트 메시지: "게임이 삭제되었습니다."
    • 모달 닫기
    • 게임 리스트 화면(CEB-BO-GZ-201)으로 이동
    • 실패 시:
      • 에러 토스트 표시
      • 모달 유지 (재시도 가능)

닫기(X) 버튼

  • 취소 버튼과 동일한 동작

4. 규칙/로직

삭제 가능 조건

조건 검증 시점 실패 시 처리
게임 상태 = Draft 또는 Ready 모달 오픈 전 버튼 비활성화
관리자 권한 확인 API 호출 시 403 에러
참여자가 없음 API 호출 시 400 에러 (Draft/Ready 상태에서는 참여자 0명)

삭제 불가 상태

상태 사유
Active 진행 중, 강제 종료 후에도 삭제 불가 (보관)
Pending 결과 입력 대기 중, 삭제 불가 (보관)
Closed 보상 지급 대기 중, 삭제 불가 (보관)
Ended 종료됨, 삭제 불가 (이력 보관)

삭제 시 처리 로직

데이터 삭제

  1. 게임 기본 정보 삭제
  2. 관련 이미지/파일 삭제 (S3 등)
  3. 연관 데이터 정리 (Draft/Ready 상태이므로 참여자 없음)

삭제 후 이동

  • 게임 리스트 화면(CEB-BO-GZ-201)으로 이동
  • 필터/정렬 상태 유지
  • 삭제된 게임이 있던 페이지로 이동 (가능한 경우)

관리자 로그

5. 에러 및 예외상황

클라이언트 에러

에러 상황 에러 메시지 처리 방법
게임 상태가 Draft 또는 Ready가 아님 "임시저장 또는 게시대기 상태의 게임만 삭제할 수 있습니다." 모달 닫기, 상세 화면 갱신
이미 삭제된 게임 "이미 삭제된 게임입니다." 모달 닫기, 리스트로 이동

서버 에러

HTTP 상태 에러 메시지 처리 방법
400 "삭제할 수 없는 게임 상태입니다." 에러 토스트, 모달 닫기, 화면 갱신
403 "게임 삭제 권한이 없습니다." 에러 토스트, 모달 닫기
404 "게임을 찾을 수 없습니다." 에러 토스트, 리스트로 이동
409 "이미 삭제 처리 중인 게임입니다." 에러 토스트, 리스트로 이동
500 "서버 오류가 발생했습니다." 에러 토스트, 모달 유지

파일 삭제 실패

에러 상황 처리 방법
이미지 파일 삭제 실패 게임 데이터는 삭제, 파일 삭제 재시도 큐에 추가
S3 접근 불가 게임 데이터는 삭제, 백그라운드에서 파일 삭제 재시도

네트워크 에러

에러 상황 처리 방법
타임아웃 "요청 시간이 초과되었습니다. 다시 시도해주세요."
연결 실패 "네트워크 연결을 확인해주세요."
