1. 화면 목적

신규 Quest를 생성하기 위한 입력 폼 화면입니다.

주요 기능

  • Quest 기본 정보 입력 (타이틀, 설명, 유저 가이드, 대표이미지) - 다국어 지원 (한국어/영어/일본어)
  • Quest 설정 정보 입력 (아티스트, 타입, 마감일시)
  • Quest 보상 정보 입력 (보상 타입, 응모권 수량, NFT 이벤트)
  • 생성된 Quest는 Draft 상태로 저장되며, 이후 상세 화면에서 게시할 수 있습니다.

진입 경로

  • [CEB-BO-FQ-101] 팬퀘스트 메인 (Quest 탭) > [생성하기] 버튼 클릭

2. UI 설명

2.1 화면 레이아웃 (Split Layout)

  • 좌측 영역 (Navigation & Status): 입력해야 할 주요 항목(타이틀, 설명, 유저 가이드 등)이 리스트 형태로 나열되며, 우측 끝에 상태 버튼이 위치합니다.
  • 우측 영역 (Input Form): 좌측에서 선택한 항목에 대한 상세 입력 폼이 노출되는 영역입니다. (예: 타이틀, 설명, 유저가이드)
  • 각 입력 항목(타이틀, 설명, 유저 가이드) 우측에 위치하며, 입력 단계에 따라 상태가 변화합니다.

3. 화면 내 기능설명

3.1 버튼 동작

  • [취소하기]: 클릭 시, 작성 취소 확인 모달인 [CEB-BO-FQ-101-MD-CANCEL] Quest 생성취소 화면을 호출합니다.
  • [생성하기]: 모든 필수 입력 항목이 입력완료 상태가 되고, 하단 규칙 설정이 완료되었을 때만 활성화됩니다. 클릭 시, 생성 완료 안내 모달인 [CEB-BO-FQ-101-MD-COMPLE] Quest 생성완료 화면을 호출합니다.

3.2 네비게이션

  • 생성 취소 시: [CEB-BO-FQ-101] 팬퀘스트 메인으로 이동
  • 생성 완료 시: [CEB-BO-FQ-201] 해당 Quest 상세 화면으로 이동

4. 규칙/로직

4.1 다국어 입력 버튼 상태

상태 버튼 텍스트 조건
미입력 입력하기 해당 항목 입력을 시작하지 않은 상태
입력중 입력중... 입력 패널이 열려있고 입력 진행 중인 상태
입력완료 입력완료 3개 언어 모두 입력 및 저장 완료된 상태

4.2 다국어 입력 패널 진행률 표시

표시 의미
(0/3) 아무 언어도 입력되지 않음
(1/3) 1개 언어 입력 완료
(2/3) 2개 언어 입력 완료
(3/3) 3개 언어 모두 입력 완료

4.3 다국어 입력 저장 시 유효성 검사

상태 메시지
미입력 언어 존재 "*모든 언어를 빠짐 없이 입력해주세요"
모든 언어 입력 완료 "*모든 언어가 성공적으로 저장되었습니다"

4.4 기간 설정 로직

  • 시작 일시: 관리자가 [생성하기] 버튼을 눌러 생성이 완료한 이후 [게시하기] 버튼을 눌러 게시한 서버 시간(Timestamp)을 시작 일시로 자동 설정합니다.
  • 종료 일시: 반드시 현재 시간(생성 시점)보다 미래의 시간으로만 설정 가능합니다.
    • 마감일 설정 필드를 누르면 캘린더 표시
    • 현재일자 +1일 부터 설정 가능
    • 시간 설정 필드를 누르면 정수만 입력 가능
    • 시: 00~23
    • 분: 00~59

4.5 보상 설정 로직

  • 응모권 수량 선택: 직접 입력이 아닌 Select Box를 통해 1~10장 중 하나를 선택합니다.
  • 동적 UI: 선택한 보상 종류(응모권+NFT, 응모권, NFT)에 따라 필요한 Select Box(수량 또는 이벤트 선택)만 노출합니다.

4.6 데이터 연동

  • NFT 민팅 이벤트 리스트: NFT 관련 옵션 선택 시, 연동된 BIVE 시스템의 '진행 가능한 NFT 민팅 이벤트' 목록을 API로 조회하여 바인딩합니다.

4.7 생성하기 버튼 활성화 조건

다음 조건을 모두 충족해야 [생성하기] 버튼이 활성화됩니다:

  • 타이틀: 3개 언어 모두 입력완료
  • 설명: 3개 언어 모두 입력완료
  • 유저 가이드: 3개 언어 모두 입력완료
  • 아티스트: 선택 완료
  • 타입: 선택 완료
  • 마감일/시간: 설정 완료
  • 보상 설정: 선택 완료
  • NFT 민팅 이벤트: 선택 완료
