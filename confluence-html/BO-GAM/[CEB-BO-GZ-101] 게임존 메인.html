4.3 게임 유형별 카드 노출 조건

게임 관리 섹션의 카드(Prediction Market, Survival Trivia 등)는 해당 게임 유형의 서비스 활성화 상태에 따라 노출 여부가 결정된다.

참고: 여기서 말하는 Active/Inactive는 개별 게임의 상태(진행중/종료 등)가 아니라, 게임 유형 자체가 서비스에 오픈되었는지 여부를 의미한다. 새로운 게임 유형이 추가될 때 서버에서 해당 유형의 서비스 상태를 Active로 설정하면 카드가 노출된다.
서비스 상태 카드 노출 바로가기 상세 항목 설명
Active (서비스 오픈) 노출 활성화 표시 해당 게임 유형의 카드가 대시보드에 표시되고, 바로가기 클릭 시 게임 리스트로 이동 가능
Inactive (서비스 미오픈) 미노출 - - 카드 자체가 화면에 표시되지 않음 (빈 공간 없이 나머지 카드가 정렬됨)

예시:

  • Phase 1에서는 Prediction Market만 Active → PM 카드 1개만 노출
  • Phase 2에서 Survival Trivia가 Active로 전환 → PM, ST 카드 2개 가로 배치
  • 향후 새로운 게임 유형 추가 시 → Active 설정된 유형만 카드 그리드에 자동 추가
