1. 화면 목적

모든 스테이지 종료 후 표시되는 최종 결과 화면입니다. 결과 화면에 도달하는 모든 사용자에게 하트 1개가 지급됩니다. 생존자에게는 상금 GP + 하트가, 모든 탈락자에게는 하트 + 팬퀘스트 응모권이 서버에서 자동 지급됩니다. 마지막 스테이지(10/10)에서 탈락한 사용자도 탈락 모달 없이 이 화면으로 직행합니다.

이 문서는 게임 종료 후 결과 화면을 다룹니다. 게임 진행 화면(CEB-GAM-206), 탈락 모달(CEB-GAM-206-MD-ELIMINATE), 관전 모드(CEB-GAM-206-SPECTATE)는 별도 문서입니다.

3. 화면 내 기능설명

3.1 보상 받기 버튼 (완주자)

트리거
  • 하단 [보상 받기] 버튼 탭
동작
  1. CEB-GAM-204(트리비아 홈)으로 이동
  2. 상금 GP + 하트 1개는 게임 종료 시점에 서버에서 자동 지급 완료 상태
  3. 별도 수령 동작 불필요

3.2 하트 받기 버튼 (탈락 관전자)

트리거
  • 하단 [❤️ 하트 받기] 버튼 탭
동작
  1. CEB-GAM-204(트리비아 홈)으로 이동
  2. 하트 1개 + 응모권은 서버에서 관전 완료 시점에 이미 자동 지급 완료 상태

3.3 뒤로가기

트리거
  • Header 좌측 [<] 버튼 탭
동작
  • CEB-GAM-204(트리비아 홈)으로 이동
  • 보상은 게임 종료 시점에 서버에서 자동 지급되므로 별도 수령 필요 없음

3.4 하트 받기 버튼 (마지막 문제 탈락자)

트리거
  • 하단 [❤️ 하트 받기] 버튼 탭
동작
  1. CEB-GAM-204(트리비아 홈)으로 이동
  2. 하트 1개 + 응모권은 서버에서 탈락 시점에 이미 자동 지급 완료 상태 (GP는 미지급)

4. 규칙/로직

보상 계산

항목 계산
적용 상금풀 게임 시작 시 모집률 기반으로 확정된 상금풀 (최대 상금풀 × 해당 단계의 상금 비율)
1인당 보상 적용 상금풀 ÷ 최종 생존자 수 (균등 분배)
소수점 처리 소수점 이하 절삭 (예: 30,000 ÷ 102 = 294 GP)
잔여 GP 소수점 절삭으로 발생한 잔여 GP는 플랫폼 보유
전원 탈락 생존자 0명인 경우 적용 상금풀 전액 플랫폼 보유
참여비 환급 없음 (참여비는 환급되지 않음)
탈락자 응모권 모든 탈락자에게 팬퀘스트 응모권 {N}장 자동 지급 (관전 여부 무관)
보상 지급 시점 게임 종료 시점에 서버에서 자동 지급 완료 (GP + 하트 + 응모권)
RESULT 화면 역할 이미 지급된 보상 결과를 표시만 함 (별도 지급 API 호출 없음)
지급 실패 시 서버에서 자동 재시도. RESULT 화면에서는 "보상 정산 중..." 표시 후 새로고침 안내
정산 중 UI 보상 표시 영역에 "+{N} GP" 대신 "보상 정산 중..." 텍스트 + 소형 로딩 스피너 표시. [확인] 버튼은 활성화 상태 유지 (탭 시 CEB-GAM-204로 이동 가능). 보상은 서버 백그라운드에서 재시도하므로 화면을 떠나도 문제 없음
뒤로가기 동작 [<] 뒤로가기와 [확인] 버튼 모두 서버에 '결과 확인 완료' API 호출 후 CEB-GAM-204로 이동. 이 API는 로깅 + 미확인 결과 상태 해소 목적 (보상 지급 API 아님). 호출 실패 시에도 CEB-GAM-204로 이동하며, 다음 204 진입 시 미확인 결과로 다시 표시될 수 있음
네트워크 실패 시 리트라이 + "일시적 오류가 발생했습니다" 에러 토스트. 재시도 실패 시에도 CEB-GAM-204로 이동 (다음 204 진입 시 다시 조회)
소멸 후 재조회 결과 확인 완료된 게임 결과는 RESULT 화면에서 재조회 불가. GP 변동 금액은 GP 변동 내역(CEB-BO-GZ-601)에서 확인 가능

나가기 선택 탈락자

  • 탈락 후 나가기를 선택한 사용자는 결과 화면 미표시 (v1.6 변경사항 유지)
  • 응모권은 서버에서 탈락 시점에 자동 지급 (팬퀘스트에서 확인 가능)

업적 배지

배지 표시 조건 설명
🏅 100% 정답 모든 문제 정답 (오답 0회) 오답 없이 전 스테이지 통과
🏅 완주 마지막 스테이지까지 생존 생존자 전원에게 부여
🏅 무하트 클리어 하트를 1개도 사용하지 않고 완주 100% 정답 달성 시 항상 동시 충족
  • 100% 정답이면 반드시 무하트 클리어 조건도 충족 (오답이 없으므로 하트가 소진되지 않음)
  • "100% 정답"과 "무하트 클리어" 배지는 중복 획득 가능 (둘 다 동시에 표시)
  • 탈락자에게는 업적 배지 영역 미표시
