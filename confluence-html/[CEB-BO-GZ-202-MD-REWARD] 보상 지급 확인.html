1. 화면 목적

게임 결과가 확정된 후(Closed 상태) 정답자에게 보상을 지급하고 모든 참여자에게 참여 GP를 환급하는 최종 단계 모달입니다. 보상 지급은 1회만 가능하며, 지급 후에는 게임이 Ended 상태로 전환되어 더 이상 관리 액션을 수행할 수 없습니다. 보상 계산 로직과 지급 대상을 명확히 안내합니다.

4. 규칙/로직

보상 지급 가능 조건

조건 검증 시점 실패 시 처리
게임 상태 = Closed 모달 오픈 전 버튼 비활성화
게임 결과 입력 완료 모달 오픈 전 버튼 비활성화
정답자가 1명 이상 API 호출 시 특수 처리 (아래 참조)
관리자 권한 확인 API 호출 시 403 에러
중복 지급 방지 API 호출 시 409 에러

보상 계산 로직

기본 원칙

총 상금 GP = 게임 생성 시 설정된 고정 금액
정답자 보상 = 총 상금 GP × (개인 배분 가중치 / 전체 정답자 가중치 합)
참여 GP 환급 = 각 참여자가 베팅한 참여 GP 전액
부스팅 GP 환급 = 없음 (환급되지 않음)

배분 가중치 계산

각 정답자의 배분 가중치:

개인 배분 가중치 = 참여 GP + (부스팅 GP × 2)

전체 정답자 가중치 합:

전체 가중치 합 = ∑(각 정답자의 배분 가중치)

개인 보상 계산

각 정답자가 받는 보상:

개인 보상 = (총 상금 GP × 개인 배분 가중치) / 전체 가중치 합
  • 소수점 이하 절사
  • 절사로 인한 단수 조정은 가중치가 가장 큰 정답자에게 가산

계산 예시

정답자 참여 GP 부스팅 GP 배분 가중치 보상 GP
A 1,000 500 1,000 + (500×2) = 2,000 (10,000 × 2,000) / 10,000 = 2,000 GP
B 2,000 1,000 2,000 + (1,000×2) = 4,000 (10,000 × 4,000) / 10,000 = 4,000 GP
C 1,000 1,000 1,000 + (1,000×2) = 3,000 (10,000 × 3,000) / 10,000 = 3,000 GP
D 500 0 500 + (0×2) = 500 (10,000 × 500) / 10,000 = 500 GP
E 500 0 500 + (0×2) = 500 (10,000 × 500) / 10,000 = 500 GP + 단수 조정
합계 - - 10,000 10,000 GP

참여 GP 환급 로직

환급 대상

  • 정답자: 참여 GP 전액 환급
  • 오답자: 참여 GP 전액 환급

모든 참여자에게 참여 GP는 무조건 환급됩니다.

부스팅 GP 처리

  • 정답자: 부스팅 GP 환급 없음 (보상 배분 가중치로만 사용)
  • 오답자: 부스팅 GP 환급 없음 (소멸)

특수 케이스 처리

정답자가 0명인 경우

처리 방식 내용
총 상금 GP 시스템으로 회수 또는 소멸
참여 GP 모든 참여자에게 전액 환급
부스팅 GP 환급 없음 (소멸)
게임 상태 Closed → Ended
안내 메시지 "정답자가 없습니다. 참여 GP만 환급됩니다."

정답자가 전체 참여자인 경우

  • 총 상금 GP를 모든 참여자에게 배분 가중치 비율로 배분
  • 참여 GP 전액 환급
  • 오답자가 없으므로 환급 대상 = 정답자 = 전체 참여자

GP 지급 실행

  1. 각 정답자에게 보상 GP 지급
  2. 각 참여자에게 참여 GP 환급
  3. 각 참여자의 GP 변동 내역에 기록 생성
    • 보상 GP: "게임 '{타이틀}' 보상"
    • 환급 GP: "게임 '{타이틀}' 참여 GP 환급"

보상 지급 후 처리

  1. 게임 상태: Closed → Ended
  2. 보상 지급 완료 플래그 설정
  3. 보상 지급 일시 기록
  4. 더 이상 관리 액션 불가 (조회만 가능)

5. 에러 및 예외상황

클라이언트 에러

에러 상황 에러 메시지 처리 방법
게임 상태가 Closed가 아님 "보상을 지급할 수 없는 상태의 게임입니다." 모달 닫기, 상세 화면 갱신
게임 결과 미입력 "게임 결과를 먼저 입력해주세요." 모달 닫기, 결과 입력 모달 안내
이미 보상이 지급된 게임 "이미 보상이 지급된 게임입니다." 모달 닫기, 상세 화면 갱신

서버 에러

HTTP 상태 에러 메시지 처리 방법
400 "보상을 지급할 수 없는 게임 상태입니다." 에러 토스트, 모달 닫기, 화면 갱신
403 "보상 지급 권한이 없습니다." 에러 토스트, 모달 닫기
404 "게임을 찾을 수 없습니다." 에러 토스트, 리스트로 이동
409 "이미 보상이 지급된 게임입니다." 에러 토스트, 상세 화면 갱신
500 "서버 오류가 발생했습니다." 에러 토스트, 모달 유지

GP 지급 실패

에러 상황 처리 방법
일부 참여자 지급 실패 트랜잭션 롤백, 게임 상태 유지, 에러 로그 기록
GP 서비스 응답 없음 타임아웃 후 재시도, 3회 실패 시 관리자 알림
GP 잔액 부족 (시스템) 긴급 알림, 재충전 후 재시도

계산 오류

에러 상황 처리 방법
보상 합계 불일치 단수 조정 재계산, 로그 기록
정답자 데이터 불일치 데이터 재조회 후 재계산

네트워크 에러

에러 상황 처리 방법
타임아웃 "요청 시간이 초과되었습니다. 다시 시도해주세요."
연결 실패 "네트워크 연결을 확인해주세요."
