1. 화면 목적

진행 중인 게임(Active 상태)을 관리자 판단에 따라 즉시 종료하는 강제 종료 확인 모달입니다. 강제 종료 시 모든 참여자에게 참여 GP와 부스팅 GP가 전액 환급되며, 보상은 지급되지 않습니다. 이 작업은 되돌릴 수 없으므로 신중한 확인이 필요합니다.

3. 화면 내 기능설명

모달 트리거

트리거 조건 동작
게임 상세(CEB-BO-GZ-202)에서 상태가 Active "강제 종료" 버튼 활성화
"강제 종료" 버튼 클릭 본 모달 오픈

버튼 동작

취소 버튼

  1. 모달 닫기
  2. 게임 상세 화면으로 복귀
  3. 상태 변경 없음

강제 종료 버튼

  1. 강제 종료 요청 실행
  2. 성공 시:
    • 게임 상태: Active → Ended
    • 모든 참여자 참여 GP 전액 환급
    • 모든 참여자 부스팅 GP 전액 환급
    • 보상 미지급
    • 토스트 메시지: "게임이 종료되었습니다."
    • 모달 닫기
    • 게임 상세 화면 갱신 (상태 및 액션 버튼 변경)
    • 실패 시:
      • 에러 토스트 표시
      • 모달 유지 (재시도 가능)

닫기(X) 버튼

  • 취소 버튼과 동일한 동작

4. 규칙/로직

강제 종료 가능 조건

조건 검증 시점 실패 시 처리
게임 상태 = Active 모달 오픈 전 버튼 비활성화
관리자 권한 확인 API 호출 시 403 에러

강제 종료 시 처리 로직

게임 상태 변경

  • Active → Ended
  • 더 이상 참여 불가
  • 결과 입력 단계 없이 바로 종료 상태로 전환

참여 GP 환급

  1. 모든 참여자의 참여 GP 계산
  2. 각 참여자에게 참여 GP 전액 환급
  3. GP 변동 내역에 환급 기록 생성

부스팅 GP 환급

  1. 모든 참여자의 부스팅 GP 계산
  2. 각 참여자에게 부스팅 GP 전액 환급
  3. GP 변동 내역에 환급 기록 생성

보상 미지급

  • 게임 결과가 없으므로 보상 계산 없음
  • 총 상금 GP는 소멸되거나 시스템으로 회수

환급 예정 GP 계산

환급 예정 GP = ∑(각 참여자의 참여 GP + 부스팅 GP)

5. 에러 및 예외상황

클라이언트 에러

에러 상황 에러 메시지 처리 방법
게임 상태가 Active가 아님 "종료할 수 없는 상태의 게임입니다." 모달 닫기, 상세 화면 갱신
이미 종료된 게임 "이미 종료된 게임입니다." 모달 닫기, 상세 화면 갱신

서버 에러

HTTP 상태 에러 메시지 처리 방법
400 "강제 종료할 수 없는 게임 상태입니다." 에러 토스트, 모달 닫기, 화면 갱신
403 "게임 종료 권한이 없습니다." 에러 토스트, 모달 닫기
404 "게임을 찾을 수 없습니다." 에러 토스트, 리스트로 이동
409 "이미 종료 처리 중인 게임입니다." 에러 토스트, 상세 화면 갱신
500 "서버 오류가 발생했습니다." 에러 토스트, 모달 유지

환급 처리 실패

에러 상황 처리 방법
일부 참여자 환급 실패 트랜잭션 롤백, 게임 상태 유지, 에러 로그 기록
GP 서비스 응답 없음 타임아웃 후 재시도, 3회 실패 시 관리자 알림

네트워크 에러

에러 상황 처리 방법
타임아웃 "요청 시간이 초과되었습니다. 다시 시도해주세요."
연결 실패 "네트워크 연결을 확인해주세요."
