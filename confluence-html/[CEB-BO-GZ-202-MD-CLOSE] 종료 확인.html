1. 화면 목적

진행 중(Active) 또는 결과 대기(Pending) 상태의 게임을 관리자 판단에 따라 즉시 종료하는 강제 종료 확인 모달입니다. 강제 종료 시 참여자에게 GP가 환불되며, 보상은 지급되지 않습니다. 이 작업은 되돌릴 수 없으므로 신중한 확인이 필요합니다.

게임유형별 강제 종료 차이

  • Prediction Market: Active 또는 Pending 상태에서 강제 종료 가능. 참여 GP + 부스팅 GP 전액 환불.
  • Survival Trivia: Active 상태에서만 강제 종료 가능 (Pending 상태 없음). 참여 GP 전액 환불 (부스팅 없음).

버튼 동작

취소 버튼

  1. 모달 닫기
  2. 게임 상세 화면으로 복귀
  3. 상태 변경 없음

  1. 강제 종료 요청 실행
  2. 성공 시:

    • ST: 모든 참여자 참여 GP 전액 환불 (부스팅 없음)

    • 토스트 메시지: "게임이 종료되었습니다."
    • 모달 닫기

  3. 실패 시:
    • 에러 토스트 표시
    • 모달 유지 (재시도 가능)

4. 규칙/로직

강제 종료 가능 조건

조건 검증 시점 실패 시 처리
게임 상태 = Active 또는 Pending (PM) / Active (ST) 모달 오픈 전 버튼 비활성화
관리자 권한 확인 API 호출 시 403 에러

강제 종료 시 처리 로직

게임 상태 변경

  • PM: Active → Ended 또는 Pending → Ended
    • Active: 진행 중인 투표가 즉시 마감되어 더 이상 참여 불가
    • Pending: 결과 입력 단계 없이 바로 종료 상태로 전환
  • ST: Active → Ended (Pending 상태 없음)
    • 게임 시작 전(Active, 아직 시작일시 미도래): 게임 일정이 취소되어 참여자 입장 불가
    • 게임 진행 중(Active, 퀴즈 진행 중): 실시간 진행 중인 게임이 즉시 중단됨

ST Active 강제 종료 시 특수 처리

  1. 진행 중인 참여자 즉시 퇴장: 현재 퀴즈에 참여 중인 모든 참여자에게 게임 중단 알림 표시
  2. 현재 스테이지까지 데이터 보존: 강제 종료 시점까지의 스테이지별 생존/탈락 기록은 이력으로 보존
  3. 보상 미지급: 생존자가 있더라도 게임이 정상 완료되지 않았으므로 보상 없음
  4. 앱 표시: 게임존에서 해당 게임이 "종료" 상태로 전환됨

참여 GP 환급

  1. 모든 참여자의 참여 GP 계산
  2. 각 참여자에게 참여 GP 전액 환급
  3. GP 변동 내역에 환급 기록 생성

강제 종료 시 환급은 "환불(REFUND_CANCEL)" 유형으로 GP 변동 내역에 기록됩니다.

부스팅 GP 환급

  1. 모든 참여자의 부스팅 GP 계산
  2. 각 참여자에게 부스팅 GP 전액 환급
  3. GP 변동 내역에 환급 기록 생성

ST: Survival Trivia는 부스팅을 지원하지 않으므로 부스팅 GP 환급이 발생하지 않습니다.

보상 미지급

  • 게임 결과가 없으므로 보상 계산 없음
  • 총 상금 GP는 소멸되거나 시스템으로 회수

환급 예정 GP 계산

게임유형별 환급 예정 GP
- PM: 환급 예정 GP = ∑(각 참여자의 참여 GP + 부스팅 GP)
- ST: 환급 예정 GP = ∑(각 참여자의 참여 GP)

5. 에러 및 예외상황

클라이언트 에러

에러 상황 에러 메시지 처리 방법
게임 상태가 Active 또는 Pending이 아님 "종료할 수 없는 상태의 게임입니다." 모달 닫기, 상세 화면 갱신
이미 종료된 게임 "이미 종료된 게임입니다." 모달 닫기, 상세 화면 갱신

서버 에러

HTTP 상태 에러 메시지 처리 방법
400 "강제 종료할 수 없는 게임 상태입니다." 에러 토스트, 모달 닫기, 화면 갱신
403 "게임 종료 권한이 없습니다." 에러 토스트, 모달 닫기
404 "게임을 찾을 수 없습니다." 에러 토스트, 리스트로 이동
409 "이미 종료 처리 중인 게임입니다." 에러 토스트, 상세 화면 갱신
500 "서버 오류가 발생했습니다." 에러 토스트, 모달 유지

환급 처리 실패

에러 상황 처리 방법
