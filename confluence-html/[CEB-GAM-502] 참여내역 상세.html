1. 화면 목적

게임 참여 관련 GP 변동 내역의 상세 정보를 보여주는 화면입니다. 참여, 부스팅, 환급, 보상, 환불 5가지 유형에 따라 다른 상세 정보를 표시하며, 게임 정보와 결과를 확인할 수 있습니다.

주요 기능

  • 게임 참여 상세 정보 조회 (참여/부스팅/환급/보상/환불)
  • GP 변동량 및 게임 정보 확인
  • 게임 상세 화면으로 이동
  • 결과 확인 링크 제공 (보상 유형)

진입 경로

CEB-GAM-501 GP 히스토리 > 참여내역 탭 > 내역 항목 탭

영역별 구성요소

Header 영역

  • 좌측: < 뒤로가기 버튼 (GP 히스토리로 복귀)
  • 중앙: "{유형} 상세" 타이틀 (참여 상세 / 부스팅 상세 / 환급 상세 / 보상 상세 / 환불 상세)

상단 카드 영역

  • 게임 타이틀 (최대 2줄, 말줄임)
  • GP 변동량 (큰 폰트, 색상)

구분선

  • 그라데이션 또는 단순 구분선

상세 정보 영역

2컬럼 레이아웃 (Label : Value)

  • 라벨: 왼쪽 정렬, 회색 텍스트
  • 값: 오른쪽 정렬, 검정 텍스트
상세 스타일은 피그마 참조

3.2 GP 변동량 표시

변동 방향 표시 형식 색상 유형
증가 (+) + {숫자} GP 파란색 (#1E90FF) 환급, 보상, 환불
감소 (-) - {숫자} GP 빨간색 (#FF4D4F) 참여, 부스팅
  • 천 단위 콤마 적용
  • 폰트 크기: 본문보다 큰 사이즈

3.3 내 선택 표시

일반 선택 (참여, 환급)

내 선택          Yes
  • "Yes" 또는 "No" 텍스트만 표시

부스팅 선택 (부스팅, 보상)

내 선택          Yes 🔥 부스팅 (2배)
  • "Yes" 또는 "No" + 불 이모지(🔥) + "부스팅" + "(배수배)"
  • 부스팅 적용 시에만 추가 표시

Survival Trivia

  • Survival Trivia는 게임 참여 시 사전 선택(Yes/No)이 없으므로 "내 선택" 필드 자체를 표시하지 않습니다.

3.4 게임 상태 표시

유형 게임 상태 표시 추가 요소
참여/부스팅 진행중 / 결과 대기 / 결과 발표 / 종료 -
환급 결과 발표 / 종료 -
보상 결과 발표 / 종료 [결과보기 >] 링크 (보상만)
환불 취소 -

결과보기 링크 (보상만)

  • 위치: "게임 상태" 필드 우측
  • 표시: [결과보기 >] (파란색 링크)
  • 탭 동작: CEB-GAM-203 결과 상세 화면 이동
  • 전달 파라미터: 게임 ID

Survival Trivia 결과보기 링크

  • ST 보상 상세에서 "결과보기" 탭 시: CEB-GAM-206 트리비아 결과 화면으로 이동
  • 전달 파라미터: 트리비아 게임 ID

3.6 뒤로가기 동작

  • Header의 < 버튼 탭 → CEB-GAM-501 GP 히스토리로 복귀
  • 복귀 시: 참여내역 탭 유지, 스크롤 위치 복원

3.7 게임 타이틀 탭

  • Prediction Market: 상단 카드의 게임 타이틀 탭 → CEB-GAM-202 PM 상세 화면 이동
  • Survival Trivia: 상단 카드의 게임 타이틀 탭 → CEB-GAM-206 트리비아 상세 화면 이동
  • 전달 파라미터: 게임 ID

3.8 사용자 시나리오

시나리오 1: 게임 참여 상세 확인

  1. GP 히스토리 > 참여내역 탭에서 "[참여]" 내역 탭
  2. 참여 상세 화면 진입
  3. Header: "참여 상세"
  4. 상단 카드: 게임 타이틀 + "- 10 GP" (빨간색)
  5. 내 선택: Yes
  6. 게임 상태: 진행중
  7. 참여일시: 2026.02.01. 09:53:32 (UTC+9)
  8. 게임 정보: Prediction Market

시나리오 2: 부스팅 상세 확인

  1. GP 히스토리 > 참여내역 탭에서 "[부스팅]" 내역 탭
  2. 부스팅 상세 화면 진입
  3. Header: "부스팅 상세"
  4. 상단 카드: 게임 타이틀 + "- 1 GP" (빨간색)
  5. 내 선택: Yes 🔥 부스팅 (2배)
  6. 게임 상태: 진행중
  7. 참여일시: 2026.02.01. 09:53:32 (UTC+9)
  8. 게임 정보: Prediction Market

시나리오 3: 환급 상세 확인

  1. GP 히스토리 > 참여내역 탭에서 "[환급]" 내역 탭
