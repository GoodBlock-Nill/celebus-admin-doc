1. 화면 목적

새로운 게임을 생성하는 폼 화면입니다. 관리자는 게임 유형을 선택하고, 기본 정보, 보상 설정, 참여 설정, 일정 설정, 결과 설정을 입력하여 게임을 임시저장하거나 즉시 게시할 수 있습니다.

3. 화면 내 기능설명

다국어 입력

타이틀 입력
  • KO/EN/JP 탭으로 언어 전환
  • 각 언어별 50자 제한
  • 실시간 글자 수 표시: "25/50"
  • 필수 입력: 모든 언어 필수 (KO/EN/JP) (권장: 모든 언어)
상세설명 입력
  • KO/EN/JP 탭으로 언어 전환
  • WYSIWYG 에디터 제공 (Toolbar + 편집 영역)
  • Toolbar 기능: Bold, Italic, Underline, 링크 삽입, 순서 없는 리스트, 순서 있는 리스트, 이미지 삽입, 좌측/중앙/우측 정렬
  • 각 언어별 500자 제한
  • 필수 입력: 모든 언어 필수 (KO/EN/JP)

힌트 링크 입력

힌트 링크
  • Segmented Button: 사용/미사용 (기본값: 미사용)
  • "사용" 선택 시 URL 입력 필드 표시
  • "미사용" 선택 시 URL 입력 필드 숨김
  • Placeholder: "https://example.com/hint"
  • 도움말 텍스트: "게임 관련 힌트 페이지 URL을 입력하세요."
  • URL 형식 검증 (입력 시)

보상설정 입력

총 상금 GP
  • Number Input 사용
  • 양수만 입력 가능
  • 천 단위 콤마 자동 표시
  • 최소값: 1 GP

참여설정 입력

참여 정원
  • 체크박스 '참여 정원 제한' 표시
  • 체크 해제 시: 무제한 (Number Input 숨김)
  • 체크 시: Number Input 표시
  • 최소값: 1명
참여 비용
  • Number Input 사용
  • 기본값: 1 GP
  • 최소값: 1 GP
부스팅 비용
  • Number Input 사용
  • 기본값: 1 GP
  • 최소값: 1 GP
부스팅 배수
  • Number Input 사용
  • 기본값: 2배
  • 최소값: 1배
  • 최대값: 10배

일정설정 입력

투표 시작일시
  • 게시 시점에 자동 설정됨
  • 사용자 입력 불가
  • 정보성 텍스트로만 표시
투표 종료일시
  • Date Picker로 날짜 선택
  • Time Picker로 시간 선택 (분 단위)
  • 현재 시간 이후만 선택 가능
  • 형식: YYYY.MM.DD HH:mm
결과 발표 예정일
  • Date Picker로 날짜 선택
  • 참여 종료일 이후만 선택 가능
  • 형식: YYYY.MM.DD

결과설정 입력

결과 확인 기준 (다국어)
  • KO/EN/JP 탭으로 언어 전환
  • 각 언어별 500자 제한
  • Textarea 사용
  • 예시 텍스트 제공: "예: FIFA 공식 홈페이지 발표 기준"
  • 필수 입력: 모든 언어 필수 (KO/EN/JP)

취소

트리거
  • [취소] 버튼 클릭
동작
  1. CEB-BO-GZ-201-MD-CANCEL (취소 확인 모달) 오픈
  2. 사용자가 [취소하기] 선택 시:
    • 입력된 모든 데이터 폐기
    • CEB-BO-GZ-201 (게임 리스트)로 이동

임시저장

트리거
  • [임시저장] 버튼 클릭
동작
  1. 필수 항목 Validation (필수 아닌 항목은 체크 안 함)
  2. 통과 시:
    • 게임 상태: Draft
    • 서버에 저장
    • Success 토스트: "게임이 임시저장되었습니다."
    • CEB-BO-GZ-201 (게임 리스트)로 이동
    • 실패 시:
      • 인라인 에러 메시지 표시
      • 첫 번째 에러 필드로 스크롤

생성하기

트리거
  • [생성하기] 버튼 클릭
동작
  1. 모든 필수 항목 Validation
  2. 통과 시:
    • 게임 상태: Ready
    • 서버에 저장
    • Success 토스트: "게임이 생성되었습니다."
    • CEB-BO-GZ-201 (게임 리스트)로 이동
    • 실패 시:
      • 인라인 에러 메시지 표시
      • 첫 번째 에러 필드로 스크롤

5. 에러 및 예외상황

필수 항목 미입력

상황
  • 필수 항목이 비어있는 상태로 저장 시도
처리
  1. 인라인 에러 메시지 표시: "{항목명}을(를) 입력해주세요."
  2. 해당 입력 필드 빨간색 테두리 표시
  3. 첫 번째 에러 필드로 자동 스크롤
  4. 버튼 비활성화 유지

날짜/시간 Validation 실패

상황
  • 투표 종료일시 < 현재 시간
처리
  • 인라인 에러 메시지: "투표 종료일시는 현재 시간 이후여야 합니다."
  • 투표 종료일시 필드 빨간색 테두리
상황
  • 결과 발표 예정일 < 참여 종료일
처리
  • 인라인 에러 메시지: "결과 발표 예정일은 참여 종료일 이후여야 합니다."
  • 결과 발표 예정일 필드 빨간색 테두리

서버 저장 실패

상황
  • API 호출 실패 또는 네트워크 오류
처리
  1. 에러 토스트 표시: "게임 저장에 실패했습니다. 다시 시도해주세요."
  2. 입력된 데이터 유지
  3. 사용자가 다시 저장 시도 가능

권한 오류

상황
  • 쓰기 권한이 없는 관리자가 화면 접근 시도
처리
  1. 접근 차단
  2. 권한 오류 페이지로 리다이렉트
  3. 에러 메시지: "게임을 생성할 권한이 없습니다."

취소 시 미저장 데이터

상황
  • 입력된 데이터가 있는 상태에서 [취소] 버튼 클릭
처리
  1. CEB-BO-GZ-201-MD-CANCEL (취소 확인 모달) 오픈
  2. 사용자 선택 대기:
    • [계속 작성]: 모달 닫기, 폼 유지
    • [취소하기]: 데이터 폐기, 리스트로 이동
