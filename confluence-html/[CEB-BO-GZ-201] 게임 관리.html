1. 화면 목적

게임존의 모든 게임을 조회하고 관리하는 메인 리스트 화면입니다. 관리자는 게임 유형, 상태, 기간 등 다양한 필터를 통해 게임을 검색하고, 새로운 게임을 생성하거나 기존 게임의 상세 정보로 이동할 수 있습니다.

3. 화면 내 기능설명

게임 생성

트리거
  • Header 우측 [+ 게임 생성] 버튼 클릭
동작
  1. CEB-BO-GZ-201-CREATE (게임 생성) 화면으로 이동
  2. 권한 검증: 쓰기 권한이 있는 관리자만 버튼 표시

게임 검색 및 필터링

게임유형 필터
  • 셀렉트 옵션: Prediction Market, Survival Trivia
  • 기본값: Prediction Market
  • "전체" 옵션 없음 (게임 유형은 반드시 선택)
  • 변경 시 즉시 리스트 갱신
  • Survival Trivia 선택 시: "개발 준비중" 메시지 표시
상태 필터
  • 셀렉트 옵션: 전체, Draft, Ready, Active, Pending, Closed, Ended
  • 기본값: 전체
  • 변경 시 즉시 리스트 갱신
검색어 입력
  • 입력 필드에 타이틀 검색어 입력
  • Enter 키 또는 검색 아이콘 클릭 시 검색
  • 부분 일치 검색 (대소문자 구분 없음)
  • 한글/영문/일본어 모두 검색 가능

게임 상세 이동

트리거
  • 테이블 ROW 클릭
동작
  1. 해당 게임의 ID를 파라미터로 전달
  2. CEB-BO-GZ-202 (게임 상세) 화면으로 이동

페이지네이션

표시 규칙
  • 페이지당 20건 표시
  • 총 페이지 수 계산: 전체 게임 수 / 20 (올림)
  • 현재 페이지 번호 강조 표시
  • 이전/다음 버튼 활성화 조건:
    • 이전: 현재 페이지 > 1
    • 다음: 현재 페이지 < 총 페이지 수
동작
  • 페이지 번호 클릭 시 해당 페이지로 이동
  • 이전/다음 버튼 클릭 시 페이지 이동
  • 페이지 이동 시 필터 조건 유지

4. 규칙/로직

정렬 기준

기본 정렬
  • 생성일 내림차순 (최신순)
  • 동일 생성일의 경우 생성 시간 내림차순

상태별 표시 규칙

상태 표시 조건
Draft 임시저장된 게임
Ready 게시 완료, 참여 시작일 > 현재 시간 (게시대기)
Active 참여 시작일 ≤ 현재 시간 ≤ 참여 종료일
Pending 참여 종료일 < 현재 시간 AND 결과 미확정
Closed 결과 확정됨
Ended 결과 발표일 + 7일 경과

권한별 기능 분기

권한 기능
읽기 권한 리스트 조회, 검색, 필터링, 상세 화면 이동
쓰기 권한 읽기 권한 + [+ 게임 생성] 버튼 표시 및 기능
권한 없음 접근 불가, 권한 오류 페이지로 이동

참여자 수 계산

  • 실시간 집계: 해당 게임에 참여한 고유 회원 수
  • 중복 제거: 동일 회원이 여러 번 참여해도 1명으로 계산
  • 초기값: 0

총 상금 GP 표시

  • 형식: {숫자} GP
  • 천 단위 콤마 표시
  • 예시: 100,000 GP

5. 에러 및 예외상황

권한 오류

상황
  • 읽기 권한이 없는 관리자가 화면 접근 시도
처리
  1. 접근 차단
  2. 권한 오류 페이지로 리다이렉트
  3. 에러 메시지: "게임 관리 화면에 접근할 권한이 없습니다."

데이터 로딩 실패

상황
  • API 호출 실패 또는 네트워크 오류
처리
  1. 에러 토스트 표시: "게임 목록을 불러오는 데 실패했습니다. 다시 시도해주세요."
  2. [다시 시도] 버튼 제공
  3. 3회 연속 실패 시 기술 지원 안내

Empty 상태

상황
  • 등록된 게임이 없거나 검색 결과가 없는 경우
처리
  1. 테이블 대신 Empty 상태 메시지 표시
  2. 필터 적용 중일 때: "검색 조건에 맞는 게임이 없습니다."
  3. 필터 미적용 시: "등록된 게임이 없습니다."
  4. 쓰기 권한이 있는 경우 [+ 게임 생성] 버튼 강조

페이지네이션 오류

상황
  • 존재하지 않는 페이지 번호 접근 시
처리
  1. 자동으로 첫 페이지(1페이지)로 이동
  2. URL 파라미터 보정

필터 조합 오류

상황
  • 기간 필터에서 종료일 < 시작일 선택
처리
  1. 인라인 에러 메시지: "종료일은 시작일 이후여야 합니다."
  2. 검색 버튼 비활성화
  3. 날짜 수정 시 자동 해제
