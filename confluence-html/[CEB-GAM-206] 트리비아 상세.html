1. 화면 목적

실시간 Survival Trivia 퀴즈 게임 진행 화면입니다. 각 스테이지(문제)별로 4지선다 선택지를 제공하고, 제한 시간(10초) 내에 정답을 선택해야 합니다. 정답/오답 결과가 실시간으로 공개되며, 참여자 간 라이브 채팅을 통해 소통할 수 있습니다.

이 문서는 Stage별 문제 풀이 진행 화면을 다룹니다. 탈락 모달(CEB-GAM-206-MD-ELIMINATE), 나가기 모달(CEB-GAM-206-MD-LEAVE), 관전 모드(CEB-GAM-206-SPECTATE), 게임 결과(CEB-GAM-206-RESULT)는 별도 문서입니다.

3. 화면 내 기능설명

3.1 선택지 탭 (응답)

트리거
  • 4개 선택지 중 하나를 탭
동작
  1. 선택 즉시 서버로 전송
  2. 선택한 선택지 하이라이트 (선택 확정 표시)
  3. 나머지 선택지 탭 비활성화 (변경 불가)
  4. 다른 참여자 응답 대기
  5. 제한시간 종료 시 서버에서 정답 공개 이벤트 수신
규칙
  • 선택 후 변경 불가 (실시간 게임 특성)

3.2 타이머 바

표시 규칙
  • 서버 기준 {N}초(기본 10초) 카운트다운
  • 프로그레스 바: 좌→우 파란색, 시간 경과에 따라 감소
  • 서버 시간 기준 (클라이언트 시간 보정)
동작
  • 시간 종료 시 자동으로 정답 공개 화면 전환

3.3 정답/오답 공개

트리거
  • 서버에서 정답 + 선택지별 선택 비율(%) 수신
동작
  1. 정답 선택지: 파란 테두리 + ⭕ 마크
  2. 오답 선택지 (내가 선택한 경우): 빨간 테두리 + "내 선택" 배지
  3. 정답 선택지 (내가 선택한 경우): 파란 테두리 + ⭕ 마크 + "내 선택" 배지
  4. 나머지 선택지: 기본 테두리 + 비율만 표시
  5. 공개 후 5초 대기 → 자동으로 다음 스테이지 전환

3.4 하트 소진

오답 시
  • 하트 1개 자동 소진 + 하트 소진 애니메이션 (❤️→💔 효과)
  • 하트 0개에서 오답 시: 오답 공개 화면 5초 표시 후 CEB-GAM-206-MD-ELIMINATE 모달 표시
  • 단, 마지막 스테이지(10/10)에서 하트 0 + 오답 시 → 모달 없이 CEB-GAM-206-RESULT 직행
미응답(시간 초과) 시
  • 타이머 만료 시 키보드가 열려 있다면 자동 닫힘
  • 정답 공개 화면 표시 (선택지에 "미선택" 표시, 선택 비율 노출)
  • 정답 공개 5초 후 CEB-GAM-206-MD-ELIMINATE 모달 표시 (오답과 동일 플로우)
  • 하트 여부 무관하게 즉시 탈락
  • 단, 마지막 스테이지(10/10)에서 미응답 시 → 모달 없이 CEB-GAM-206-RESULT 직행

3.5 채팅 기능

메시지 표시
  • 화면 하단 채팅 영역에 실시간 메시지 스트림 표시
  • 닉네임 + 메시지 형태로 좌측 정렬
  • 새 메시지 수신 시 자동 하단 스크롤
메시지 입력/전송
  • 💬 버튼 탭 → 키보드 오픈 + 입력 필드 활성화
  • ▶ 전송 버튼 탭 → 메시지 서버 전송
제한 규칙
규칙 내용
욕설 필터링 금칙어 목록 기반 자동 마스킹 처리 ("***")
발송 제한 초당 1회 (rate limit), 초과 시 발송 무시
문자 수 제한 최대 100자, 초과 입력 불가
키보드 활성화 시
  • 상태 바(하트/스테이지/참여자) 숨김
  • 타이머 바와 문제/선택지는 축소 표시
  • 채팅 중에도 선택지 탭 가능 (키보드가 선택지를 가릴 수 있지만 스크롤로 접근 가능)
  • 키보드 닫기: 빈 영역 탭, 전송 완료 후, 또는 타이머 만료 시 자동 닫힘

3.6 뒤로가기

트리거
  • Header 좌측 [<] 버튼 탭
동작
  1. CEB-GAM-206-MD-LEAVE (나가기 확인) 모달 표시
  2. 나가기 확인 시: 게임 포기 + CEB-GAM-204(트리비아 홈)로 이동
  3. GP 환불 없음

3.7 스테이지 전환

트리거
  • 정답/오답 공개 후 5초 대기
동작 (자동 전환 시 화면 초기화)
  1. 타이머 바 리셋 (풀 파란색)
  2. 선택지 활성화 (선택 가능 상태)
  3. 결과 텍스트(정답/오답) 제거
  4. 새 문제 텍스트 + 선택지 표시
  5. 스테이지 번호 +1 (예: 1/10 → 2/10)
마지막 스테이지 완료 시
  • 마지막 스테이지(10/10) 정답 후 → CEB-GAM-206-RESULT (완주 성공)
  • 마지막 스테이지(10/10) 오답/미응답으로 탈락 시 → 탈락 모달 없이 CEB-GAM-206-RESULT 직행 (탈락자 결과)

선택 비율 계산

  • 각 선택지별 선택 비율 = (해당 선택지 선택 인원 / 전체 응답 인원) × 100
  • 소수점 없이 정수로 표시 (반올림)
  • 미응답자는 비율 계산에서 제외

5. 에러 및 예외상황

네트워크 끊김 중 응답

상황
  • 선택지 탭 후 서버 전송 실패
처리
  1. 자동 재전송 시도 (최대 3회)
  2. 성공 시: 정상 진행
  3. 실패 시: 미응답 처리 → 즉시 탈락

서버 오류

상황
  • 문제 수신 또는 정답 수신 실패
처리
  1. 에러 토스트: "일시적 오류가 발생했습니다."

라이브 서버 다운

상황
  • 게임 진행 중 서버 전체 장애
처리
  1. "서버 장애로 게임이 중단되었습니다." 안내
  2. GP 100% 환불 처리 (서버 자동)
  3. 공지 발송

채팅 전송 실패

상황
  • 채팅 메시지 전송 실패
처리
  1. 전송 실패 표시 (빨간색 느낌표)
  2. 재전송 없음 (다시 입력)

채팅 금칙어 감지

상황
  • 욕설/금칙어가 포함된 메시지 전송 시도
처리
  1. 메시지 내 금칙어 부분 "***" 마스킹 후 전송 (차단이 아닌 마스킹)

운영자 강제 종료

상황
  • 운영자가 게임을 강제 종료한 경우
처리
  1. 안내 모달: "운영자에 의해 게임이 종료되었습니다."
  2. GP 100% 환불 처리 (서버 자동)
  3. 모달 확인 시 CEB-GAM-204(트리비아 홈)로 이동

앱 백그라운드 복귀

상황
  • 게임 진행 중 앱이 백그라운드로 전환된 후 다시 포그라운드로 복귀
처리
  1. 게임이 아직 진행 중인 경우: 현재 스테이지부터 게임 합류
  2. 복귀 시 놓친 문제가 있는 경우: 해당 문제는 미응답 처리 → 즉시 탈락
  3. 게임이 이미 종료된 경우: CEB-GAM-204(트리비아 홈)로 이동
