1. 화면 목적

네트워크 끊김 후 재연결 시, 이미 놓친 스테이지가 존재하여 탈락 처리된 사용자에게 상황을 안내하는 모달입니다. 관전 모드로 전환하거나 게임을 나갈 수 있는 선택지를 제공합니다.

기존 탈락 모달(CEB-GAM-206-MD-ELIMINATE)은 게임 중 오답/미응답에 의한 탈락 시 표시되며, 본 모달은 네트워크 끊김으로 인한 탈락 시에만 표시됩니다.

3. 화면 내 기능설명

모달 트리거

트리거 조건 동작
CEB-GAM-205 카운트다운 중 네트워크 끊김 → 재연결 성공 → 놓친 스테이지 존재 본 모달 오픈
CEB-GAM-206 탭 비활성화/백그라운드 → 탭 복귀 → 놓친 스테이지 존재 본 모달 오픈

놓친 스테이지가 없는 경우 (아직 Stage 1 진행 중) → 본 모달 미표시, CEB-GAM-206 정상 합류

버튼 동작

나가기 버튼

  1. 본 모달 닫기
  2. CEB-GAM-204(트리비아 홈)로 즉시 이동
  3. 하트 보상 미지급

관전하고 하트 받기 버튼

  1. 본 모달 닫기
  2. CEB-GAM-206-SPECTATE(관전 모드)로 전환
  3. 끝까지 관전 완료 시 하트 1개 지급 (서버 처리)

모달 닫기 동작

  • 나가기 / 관전하고 하트 받기 버튼으로만 닫기 가능
  • 외부 영역(딤) 탭: 닫히지 않음 (강제 선택 필요)
  • 뒤로가기 제스처: 닫히지 않음

자동 닫기 규칙

4. 규칙/로직

탈락 처리 기준

놓친 스테이지에 대해 "미응답 = 즉시 탈락" 규칙이 동일하게 적용됩니다. 네트워크 끊김 여부와 관계없이 미응답은 즉시 탈락입니다.

상황 처리
재연결 시 놓친 스테이지 0개 본 모달 미표시, CEB-GAM-206 정상 합류
재연결 시 놓친 스테이지 1개 이상 서버에서 즉시 탈락 처리 완료 → 본 모달 표시
재연결 시 게임 이미 종료 본 모달 미표시, CEB-GAM-206-RESULT 이동

게임 종료 시 예외

재연결 시점에 게임이 이미 종료된 경우 본 모달을 표시하지 않고 CEB-GAM-206-RESULT로 직접 이동합니다.
