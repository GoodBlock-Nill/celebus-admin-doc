1. 화면 목적

CELB 토큰을 GP로 교환하기 위한 입금 지갑 주소를 안내하는 화면입니다. 사용자는 QR 코드 스캔 또는 주소 복사를 통해 CELB를 입금할 수 있습니다.

주요 기능

  • CELB 입금 주소 QR 코드 표시
  • 지갑 주소 텍스트 표시 및 복사
  • 네트워크 정보 표시 (BSC)
  • 입금 시 주의사항 안내

접근 경로

  • GP 교환소(CEB-GAM-301) > [CELB 입금주소] 버튼 탭

3. 화면 내 기능설명

뒤로가기

트리거
  • Header 좌측 [<] 버튼 탭
동작
  1. GP 교환소(CEB-GAM-301) 화면으로 복귀
  2. 네비게이션 스택에서 현재 화면 제거

QR 코드

생성 규칙
  • 입력 데이터: CELB 입금 지갑 주소 (0x로 시작하는 42자)
  • 크기: 256x256px
  • 오류 정정 레벨: M (중간, 15% 복구 가능)
  • 배경색: 흰색
  • 전경색: 검정색
동작
  • QR 코드 롱프레스 시 이미지 저장 옵션 표시
  • 저장 성공 시 토스트: "QR 코드가 저장되었습니다."

입금 주소 표시

표시 규칙
  • 형식: 0x로 시작하는 42자 Ethereum 주소
  • 폰트: Monospace (고정폭)
  • 크기: 14px
  • 색상: 검정 (Primary Text)
  • 줄바꿈: 2줄 (21자마다 줄바꿈) 또는 1줄 (가로 스크롤)
  • 예시:
    • 0x6B175474E89094C44Da98b954Ee
    • deAC495271d0F
복사 아이콘
  • 아이콘: 📋 (클립보드)
  • 위치: 주소 텍스트 우측
  • 탭 시 주소 복사 (아래 "주소 복사하기" 버튼과 동일 기능)

네트워크 정보

표시 규칙
  • 텍스트: "BSC (BNB Smart Chain)" (고정)
  • 색상: 검정 (Primary Text)
  • 크기: 16px
  • 설명: CELB 토큰이 발행된 블록체인 네트워크 정보
네트워크 종류
  • BSC (BNB Smart Chain): Binance Smart Chain (현재 고정)
  • 향후 다른 네트워크 추가 가능 (Ethereum, Polygon 등)

안내사항

표시 내용
  1. CELB 토큰만 입금: "이 주소로 CELB 토큰만 입금해주세요."
  2. 자산 손실 경고: "다른 토큰을 보내면 자산이 손실될 수 있습니다."
  3. 입금 확인 시간: "입금 확인에는 네트워크 상태에 따라 시간이 걸릴 수 있습니다."
표시 규칙
  • 불릿 포인트(•) 형식
  • 다국어 지원 (KO/EN/JP)
  • 색상: 회색 (Secondary Text)
  • 크기: 14px
  • 줄 간격: 1.5

[주소 복사하기] 버튼

표시 규칙
  • 라벨: "주소 복사하기" (다국어)
  • 스타일: Primary 버튼 (파란색)
  • 위치: 하단 고정 (Safe Area 내)
  • 너비: 화면 너비 - 좌우 패딩 (32px)
동작
  1. 버튼 탭 시 CELB 입금 주소를 클립보드에 복사
  2. 성공 토스트: "주소가 복사되었습니다."
  3. 복사 실패 시 에러 토스트: "주소 복사에 실패했습니다."

4. 규칙/로직

지갑 주소 제공 방식

백오피스 연동
  • 백오피스 지갑 관리(CEB-BO-GZ-503)에서 등록된 충전용 대표 지갑의 주소를 표시
  • 대표 지갑이 없는 경우 Error 상태 표시 (아래 "에러 및 예외상황" 참조)
지갑 변경 시
  • 백오피스에서 충전용 대표 지갑이 변경되면 앱에서도 즉시 반영
  • 화면 진입 시마다 최신 주소 조회

QR 코드 생성 로직

생성 시점
  • 화면 진입 시 서버에서 생성된 QR 코드 이미지 URL 전달
  • 또는 클라이언트에서 주소를 받아 QR 코드 라이브러리로 생성
캐싱 정책
  • 동일 주소의 QR 코드는 캐싱하여 재사용
  • 주소 변경 시 캐시 무효화

주소 복사 로직

복사 대상
  • 지갑 주소 전체 (42자)
  • 공백, 줄바꿈 없이 순수 주소만 복사
복사 성공 조건
  • 클립보드 API 사용 가능
  • 권한 허용 상태
토스트 표시
  • 성공: "주소가 복사되었습니다." (Success 토스트, 초록색)
  • 실패: "주소 복사에 실패했습니다." (Error 토스트, 빨간색)

다국어 처리

지원 언어
  • 한국어 (KO)
  • 영어 (EN)
  • 일본어 (JP)
적용 대상
  • Header 타이틀: "CELB 입금주소"
  • 라벨: "입금 주소", "네트워크"
  • 안내사항 (3개 텍스트)
  • [주소 복사하기] 버튼 라벨
  • 토스트 메시지
주소 및 네트워크 정보
  • 지갑 주소: 다국어 처리 불필요 (숫자와 영문자만 사용)
  • 네트워크: "BSC (BNB Smart Chain)" (고정, 다국어 처리 불필요)

입금 확인 처리

입금 모니터링
  • 사용자가 CELB를 입금하면 서버에서 블록체인 트랜잭션 모니터링
  • 입금 확인 시 자동으로 GP 충전 (CEB-GAM-301의 교환 내역에 표시)
  • 사용자에게 푸시 알림 또는 인앱 알림 전송
입금 확인 시간
  • 블록체인 네트워크 상태에 따라 1~10분 소요 (BSC 기준)
  • 사용자에게 "입금 확인에는 네트워크 상태에 따라 시간이 걸릴 수 있습니다." 안내

5. 에러 및 예외상황

대표 지갑 미등록

상황
  • 백오피스에서 충전용 대표 지갑이 등록되지 않은 경우
처리
  1. Error 상태 표시
  2. 에러 메시지: "현재 CELB 입금이 불가능합니다. 나중에 다시 시도해주세요."
  3. [뒤로가기] 버튼 제공
  4. 버튼 탭 시 GP 교환소(CEB-GAM-301) 화면으로 복귀

데이터 로딩 실패

상황
  • API 호출 실패 또는 네트워크 오류
처리
  1. Error 상태 표시
  2. 에러 메시지: "입금 주소를 불러오는 데 실패했습니다."
  3. [다시 시도] 버튼 제공
  4. 버튼 탭 시 데이터 재로딩

QR 코드 생성 실패

상황
  • QR 코드 이미지 생성 또는 로딩 실패
처리
  1. QR 코드 영역에 에러 메시지 표시: "QR 코드를 불러올 수 없습니다."
  2. 주소 텍스트 및 복사 기능은 정상 제공
  3. [다시 시도] 버튼 제공 (QR 코드 영역 내)

주소 복사 실패

상황
  • 클립보드 권한 거부 또는 API 오류
처리
  1. 에러 토스트: "주소 복사에 실패했습니다."
  2. 권한 요청 안내 (iOS/Android 시스템 설정 안내)
  3. 사용자가 수동으로 주소 선택 및 복사 가능

QR 코드 저장 실패

상황
  • 갤러리 저장 권한 거부 또는 저장 공간 부족
처리
  1. 에러 토스트: "QR 코드 저장에 실패했습니다."
  2. 권한 요청 안내 (갤러리 접근 권한)

Loading 상태

상황
  • 데이터 로딩 중
처리
  1. Skeleton UI 표시
    • QR 코드: 회색 사각형 (256x256px)
    • 주소 텍스트: 회색 박스 (2줄)
    • 네트워크: 회색 박스 (1줄)
    • 안내사항: 회색 박스 (3줄)
  2. 로딩 완료 시 실제 데이터로 전환 (페이드인 애니메이션)

네트워크 연결 끊김

상황
  • 오프라인 상태에서 화면 진입 또는 데이터 로딩 시도
처리
  1. Error 상태 표시
  2. 에러 메시지: "네트워크 연결을 확인해주세요."
  3. [다시 시도] 버튼 제공
  4. 온라인 복구 시 자동 재로딩
