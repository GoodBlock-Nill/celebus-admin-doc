1. 화면 목적

모든 스테이지 종료 후 표시되는 최종 결과 화면입니다. 생존자의 상금 GP는 서버에서 자동 지급되며, 탈락 관전자에게는 하트 보상이 자동 지급됩니다. 마지막 스테이지(10/10)에서 탈락한 사용자도 탈락 모달 없이 이 화면으로 직행합니다. 탈락 후 나가기를 선택한 사용자(미관전)에게도 게임 종료 후 결과를 제공합니다.

이 문서는 게임 종료 후 결과 화면을 다룹니다. 게임 진행 화면(CEB-GAM-206), 탈락 모달(CEB-GAM-206-MD-ELIMINATE), 관전 모드(CEB-GAM-206-SPECTATE)는 별도 문서입니다.

3. 화면 내 기능설명

3.1 확인 버튼 (완주자)

트리거
  • 하단 [확인] 버튼 탭
동작
  1. CEB-GAM-204(트리비아 홈)으로 이동
  2. 상금 GP는 RESULT 화면 진입 시 서버에서 자동 지급 완료 상태
  3. 별도 수령 동작 불필요

3.2 확인 버튼 (탈락 관전자)

트리거
  • 하단 [확인] 버튼 탭
동작
  1. CEB-GAM-204(트리비아 홈)으로 이동
  2. 하트 1개는 서버에서 관전 완료 시점에 이미 자동 지급 완료 상태

3.3 뒤로가기

트리거
  • Header 좌측 [<] 버튼 탭
동작
  • CEB-GAM-204(트리비아 홈)으로 이동
  • 보상은 RESULT 화면 진입 시 서버에서 자동 지급되므로 별도 수령 필요 없음

3.4 확인 버튼 (탈락 미관전자)

트리거
  • 하단 [확인] 버튼 탭
동작
  1. CEB-GAM-204(트리비아 홈)으로 이동
  2. 보상 없음 (GP, 하트 모두 미지급)

4. 규칙/로직

보상 계산

항목 계산
1인당 보상 총 상금 GP ÷ 최종 생존자 수 (균등 분배)
소수점 처리 소수점 이하 절삭 (예: 30,000 ÷ 102 = 294 GP)
잔여 GP 소수점 절삭으로 발생한 잔여 GP는 플랫폼 보유
전원 탈락 생존자 0명인 경우 총 상금 GP 전액 플랫폼 보유
참여비 환급 없음 (참여비 10 GP는 환급되지 않음)

업적 배지

배지 표시 조건 설명
🏅 100% 정답 모든 문제 정답 (오답 0회) 오답 없이 전 스테이지 통과
🏅 완주 마지막 스테이지까지 생존 생존자 전원에게 부여
🏅 무하트 클리어 하트를 1개도 사용하지 않고 완주 100% 정답 달성 시 항상 동시 충족
  • 100% 정답이면 반드시 무하트 클리어 조건도 충족 (오답이 없으므로 하트가 소진되지 않음)
  • "100% 정답"과 "무하트 클리어" 배지는 중복 획득 가능 (둘 다 동시에 표시)
  • 탈락자에게는 업적 배지 영역 미표시

화면 진입 경로별 표시

진입 경로 결과 화면 타입 보상
CEB-GAM-206 마지막 스테이지 정답 (생존) A 또는 B (완주 성공) GP 보상 (균등 분배)
CEB-GAM-206-SPECTATE 게임 종료 (관전 완료) C (탈락 관전 완료) 하트 1개 (자동 지급)
CEB-GAM-206 마지막 스테이지 탈락 (10/10 오답/미응답) D (탈락 미관전) 보상 없음 (탈락 모달 없이 직행)
CEB-GAM-204 게임 종료 이벤트 수신 (탈락 후 나가기) D (탈락 미관전) 보상 없음 (자동 전환)
CEB-GAM-204 재방문 시 미확인 결과 존재 D (탈락 미관전) 보상 없음 (자동 전환)
