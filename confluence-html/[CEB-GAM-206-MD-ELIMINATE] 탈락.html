1. 화면 목적

하트 소진(0개에서 오답) 또는 미응답(시간 초과)으로 탈락 시 표시되는 모달입니다. 사용자에게 관전 모드로 전환하거나 게임을 나갈 수 있는 선택지를 제공합니다. 끝까지 관전하면 하트 1개를 보상으로 받을 수 있음을 안내합니다.

3. 화면 내 기능설명

모달 트리거

트리거 조건 동작
CEB-GAM-206에서 하트 0개 + 오답 오답 공개 후 본 모달 오픈
CEB-GAM-206에서 미응답(시간 초과) 시간 초과 즉시 본 모달 오픈 (하트 수 무관)

버튼 동작

나가기 버튼

  1. 본 모달 닫기
  2. CEB-GAM-204(트리비아 홈)로 즉시 이동
  3. 하트 보상 미지급
  4. 게임 종료 시 204에서 자동으로 결과 화면(CEB-GAM-206-RESULT D타입)으로 전환

관전하고 하트 받기 버튼

  1. 본 모달 닫기
  2. CEB-GAM-206-SPECTATE(관전 모드)로 전환
  3. 끝까지 관전 완료 시 하트 1개 지급 (서버 처리)

모달 닫기 동작

  • 나가기 / 관전하고 하트 받기 버튼으로만 닫기 가능
  • 외부 영역(딤) 탭: 닫히지 않음 (강제 선택 필요)
  • 뒤로가기 제스처: 닫히지 않음

자동 닫기 규칙

  • 다음 스테이지 시작 전까지 선택하지 않으면 자동으로 CEB-GAM-204(트리비아 홈)로 이동
  • 사유: 게임이 진행 중이므로 무한 대기 불가

4. 규칙/로직

탈락 조건

조건 하트 변화 결과
하트 1개 이상 + 오답 -1개 계속 진행 (본 모달 미표시)
하트 0개 + 오답 0 유지 탈락 → 본 모달 표시
미응답 (시간 초과) 변화 없음 즉시 탈락 → 본 모달 표시

5. 에러 및 예외상황

관전 전환 실패

에러 상황 처리 방법

네트워크 끊김 "네트워크 연결을 확인해주세요." 에러 토스트, CEB-GAM-204로 이동
