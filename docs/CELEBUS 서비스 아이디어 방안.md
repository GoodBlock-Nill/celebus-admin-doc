## (Part 1 배경) 우리가 가지고 있는것

현재 저희 서비스에서 제공할수 있는 서비스들입니다.

- 팬 퀘스트 / 응모권
- 팬즈
- NFT : 드랍 / 합성 / 뽑기
- Prediction Market
- Trivia

**CELEBUS만의 핵심 강점:** 위 5가지 기능을 동시에 보유한 K-pop 플랫폼은 CELEBUS가 유일합니다. Weverse는 커뮤니티+커머스, Bubble은 1:1 메시지, 최애돌은 투표 중심입니다. CELEBUS는 퀘스트+Trivia+PM이라는 **"게임화된 덕질 경험"**을 제공할 수 있는 유일한 플랫폼입니다.

**현재 상황:** CELEBUS는 클로즈드 앱으로 운영 중이며, 이번 V01D 캠페인을 계기로 오픈 전환이 필요합니다.

**추가 자산:** V01D 소속사 IX Entertainment와 긴밀한 협력 관계, iKON 온보딩 예정

## (Part 2 서비스 아이디어) 서비스 설명

### 핵심 전략: "CELEBUS에서 덕질하면 V01D를 만날 수 있다"

모든 서비스가 하나의 최종 목표로 수렴합니다: **V01D 팬싸인회/콘서트 VIP/영상통화**. CELEBUS가 마케팅 예산으로 팬싸 참가권과 콘서트 티켓을 직접 구매하여, CELEBUS 활동 유저에게만 경품으로 제공합니다.

| 서비스 | 핵심 역할 | 팬에게 한 마디 |
| ---- | ------- | --------- |
| 서비스 1: V01D 챌린지 | **Acquisition** | "V01D 퀘스트를 클리어하면 사인 포카 + 독점 콘텐츠가 열려! (팬싸 시즌에는 팬싸 응모까지!)" |
| 서비스 2: V01D 덕력 시즌 랭킹 | **Retention** | "매달 덕력 1위는 사인 앨범 확정! 팬싸 시즌에는 팬싸까지 확정!" |
| 서비스 3: V01D 팬덤 레벨업 | **Viral** | "팬덤 레벨이 올라가면 독점 콘텐츠→굿즈→영통까지 열려!" |

### 보상 체계 이원화: 상시 보상 + 이벤트 보상

팬싸인회와 콘서트는 연 2~3회로 상시 진행되는 이벤트가 아닙니다. 보상이 팬싸/콘서트에만 의존하면 행사 공백 기간에 유저 동기가 급락합니다. 이를 방지하기 위해 보상을 이원화합니다.

**상시 보상 (Always Available) — 팬싸/콘서트 일정 무관, CELEBUS가 독립적으로 제공:**

| 보상 | 설명 | 비용 | IX 의존도 |
| ---- | ---- | ---- | --------- |
| V01D 사인 포토카드 | IX에서 소량 매입 후 경품 제공 (월 3~5장) | ~5만원/월 | 낮음 (매입) |
| V01D 독점 콘텐츠 해금 | 비하인드 사진, 미공개 클립 (IX 월 1회 제공) | 무료 | 중간 (월 1회 협조) |
| V01D 셀카/보이스 메시지 | 멤버별 월 1건 제공 (IX 협조) | 무료 | 중간 (월 1회 협조) |
| CELEBUS 한정판 디지털 포토카드 | CELEBUS 자체 제작 디지털 수집품 (NFT 기반) | ~10만원/회 | 없음 |
| V01D TOP FAN 인증 뱃지 | 시즌 랭킹 결과 기반 디지털 뱃지 | 무료 | 없음 |
| 커스텀 프로필 프레임/테마 | V01D 테마 프로필 꾸미기 아이템 | 무료 | 없음 |
| V01D 기념일 특별 굿즈 | 생일/데뷔일 등에 맞춘 한정 굿즈 (사전 기획) | ~20만원/회 | 낮음 (매입) |

**이벤트 보상 (Event-based) — 팬싸/콘서트 일정 확정 시 해금:**

| 보상 | 설명 | 비용 | IX 의존도 |

**운영 원칙:** 이벤트 보상은 "보너스"로 포지셔닝하고, 상시 보상만으로도 팬이 충분히 매력을 느끼는 구조를 기본으로 합니다. 팬싸/콘서트 일정이 확정되면 이벤트 보상이 "추가 해금"되는 서프라이즈 효과를 줍니다.

### 서비스 1: V01D 챌린지 — 덕질하면 팬싸 응모가 열린다

**1. 가설:**

K-pop 팬에게 팬싸인회는 최고의 아티스트 접점입니다. 팬싸 한 번 가려고 앨범 50장을 사는 것이 이 시장의 현실입니다. 하지만 CELEBUS는 앨범을 파는 플랫폼이 아닙니다.

만약 "CELEBUS에서 V01D 퀘스트를 클리어하면 팬싸 응모 자격이 열린다"면, 팬은 앨범 구매 대신 **CELEBUS에서 덕질을 하는 것이 팬싸에 갈 수 있는 새로운 경로**가 됩니다. 이것은 기존 K-pop 팬 플랫폼 어디에서도 제공하지 않는 경험입니다.

CELEBUS의 퀘스트→Trivia→PM을 순차적으로 거치는 스토리형 구조는 단순 "활동 = 응모권"이 아니라 **"V01D를 알아가는 여정"** 자체입니다. 이 구조는 퀘스트+Trivia+PM을 동시에 보유한 CELEBUS에서만 가능합니다.

**2. 서비스내용:**

유저가 CELEBUS에 가입하면 "V01D 탐험"이라는 스토리형 퀘스트가 시작됩니다. 총 5장(Chapter)으로 구성되며, 각 장을 클리어하면 다음 장이 열립니다. **마지막 5장을 클리어하면 상시 보상이 즉시 해금되고, 팬싸 일정 확정 시 이벤트 보상까지 추가됩니다.**

| 장 | 제목 | 미션 | CELEBUS 기능 활용 | 보상 |
| -- | ---- | ---- | -------------- | ---- |
| 1장 | "V01D를 만나다" | V01D 공식 X/IG/YouTube 팔로우 인증 | 퀘스트 | 응모권 3장 + 2장 해금 |
| 2장 | "V01D를 듣다" | Tug of War + ROCKROCK 스트리밍 인증 | 퀘스트 | 응모권 4장 + 3장 해금 |
| 3장 | "V01D를 알다" | V01D Trivia 10문제 중 7개 이상 정답 | Trivia | 응모권 3장 + 4장 해금 |
| 4장 | "V01D를 예측하다" | V01D 관련 PM 3개 참여 | Prediction Market | 응모권 5장 + 5장 해금 |
| 5장 | "V01D를 응원하다" | V01D 응원 SNS 게시 (@celebus_official 태그) | 퀘스트 | **상시 보상 즉시 해금** + 응모권 5장 |

**5장 클리어 보상 (상시):**

- V01D 사인 포토카드 추첨 응모 자격 해금 (매월 추첨, 3~5명)
- V01D 독점 콘텐츠 즉시 해금 (비하인드 사진 3장 + 미공개 클립 1개)
- "V01D 탐험 완료" 한정 디지털 포토카드 1장 (소장 가치)
- 응모권 1장 = 1회 응모. 많이 모을수록 당첨 확률 UP
- 매월 추첨 → 당첨자 발표

**이벤트 보상 (팬싸/콘서트 일정 확정 시 추가 해금):**

- 5장 클리어 유저만 팬싸 참가권 추첨 응모 가능
- CELEBUS가 마케팅 예산으로 팬싸 참가권 사전 구매 (1~2장)
- 팬싸 일정 확정 시 앱 내 특별 배너로 서프라이즈 공지

**왜 CELEBUS에서만 가능한가:** 5장 구조가 퀘스트(1~2장) → Trivia(3장) → PM(4장) → 퀘스트(5장)로 CELEBUS의 4가지 기능을 순차적으로 거침. Weverse에서는 퀘스트만, 최애돌에서는 투표만 가능. **이 스토리형 덕질 여정은 CELEBUS만 가능.**

### 서비스 2: V01D 덕력 시즌 랭킹 — 1위는 팬싸 확정

**1. 가설:**

K-pop 팬덤에서 "덕력 랭킹 1위"는 최고의 자부심입니다. 최애돌에서 투표 1위 하려고 밤새는 팬들이 있는데, 만약 그 1위의 보상이 **"사인 앨범 확정, 팬싸 시즌에는 팬싸까지 확정"**이라면?

CELEBUS의 덕력 랭킹은 투표 한 가지가 아니라 퀘스트+Trivia+PM 종합 활동량으로 측정됩니다. "진짜 V01D 덕후"가 1위를 차지하는 구조이며, 이 종합 랭킹은 CELEBUS에서만 가능합니다.

**2. 서비스내용:**

월간 시즌제로 운영됩니다. 매월 1일~말일까지가 하나의 시즌이며, 시즌 종료 시 랭킹을 정산합니다.

**덕력 포인트 적립:**

| 활동 | 포인트 | 비고 |
| ---- | ----- | ---- |
| 퀘스트 완료 | 50~100 | 미션 난이도별 차등 |
| Trivia 정답 | 30 | 문제당 |
| PM 참여 | 20 | 베팅 참여 자체 |
| PM 적중 | 100 | 보너스 |
| SNS 공유 | 50 | 주 1회 한정 |
| 7일 연속 접속 스트릭 | 200 | 보너스 |

**랭킹 보상 (상시):**

| 순위 | 상시 보상 | 팬 반응 |
| ---- | -------- | ----- |
| 시즌 1위 | **V01D 사인 앨범 확정** + "덕력왕" 레전드 뱃지 + V01D 독점 셀카 1장 | "1위 무조건 해야 해!" |
| 시즌 2~10위 | V01D 사인 포토카드 1장 + "V01D TOP10" 뱃지 + 독점 콘텐츠 해금 | "TOP10 안에는 들어야지" |
| 시즌 TOP 10% | "슈퍼팬" 뱃지 + 한정판 디지털 포토카드 + 커스텀 프로필 프레임 | "슈퍼팬 뱃지 자랑해야지" |
| 나머지 | 참여 뱃지 + 응모권 보너스 | "다음 달에는 더 열심히" |

**이벤트 보상 가산 (팬싸/콘서트 일정 확정 시):**

| 순위 | 추가 보상 |
| ---- | -------- |
| 시즌 1위 | 팬싸 참가권 1장 확정 (추첨 아님) |
| 시즌 2~10위 | 팬싸 추첨 가중치 3배 |
| 시즌 TOP 10% | 팬싸 추첨 가중치 1.5배 |

시즌 종료 시 **TOP 10 닉네임을 V01D 공식 계정에서 공개** (IX 협조: 이미지 1장, V01D 로고+TOP10 닉네임)

**팬덤 문화 적합성:** 최애돌 투표 랭킹의 진화형. 차이점은 최애돌 1위 = 광고판, CELEBUS 1위 = **사인 앨범 + 팬싸 (시즌 시)**. 보상의 차원이 다릅니다.

### 서비스 3: V01D 팬덤 레벨업 — 함께 올리면 보상이 커진다

**1. 가설:**

K-pop 팬덤의 가장 강력한 DNA는 "총공" 문화입니다. 개인이 아닌 팬덤 전체가 힘을 모아 목표를 달성하는 것이 K-pop 팬덤의 핵심 행동 양식입니다.

"팬덤 전체 활동량이 올라가면 보상이 커진다"는 구조는 총공 문화와 정확히 맞습니다. 그리고 상시 보상이 먼저 쌓이고, 팬싸/콘서트 시즌에 이벤트 보상이 추가되면, 팬들은 "CELEBUS 가입해! 우리 레벨 올려야 해!"라고 서로에게 자발적으로 홍보합니다.

**2. 서비스내용:**

팬덤 전체의 CELEBUS 활동량이 합산되어 "팬덤 레벨"이 올라갑니다. 레벨이 올라갈수록 CELEBUS가 제공하는 경품이 추가됩니다.

| 레벨 | 조건 (누적 활동 건수) | 상시 해금 보상 | 이벤트 추가 보상 (일정 확정 시) | 예상 도달 시점 |
| ---- | -------------- | ----------- | ---------------------- | --------- |
| Lv.1 | 500건 | V01D 독점 비하인드 콘텐츠 팩 해금 | 팬싸 참가권 +1장 (총 2장) | 1~2주차 |
| Lv.2 | 2,000건 | V01D 멤버 보이스 메시지 해금 + 사인 포카 추첨 2명 | 팬싸 참가권 +2장 (총 4장) | 3~4주차 |
| Lv.3 | 5,000건 | V01D 한정판 디지털 포카 세트 + 사인 앨범 추첨 1명 | 콘서트 VIP 티켓 2장 추가 | 5~6주차 |
| Lv.4 | 10,000건 | V01D 멤버별 독점 셀카 + 사인 앨범 추첨 3명 | 콘서트 VIP 3장 + 영상통화 추첨 1명 | 7~8주차 |
| Lv.5 | 20,000건 | V01D 특별 영상 메시지 (팬덤 전체 대상) + 기념 굿즈 | 영상통화 3명 + V01D 감사 라이브 | 10주차+ |

**개인 기여도 가시화:**

- 홈 화면에 실시간 프로그레스 바: "현재 Lv.2 / 2,847 / 5,000건 (다음: Lv.3 디지털 포카 세트 + 사인 앨범)"
- 내 기여도 표시: "내 기여: 47건 (전체의 1.7%)"
- 기여도 TOP 10 닉네임 실시간 표시
- **기여도 TOP 3는 팬싸 추첨 추가 응모권 보너스** (서비스 2 랭킹과 연동)

**바이럴 구조:**

- 레벨업 달성 시 축하 이미지 자동 생성 → SNS 공유
- "우리 팬덤 Lv.3 달성! 디지털 포카 세트 + 사인 앨범 추첨 해금!" → X/IG에서 팬들이 자발적 확산
- "아직 Lv.4까지 3,000건 남았어! CELEBUS 가입해서 같이 올리자!" → 팬 주도 바이럴
