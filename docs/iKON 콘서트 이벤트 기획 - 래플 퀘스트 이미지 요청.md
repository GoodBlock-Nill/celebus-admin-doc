# iKON 서울 콘서트 래플/퀘스트 이벤트 기획 + 소속사 이미지 요청 리스트

## Context

5월 iKON 서울 콘서트(장충체육관)에 맞춰 CELEBUS에서 래플 이벤트(경품: 콘서트 티켓)와 퀘스트를 진행한다. 고도화 적용 전이므로 기존 v1 래플/퀘스트 시스템을 사용한다. 실물 이미지 NFT는 제공 불가하나, 콘서트 티켓 스타일 디자인 이미지는 제작 가능하다. 소속사에 필요한 이미지 리스트를 정리하여 전달해야 한다.

**확정 사항:**

- 래플 경품: 콘서트 티켓 (수량 소속사 협의 중)
- 퀘스트: 캡처 제출형 (v1 시스템)
- 콘서트: 5월 예정, 장충체육관
- BIVE: 콘서트 티켓 스타일 디자인 이미지 사용 (멤버 초상권 이미지 제공 불가)

**제약 사항:**

- 멤버 초상권 관련 이미지(멤버 개인 프로필, 멤버 사진 등)는 소속사에서 제공 불가
- 사용 가능 범위: 공식 로고, 콘서트 포스터, 티켓 디자인, 무대/공연장 이미지, MD 제품 이미지, 텍스트/사인

|-----|--------|------|------|------|

## 2. BIVE 디자인 컨셉 (콘서트 티켓 스타일)

멤버 초상권 이미지가 불가하므로, **콘서트 티켓/무대/로고 기반 디자인**으로 BIVE를 자체 제작한다.

| BIVE | 등급 | 디자인 컨셉 | 소스 이미지 | 획득 방법 |
|------|------|------------|------------|----------|
| iKON 콘서트 티켓 (일반) | Grade 1 | 콘서트 티켓 레이아웃 + iKON 로고 + 일시/장소 | No.1 로고 + No.3 티켓 시안 | 퀘스트 미션 1 완료 |
| iKON 콘서트 포스터 에디션 | Grade 2 | 콘서트 포스터 기반 + 티켓 프레임 + 시리얼 넘버 | No.2 포스터 + No.8 타이틀 | 퀘스트 미션 2 완료 |
| iKON 스테이지 에디션 | Grade 3 | 무대/조명 이미지 + 골드 티켓 프레임 | No.4 무대 이미지 | 퀘스트 미션 3 완료 |
| iKON 콘서트 골드 티켓 | Grade 4 | 골드 호일 + 로고 엠보싱 + 콘서트 정보 | No.1 로고 + No.8 타이틀 | 퀘스트 전체 완료 |
| iKON 스페셜 에디션 | 스페셜 | 자필 메시지 + 홀로그램 티켓 + 응원 컬러 | No.5 자필 메시지 + No.7 응원 컬러 | Grade 1~4 합성 |

> 멤버 사진 없이도 콘서트 티켓 고유의 소장 가치를 만드는 전략: 시리얼 넘버, 골드 호일, 홀로그램 등 프리미엄 디자인 요소로 차별화.

## 3. 래플 이벤트 기획 (v1 시스템)

| 항목 | 내용 |
|------|------|
| 래플명 | iKON 서울 콘서트 티켓 래플 |
| 경품 | 콘서트 티켓 (수량 소속사 협의 중) |
| 응모 조건 | 응모권 소모 (1장 = 1회 응모) |
| 응모 기간 | 콘서트 2주 전 ~ 콘서트 3일 전 (약 11일) |
| 추첨 | 응모 마감 후 BO에서 추첨 |
| 당첨 안내 | 앱 푸시 + 래플 상세 화면에서 확인 |
| 경품 수령 | 당첨자 개별 연락 (DM 또는 앱 내 메시지) |

### 응모권 획득 경로 (팬 활동 유도)

- 퀘스트 미션 완료 시 응모권 지급
- 일일 미션 완료 시 응모권 지급
- 출석 스트릭 보너스로 응모권 지급

## 4. 퀘스트 기획 (v1 시스템, 캡처 제출형)

콘서트 전에 완료해야 하므로 콘서트 약 3~4주 전 오픈.

| 미션 | 유형 | 내용 | 보상 |
|------|------|------|------|
| 미션 1 | 캡처 제출 | "iKON 최애곡 스트리밍 인증" — 음원 앱 스트리밍 화면 캡처 제출 | 덕력 + 응모권 + BIVE Grade 1 |
| 미션 2 | 캡처 제출 | "iKON 응원 메시지 SNS 공유 인증" — SNS에 #iKON_콘서트 해시태그 게시 캡처 | 덕력 + 응모권 + BIVE Grade 2 |
| 미션 3 | 캡처 제출 | "iKON 플레이리스트 공유 인증" — iKON 플레이리스트 캡처 또는 음악 앱 공유 화면 | 덕력 + 응모권 + BIVE Grade 3 |
| 미션 4 | 캡처 제출 | "iKON 팬레터 작성" — 기억저장소에 iKON에게 편지 작성 후 캡처 | 덕력 + 응모권 + BIVE Grade 4 |
| 미션 5 | 캡처 제출 | "콘서트 D-Day 카운트다운" — 콘서트 D-Day 위젯/캘린더 캡처 | 덕력 + 응모권 |

> 전체 미션 완료 시: 추가 응모권 보너스 + BIVE 스페셜 에디션 합성 자격
