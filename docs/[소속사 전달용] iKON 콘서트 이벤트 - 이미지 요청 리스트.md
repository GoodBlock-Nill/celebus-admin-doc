# iKON 서울 콘서트 x CELEBUS 이벤트 — 이미지 요청 리스트

## 안녕하세요, CELEBUS 팀입니다.

5월 iKON 서울 콘서트(장충체육관)를 맞아 CELEBUS 앱에서 **콘서트 티켓 래플 이벤트**와 **팬 참여 퀘스트**를 진행하고자 합니다. 이벤트 운영 및 디지털 굿즈(BIVE) 디자인을 위해 아래 이미지 소스를 요청드립니다.

> 멤버 초상권 이미지가 제공 불가한 점을 확인하였으며, 아래 리스트는 **초상권과 무관한 이미지만** 포함하고 있습니다.

---

## 이벤트 개요

| 항목 | 내용 |
|------|------|
| 이벤트명 | iKON 서울 콘서트 티켓 래플 |
| 경품 | 콘서트 티켓 (수량 협의 중) |

| 참여 방식 | 앱 내 퀘스트 미션 완료 → 응모권 획득 → 래플 응모 |
| 디지털 굿즈 | 콘서트 티켓 스타일 디자인의 BIVE(디지털 수집 카드) 제작 및 배포 |

## 이미지 사용처 안내

제공해주신 이미지는 아래와 같이 사용됩니다.

| 사용처 | 해당 이미지 | 설명 |
|--------|------------|------|
| 앱 메인 이벤트 배너 | 콘서트 포스터 | 앱 홈 화면 상단 캐러셀 배너로 노출 |
| 래플 경품 미리보기 | 티켓 이미지 | 래플 참여 화면에서 경품 이미지로 표시 |
| 당첨 축하 화면 | 티켓 이미지 | 당첨자에게 보여주는 축하 화면 |
| 디지털 굿즈 (BIVE) | 로고 + 포스터 + 티켓 + 무대 + 자필 | 콘서트 티켓 스타일 디자인의 수집형 디지털 카드 제작 |
| 아티스트 프로필 | 로고 | 앱 내 iKON 프로필 아이콘 |

> 모든 이미지는 CELEBUS 앱 내에서만 사용되며, 외부 유출 없이 이벤트 목적으로만 활용됩니다.

## 디지털 굿즈 (BIVE) 제작 안내

제공해주신 이미지를 기반으로 CELEBUS에서 **콘서트 티켓 스타일의 디지털 수집 카드**를 디자인하여 팬들에게 퀘스트 보상으로 지급합니다.

| 카드명 | 디자인 컨셉 | 필요 이미지 |
|--------|------------|------------|
| 콘서트 티켓 (일반) | 실제 티켓 레이아웃 + iKON 로고 + 일시/장소 | 로고, 티켓 시안 |
| 포스터 에디션 | 콘서트 포스터 기반 + 티켓 프레임 | 포스터, 타이틀 로고 |
| 스테이지 에디션 | 무대/조명 이미지 + 골드 프레임 | 무대 이미지 |
| 골드 티켓 | 골드 호일 + 로고 엠보싱 | 로고, 타이틀 로고 |
| 스페셜 에디션 | 자필 메시지 + 홀로그램 디자인 | 자필 메시지, 응원 컬러 |

> 디자인 시안은 제작 후 사전에 공유드려 확인받겠습니다.
