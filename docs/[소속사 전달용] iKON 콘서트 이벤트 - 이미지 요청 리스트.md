# iKON 서울 콘서트 x CELEBUS — 이미지 요청 리스트

안녕하세요, CELEBUS 팀입니다.

5월 iKON 서울 콘서트(장충체육관)를 맞아 CELEBUS 앱에서 콘서트 티켓 래플 이벤트와 팬 참여 퀘스트를 진행합니다. 이벤트 운영 및 디지털 굿즈 디자인을 위해 아래 이미지를 요청드립니다.

> 멤버 초상권 이미지가 제공 불가한 점을 확인하였으며, 아래 리스트는 **초상권과 무관한 이미지만** 포함하고 있습니다.

---

## 필수 이미지 (3종)

| No. | 이미지 | 사용 목적 | 요청 규격 |
|-----|--------|----------|----------|
| 1 | **iKON 공식 로고** | 앱 내 아티스트 아이콘, 디지털 굿즈 디자인 | 투명 배경 PNG, 512x512px 이상, 정사각형 |
| 2 | **콘서트 공식 포스터** | 앱 이벤트 배너, 래플 상세 화면, 디지털 굿즈 소스 | 고해상도 세로형 (1080x1920) + 가로형 (1920x1080). 가능하면 텍스트 포함/미포함 2버전 |
| 3 | **콘서트 티켓 이미지 또는 디자인 시안** | 래플 경품 이미지, 당첨 화면, 디지털 굿즈 소스 | 고해상도 (1080x720px 이상). 미확정 시 목업도 가능 |

## 이미지 사용처

| 사용처 | 해당 이미지 |
|--------|------------|
| 앱 메인 이벤트 배너 | 콘서트 포스터 |
| 래플 경품 미리보기 / 당첨 화면 | 티켓 이미지 |
| 디지털 굿즈 (수집형 카드) 제작 | 로고, 포스터, 티켓, 무대, 자필, 타이틀 로고 |
| 앱 내 아티스트 프로필 | 로고 |
| SNS 이벤트 홍보 게시물 (Instagram, X, TikTok 등) | 포스터, 티켓, 로고 |

> 모든 이미지는 CELEBUS 앱 내 이벤트 운영 및 SNS 이벤트 홍보 목적으로만 활용됩니다.
> 디지털 굿즈 디자인 시안은 제작 후 사전에 공유드려 확인받겠습니다.
