## 디자이너 참고사항

### Summary Cards 스펙
- 카드 배치: 가로 4등분 그리드 (3개 카드, 1개 여백)
- 카드 간 간격: 16px
- 카드 높이: 120px
- 카드 내부 패딩: 20px (상하좌우)
- 카드 배경: 흰색 (#FFFFFF)
- 카드 테두리: 1px 연한 회색 (#E5E7EB)
- 카드 모서리: 8px 라운드
- 라벨 스타일: 14px, 회색 (상단 배치)
- 수치 스타일: 28px, 볼드 (하단 배치)
- 카드 내부 정렬: 세로 가운데 정렬

### 간격 (Spacing)
- Header 하단 여백: 24px
- Summary Cards 하단 여백: 24px
- Filter 영역 하단 여백: 16px
- Table 행 높이: 56px
- Pagination 상단 여백: 24px

### 인터랙션
- Table Row 클릭 → CEB-BO-GZ-501-MD-DETAIL 모달 오픈
- [지갑 관리] 버튼 클릭 → CEB-BO-GZ-503 화면 이동
- [교환 설정] 버튼 클릭 → CEB-BO-GZ-502 화면 이동
- [커스텀] 기간 선택 시 → DateRangePicker 오픈 (최대 90일)
- 정렬 가능 컬럼 헤더 클릭 → 화살표 아이콘 회전

### DateRangePicker 스펙
- 위치: 기간 Select 하단
- 너비: 400px
- 최대 선택 범위: 90일
- 범위 초과 시: 에러 메시지 표시

### 정렬 아이콘
- 정렬 전: ⬍ (회색, 비활성)
- 오름차순: ▲ (파란색, 활성)
- 내림차순: ▼ (파란색, 활성)
- 기본 정렬: 교환일시 내림차순 (▼ 활성)

### 실시간 업데이트
- Summary Cards: 30초마다 자동 갱신
- 갱신 중: 카드 우측 상단에 Spinner 표시

### 접근성
- Table Header에 aria-sort 속성
- Row에 role="button", tabindex="0"
- Summary Cards에 aria-label
- Badge에 적절한 색상 대비 (WCAG AA)

### 반응형
- 테이블 최소 너비: 1200px
- 화면 너비 부족 시: 가로 스크롤 허용
- Summary Cards: 화면 좁아지면 세로 배치
