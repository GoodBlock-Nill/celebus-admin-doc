## 디자이너 참고사항

### TOP 10 Section 스펙 (5열 그리드)
- 레이아웃: 5열 × 2행 그리드
- 카드 간 간격: 12px
- 카드 스타일: 배경 흰색, 테두리 gray-200, 모서리 8px 라운드
- 카드 패딩: 16px
- 텍스트 정렬: 중앙 정렬
- 검색 시: TOP 10 섹션 비표시 (검색 결과 테이블만 표시)

### 색상 정의
- 1위 금색 배경: linear-gradient(135deg, #FEF3C7 0%, #FDE68A 100%)
- 2위 은색 배경: linear-gradient(135deg, #F3F4F6 0%, #E5E7EB 100%)
- 3위 동색 배경: linear-gradient(135deg, #FFEDD5 0%, #FED7AA 100%)
- 누적 GP: 파란색 (#3B82F6)

### 간격 (Spacing)
- Header 하단 여백: 24px
- Filter 하단 여백: 16px
- TOP 10 Section 하단 여백: 32px
- Table 행 높이: 56px
- Pagination 상단 여백: 24px

### TOP 10 카드 내부 레이아웃 (중앙 정렬)
```
┌─────────────────────────────┐
│      [순위 Badge/메달]       │
│       [닉네임, truncate]     │
│      [누적 GP, 파란색 강조]   │
│       [참여 {n}회]           │
│        [승률 {n}%]           │
└─────────────────────────────┘
```

### 인터랙션
- Table Row 클릭 → CEB-BO-USR-201 (회원 상세) 게임존 탭 이동
- TOP 10 카드 클릭 → CEB-BO-USR-201 (회원 상세) 게임존 탭 이동
- [랭킹 설정 →] 버튼 클릭 → /game-zone/ranking/settings 이동
- 정렬 가능 컬럼 헤더 클릭 → 화살표 아이콘 회전
- 검색어 입력 후 Enter → 검색 실행, TOP 10 섹션 숨김

### 정렬 아이콘
- 정렬 전: ⬍ (회색, 비활성)
- 오름차순: ▲ (파란색, 활성)
- 내림차순: ▼ (파란색, 활성)
- 기본 정렬: 누적 GP 내림차순 (▼ 활성)

### 접근성
- Table Header에 aria-sort 속성
- Row에 role="button", tabindex="0"
- TOP 10 카드에 role="button", tabindex="0"
- 메달 아이콘에 alt 텍스트

### 반응형
- 테이블 최소 너비: 1200px
- 화면 너비 부족 시: 가로 스크롤 허용
- TOP 10 카드: 세로 스크롤 허용 (높이 제한)
