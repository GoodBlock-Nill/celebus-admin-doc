## 디자이너 참고사항

### 테이블 스펙

- Header Row: background: gray-100, font-weight: 600, height: 48px
- Body Row: height: 72px, border-bottom: 1px solid gray-200
- 복합 셀 (지갑 정보, 잔액): flex-direction: column, gap: 2px
- 액션 컬럼: justify-content: flex-end

### 잔액 표시

- 폰트: font-semibold
- 형식: 소수점 8자리 (1,234.56789012 CELB)
- 정렬: 우측 정렬

### 지갑 주소 표시

- 폰트: font-mono, text-sm
- 색상: text-blue-600
- Truncate: 앞 12자 + "..." + 뒤 8자
- 클릭: BSCScan 새 탭 열기

### PK 표시 (출금용 지갑만, 지갑 정보 셀 Line 2)

- 폰트: font-mono, text-xs
- 색상: text-gray-400
- 접두사: "PK: " (라벨 포함)
- 마스킹: 앞 4자 + "..." + 뒤 4자
- 충전용: Line 2 없음 (주소만 표시)

### BNB 잔액 표시 (전체 지갑, 잔액 셀 Line 2)

- 폰트: text-xs
- 색상: text-gray-500
- 형식: 소수점 6자리 (0.025000 BNB)
- 정렬: 우측 정렬
- 경고: 0.01 BNB 미만 시 text-red-600 + ⚠ 아이콘 (출금용 지갑에만 적용)
- 모든 유형의 지갑에 표시

### 간격 (Spacing)

- Header 하단 여백: 24px
- Table 행 높이: 72px
- 액션 버튼 간 간격: 8px

### 인터랙션

- 지갑 주소 클릭 → BSCScan 새 탭 (https://bscscan.com/address/{address})
- PK 표시 클릭 → 없음 (보안상 복사 불가)
- [잔액 새로고침] 클릭 → Toast "잔액이 갱신되었습니다."
- [+ 지갑 등록] 클릭 → 지갑 등록 모달 오픈
- [수정] 클릭 → 지갑 수정 모달 오픈
- [삭제] 클릭 → 삭제 확인 모달 오픈
- [활성화/비활성화] 클릭 → 즉시 상태 변경 + Toast

### 접근성

- 모든 버튼에 aria-label
- Table Row에 role="button" 없음 (액션 버튼으로 동작)
- Badge에 적절한 색상 대비 (WCAG AA)

### 반응형

- 테이블 최소 너비: 1060px
- 화면 너비 부족 시: 가로 스크롤 허용
