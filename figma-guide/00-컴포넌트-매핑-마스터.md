## 사용 가이드

1. **Preline UI Figma Kit 설치**
   - 위 링크에서 Community 파일 복제
   - 프로젝트에 라이브러리로 추가

2. **컴포넌트 선택**
   - 화면설명서의 UI 요소를 이 매핑표와 대조
   - 적용 화면 열에서 해당 화면 ID 확인
   - Preline에서 해당 컴포넌트 드래그 & 드롭

3. **Variant 적용**
   - Figma 우측 패널에서 Variant 속성 변경
   - 테이블 표의 "Variant/속성" 열 참고

4. **커스텀 컴포넌트**
   - Preline에 없는 요소는 `00-커스텀-컴포넌트-명세.md` 참조
   - 별도로 생성 후 로컬 컴포넌트로 등록

5. **일관성 유지**
   - 같은 UI 요소는 반드시 같은 Preline 컴포넌트 사용
   - 색상, 간격, 폰트는 Preline 기본값 유지
   - 커스텀이 필요한 경우 명세서 작성 후 공유
