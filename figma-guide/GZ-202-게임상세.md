- **Variant**: Line (하단 밑줄)
- **Active Tab**: blue-600 텍스트 + 2px 밑줄
- **Inactive Tab**: gray-600 텍스트
- **Tab 높이**: 48px

### Section (Card) 스타일
- **Container**: background: white, border: gray-200, border-radius: 8px, padding: 24px
- **Title**: 16px, Semibold, margin-bottom: 16px
- **Field Label**: 14px, Medium, color: gray-600
- **Field Value**: 14px, Regular, color: gray-900

### 테이블 스타일
- **Header Row**: background: gray-100, font-weight: 600, height: 48px
- **Body Row**: height: 56px, border-bottom: gray-200
- **Striped**: 짝수 행 background: gray-50
- **Hover**: Row 전체 background: gray-100, cursor: pointer (참여내역 테이블만)

### 타이포그래피
- **Page Title (H1)**: 28px, Bold
- **Section Title (H3)**: 16px, Semibold
- **Field Label**: 14px, Medium, gray-600
- **Field Value**: 14px, Regular, gray-900
- **Badge**: 12px, Medium
- **Table Header**: 14px, Semibold, gray-700
- **Table Cell**: 14px, Regular, gray-900

### 간격 (상세)
- Breadcrumb ~ Title Row: 8px
- Title Row ~ Tabs: 0px (붙어있음)
- Tabs ~ Tab Content: 0px (붙어있음)
- Tab Content 내부 Section 간: 32px
- Section 내부 Field 간: 12px

### 이미지 미리보기
- **썸네일**: width: 320px, aspect-ratio: 16:9, object-fit: cover
- **클릭 인터랙션**: 클릭 시 모달 또는 새 탭에서 확대 보기

### 보상 계산 표시
- **보상 GP**: 굵은 글씨 (font-weight: 600) 또는 색상 강조 (blue-700)
- **환급 GP**: 일반 글씨

### 결과 Badge
- **YES**: Badge (Success, Soft) - green-100 배경, green-700 텍스트
- **NO**: Badge (Danger, Soft) - red-100 배경, red-700 텍스트

### Empty State (참여내역)
- **Container**: 중앙 정렬, padding: 80px 0
- **Icon**: 큰 회색 아이콘 (예: users icon)
- **Message**: 16px, gray-500, "참여 내역이 없습니다."

### 정렬 표시
- **Sortable Column**: Header에 정렬 아이콘 (↕︎) 표시
- **Active Sort**: 화살표 방향 표시 (↑ 오름차순, ↓ 내림차순)
- **기본 정렬**: 참여일시 내림차순 (↓ 아이콘)

### 인터랙션
- **Tab 클릭**: 해당 Tab Content로 전환, URL 파라미터 변경
- **액션 버튼**: 상태별로 다른 모달 또는 화면 이동
- **테이블 Row 클릭**: 회원 상세(CEB-BO-USR-201) 화면으로 이동 (참여내역 테이블만)
- **닉네임 클릭**: 회원 상세 화면으로 이동 (색상: blue-600, hover: underline)

### 접근성
- **Tab**: role="tab", aria-selected="true/false"
- **Tab Panel**: role="tabpanel", aria-labelledby="tab-id"
- **Sortable Column**: aria-sort="ascending/descending/none"
- **Empty State**: role="status", aria-live="polite"

### 반응형
- 1440px 이상: 위 레이아웃 그대로
- 1024px ~ 1439px: Section Grid → 1 column, Table 가로 스크롤
- 768px ~ 1023px: Tab → Accordion 형태로 변경 고려
- 768px 미만: 모든 컨텐츠 세로 스택, Table → Card List 변환
