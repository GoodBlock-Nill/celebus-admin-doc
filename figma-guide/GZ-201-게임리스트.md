## 디자이너 참고사항

### 테이블 스타일
- **Header Row**: background: gray-100, font-weight: 600, color: gray-700, height: 48px
- **Body Row**: height: 64px, border-bottom: 1px solid gray-200
- **Striped**: 짝수 행 background: gray-50
- **Hover**: Row 전체 background: gray-100, cursor: pointer
- **타이틀 Cell**: max-width 초과 시 말줄임(...) 적용

### 필터 바 레이아웃
- **Container**: Flex, gap: 12px, padding: 16px, background: white, border-bottom: 1px solid gray-200
- **필터 순서**: 게임유형 → 상태 → 검색어 (좌측부터)
- **높이**: 56px (내부 컴포넌트 height: 40px)

### 간격
- Breadcrumb ~ Header: 8px
- Header ~ Filter Bar: 24px
- Filter Bar ~ Table: 0px (붙어있음)
- Table ~ Pagination: 24px

### 타이포그래피
- **Page Title (H1)**: 28px, Bold
- **Table Header**: 14px, Semibold, gray-700
- **Table Cell**: 14px, Regular, gray-900
- **Badge**: 12px, Medium
- **Empty State**: 16px, Regular, gray-500

### 인터랙션
- **Row 클릭**: 게임 상세 화면 이동
- **Filter 변경**: 즉시 리스트 갱신 (debounce 없음)
- **검색어 입력**: Enter 키 또는 검색 아이콘 클릭 시 검색
- **Pagination**: 페이지 번호 클릭 시 해당 페이지로 이동, 필터 조건 유지

### 정렬 (향후 확장)
- 현재는 정렬 기능 없음 (기본 정렬: 생성일 내림차순)
- 향후 컬럼 Header 클릭 시 정렬 기능 추가 예정

### 반응형
- 1440px 이상: 모든 컬럼 표시
- 1024px ~ 1439px: "관리자" 컬럼 숨김
- 768px ~ 1023px: "참여 기간", "생성일" 컬럼 숨김
- 768px 미만: Table → Card List 형태로 변경 (모바일 최적화)

### 권한별 표시
- **읽기 권한**: 리스트 조회, 필터링, Row 클릭 가능
- **쓰기 권한**: 읽기 권한 + [+ 게임 생성] 버튼 표시
- **권한 없음**: 화면 접근 불가, 권한 오류 페이지로 리다이렉트

### Empty State 디자인
- **Container**: 중앙 정렬, padding: 80px 0
- **Icon**: 큰 회색 아이콘 (예: empty box, search icon)
- **Message**: 16px, gray-500
- **Button**: Primary 또는 Secondary (상황에 따라)

### 페이지네이션 계산
- 페이지당 20건
- 총 페이지 수 = ceil(전체 게임 수 / 20)
- 최대 표시 페이지 번호: 5개 (1 2 3 ... 10 형태)
