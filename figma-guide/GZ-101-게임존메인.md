| 상태 | 변경 요소 | 표시 |

## 디자이너 참고사항

### 카드 스타일
- **Stats Card**: 중앙 정렬, Label은 위, Value는 아래 (큰 폰트 사이즈)
- **Simple Card**: 좌측 정렬, Title과 바로가기는 Header 영역에 배치, Body는 항목별 Label+Value 수평 배치
- **Card Grid 간격**: 16px (섹션 내), 24px (섹션 간)

### 레이아웃 그리드
- **전체 현황**: 4 columns (1fr 1fr 1fr 1fr)
- **게임 관리 카드 그리드**: 2 columns (1fr 1fr)
- **하단 정보 카드**: Row 1은 2 columns (1fr 1fr), Row 2는 1 column (full width)

### 타이포그래피
- **Page Title (H1)**: 28px, Bold
- **Section Title (H2)**: 20px, Bold
- **Card Title**: 16px, Bold
- **Label**: 14px, Regular, color: gray-600
- **Value**: 16px, Semibold, color: gray-900
- **Stats Value**: 24px, Bold, color: gray-900

### 간격
- Breadcrumb ~ Page Title: 8px
- Page Title ~ 섹션: 32px
- 섹션 간: 40px
- Card 내부 Header ~ Body: 16px
- Card 내부 항목 간: 12px

### 색상
- Badge (Success, ON): green-100 배경, green-700 텍스트
- Badge (Gray, 준비중): gray-100 배경, gray-700 텍스트
- Link: blue-600, hover: blue-700
- Link (disabled): gray-400

### 인터랙션
- **Card 전체 바로가기**: 마우스 호버 시 카드에 subtle shadow 추가
- **Link 호버**: 색상 변화 + underline
- **재시도 Link**: 에러 상태에서만 표시, 클릭 시 데이터 재조회

### 반응형
- 1440px 이상: 3-2-2-1 그리드 유지
- 1024px ~ 1439px: 전체 현황 3 columns, 게임 관리 2 columns, 하단 카드 2 columns
- 768px ~ 1023px: 전체 현황 2 columns, 나머지 1 column
- 768px 미만: 모든 카드 1 column (세로 스택)
