## 디자이너 참고사항

### Card 스타일
- 배경: 흰색 (#FFFFFF)
- 테두리: 1px 연한 회색 (#E5E7EB)
- 모서리 라운드: 8px
- 패딩: 24px (상하좌우)

### 설정 항목 간격
- Label과 Control 간격: 12px
- 설정 항목 간 Divider: 1px 연한 회색 (#E5E7EB)
- Divider 상하 여백: 24px
- 설명 텍스트 상단 여백: 8px

### Toggle Switch 스펙
- 너비: 44px
- 높이: 24px
- 슬라이더 크기: 20×20px
- ON 색상: 파란색 (#3B82F6)
- OFF 색상: 회색 (#D1D5DB)
- Label과 Toggle 배치: 같은 행, Toggle 우측 정렬

### Radio Button 스펙
- 라디오 버튼 크기: 20×20px
- Label과 버튼 간격: 8px
- 옵션 간 간격: 12px (세로 배치)
- 들여쓰기: 0px (라벨 아래 바로 배치)
- Checked 상태: 파란색 원형 채움
- Unchecked 상태: 회색 원형 외곽선

### Advanced Select 스펙
- 높이: 40px
- 너비: 200px
- 우측 화살표 아이콘: ▼
- 드롭다운 메뉴: 하단 펼침
- 옵션 높이: 40px
- 옵션 hover: 배경 연한 파란색 (#EFF6FF)

### Footer 배치
- 버튼 간 간격: 12px
- 우측 정렬
- 상단 여백: 32px

### 인터랙션
- [초기화] 클릭 → Confirm 모달 표시
- [저장] 클릭 → 유효성 검증 → API 호출 → Toast 메시지
- Toggle 변경 → 즉시 반영 (저장 버튼 클릭 필요)
- Radio 변경 → 선택 상태 업데이트
- Select 변경 → 선택된 옵션 표시
